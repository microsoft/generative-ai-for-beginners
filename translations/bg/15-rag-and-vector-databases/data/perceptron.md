## Модел на перцептрон

Да предположим, че имаме N характеристики в нашия модел, в който случай входният вектор би бил вектор с размер N. Перцептронът е модел за **бинарна класификация**, т.е. може да различава между два класа входни данни. Ще приемем, че за всеки входен вектор x изходът на нашия перцептрон би бил или +1, или -1, в зависимост от класа. Изходът ще се изчислява по формулата:

y(x) = f(w<sup>T</sup>x)

където f е стъпкова активационна функция

## Обучение на перцептрон

За да обучим перцептрон, трябва да намерим вектор на тежестите w, който класифицира повечето от стойностите правилно, т.е. води до най-малката **грешка**. Тази грешка се определя от **критерия на перцептрона** по следния начин:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

където:

* сумата се взема върху онези тренировъчни точки i, които водят до неправилна класификация
* x<sub>i</sub> са входните данни, а t<sub>i</sub> е -1 или +1 за съответно отрицателни и положителни примери.

Този критерий се разглежда като функция на тежестите w и трябва да го минимизираме. Често се използва метод, наречен **градиентен спуск**, при който започваме с някакви начални тежести w<sup>(0)</sup>, и след това на всяка стъпка обновяваме тежестите според формулата:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

Тук η е така наречената **скорост на обучение**, а ∇E(w) означава **градиента** на E. След като изчислим градиента, получаваме

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

Алгоритъмът на Python изглежда така:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
