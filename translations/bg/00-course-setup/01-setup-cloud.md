# Настройка в облака ☁️ – GitHub Codespaces

**Използвайте това ръководство, ако не искате да инсталирате нищо локално.**  
Codespaces ви предоставя безплатен VS Code в браузъра с всички зависимости предварително инсталирани.

---

## 1. Защо Codespaces?

| Полза | Какво означава това за вас |
|---------|----------------------|
| ✅ Без инсталации | Работи на Chromebook, iPad, компютри в училищни лаборатории… |
| ✅ Предварително изградена развойна среда | Python 3, Node.js, .NET, Java вече вътре |
| ✅ Безплатен квота | Лични акаунти получават **120 ядро-часа / 60 GB-часа на месец** |

> 💡 **Съвет**  
> Поддържайте здравословна квота чрез **спиране** или **изтриване** на неактивни codespaces  
> (Прегледайте ▸ Command Palette ▸ *Codespaces: Stop Codespace*).

---

## 2. Създаване на Codespace (с един клик)

1. **Форк** на това хранилище (бутон **Fork** горе вдясно).  
2. Във вашия форк изберете **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Диалог с бутони за създаване на codespace](../../../translated_images/bg/who-will-pay.4c0609b1c7780f44.webp)

✅ Отваря се прозорец на VS Code в браузъра и започва изграждането на развойната среда.
Това отнема **~2 минути** при първото пускане.

## 3. Добавете вашия API ключ (безопасният начин)

### Опция A Secrets на Codespaces — Препоръчително

1. ⚙️ Икона на зъбно колело -> Command Palette -> Codespaces : Manage user secret -> Add a new secret.
2. Име: OPENAI_API_KEY
3. Стойност: поставете вашия ключ → Add secret

Това е всичко — нашият код ще го вземе автоматично.

### Опция B .env файл (ако наистина имате нужда)

```bash
cp .env.copy .env
code .env         # попълнете OPENAI_API_KEY=your_key_here
```
