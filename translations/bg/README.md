# Генеративен изкуствен интелект за начинаещи (Версия 3) - Курс

Научете основите на изграждането на приложения с генеративен изкуствен интелект с нашия всеобхватен курс от 21 урока от Microsoft Cloud Advocates.

## 🌱 Започване

Този курс включва 21 урока. Всеки урок покрива своя собствена тема, така че започнете откъдето искате!

Уроците са маркирани като "Учене" с уроци, обясняващи концепция за генеративен ИИ, или като "Изграждане", които обясняват концепция и примери с код на **Python** и **TypeScript**, когато е възможно.

За .NET разработчици разгледайте [Генеративен ИИ за начинаещи (NET издание)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Всеки урок също включва раздел "Продължи да учиш" с допълнителни учебни материали.
