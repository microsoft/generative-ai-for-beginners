# Генеративен AI за начинаещи (Версия 3) - Един курс

Научете основите на изграждането на приложения с генеративен AI с нашия комплексен курс от 21 урока, предоставен от Microsoft Cloud Advocates.

## 🌱 Започване

Този курс съдържа 21 урока. Всеки урок покрива своя собствена тема, така че започнете от където искате!

Уроците са обозначени като "Научи" за обяснение на концепция за генеративен AI или "Създай" за обяснение и кодови примери както на **Python**, така и на **TypeScript** когато е възможно.

За .NET разработчици разгледайте [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Всеки урок включва и секция "Продължавай да учиш" с допълнителни учебни ресурси.

## Какво ви е нужно
### За да пускате кода от този курс, можете да използвате: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Уроци:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Уроци:** "githubmodels" (GitHub Models се оттегля в края на юли 2026 г. – използвайте Microsoft Foundry Models вместо това)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Уроци:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Стартиране на модели напълно офлайн на вашето собствено устройство, без необходимост от облачен абонамент
   
- Основни познания по Python или TypeScript са полезни - \*За напълно начинаещи разгледайте тези курсове по [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) и [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- GitHub акаунт, за да [форкнете цялото това хранилище](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) в собствения си GitHub акаунт

Създадохме урок **[Настройка на курса](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**, който да ви помогне с настройката на вашата среда за разработка.

Не забравяйте да [маркирате с звезда (🌟) това хранилище](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), за да го намирате по-лесно по-късно.
