# ক্লাউড সেটআপ ☁️ – GitHub Codespaces

**এই গাইডটি ব্যবহার করুন যদি আপনি লোকালি কিছুই ইনস্টল করতে না চান।**  
Codespaces আপনাকে একটি বিনামূল্যের, ব্রাউজার-ভিত্তিক VS Code ইন্সট্যান্স দেয় যার সব নির্ভরশীলতা আগেই ইনস্টল করা হয়েছে।

---

## ১. কেন Codespaces?

| সুবিধা | আপনার জন্য এর অর্থ কী |
|---------|----------------------|
| ✅ কোন ইনস্টলেশন লাগবে না | Chromebook, iPad, স্কুল ল্যাব পিসিগুলিতে কাজ করে… |
| ✅ প্রস্তুত dev container | Python 3, Node.js, .NET, Java আগেই অন্তর্ভুক্ত |
| ✅ ফ্রি квота | ব্যক্তিগত অ্যাকাউন্টগুলো পায় **১২০ কোর-ঘণ্টা / ৬০ GB-ঘণ্টা প্রতি মাসে** |

> 💡 **টিপ**  
> আপনার квота সুস্থ রাখার জন্য অব্যবহৃত codespaces **বন্দ** বা **মুছে** ফেলুন  
> (ভিউ ▸ কমান্ড প্যালেট ▸ *Codespaces: Stop Codespace*)।

---

## ২. Codespace তৈরি করুন (এক ক্লিকে)

১. এই রেপো **Fork** করুন (টপ-রাইট **Fork** বোতাম)।  
২. আপনার fork-এ, ক্লিক করুন **Code ▸ Codespaces ▸ Create codespace on main**।  
   ![Dialog showing buttons to create a codespace](../../../translated_images/bn/who-will-pay.4c0609b1c7780f44.webp)

✅ একটি ব্রাউজার VS Code উইন্ডো খুলবে এবং dev container তৈরি হতে শুরু করবে।
প্রথমবারে এটি **~২ মিনিট** সময় নেয়।

## ৩. আপনার API কী যোগ করুন (নিরাপদ উপায়)

### অপশন A Codespaces Secrets — সুপারিশকৃত

১. ⚙️ গিয়ার আইকন -> কমান্ড প্যালেট -> Codespaces : Manage user secret -> Add a new secret.
২. নাম: OPENAI_API_KEY
৩. মান: আপনার কী পেস্ট করুন → Add secret

এটাই—আমাদের কোড নিজে থেকেই এটি নিবে।

### অপশন B .env ফাইল (যদি সত্যিই প্রয়োজন হয়)

```bash
cp .env.copy .env
code .env         # এখানে আপনার কী দিয়ে OPENAI_API_KEY পূরণ করুন
```
