# শর্টস সূচনা জেনারেটিভ AI (ভার্সন ৩) - একটি কোর্স

Microsoft Cloud Advocates কর্তৃক নির্মিত ২১ পাঠের বিস্তৃত কোর্সের মাধ্যমে জেনারেটিভ AI অ্যাপ্লিকেশন তৈরির মৌলিক বিষয়গুলি শিখুন।

## 🌱 শুরু করা

এই কোর্সে রয়েছে ২১টি পাঠ। প্রত্যেকটি পাঠ তার নিজস্ব বিষয় নিয়ে কাজ করে, তাই আপনি যেখান থেকেই শুরু করতে পারেন!

পাঠগুলো “Learn” লেবেলযুক্ত হয় যা একটি জেনারেটিভ AI ধারণা ব্যাখ্যা করে, কিংবা “Build” লেবেলযুক্ত হয় যা একটি ধারণা এবং কোড উদাহরণ উল্লেখ করে **Python** এবং **TypeScript** এ যেখানে সম্ভব।

.NET ডেভেলপারদের জন্য দেখুন [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

প্রত্যেকটি পাঠে অতিরিক্ত শেখার সরঞ্জাম সহ একটি "Keep Learning" বিভাগও রয়েছে।
