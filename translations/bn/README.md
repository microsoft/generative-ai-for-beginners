# শুরু করার জন্য জেনারেটিভ এআই (ভার্সন ৩) - একটি কোর্স

মাইক্রোসফট ক্লাউড অ্যাডভোকেটস দ্বারা তৈরি ২১-টিপাঠের বিস্তৃত কোর্সের মাধ্যমে জেনারেটিভ এআই অ্যাপ্লিকেশন তৈরির মৌলিক ধারণা শিখুন।

## 🌱 শুরু করা

এই কোর্সে ২১ টি পাঠ আছে। প্রতিটি পাঠ আলাদা বিষয় আচ্ছাদিত করে, তাই আপনি আপনার পছন্দমতো যেখান থেকে খুশি শুরু করতে পারেন!

পাঠগুলো হয় "শিখুন" নামক, যেখানে একটি জেনারেটিভ এআই ধারণা ব্যাখ্যা করা হয় অথবা "তৈরি করুন" নামক, যা ধারণা এবং কোড উদাহরণ (যেখানে সম্ভব) **Python** এবং **TypeScript** এ ব্যাখ্যা করে।

.NET ডেভেলপারদের জন্য দেখুন [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

প্রতিটি পাঠে একটি "শেখা চালিয়ে যান" বিভাগ রয়েছে যেখানে অতিরিক্ত শেখার সরঞ্জাম রয়েছে।

## আপনি কি প্রয়োজন
### এই কোর্সের কোড চালানোর জন্য, আপনি নিম্নলিখিতগুলোর যেকোনো ব্যবহার করতে পারেন:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **পাঠ:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **পাঠ:** "githubmodels" (GitHub Models জুলাই ২০২৬ এর শেষে অবসর নিচ্ছে - পরিবর্তে Microsoft Foundry Models ব্যবহার করুন)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **পাঠ:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - নিজের ডিভাইসে সম্পূর্ণ অফলাইনে মডেল চালান, কোন ক্লাউড সাবস্ক্রিপশন প্রয়োজন নেই
   
- Python বা TypeScript এর মৌলিক জ্ঞান সাহায্যকারী - *সম্পূর্ণ নতুনদের জন্য এই [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) এবং [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) কোর্সগুলো দেখুন
- একটি গিটহাব অ্যাকাউন্ট যার মাধ্যমে আপনি [এই পুরো রেপো ফর্ক করতে পারেন](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst)

আমরা আপনার ডেভেলপমেন্ট পরিবেশ সেটআপে সাহায্যের জন্য একটি **[কোর্স সেটআপ](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** পাঠ তৈরি করেছি।

পরবর্তীতে সহজে খুঁজে পেতে [এই রেপোতে (🌟) স্টার দিতে ভুলবেন না](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)।
