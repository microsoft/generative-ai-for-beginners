# জেনারেটিভ এআই ফর বিগিনার্স (সংস্করণ ৩) - একটি কোর্স

মাইক্রোসফট ক্লাউড অ্যাডভোকেটস দ্বারা প্রদত্ত আমাদের ২১-লেসনের ব্যাপক কোর্সের মাধ্যমে জেনারেটিভ এআই অ্যাপ্লিকেশন তৈরি করার মৌলিক বিষয়গুলো শিখুন।

## 🌱 শুরু করা যাক

এই কোর্সে মোট ২১টি লেসন রয়েছে। প্রতিটি লেসন নিজস্ব বিষয়বস্তু নিয়ে আলোচনা করে, তাই আপনি যেখান থেকে ইচ্ছা শুরু করতে পারেন!

লেসনগুলোকে "শেখা" লেসন হিসেবে লেবেল করা হয়েছে যা জেনারেটিভ এআই ধারণা ব্যাখ্যা করে অথবা "তৈরি করা" লেসন যা একটি ধারণা এবং কোড উদাহরণ ব্যাখ্যা করে যখন সম্ভব **Python** এবং **TypeScript** উভয় ভাষায়।

.NET ডেভেলপারদের জন্য [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) দেখুন!

প্রতিটি লেসনেও একটি "অধিক শিখুন" অংশ অন্তর্ভুক্ত রয়েছে অতিরিক্ত শেখার সরঞ্জামসহ।
