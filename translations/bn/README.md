# শুরু করার জন্য জেনারেটিভ এআই (ভার্সন ৩) - একটি কোর্স

Microsoft Cloud Advocates দ্বারা আমাদের ২১টি পাঠের পূর্ণাঙ্গ কোর্সের মাধ্যমে জেনারেটিভ এআই অ্যাপ্লিকেশন তৈরি করার মৌলিক বিষয়গুলি শিখুন।

## 🌱 শুরু করা

এই কোর্সে ২১টি পাঠ রয়েছে। প্রতিটি পাঠ তার নিজস্ব বিষয় কভার করে তাই যেখান থেকে আপনি চান সেখান থেকে শুরু করুন!

পাঠগুলি "Learn" নামক পাঠ বা "Build" নামক পাঠ হিসেবে লেবেল করা হয়েছে, যেখানে "Learn" পাঠ জেনারেটিভ এআই ধারণা বোঝায় এবং "Build" পাঠ একটি ধারণা এবং প্রয়োজনে **Python** ও **TypeScript** দুটিতে কোড উদাহরণ ব্যাখ্যা করে।

.NET ডেভেলপারদের জন্য দেখুন [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

প্রতিটি পাঠে একটি "Keep Learning" বিভাগও রয়েছে যেখানে অতিরিক্ত শেখার উপকরণ দেওয়া হয়েছে।

## আপনার যা প্রয়োজন
### এই কোর্সের কোড চালানোর জন্য আপনি নিম্নলিখিত কোনোটিই ব্যবহার করতে পারেন:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **পাঠ:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **পাঠ:** "githubmodels" (GitHub Models জুলাই ২০২৬ সালের শেষে অবসর নিচ্ছে - তার পরিবর্তে Microsoft Foundry Models ব্যবহার করুন)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **পাঠ:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - আপনার নিজের ডিভাইসে সম্পূর্ণ অফলাইন মডেল চালান, কোন ক্লাউড সাবস্ক্রিপশন প্রয়োজন নেই
   
- পাইথন বা টাইপস্ক্রিপ্টের সাধারণ জ্ঞান সহায়ক - \*সম্পূর্ণ নতুনদের জন্য এই [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ও [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) কোর্সগুলি দেখুন
- একটি GitHub অ্যাকাউন্ট আপনার নিজের জন্য এই পুরো রিপোটি [ফর্ক](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) করার জন্য

আমরা আপনার ডেভেলপমেন্ট পরিবেশ সেটআপে সাহায্যের জন্য একটি **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** পাঠ তৈরি করেছি।
