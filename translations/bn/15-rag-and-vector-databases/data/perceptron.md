# নিউরাল নেটওয়ার্কের পরিচয়: পার্সেপট্রন

আধুনিক নিউরাল নেটওয়ার্কের মতো কিছু তৈরি করার প্রথম প্রচেষ্টাগুলির মধ্যে একটি ১৯৫৭ সালে কর্নেল এরোনটিক্যাল ল্যাবরেটরির ফ্র্যাঙ্ক রোজেনব্লাট দ্বারা করা হয়েছিল। এটি ছিল "মার্ক-১" নামে একটি হার্ডওয়্যার বাস্তবায়ন, যা প্রাথমিক জ্যামিতিক চিত্রগুলি যেমন ত্রিভুজ, বর্গক্ষেত্র এবং বৃত্ত চিনতে ডিজাইন করা হয়েছিল।

> উইকিপিডিয়া থেকে ছবি

একটি ইনপুট ইমেজ ২০x২০ ফটোসেল অ্যারে দ্বারা প্রতিনিধিত্ব করা হয়েছিল, তাই নিউরাল নেটওয়ার্কের ৪০০টি ইনপুট এবং একটি বাইনারি আউটপুট ছিল। একটি সাধারণ নেটওয়ার্কে একটি নিউরন ছিল, যাকে **থ্রেশহোল্ড লজিক ইউনিট** ও বলা হয়। নিউরাল নেটওয়ার্কের ওজনগুলি পোটেনশিওমিটারের মতো কাজ করত যা প্রশিক্ষণ পর্বে ম্যানুয়াল সমন্বয় প্রয়োজন।

> ✅ একটি পোটেনশিওমিটার এমন একটি ডিভাইস যা ব্যবহারকারীকে একটি সার্কিটের প্রতিরোধ ক্ষমতা সমন্বয় করতে দেয়।

> নিউ ইয়র্ক টাইমস সেই সময়ে পার্সেপট্রন সম্পর্কে লিখেছিল: *একটি ইলেকট্রনিক কম্পিউটারের ভ্রূণ যা [নেভি] আশা করে হাঁটতে, কথা বলতে, দেখতে, লিখতে, নিজেকে পুনরুৎপাদন করতে এবং তার অস্তিত্ব সম্পর্কে সচেতন হতে সক্ষম হবে।*

## পার্সেপট্রন মডেল

ধরা যাক আমাদের মডেলে N বৈশিষ্ট্য রয়েছে, সেক্ষেত্রে ইনপুট ভেক্টর হবে N আকারের একটি ভেক্টর। একটি পার্সেপট্রন একটি **বাইনারি শ্রেণীবিভাজন** মডেল, অর্থাৎ এটি ইনপুট ডেটার দুটি শ্রেণীর মধ্যে পার্থক্য করতে পারে। আমরা ধরে নেব যে প্রতিটি ইনপুট ভেক্টর x এর জন্য আমাদের পার্সেপট্রনের আউটপুট হয় +1 বা -1 হবে, শ্রেণির উপর নির্ভর করে। আউটপুটটি নিম্নলিখিত সূত্র ব্যবহার করে গণনা করা হবে:

y(x) = f(w<sup>T</sup>x)

যেখানে f হল একটি ধাপ সক্রিয়করণ ফাংশন

## পার্সেপট্রন প্রশিক্ষণ

পার্সেপট্রন প্রশিক্ষণ দিতে আমাদের একটি ওজন ভেক্টর w খুঁজে বের করতে হবে যা বেশিরভাগ মান সঠিকভাবে শ্রেণীবদ্ধ করে, অর্থাৎ সবচেয়ে ছোট **ত্রুটি** দেয়। এই ত্রুটি **পার্সেপট্রন মানদণ্ড** দ্বারা নিম্নলিখিতভাবে সংজ্ঞায়িত করা হয়েছে:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

যেখানে:

* যোগফলটি সেই প্রশিক্ষণ ডেটা পয়েন্টগুলির উপর নেওয়া হয় যা ভুল শ্রেণীবিভাজন দেয়
* x<sub>i</sub> হল ইনপুট ডেটা, এবং t<sub>i</sub> নেতিবাচক এবং ইতিবাচক উদাহরণগুলির জন্য যথাক্রমে -1 বা +1।

এই মানদণ্ডটিকে ওজন w এর একটি ফাংশন হিসাবে বিবেচনা করা হয়, এবং আমাদের এটি সর্বনিম্ন করতে হবে। প্রায়শই, **গ্রেডিয়েন্ট ডিসেন্ট** নামে একটি পদ্ধতি ব্যবহার করা হয়, যেখানে আমরা কিছু প্রাথমিক ওজন w<sup>(0)</sup> দিয়ে শুরু করি এবং তারপর প্রতিটি পদক্ষেপে ওজনগুলি নিম্নলিখিত সূত্র অনুযায়ী আপডেট করি:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

এখানে η হল তথাকথিত **লার্নিং রেট**, এবং ∇E(w) E এর **গ্রেডিয়েন্ট** নির্দেশ করে। আমরা যখন গ্রেডিয়েন্ট গণনা করি, তখন আমাদের শেষ হয়

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

পাইথনে অ্যালগরিদমটি এভাবে দেখায়:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
