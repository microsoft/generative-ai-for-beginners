бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһҡбһҝбһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҡбһ бһјбһҸбһҠбһӣбҹӢбһҘбһЎбһјбһңбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ—бҹ” бһ‘бҹ„бҹҮбһҷбҹүбһ¶бһ„бһҺбһ¶бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ бһҷбһҫбһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бһ‘бҹҖбһҸбҹ” бһҡбһҝбһ„бһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһ¶бһ“ бһӮбһәбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбһ”бһ“бҹ’бһҗбҹӮбһҳ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбҹ” бһ бһҫбһҷбһҷбһҫбһ„бһҖбҹҸбһўбһ¶бһ…бһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҹбһҳбҹ’бһ”бһјбһҡбһ”бҹӮбһ”бҹ”

бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҒбһ¶бһ„бһӣбһҫ бһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ…бһ„бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

- бһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһўбҹҶбһ–бһёбһўбҹ’бһңбһёбһ‘бҹ…бһҮбһ¶ function calling бһ“бһ·бһ„бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶бҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Azure OpenAIбҹ”
- бһҡбһ”бҹҖбһ”бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҪбһҷбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбҹҖбһ“

бһҠбһӣбҹӢбһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһ–бһёбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ function callingбҹ”
- бһҖбҹҶбһҺбһҸбҹӢ Function Call бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Azure OpenAI Serviceбҹ”
- бһҡбһ…бһ“бһ¶бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһҖбһҡбһҺбһёбҹ– бһҖбҹӮбһӣбһҳбҹ’бһў chatbot бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһўбһ”бҹӢбһҡбҹҶбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ chatbot бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһңбһӮбҹ’бһӮбһҠбҹӮбһӣбһҹбһҳбһ бҹҠбһ»бҹҶбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҮбҹҶбһ“бһ¶бһү бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҸбһҪбһ“бһ¶бһ‘бһё бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһ–бһҪбһҖбһӮбҹҒбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһӣбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбҹ–

- `Azure OpenAI` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҮбһ¶бһҹбһ“бҹ’бһ‘бһ“бһ¶ (chat) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”
- `Microsoft Learn Catalog API` бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбҹҶбһҺбһҫбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ”
- `Function Calling` бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһҖбһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ бһҫбһҷбһ•бҹ’бһүбһҫбһңбһ¶бһ‘бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҹбҹҶбһҺбһҫ APIбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҳбһҖбһҳбһҫбһӣбһ–бһёбһҳбһјбһӣбһ бҹҒбһҸбһ»бһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ function calling бһҮбһ¶бһҳбһ»бһ“бһҮбһ¶бһ„бһўбҹ’бһңбһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ–

## бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Function Calling

бһҳбһ»бһ“бһ–бҹҒбһӣбһҳбһ¶бһ“ function calling бһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһё LLM бһҳбһ·бһ“бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶ бһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷбһ”бһ¶бһ“бҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһ·бһ“бһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбһјбһ…бһҮбһ¶ "бһҸбһҫбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ“бҹ…бһ‘бһёбһҖбҹ’бһҡбһ»бһ„бһҹбҹ’бһҸбһ»бһҖбһ бҹ„бһӣбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҠбһ»бһҖбһ бҹ„бһӣбһҳбҹӮбһ“бһ‘бҹҒ?"бҹ” бһ“бҹҒбҹҮбһӮбһәбһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһ–бҹҒбһӣбһҠбҹӮбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ”

Function Calling бһӮбһәбһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҳбһҪбһҷбһ“бҹғ Azure OpenAI Service бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷ бһ“бһ·бһ„бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһҡбһ”бһҹбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹ„бҹҮбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `search_courses` бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“бҹ”

> **бһҹбҹҶбһҒбһ¶бһ“бҹӢ** : бһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбһ¶бһҡвҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ LLM бһ бһҫбһҷбһ“бһ№бһ„бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһ“бһҪбһ“бһҸбһјбһҖбҹӮбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбҹ”

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ бһҮбһ¶ бһўбһ¶бһҡбҹҒбһӣбһӮбҹ’бһҡбһҝбһ„бһҖбҹ’бһ“бһ»бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Responses API бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ `type`, `name`, `description` бһ“бһ·бһ„ `parameters`бҹ–

```python
functions = [
   {
      "type":"function",
      "name":"search_courses",
      "description":"Retrieves courses from the search index based on the parameters provided",
      "parameters":{
         "type":"object",
         "properties":{
            "role":{
               "type":"string",
               "description":"The role of the learner (i.e. developer, data scientist, student, etc.)"
            },
            "product":{
               "type":"string",
               "description":"The product that the lesson is covering (i.e. Azure, Power BI, etc.)"
            },
            "level":{
               "type":"string",
               "description":"The level of experience the learner has prior to taking the course (i.e. beginner, intermediate, advanced)"
            }
         },
         "required":[
            "role"
         ]
      }
   }
]
```

бһҳбһҖбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бһёбһҳбһҪбһҷбҹ—бһӣбһҳбҹ’бһўбһ·бһҸбһҮбһ¶бһ„бһ“бҹҒбҹҮбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- `name` - бһҲбҹ’бһҳбҹ„бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһІбҹ’бһҷбһ бҹ…бҹ”
- `description` - бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһ”бҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”
- `parameters` - бһ”бһүбҹ’бһҮбһёбһҸбһҳбҹ’бһӣбҹғ бһ“бһ·бһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ” бһўбһ¶бһҡбҹҒбһӣ parameters бһҳбһ¶бһ“=[items] бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
  1.  `type` - бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбҹӮбһӣбһ“бһ№бһ„бһ•бҹ’бһ‘бһ»бһҖбҹ”
  1.  `properties` - бһ”бһүбҹ’бһҮбһёбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҡбһ”бһҹбҹӢбһңбһ¶бҹ”
      1. `name` - бһҖбһјбһ“бһҹбҹ„бһӮбһәбһҲбҹ’бһҳбҹ„бҹҮбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҡбһ”бһҹбҹӢбһңбһ¶ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ `product`бҹ”
      1. `type` - бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһ“бҹҒбҹҮ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ `string`бҹ”
      1. `description` - бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ–бһёбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”

бһҳбһ¶бһ“бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҳбһҪбһҷбһ•бһ„ `required` - бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһІбҹ’бһҷбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бҹ”

### бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 3 - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ бҹ… Responses APIбҹ” бһҷбһҫбһ„бһ’бҹ’бһңбһҫбһңбһ¶бһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳ `tools` бһ‘бҹ…бһҹбҹҶбһҺбһҫбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ `tools=functions`бҹ”

бһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢ `tool_choice` бһ‘бҹ… `auto`бҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҷбһҫбһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ LLM бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ бҹ…бһҳбҹӮбһ“бһ‘бҹ…бһӣбһҫбһҹбһ¶бһҡ user бһҮбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҷбһҫбһ„бҹ”

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҮбһ¶бһҖбһјбһҠбһҠбҹӮбһӣбһҷбһҫбһ„бһ бҹ… `client.responses.create` бһҹбһјбһҳбһ…бҹҶбһҺбһ¶бҹҶбһҖбһ¶бһӣбһҺбһ¶бһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢ `tools=functions` бһ“бһ·бһ„ `tool_choice="auto"` бһ”бҹ’бһҡбһ¶бһ”бҹӢ LLM бһҮбҹ’бһҡбһҫбһҹбһҮбһ¶бһҸбһ»бҹҶбһҖбһ¶бһӣбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҷбһҫбһ„бһ•бҹ’бһҸбһӣбҹӢбһ‘бҹ…бһңбһ¶бҹ–

```python
response = client.responses.create(model=deployment,
                                        input=messages,
                                        tools=functions,
                                        tool_choice="auto",
                                        store=False)

print(response.output)
```

бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбҹ’бһҡбһЎбһ”бҹӢбһ•бһ„бһҳбһ¶бһ“бһ’бһ¶бһҸбһ» `function_call` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `response.output` бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```json
{
  "type": "function_call",
  "name": "search_courses",
  "call_id": "call_abc123",
  "arguments": "{\n  \"role\": \"student\",\n  \"product\": \"Azure\",\n  \"level\": \"beginner\"\n}"
}
```

бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһҷбһҫбһ„бһўбһ¶бһ…бһғбһҫбһүбһҗбһ¶ function `search_courses` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ… бһ бһҫбһҷбһҳбһ¶бһ“бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбҹӢбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ `arguments` бһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷ JSONбҹ”

бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһӮбһә LLM бһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ arguments бһ“бҹғбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹ„бһҷбһңбһ¶ бһ бһјбһ”бһ‘бҹ…бһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷ `input` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ бҹ… Responses APIбҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһҸбһҳбҹ’бһӣбҹғ `messages`бҹ–

```python
messages= [ {"role": "user", "content": "Find me a good course for a beginner student to learn Azure."} ]
```

1. бһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„ бһҷбһҫбһ„бһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ OpenAI бһ бһҫбһҷбһҷбһҖбһ’бһ¶бһҸбһ» function call бһ–бһёбһ…бһҳбҹ’бһӣбһҫбһҷ `output`бҹ”

1. бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһ“бһ№бһ„бһ бҹ… Microsoft Learn API бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһёбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ–

   ```python
   import requests

   def search_courses(role, product, level):
     url = "https://learn.microsoft.com/api/catalog/"
     params = {
        "role": role,
        "product": product,
        "level": level
     }
     response = requests.get(url, params=params)
     modules = response.json()["modules"]
     results = []
     for module in modules[:5]:
        title = module["title"]
        url = module["url"]
        results.append({"title": title, "url": url})
     return str(results)
   ```

бһҷбҹүбһ¶бһ„бһҺбһ¶бһҳбһ·бһү бһҷбһҫбһ„бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҗбҹҒбһҡ `functions` бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ Python бһҸбҹ’бһ”бһ·бһҸ бһҸбһҫбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһ¶бһ”бҹӢ LLM бһҡбһ”бҹҖбһ”бһ•бҹ’бһӮбһјбһҡбһ•бҹ’бһӮбһ„бһңбһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбһ»бһҒбһ„бһ¶бһҡ Python бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…?

1. бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ Python бһ¬бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ бҹ… бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһё LLM бһ бһҫбһҷбһҳбһҫбһӣбһҗбһ¶бһҠбһӣбҹӢбһ’бһ¶бһҸбһ» `function_call` бһ¬бһўбһҸбҹӢ бһ бһҫбһҷбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбҹ” бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

   ```python
   # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶вҖӢбһӮбҹҶбһҡбһјвҖӢбһ…бһ„бҹӢбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷвҖӢбһ‘бҹҒвҖӢбһ¬бһўбһҸбҹӢ
   if tool_calls:

     print(tool_call.name)

     # бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбҹ”

     print(function_response)
     print(type(function_response))

     # бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ”бһҹбҹӢбһңбһ¶бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһңбһ·бһүбҹ”
     # бһ’бһ¶бһҸбһ» function_call бһҡбһ”бһҹбҹӢбһӮбҹҶбһҡбһјбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ”бһҹбҹӢбһңбһ¶бҹ”
     messages.append(tool_call)  # бһ’бһ¶бһҸбһ» function_call бһҡбһ”бһҹбҹӢбһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡ
     messages.append( # бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҳбһ»бһҒбһ„бһ¶бһҡ
         {
             "type": "function_call_output",
   ```
