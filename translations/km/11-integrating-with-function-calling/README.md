бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһҡбһҪбһ…бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһӣбҹ’бһўбһ бһҫбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ—бҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһҷбһҫбһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бһ‘бҹҖбһҸбҹ” бһҳбһ¶бһ“бһҡбһҝбһ„бһҒбҹ’бһӣбҹҮбҹ—бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһ¶бһ“ бһӮбһәбһҸбһҫбһҷбһҫбһ„бһўбһ¶бһ…бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҡбһ»бһ”бһҡбһ№бһ„бһҳбһ¶бҹҶбһҠбһҫбһҳбҹ’бһ”бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бҹ…бһҶбҹ’бһ–бҹ„бҹҮбһ‘бҹ…бһҳбһ»бһҒбһ¬бһ‘бҹҒбҹ” бһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһҷбһҫбһ„бһўбһ¶бһ…бһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһ‘бҹғбһ‘бҹҖбһҸбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҠбһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”

бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбҹ„бһ„бһҒбһ¶бһ„бһӣбһҫбһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһҖбҹҶбһ–бһ»бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ”

## бһҳбҹҒбһҡбҹҖбһ“бһ”бһҫбһҖбһ”бһҡ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ:

- бһ–бһ“бҹ’бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ¶бҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ Azure OpenAIбҹ”
- бһҡбһ”бҹҖбһ”бһ…бһјбһӣбһҡбһҪбһҳбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһҪбһҷбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһўбһ”бҹӢбһҡбҹҶ

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

- бһ–бһ“бҹ’бһҷбһӣбҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбҹ”
- бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ Function Call бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure OpenAIбҹ”
- бһҡбһ…бһ“бһ¶бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ– бһҖбҹӮбһӣбһҳбҹ’бһў chatbot бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҸбһ¶бһҸбһ¶бһ”бҹӢбһўбһ”бҹӢбһҡбҹҶбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһўбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ chatbot бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһӮбҹ’бһӮбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһ‘бҹ…бһ“бһ№бһ„бһҖбҹҶбһҡбһ·бһҸбһҮбҹҶбһ“бһ¶бһү бһҸбһҪбһ“бһ¶бһ‘бһёбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбҹ–

- `Azure OpenAI` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҮбһҮбҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”
- `Microsoft Learn Catalog API` бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”
- `Function Calling` бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһҖбһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ бһҫбһҷбһ•бҹ’бһүбһҫбһңбһ¶бһ‘бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҹбҹҶбһҺбһҫ APIбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ бһҳбһҖбһҳбһҫбһӣбһҗбһ¶бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ Function Calling бһҮбһ¶бһ…бҹҶбһҺбҹӮбһҖбһҠбҹҶбһ”бһјбһ„бҹ–

## бһ бҹҒбһҸбһ»бһ•бһӣбһ“бҹғбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ

- **бһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Structured Data**бҹ” бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ¶бһ…бһҷбһҖбһўбһҸбҹ’бһҗбһ”бһ‘бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһ¬ CSV бһ бһҫбһҷбһ”бҹ’бһҡбһҫ LLM бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһҖбһҹбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ…бҹҒбһүбһ–бһёбһңбһ¶бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҹбһ·бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбһўбһ¶бһ…бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһңбһ·бһ…бһ·бһ”бҹҒбһҢбһ¶бһҷбҹүбһ¶бһўбҹҶбһ–бһёбһ—бһ¶бһ–бһҹбһ“бҹ’бһҸбһ·бһ—бһ¶бһ– бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҸбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ AIбҹ” бһ“бҹҒбҹҮбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `get_important_facts(agreement_name: string, date_signed: string, parties_involved: list)`бҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһ¶бһ…бһ”бҹӮбһ„бһ…бҹӮбһҖбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһ…бһҳбҹ’бһ”бһ„бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ–

1. **бһ бҹ…** API Chat Completions бһҮбһ¶бһҳбһҪбһҷбһ”бһүбҹ’бһҮбһёбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҹбһ¶бһҡ user бһҳбһҪбһҷбҹ”
2. **бһўбһ¶бһ“** бһ…бһҳбҹ’бһӣбһҫбһҷбһҳбҹүбһјбһҠбҹӮбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ§. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбһ»бһҒбһ„бһ¶бһҡ бһ¬ бһ бҹ… APIбҹ”
3. **бһ’бҹ’бһңбһҫ**бһҖбһ¶бһҡбһ бҹ…бһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸбһ‘бҹ… API Chat Completions бһҮбһ¶бһҳбһҪбһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹ„бҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бҹ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”

### бһҮбҹҶбһ бһ¶бһ“ 1 - бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҡ

бһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһӮбһәбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһҪбһҷбҹ” бһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһ”бҹӮбһ”бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹ„бһҷбһҷбһҖбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ”бһ‘ бһ¬бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһ•бһ„бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҮбһ¶бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ API Chat Completions бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢ `role` бһ“бһ·бһ„ `content` бһҡбһ”бһҹбҹӢбһҹбһ¶бһҡбҹ”

`role` бһўбһ¶бһ…бһҮбһ¶бһҸбһҪбһ“бһ¶бһ‘бһё `system` (бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢ), `assistant` (бһҳбҹүбһјбһҠбҹӮбһӣ) бһ¬ `user` (бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ)бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ function calling бһҷбһҫбһ„бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһңбһ¶бһҮбһ¶ `user` бһ бһҫбһҷбһ•бҹ’бһҠбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбҹҶбһҺбһҪбһҡбҹ”

```python
messages= [ {"role": "user", "content": "Find me a good course for a beginner student to learn Azure."} ]
```

бһҠбҹ„бһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбһҪбһ“бһ¶бһ‘бһёбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶ бһ’бҹ’бһңбһҫбһІбҹ’бһҷ LLM бһҷбһӣбҹӢбһҗбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ“бҹҒбҹҮбһ‘бҹ’бһ”бһ¶бһ“бһ”бҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһңбһёбһҳбһҪбһҷ бһ¬бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбҹҶбһ–бһ»бһ„бһ“бһ·бһҷбһ¶бһҷ бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҮбһҪбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһҠбҹӮбһӣ LLM бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бҹ”

### бһҮбҹҶбһ бһ¶бһ“ 2 - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ бһҸбҹӮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“бҹ”

> **бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бҹ… LLM бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһ“бһҪбһ“ token бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ”

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһўбһ¶бһҡбҹҒбһ“бҹғбһ’бһ¶бһҸбһ»бһҳбһҪбһҷбҹ—бҹ” бһ’бһ¶бһҸбһ»бһҳбһҪбһҷбһ“бһёбһҳбһҪбһҷбҹ—бһӮбһәбһҮбһ¶вҖӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҳбһҪбһҷвҖӢ бһ бһҫбһҷбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲ `name`, `description` бһ“бһ·бһ„ `parameters`бҹ–

```python
functions = [

            }

      }
   }
```

бһҷбһҫбһ„бһҳбһҖбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһ“бһёбһҳбһҪбһҷбҹ—бһӣбһҳбҹ’бһўбһ·бһҸбҹ–

- `name` - бһҲбҹ’бһҳбҹ„бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ бҹ…бҹ”
- `description` - бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбҹ” бһҖбҹ’бһ“бһ»бһ„бһ“бҹҒбҹҮбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”
- `parameters` - бһ”бһүбҹ’бһҮбһёбһҸбһҳбҹ’бһӣбҹғ бһ“бһ·бһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһұбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ•бһӣбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһўбһ¶бһҡбҹҒ parameters бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһ’бһ¶бһҸбһ» бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
  1.  `type` - бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһҖбҹ’бһҒбһҺбҹҲбҹ”
  1.  `properties` - бһ”бһүбҹ’бһҮбһёбһҸбһҳбҹ’бһӣбҹғбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбҹ”
      1. `name` - бһҹбҹ„бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲ бһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ бһҠбһјбһ…бһҮбһ¶ `product`бҹ”
      1. `type` - бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһ“бҹҒбҹҮ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ `string`бҹ”
      1. `description` - бһҖбһ¶бһҡбһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһӣбһҖбҹ’бһҒбһҺбҹҲбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”

бһҖбҹҸбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ `required` - бһӣбһҖбҹ’бһҒбһҺбҹҲбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбҹ”

### бһҮбҹҶбһ бһ¶бһ“ 3 - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһңбһ¶бһӣбһҫбһҖбһ¶бһҡбһ бҹ…бһ‘бҹ… Chat Completion APIбҹ” бһҷбһҫбһ„бһ’бҹ’бһңбһҫбһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳ `functions` бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһҫбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ `functions=functions`бҹ”

бһҖбҹҸбһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ `function_call` бһ‘бҹ…бһҮбһ¶ `auto`бҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҷбһҫбһ„бһўбҹ„бһҷ LLM бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбһ¶бһҡ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бҹ…бһ“бһ№бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҠбҹғбҹ”

бһ“бҹҒбҹҮбһҮбһ¶бһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһҠбҹӮбһӣбһ бҹ… `ChatCompletion.create` бһ бһҫбһҷбһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢ `functions=functions` бһ“бһ·бһ„ `function_call="auto"` бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҮбһҳбҹ’бһҡбһҫбһҹбһҠбһӣбҹӢ LLM бһҮбһҪбһҷбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ–бҹҒбһӣбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҷбһҫбһ„бһ•бҹ’бһҸбһӣбҹӢбҹ–

```python
response = client.chat.completions.create(model=deployment,
```

### бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбҹҶбһ бһјбһҡ

1. бһҘбһЎбһјбһңбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһ бҹ… Microsoft Learn API бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһүбҹ’бһҮбһёбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ–

   ```python
   import requests

   def search_courses(role, product, level):
     url = "https://learn.microsoft.com/api/catalog/"
     params = {
        "role": role,
        "product": product,
        "level": level
     }
     response = requests.get(url, params=params)
     modules = response.json()["modules"]
     results = []
     for module in modules[:5]:
        title = module["title"]
        url = module["url"]
        results.append({"title": title, "url": url})
     return str(results)
   ```
