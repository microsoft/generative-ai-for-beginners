# бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘

[![Building Text Generation Applications](../../../translated_images/km/06-lesson-banner.a5c629f990a636c8.webp)](https://youtu.be/0Y5Luf5sRQA?si=t_xVg0clnAI4oUFZ)

> _(бһ…бһ»бһ…бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫ бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ)_

бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһҡбһ бһјбһҸбһҳбһҖбһҠбһӣбҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮбһҸбһ¶бһҳбһҡбһҷбҹҲбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҗбһ¶бһҳбһ¶бһ“бһҷбҹ„бһ”бһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбһјбһ…бһҮбһ¶ prompt бһ бһҫбһҷбһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһҳбһ¶бһ“бһңбһ·бһҹбҹҗбһҷбһҳбһҪбһҷбһ‘бҹҖбһҸбһ бҹ…бһҗбһ¶ "prompt engineering"бҹ” бһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҠбһјбһ…бһҮбһ¶ ChatGPT, Office 365, Microsoft Power Platform бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ prompt бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһўбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ”бҹӮбһ”бһ“бҹҒбҹҮбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҷбһӣбҹӢбһ–бһёбһҷбҹ„бһ”бһӣбҹӢбһҠбһјбһ…бһҮбһ¶ prompt, completions бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһңбһ¶бҹ” бһ“бҹҒбҹҮбһҮбһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бҹ–

- бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒ openai бһ“бһ·бһ„бһҷбҹ„бһ”бһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһңбһ¶бҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ openaiбҹ”
- бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҷбҹ„бһ”бһӣбҹӢбһҠбһјбһ…бһҮбһ¶ prompt, temperature бһ“бһ·бһ„ tokens бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһўбҹ’бһңбһёбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ openaiбҹ”
- бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһ”бҹ’бһҡбһҫ tokens бһұбҹ’бһҷбһ…бҹ’бһҡбһҫбһ“ бһ¬бһҸбһ·бһ… бһ“бһ·бһ„бһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘ temperature бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

## бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘?

бһ’бһҳбҹ’бһҳбһҸбһ¶ бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбһ¶бһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһҳбһҪбһҷбһҠбҹӮбһӣбһҠбһјбһ…бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбһјбһ…бһҮбһ¶бҹ–

- бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһүбҹ’бһҮбһ¶ Command-basedбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё console бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһңбһ¶бһҷбһ”бһүбҹ’бһҮбһ¶ бһ бһҫбһҷбһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбһ¶бһҳбһ”бһүбҹ’бһҮбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ `git` бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё command-basedбҹ”
- бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ (UI)бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҒбҹ’бһӣбҹҮбһҳбһ¶бһ“ graphical user interfaces (GUIs) бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ бһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ”бһ‘ бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбҹҶбһҡбһҫбһҹ бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”

### бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Console бһ“бһ·бһ„ UI бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ

бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһңбһ¶бһ“бһ№бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё command-based бһҠбҹӮбһӣ бһўбҹ’бһ“бһҖбһңбһ¶бһҷбһ”бһүбҹ’бһҮбһ¶бҹ–

- **бһңбһ¶бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ**бҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһңбһ¶бһҷбһ”бһүбҹ’бһҮбһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһ“бһ‘бҹҒ бһҳбһ¶бһ“бһҸбҹӮбһ”бһүбҹ’бһҮбһ¶бһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”
- **бһҮбһ¶бһ—бһ¶бһҹбһ¶бһ–бһ·бһҹбҹҒбһҹ**бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҒбҹ’бһӣбҹҮбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһ–бһ·бһҹбҹҒбһҹбһҳбһҪбһҷбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ‘бҹ„бҹҮбһ”бһёбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹҸбһҠбҹ„бһҷбҹ”

### бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘

бһ бһҫбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҒбһ»бһҹбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҠбҹҒбһ…?

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘ бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ—бһ¶бһ–бһўбһ¶бһ…бһ”бһҸбҹӢбһ”бҹӮбһ“бһ…бҹ’бһҡбһҫбһ“ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһӣбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһүбҹ’бһҮбһ¶ бһ¬бһ—бһ¶бһҹбһ¶бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ‘бҹҒбҹ” бһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бҹ’бһҸбһҡбһ¶бһӮбһҳбһ“бҹҚбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбһә бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӣбһҫбһ–бһ бһ»бһўбһҸбҹ’бһҗбһ”бһ‘бһңбҹӮбһ„ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹӮбһ”бһ”бһ»бһҡбһ¶бһҺбһҳбһҪбһҷбһўбһ¶бһ…бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһӣбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

## бһҸбһҫбһ’бҹ’бһңбһҫбһҳбҹүбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ?

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҡбһҖбһңбһ·бһ’бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ LLM бһҠбҹӮбһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бҹ…бһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- бһ”бҹ’бһҡбһҫ APIбҹ” бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҖбһҹбһ¶бһ„бһҹбҹҶбһҺбһҫбһңбҹӮбһ”бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ prompt бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ·бһүбҹ”

бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһӣбҹ’бһ”бһёбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ LLM бһҠбһјбһ…бһҮбһ¶бҹ–

- **openai** бһҠбҹӮбһӣбһ–бһёбһҡбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһ“бҹҒбҹҮбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ•бҹ’бһүбһҫ prompt бҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бҹ–

- **Semantic Kernel**бҹ” Semantic Kernel бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһ–бһёбһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖ Microsoft бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶ C#, Python бһ“бһ·бһ„ Javaбҹ”

## бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһ”бһјбһ„ бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ openai

бһҷбһҫбһ„бһҳбһҖбһҳбһҫбһӣбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһ”бһјбһ„ бһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһҠбҹӮбһӣбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

бһ“бҹ…бһ–бҹҒбһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒ `openai` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҗбһ¶бһҸбһҫбһҳбһ¶бһ“ API key бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҺбһ¶бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖ API key бһҹбһјбһҳбһ‘бҹ…бһ…бҹҶбһҺбһ»бһ… "Keys and Endpoint" бһ“бҹғбһ’бһ“бһ’бһ¶бһ“ Azure OpenAI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһ…бһҳбҹ’бһӣбһ„бһҸбһҳбҹ’бһӣбҹғ "Key 1"бҹ”

![Keys and Endpoint resource blade in Azure Portal](https://learn.microsoft.com/azure/ai-services/openai/media/quickstarts/endpoint.png?WT.mc_id=academic-105485-koreyst)

бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбһ бһҫбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҺбҹӮбһ“бһ¶бҹҶбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһңбһ¶бһ”бһ¶бһ“бһҡбһҪбһ…бһ бһҫбһҷбҹ”

> [!NOTE]
> бһңбһ¶бһҮбһ¶бһҡбһҝбһ„бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ API key бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бҹҒбһүбһ–бһёбһҖбһјбһҠбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһңбһ¶вҖӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ environment variablesбҹ”
>
> - бһ”бһ„бҹ’бһҖбһҫбһҸ environment variable `OPENAI_API_KEY` бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„ API key бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
>   `export OPENAI_API_KEY='sk-...'`

### бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбҹ’бһ“бһ»бһ„ Azure

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Azure OpenAI бһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбҹҖбһ”бһ…бҹҶбһӮбһәбҹ–

```python
openai.api_type = 'azure'
openai.api_key = os.environ["OPENAI_API_KEY"]
openai.api_version = '2023-05-15'
openai.api_base = os.getenv("API_BASE")
```

бһҒбһ¶бһ„бһӣбһҫ бһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбҹ–

- `api_type` бһ‘бҹ…бһҮбһ¶ `azure`бҹ” бһңбһ¶бһ”бҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһІбҹ’бһҷбһ”бҹ’бһҡбһҫ Azure OpenAI бһҳбһ·бһ“бһҳбҹӮбһ“ OpenAI бһ‘бҹҒбҹ”
- `api_key` бһӮбһәбһҮбһ¶бһӮбһ“бҹ’бһӣбһ„ API key бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure Portalбҹ”
- `api_version` бһӮбһәбһҮбһ¶бһҖбҹҶбһҺбҹӮ API бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбҹ” бһ“бҹ…бһ–бҹҒбһӣбһҹбһҡбһҹбҹҒбһҡ бһҖбҹҶбһҺбҹӮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһӮбһә `2023-05-15`бҹ”
- `api_base` бһӮбһәбһҮбһ¶ endpoint бһ“бҹғ APIбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure Portal бһ“бҹ…бһҮбһ·бһҸ API key бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

> [!NOTE] > `os.getenv` бһӮбһәбһҮбһ¶ function бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһ“ environment variablesбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“ environment variables бһҠбһјбһ…бһҮбһ¶ `OPENAI_API_KEY` бһ“бһ·бһ„ `API_BASE`бҹ” бһҹбһјбһҳбһҖбҹҶбһҺбһҸбҹӢ environment variables бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminal бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ¬бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҒбһҒбһҠбһјбһ…бһҮбһ¶ `dotenv`бҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘

бһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһӮбһәбһ”бҹ’бһҡбһҫ class `Completion`бҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһӮбһәбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–

```python
prompt = "Complete the following: Once upon a time there was a"

completion = openai.Completion.create(model="davinci-002", prompt=prompt)
print(completion.choices[0].text)
```

бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҒбһ¶бһ„бһӣбһҫ бһҷбһҫбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһў к°қмІҙ completion бһ бһҫбһҷбһ•бҹ’бһүбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„ prompt бһҳбһҖбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҷбһҫбһ„бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”

### Chat completions

бһҡбһ бһјбһҸбһҳбһҖбһҠбһӣбҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһғбһҫбһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫ `Completion` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹҸбһҳбһ¶бһ“ class бһҳбһҪбһҷбһ‘бҹҖбһҸбһ бҹ…бһҗбһ¶ `ChatCompletion` бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ chatbot бһҮбһ¶бһ„бҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһӮбҹҶбһҡбһјбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһңбһ¶бҹ–

```python
import openai

openai.api_key = "sk-..."

completion = openai.ChatCompletion.create(model="gpt-3.5-turbo", messages=[{"role": "user", "content": "Hello world"}])
print(completion.choices[0].message.content)
```

бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһҖбҹ’бһҡбҹ„бһҷбҹ”

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһӣбһ„ - бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҘбһЎбһјбһңбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһҡбһ”бҹҖбһ”бһҡбҹҖбһ”бһ…бҹҶ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ openai бһңбһ·бһү бһ–бҹҒбһӣбһ“бҹҒбҹҮбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҹбһјбһҳбһ’бҹ’бһңбһҫбһҸбһ¶бһҳбһҮбҹҶбһ бһ¶бһ“бҹ–

1. бһ”бһ„бҹ’бһҖбһҫбһҸ virtual environment бһ бһҫбһҷбһҠбҹҶбһЎбһҫбһ„ openaiбҹ–

   ```bash

   source venv/bin/activate

   ```

   > [!NOTE]
   > бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Windows бһҹбһјбһҳбһңбһ¶бһҷ `venv\Scripts\activate` бһҮбҹҶбһ“бһҪбһҹ `source venv/bin/activate`бҹ”

   > [!NOTE]
   > бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбһјбһ“бһҹбҹ„ Azure OpenAI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ‘бҹ…бһҖбһ¶бһ“бҹӢ [https://portal.azure.com/](https://portal.azure.com/?WT.mc_id=academic-105485-koreyst) бһ бһҫбһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ `Open AI` бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ `Open AI resource` бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҮбҹ’бһҡбһҫбһҹ `Keys and Endpoint` бһ бһҫбһҷбһ…бһҳбҹ’бһӣбһ„бһҸбһҳбҹ’бһӣбҹғ `Key 1`бҹ”

1. бһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһҜбһҖбһҹбһ¶бһҡ _app.py_ бһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһҖбһјбһҠбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

   ```python
   import openai

   openai.api_key = "<replace this value with your open ai key or Azure OpenAI key>"

   openai.api_type = 'azure'
   openai.api_version = '2023-05-15'
   openai.api_base = "<endpoint found in Azure Portal where your API key is>"
   deployment_name = "<deployment name>"

   # бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһҠбһ”бһүбҹ’бһ…бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
   prompt = "Complete the following: Once upon a time there was a"

   # бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢ
   completion = openai.chat.completions.create(model=deployment_name, messages=messages)

   # бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ…бҹҒбһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһҫбһҷбһҸбһ”
   print(completion.choices[0].message.content)
   ```

   > [!NOTE]
   > бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Azure OpenAI бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢ `api_type` бһ‘бҹ… `azure` бһ бһҫбһҷбһҖбҹҶбһҺбһҸбҹӢ `api_key` бһ‘бҹ…бһҖбһјбһ“бһҹбҹ„ Azure OpenAI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”


## бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ

бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһӮбһәбһҮбһ¶ prompt бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҳбһҫбһӣбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶ бһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһңбһ¶бһІбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһ…бҹҶбһ–бҹ„бҹҮбһўбҹ’бһ“бһҖбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ”
