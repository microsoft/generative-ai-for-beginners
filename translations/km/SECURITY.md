## бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ

Microsoft бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ…бҹҶбһ–бҹ„бҹҮбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһ“бҹғбһ•бһӣбһ·бһҸбһ•бһӣбһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ¶бҹҶбһ„бһ”бһҺбҹ’бһҠбһ»бҹҶбһҖбһјбһҠбһ”бҹ’бһҡбһ—бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҮбһ¶ GitHub бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһҡбһҪбһҳбһҳбһ¶бһ“ [Microsoft](https://github.com/microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet), [Xamarin](https://github.com/xamarin), бһ“бһ·бһ„ [бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҮбһ¶ GitHub бһҡбһ”бһҹбҹӢбһҷбһҫбһ„](https://opensource.microsoft.com/)бҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҮбһҝбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһүбһҖбһ„бҹ’бһңбҹҮбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҮбһ¶бһҖбһ¶бһҡбһҖбһ¶бһ“бҹӢбһҖбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢ Microsoft бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбһ¶бһҳ [бһ“бһ·бһҷбһҳбһ“бҹҗбһҷбһҖбһ„бҹ’бһңбҹҮбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҡбһ”бһҹбҹӢ Microsoft](https://aka.ms/opensource/security/definition) бһҹбһјбһҳбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһІбҹ’бһҷбһҷбһҫбһ„бһҠбһјбһ…бһ”бһ¶бһ“бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбҹ”

## бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ”бһүбҹ’бһ бһ¶бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ
