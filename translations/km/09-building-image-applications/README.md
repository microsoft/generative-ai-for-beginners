# бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–

[![Building Image Generation Applications](../../../translated_images/km/09-lesson-banner.906e408c741f4411.webp)](https://aka.ms/gen-ai-lesson9-gh?WT.mc_id=academic-105485-koreyst)

бһҳбһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбҹӮбһҳбҹ’бһҠбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ LLMsбҹ” бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһёбһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ¶бһ“бһ•бһ„бһҠбҹӮбһҡбҹ” бһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶бһҡбһ”бҹҖбһ”бһҳбһҪбһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҳбҹҒбһҢбһ·бһ…, бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ, бһ‘бҹҒбһҹбһ…бһҡбһҺбҹҚ, бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ бҹ’бһӮбҹҒбһҳ, бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҹбҹҶбһ—бһ¶бһҡбҹҲбһ‘бҹ…бһӣбһҫбһҳбҹүбһјбһҠбҹӮбһӣ **GPT Image** бһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғ бһ“бһ·бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бҹ”

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ•бҹ’бһҸбһҫбһҳ

бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹҶбһӣбҹӮбһ„бһҹбҹҶбһҺбһҫбһҠбҹ„бһҷбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бҹ…бһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһҪбһҷбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ **`gpt-image`** бһ–бһё OpenAI вҖ” бһҮбһ¶бһҖбҹҶбһҺбһҫбһҸбһҹбһҳбҹҗбһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһӣбһҫ **[Microsoft Foundry](https://ai.azure.com?WT.mc_id=academic-105485-koreyst)** бһ“бһ·бһ„бһңбҹҒбһ‘бһ·бһҖбһ¶ OpenAIбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбҹҶбһ“бһҪбһҹбһҳбҹүбһјбһҠбҹӮбһӣ DALLВ·E бһ…бһ¶бһҹбҹӢбҹ— (DALLВ·E 2/3 бһӮбһәбһҮбһ¶бһҡбһ”бһҹбҹӢбһ–бһ“бҹ’бһӣбһәбһ…бһ¶бһҹбҹӢ)бҹ”

бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮ **Edu4All** бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбҹҖбһ“бҹ” бһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡбһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ§бһ‘бһ¶бһ бһҡбһҺбҹҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ“бһ·бһ„бһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ”

## бһӮбҹ„бһӣбһҠбҹ…бһҡбҹҖбһ“

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҗбһ¶ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһҠбҹӮбһӣбһңбһ¶бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбҹ”
- бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ `gpt-image` бһ“бһ·бһ„бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһңбһ¶бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣ DALLВ·E бһ…бһ¶бһҹбҹӢбҹ—бҹ”
- бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ Python (бһ“бһ·бһ„ TypeScript / .NET)бҹ”
- бһҖбҹӮбһӣбһҳбҹ’бһўбһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷ metapromptsбҹ”

## бһҸбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶бһўбҹ’бһңбһё?

бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ–бһёбһҹбҹҶбһҺбһҫбһўбһҸбҹ’бһҗбһ”бһ‘бһҳбһҪбһҷбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ”бһҠбһјбһ…бһҮбһ¶ `gpt-image` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ transformer + diffusion: бһҳбҹүбһјбһҠбҹӮбһӣбһҡбҹҖбһ“бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһ“бҹ…бһ–бҹҒбһӣбһ”бһ¶бһ“бһҹбҹҶбһҺбһҫ бһңбһ¶бһӮбһәбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӣбһ»бһ”бһҹбҹҶбһӣбҹҒбһ„бһҒбҹ’бһ“бҹӮбһ„бһҶбһ¶бһ”бҹӢбҹ—бһ‘бҹ…бһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҹбҹ’бһҡбһ”бһ“бһ№бһ„бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бҹ”

бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһёбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһӣбҹ’бһ”бһёбһӮбһәбҹ–

- **`gpt-image` (OpenAI)** - бһҖбҹҶбһҺбҹӮбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбҹ” бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ–бһёбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһҡбһјбһ”бһ—бһ¶бһ– (inpainting бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹӢ)бҹ”
- **Midjourney** - бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ–бһёбһ—бһ¶бһӮбһёбһ‘бһёбһ”бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ бһ»бһҳбҹҒбһҢбҹҖбһӣ Discordбҹ”

> бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ– OpenAI бһ…бһ¶бһҹбҹӢбҹ— - **DALLВ·E 2** бһ“бһ·бһ„ **DALLВ·E 3** - бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһҳбһ»бһ“бҹ” DALLВ·E 3 бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҗбҹ’бһҳбһёбҹ—бһ‘бҹҖбһҸбһ бһҫбһҷ бһ бһҫбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһјбһ…бһҮбһ¶ `create_variation` бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ DALLВ·E 2 бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ `gpt-image` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҗбҹ’бһҳбһёбҹ—бҹ”

### бһҸбһҫбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ `gpt-image` бһҺбһ¶бһҠбҹӮбһӣбһҒбҹ’бһүбһ»бҹҶбһӮбһҪбһҡбһҸбҹӮбһ”бҹ’бһҡбһҫ?

бһ“бҹ…бһӣбһҫ Microsoft Foundry бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ **бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ‘бһјбһ‘бҹ…**:

бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһҮбһёбһҳбҹүбһјбһҠбҹӮбһӣ [Foundry image models list](https://learn.microsoft.com/azure/ai-foundry/openai/concepts/models?WT.mc_id=academic-105485-koreyst) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“ бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбҹ”

> **бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹҲ** бһҳбҹүбһјбһҠбҹӮбһӣ `gpt-image` бһ”бһүбҹ’бһ…бҹҒбһүбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ **base64** (`b64_json`), бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶ URL бһ‘бҹҒбҹ” бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ decode бһҹбһ¶бһ…бҹӢбһўбһҖбҹ’бһҹбһҡ base64 бһ‘бҹ…бһҮбһ¶бһўбһ»бһҖбҹ’бһҸбҹҒбһҸбһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ - бһӮбҹ’бһҳбһ¶бһ“ URL бһҡбһјбһ”бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖбҹ”

## бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбҹҶбһҖбһ·бһӣбһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ“бҹ…бһӣбһҫ **Azure OpenAI бһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry** (бһӮбҹҶбһҡбһј `aoai-*`) бһ¬бһңбҹҒбһ‘бһ·бһҖбһ¶ **OpenAI** (бһӮбҹҶбһҡбһј `oai-*`)бҹ”

### 1. бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһҠбҹҶбһЎбһҫбһ„бһҳбҹүбһјбһҠбҹӮбһӣ

бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ‘бҖҠбһ№бһҖбһ“бһ¶бҹҶ [бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/create-resource?pivots=web-portal&WT.mc_id=academic-105485-koreyst) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Microsoft Foundry бһҡбһҪбһ…бһҠбҹҶбһЎбһҫбһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ– - **`gpt-image-2`** бһӮбһәбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбҹ”

### 3. бһҸбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ

## бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё

бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ `app.py` бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ” бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҳбһҪбһҷ бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҮбһ¶бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ PNGбҹ”

```python
import os
import base64
from openai import AzureOpenAI
from PIL import Image
import dotenv

dotenv.load_dotenv()

# бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ’бһ“бһ’бһ¶бһ“ Azure OpenAI (Microsoft Foundry) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
# бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһ¶бһ“бҹӢ API бһҗбҹ’бһҳбһё - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҜбһҖбһҹбһ¶бһҡ Foundry бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ“бһ¶бһ“бҹӢбһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”
client = AzureOpenAI(
    api_key=os.environ["AZURE_OPENAI_API_KEY"],
    api_version="2025-04-01-preview",
    azure_endpoint=os.environ["AZURE_OPENAI_ENDPOINT"],
)

deployment = os.environ["AZURE_OPENAI_DEPLOYMENT"]  # бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ "gpt-image-2"

result = client.images.generate(
    model=deployment,
    prompt='Bunny on a horse, holding a lollipop, on a foggy meadow where it grows daffodils',
    size="1024x1024",   # бһ•бһ„бһҠбҹӮбһҡ 1536x1024 (бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһӣбһ¶бһҷ), 1024x1536 (бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһҖбҹҶбһ–бһҹбҹӢ), бһ¬ "бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·"
    n=1,
)

# бһҳбҹүбһјбһҠбҹӮбһӣ gpt-image бһ”бһүбҹ’бһҮбһјбһ“бһҸбһҳбҹ’бһӣбҹғ base64 (b64_json) бһҳбһ·бһ“бһҳбҹӮбһ“ URL - бһҒбһҖбһ‘бһёбһңбһ¶бһңбһ¶бһ‘бҹ…бһҮбһ¶ bytes.
image_bytes = base64.b64decode(result.data[0].b64_json)

os.makedirs("images", exist_ok=True)
image_path = os.path.join("images", "generated-image.png")
with open(image_path, "wb") as f:
    f.write(image_bytes)

Image.open(image_path).show()
```

бһ”бһҫбһҖбһңбһ¶бһҠбҹ„бһҷ `python app.py`бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбҹ’бһ“бһҖбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ PNG бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `images/`бҹ”

> бһҖбһ¶бһҡбһ бҹ…бһҳбһҪбһҷбһ‘бҹ… `images.generate` бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһ“бһ¶бһҳбһҹбҹҶбһҺбһҫбһҠбһҠбһҠбҹӮбһӣ вҖ” бһҳбҹүбһјбһҠбҹӮбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ `temperature` (бһҠбҹӮбһӣбһҮбһ¶бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘)бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ бһ…бҹ„бһӣбһ бҹ… API бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ; бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҹбҹҶбһҺбһҫбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҮбһ¶бһ„бһҳбһ»бһ“бҹ”

## бһҖбҹӮбһӣбһҳбҹ’бһўбһҡбһјбһ”бһ—бһ¶бһ–

бһҳбҹүбһјбһҠбҹӮбһӣ `gpt-image` бһўбһ¶бһ… **бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ** бһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ: бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһҡбһјбһ”бһ—бһ¶бһ– бһҳбһ¶бһҹ (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһҠбҹӮбһӣбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҠбһјбһҡ бһ“бһ·бһ„бһҹбҹҶбһҺбһҫбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбҹ” бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһҖбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҪбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бҹҒбһҡбһҮбһ¶бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ base64 бһ•бһ„бһҠбҹӮбһҡбҹ”

```python
result = client.images.edit(
    model=deployment,
    image=open("sunlit_lounge.png", "rb"),
    mask=open("mask.png", "rb"),
    prompt="A sunlit indoor lounge area with a pool containing a flamingo",
)
image_bytes = base64.b64decode(result.data[0].b64_json)
with open("images/edited-image.png", "wb") as f:
    f.write(image_bytes)
```

## бһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҳбһҪбһҷ metaprompts

бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ¶бһ“ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бһүбҹ’бһ…бҹҒбһүбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ¬бһҳбһ·бһ“бһҹбһҳбһҡбһҳбҹ’бһҷбҹ” **metaprompt** бһӮбһәбһҮбһ¶бһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһ…бҹҶбһ–бҹ„бҹҮбһҳбһ»бһҒбһҹбҹҶбһҺбһҫбһҡбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ…бҹҒбһүбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбҹ”

```python
disallow_list = "swords, violence, blood, gore, nudity, sexual content, adult content, adult themes, adult language"

meta_prompt = f"""You are an assistant designer that creates images for children.

The image needs to be safe for work and appropriate for children.
```
