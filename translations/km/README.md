# AI បង្កើតសម្រាប់អ្នកដំបូង (Version 3) - វគ្គសិក្សា

រៀនមូលដ្ឋាននៃការបង្កើតកម្មវិធី AI បង្កើតជាមួយវគ្គសិក្សាសង្ខេប 21 មេរៀនដោយអ្នកផ្សព្វផ្សាយ Microsoft Cloud។

## 🌱 ការចាប់ផ្តើម

វគ្គសិក្សានេះមាន 21 មេរៀន ហើយមេរៀននីមួយៗនិយាយពីប្រធានបទផ្ទាល់ខ្លួន ដូច្នេះចាប់ផ្តើមធ្វើយ៉ាងណាក៏បានតាមចំណង់ចំណូលចិត្ត!

មេរៀនត្រូវបានសម្គាល់ថាជា "Learn" ដែលពន្យល់ពីមូលដ្ឋាន AI បង្កើត ឬ "Build" ដែលពន្យល់ទៅលើមូលដ្ឋាន និងឧទាហរណ៍កូដទាំង **Python** និង **TypeScript** នៅពេលអាចធ្វើទៅបាន។

សម្រាប់អ្នកអភិវឌ្ឍន៍ .NET សូមចុះទៅ [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

មេរៀននីមួយៗក៏មានផ្នែក "Keep Learning" ដែលផ្តល់ឧបករណ៍សម្រាប់រៀនបន្ថែម។
