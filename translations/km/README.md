# Generative AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (бһҖбҹҶбһҺбҹӮ бҹЈ) - бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбһҪбһҷ

бһҹбһјбһҳбһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Generative AI бһҮбһ¶бһҳбһҪбһҷбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҳбһ¶бһҸбһ·бһҖбһ¶ бҹўбҹЎ бһҳбҹҒбһҡбҹҖбһ“бһҠбҹ„бһҷ Microsoft Cloud Advocatesбҹ”

## рҹҢұ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“ бҹўбҹЎ бһҳбҹҒбһҡбҹҖбһ“бҹ” бһҳбҹҒбһҡбҹҖбһ“бһҳбһҪбһҷбҹ—бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“!

бһҳбҹҒбһҡбҹҖбһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһҖбҹӢбһҹбҹ’бһӣбҹҖбһҖбһҮбһ¶бһҳбҹҒбһҡбҹҖбһ“ "бһҡбҹҖбһ“" бһҠбҹӮбһӣбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸ Generative AI бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ бһ¬бһҳбҹҒбһҡбҹҖбһ“ "бһҹбһ„бҹӢ" бһҠбҹӮбһӣбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸ бһ“бһ·бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ **Python** бһ“бһ·бһ„ **TypeScript** бһ“бҹ…бһ–бҹҒбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ .NET бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)бҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҮбһҪбһ”бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖ бһ¬бһҳбһ¶бһ“бһҹбҹҶбһҺбһҪбһҡбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AIбҹ” бһ…бһјбһӣбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҹбһ·бһҹбҹ’бһҹбһҡбҹҖбһ“ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһё MCPбҹ” бһңбһ¶бһҮбһ¶бһҹбһ бһӮбһҳбһ“бҹҚбһӮбһ¶бҹҶбһ‘бҹ’бһҡ бһҠбҹӮбһӣбһҹбҹҶбһҺбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚ бһ бһҫбһҷбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҠбҹ„бһҷбһҹбҹҒбһҡбһёбҹ”
