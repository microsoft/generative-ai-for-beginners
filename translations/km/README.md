# Generative AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ– 3) - бһҳбҹҒбһҡбҹҖбһ“бһҳбһҪбһҷ

бһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Generative AI бһҮбһ¶бһҳбһҪбһҷбһҳбҹҒбһҡбҹҖбһ“бһ–бҹҒбһүбһӣбҹҒбһү 21 бһҳбҹҒбһҡбҹҖбһ“бһҠбҹ„бһҷ Microsoft Cloud Advocatesбҹ”

## рҹҢұ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҳбһ¶бһ“ 21 бһҳбҹҒбһҡбҹҖбһ“бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбҹ—бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ!

бһҳбҹҒбһҡбҹҖбһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҮбһ¶бһҳбҹҒбһҡбҹҖбһ“ "бһҹбһ·бһҖбҹ’бһҹбһ¶" бһҠбҹӮбһӣбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸ Generative AI бһ¬бһҳбҹҒбһҡбҹҖбһ“ "бһ”бһ„бҹ’бһҖбһҫбһҸ" бһҠбҹӮбһӣбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһ“бһ·бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһ‘бһ¶бҹҶбһ„бһҖбҹ’бһ“бһ»бһ„ **Python** бһ“бһ·бһ„ **TypeScript** бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ .NET бһҹбһјбһҳбһҳбһҫбһӣ [Generative AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

бһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹҸбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһ“бҹӮбһҖ "бһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶" бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”
