## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһ–бһ·бһ—бһ–бһ“бҹғ LLM бһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ  бһӮбһәбһӮбһҪбһҡбһұбҹ’бһҷбһҡбҹҶбһ—бһҫбһ” бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһңбһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҮбһ¶бһ“бһ·бһҡбһ“бҹ’бһҸбһҡбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһўбҹҶбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҖбһҳбҹ’бһҳбһҹбһ·бһ‘бҹ’бһ’бһ·бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ“бһ№бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ  бһҹбһјбһҳбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҳбҹҒбһҡбҹҖбһ“ ["бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” LLMs бһ•бҹ’бһҹбҹҒбһ„бҹ—"](../02-exploring-and-comparing-different-llms/README.md?WT.mc_id=academic-105485-koreyst)бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҖбҹҸбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘ fine-tuning бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһӣбһҳбҹ’бһўбһ·бһҸбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“ ["Fine-Tuning LLMs"](../18-fine-tuning/README.md?WT.mc_id=academic-105485-koreyst)бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бһ–бһ…бҹҶбһ 
- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ 
- бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Hugging Face бһ“бһ·бһ„бһҖбһ¶бһҸбһ¶бһЎбһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry

## бһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бһ–бһ…бҹҶбһ  бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бһ”бһ¶бһ“бһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҷбҹүбһ¶бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӣбһјбһҸбһӣбһ¶бһҹбҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ‘бһјбһ‘бһ¶бҹҶбһ„бһңбһ·бһҹбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” Open Source Initiative (OSI) бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ[бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ 10бһ…бҹҶбһҺбһ»бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё](https://web.archive.org/web/20241126001143/https://opensource.org/osd?WT.mc_id=academic-105485-koreyst)бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бҹ” бһҖбһјбһҠбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ”бһ¶бһ“бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҷбҹүбһ¶бһ„бһ”бһҫбһҖбһ…бҹҶбһ бһҖбҹ’бһҡбҹ„бһҳбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ”бһҺбҹ’бһҺбһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸбһҠбҹ„бһҷ OSIбҹ”

бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶ бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ LLM бһҳбһ¶бһ“бһ’бһ¶бһҸбһ»бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҳбһ·бһ“бһҳбҹӮбһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ–бҹҒбһүбһӣбҹҒбһүбһ‘бҹҒбҹ” бһңбһ¶бһ”бһ¶бһ“бһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбһҳбһ“бҹҚбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһ“бҹғбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘ LLMбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһўбҹ„бһҷбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бһ”бһ»бһҡбһ¶бһҺ бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ“бҹҒбҹҮбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бҹӮбһҖбһ…бһ¶бһҷбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбҹ–

- бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҠбҹӮбһӣбҹ”
- бһ‘бҹҶбһ„бһ“бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ–бҹҒбһүбһӣбҹҒбһүбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбҹ”
- бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбҹ”
- бһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһ„бҹӢ (fine-tuning)бҹ”
- бһ‘бҹҶбһ„бһ“бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ–бҹҒбһүбһӣбҹҒбһү бһ“бһ·бһ„бһҳбҹүбҹҒбһҸбҹ’бһҡбһ·бһҖбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбҹ”

бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҳбһ¶бһ“бһҸбҹӮбһҳбҹүбһјбһҠбҹӮбһӣбһҸбһ·бһ…бһҸбһҪбһ…бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹӮбһӣбһҹбһҳбһҡбһҖбҹӢбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ“бҹҒбҹҮбҹ” [бһҳбҹүбһјбһҠбҹӮбһӣ OLMo бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ Allen Institute for Artificial Intelligence (AllenAI)](https://huggingface.co/allenai/OLMo-7B?WT.mc_id=academic-105485-koreyst) бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһҸбһ¶бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҷбҹ„бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶ "бһҳбҹүбһјбһҠбҹӮбһӣвҖӢбһ…бҹҶбһ " бһ“бҹ…бһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһӣбһ“бҹҒбҹҮ бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҳбһ·бһ“бһ–бһ·бһҸбһҮбһ¶бһҹбһҳбһҡбһҳбҹ’бһҷбһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбһ¶бһ„бһӣбһҫбһ“бҹ…бһ–бҹҒбһӣбһҹбһҡбһҹбҹҒбһҡбһ“бҹҒбҹҮбһ‘бҹҒбҹ”

[Falcon](https://huggingface.co/tiiuae?WT.mc_id=academic-105485-koreyst) бһӮбһәбһҮбһ¶бһҳбҹүбһјбһҠбҹӮбһӣ LLM бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ Technology Innovation Institute (**TII**)бҹ” Falcon-40B бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһӣбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ 40 бһ–бһ¶бһ“бҹӢбһӣбһ¶бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбҹ’бһўбһҮбһ¶бһ„ GPT-3 бһҠбҹ„бһҷбһҳбһ¶бһ“бһҗбһңбһ·бһҖбһ¶бһӮбһҺбһ“бһ¶бһҗбһҷбһ…бһ»бҹҮбҹ” бһ“бҹҒбҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбһҖбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳ FlashAttention бһ“бһ·бһ„ multiquery attention бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһңбһ¶бһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ–бҹҒбһӣбһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” бһҮбһ¶бһҳбһҪбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһҒбҹ’бһңбһҖбҹӢбһ“бҹҒбҹҮ Falcon-40B бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһҮбҹӮбһҖбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҒбҹ’бһӣбҹҮбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣ Falcon бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбһ№бһҳбһҡбһҪбһҳбһҳбһ¶бһ“ [OpenAssistant](https://huggingface.co/OpenAssistant/falcon-40b-sft-top1-560?WT.mc_id=academic-105485-koreyst) бһҮбһ¶бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһҠбҹӮбһӣбһҹбһ¶бһ„бһҹбһ„бҹӢбһӣбһҫбһҳбҹүбһјбһҠбҹӮбһӣвҖӢбһ…бҹҶбһ  бһ“бһ·бһ„ [GPT4ALL](https://huggingface.co/nomic-ai/gpt4all-falcon?WT.mc_id=academic-105485-koreyst) бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ”бһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ”

## бһҡбһ”бҹҖбһ”бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ

бһҳбһ·бһ“бһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣвҖӢбһ…бҹҶбһ бһ‘бҹҒбҹ” бһҖбһ“бҹ’бһӣбҹӮбһ„бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбһәбһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһҡбһ„вҖӢбһҸбһ¶бһҳбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҸбһ¶бһЎбһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundryбҹ” бһңбһ¶бһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһ–бһёбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбҹ” Hugging Face бһҖбҹҸбһ”бһ¶бһ“бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ LLM Leaderboard бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҳбҹүбһјбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ“бҹ…бһӣбһҫбһҳбҹҒбһҸбҹ’бһҡбһ·бһҖбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ”

бһ–бҹҒбһӣбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” LLMs бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ— [Artificial Analysis](https://artificialanalysis.ai/?WT.mc_id=academic-105485-koreyst) бһӮбһәбһҮбһ¶бһ’бһ“бһ’бһ¶бһ“бһӣбҹ’бһўбһҳбһҪбһҷбһ‘бҹҖбһҸбҹ–

бһ¬бһҹбҹ’бһҹ: Artificial Analysis

бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһӣбһҫбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбһҳбҹ’бһҡбһ№бһҳбһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһңбһ·бһҹбҹҗбһҷбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһӣбһ„бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣвҖӢбһ…бҹҶбһ бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҸбһ¶бһҳбһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһҪбһӣбһҳбһҪбһҷбһ‘бҹҖбһҸбҹ”

## бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ

бһ•бҹ’бһ“бҹӮбһҖбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣвҖӢбһ…бҹҶбһ бһӮбһәбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ–бһҪбһҖбһңбһ¶бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбҹ” бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҸбһ¶бһЎбһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry (https://ai.azure.com?WT.mc_id=academic-105485-koreyst) бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһҸбһ»бҹҶбһҜбһҖбһ‘бҹҒбһҹбһҖбҹ’бһ“бһ»бһ„ Hugging Face бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһ”бһ¶бһ“бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮбһ•бһ„бһҠбҹӮбһҡбҹ”
