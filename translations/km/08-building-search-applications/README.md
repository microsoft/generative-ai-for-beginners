### бһҸбһҫ cosine similarity бһҮбһ¶бһўбҹ’бһңбһё?

Cosine similarity бһӮбһәбһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһңбһ¶бһҹбҹӢбһ–бһёбһ—бһ¶бһ–бһҹбҹ’бһҡбһҠбҹҖбһ„бһҡбһңбһ¶бһ„бһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһ–бһёбһҡбһӮбҹ’бһ“бһ¶ бһўбҹ’бһ“бһҖбһ“бһ·бһ„бһҹбҹ’бһҸбһ¶бһ”бҹӢбһңбһ¶бһҗбһ¶ `nearest neighbor search`бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҮбһ¶бһҳбһҪбһҷ cosine similarity бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮ _бһ”бһҳбҹ’бһӣбҹӮбһ„_ бһўбһҸбҹ’бһҗбһ”бһ‘бһҹбҹҶбһҺбһҪбһҡбһҮбһ¶бһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ OpenAI Embedding APIбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһӮбһҺбһ“бһ¶бһҡбҹүбҹҒбһҹбҹҠбһёбһҷбҹүбһј cosine similarity бһҡбһңбһ¶бһ„вҖӢбһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһҹбҹҶбһҺбһҪбһҡ бһ“бһ·бһ„бһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ Embeddingбҹ” бһҹбһјбһҳбһ…бһ¶бҹҶбһҗбһ¶ бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ Embedding бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһ„бһҹбҹҶбһЎбҹҒбһ„ YouTube бһҡбһ¶бһӣбҹӢбһ•бҹ’бһ“бҹӮбһҖбһўбһҸбҹ’бһҗбһ”бһ‘бҹ” бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҳбһҫбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбҹҖбһ” cosine similarity бһ бһҫбһҷбһ•бҹ’бһ“бҹӮбһҖбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҳбһ¶бһ“ cosine similarity бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҠбҹӮбһӣбһҹбҹ’бһҡбһҠбҹҖбһ„бһҮбһ¶бһ„бһӮбҹҒбһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһҺбһҪбһҡбҹ”

бһҷбҹ„бһ„бһ‘бһҹбҹ’бһҹбһ“бһңбһ·бһҮбҹ’бһҮбһ¶бһҳбһ¶бһ“бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ cosine similarity бһңбһ¶бһҹбҹӢбһҖбһҳбҹ’бһ–бһҹбҹӢ acute angle бһҡбһңбһ¶бһ„бһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһ–бһёбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһңбҹӮбһ„бһ–бһ бһ»бһңбһ·бһҳбһ¶бһҸбҹ’бһҡбҹ” бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢбһ“бҹҒбҹҮбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһ”бһҫбһҜбһҖбһҹбһ¶бһҡбһ–бһёбһҡбһҳбһ¶бһ“бһ…бһҳбҹ’бһ„бһ¶бһҷ Euclidean бһҶбҹ’бһ„бһ¶бһҷбһ–бһёбһӮбҹ’бһ“бһ¶бһҖбҹ’бһҡбҹ„бһҳбһ”бҹ’бһҡбһңбҹӮбһ„ бһҒбһҺбҹҲбһңбһ·бһ»бһҳбһ¶бһҸбҹ’бһҡбһҒбһјбһ…бһҸбһ¶бһҳбһңбһ·бһ»бһҳбһ¶бһҸбҹ’бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹ’бһҡбһ бҹӮбһӣ angle бһҡбһңбһ¶бһ„бһңбҹүбһ·бһҖбһ‘бҹҗбһҡбһ–бһҪбһҖбһңбһ¶бһўбһ¶бһ…бһҸбһ·бһ…бһ бһҫбһҷбһҳбһ¶бһ“ cosine similarity бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһҹбҹҶбһҡбһ¶бһ”бҹӢ cosine similarity бһҹбһјбһҳбһҳбһҫбһӣ [Cosine similarity](https://en.wikipedia.org/wiki/Cosine_similarity?WT.mc_id=academic-105485-koreyst)бҹ”

## бһҖбһ¶бһҡбһҖбһҹбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҠбҹҶбһ”бһјбһ„

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһ“бһ№бһ„бһҡбҹҖбһ“бһҡбһ”бҹҖбһ”бһҖбһҹбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бҹ’бһҡбһҫ Embeddingsбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бһ№бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ“бһ·бһҹбҹ’бһҹбһ·бһҸбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһёбһҠбҹҒбһўбһјбһҠбҹ„бһҷбһңбһ¶бһҷбһҹбҹҶбһҺбһҪбһҡбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһүбҹ’бһҮбһёбһңбһёбһҠбҹҒбһўбһјбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҹбҹҶбһҺбһҪбһҡбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһёбһҠбҹҒбһўбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбһ“бҹ„бҹҮбҹ”
