бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҸбһ¶бһҸбһ’бҹ’бһңбһҫбһўбһ”бҹӢбһҡбҹҶбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ” бһҹбҹ’бһҸбһ¶бһҸбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҳбһ·бһ“бһҡбһҖбһ”бҹ’бһҡбһ¶бһҖбҹӢбһ…бҹҶбһҺбҹҒбһүбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҠбһӣбҹӢбһҹбһ·бһҹбҹ’бһҹбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ‘бҹҒбһҹбһҖбҹҶбһ–бһ»бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбҹ” бһҹбҹ’бһҸбһ¶бһҸбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҳбһ¶бһ“бһңбһёбһҠбҹҒбһўбһј YouTube бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҹбһ·бһҹбҹ’бһҹбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ“бһўбҹҶбһ–бһё AIбҹ” бһҹбҹ’бһҸбһ¶бһҸбһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҹбһ·бһҹбҹ’бһҹбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһёбһҠбҹҒбһўбһј YouTube бһҠбҹ„бһҷбһңбһ¶бһҷбһҹбҹҶбһҺбһҪбһҡбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҹбһ·бһҹбҹ’бһҹбһўбһ¶бһ…бһңбһ¶бһҷ 'What are Jupyter Notebooks?' бһ¬ 'What is Azure ML' бһ бһҫбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бһ№бһ„бһ”бһүбҹ’бһҮбһјбһ“бһ”бһүбҹ’бһҮбһёбһңбһёбһҠбҹҒбһўбһј YouTube бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҹбҹҶбһҺбһҪбһҡбһ“бҹ„бҹҮ бһ бһҫбһҷбһӣбҹ’бһўбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһ‘бҹҖбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһҸбҹҶбһҺбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһёбһҠбҹҒбһўбһј бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҪбһҡбһ“бҹ„бҹҮбҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбҹ–

- бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮбҹ”
- Text Embeddings бһҮбһ¶бһўбҹ’бһңбһёбҹ”
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ Text Embeddingsбҹ”
- бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹ’бһ“бһ»бһ„бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ Text Embeddingsбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһ•бҹ’бһҸбһӣбҹӢбһҮбһ¶бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһ¶бһҖбҹ’бһҷбһӮбһ“бҹ’бһӣбһ№бҹҮбҹ”
- бһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһўбҹҶбһ–бһё Text Embeddingsбҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Embeddings бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

## бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ?

Cosine similarity бһӮбһәбһҮбһ¶бһҳбһ¶бһҸбҹ’бһҡбһҮбһ¶бһҸбһ·бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ¶бһҹбҹӢбһ—бһ¶бһ–бһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ–бһёбһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ¶бһ“бһҹбҹ’бһҠбһ¶бһ”бҹӢбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһҲбҹ’бһҳбҹ„бҹҮ `nearest neighbor search` бһ•бһ„бһҠбҹӮбһҡбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ cosine similarity бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮ _vectorize_ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘ _query_ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ OpenAI Embedding APIбҹ” бһ–бҹ’бһҡбҹ„бҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһӮбһҺбһ“бһ¶бһ‘бҹҶбһ бҹҶбһ“бҹғ _cosine similarity_ бһҡбһңбһ¶бһ„ query vector бһ“бһ·бһ„бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ“бһёбһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ Embeddingбҹ” бһ…бһ„бһ…бһ¶бҹҶбһҗбһ¶ бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚ Embedding бһҳбһ¶бһ“бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһңбһёбһҠбҹҒбһўбһј YouTube бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбҹ—бҹ” бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбһ¶бһҳвҖӢбһҸбһҳбҹ’бһӣбҹғ cosine similarity бһ бһҫбһҷбһ•бҹ’бһ“бҹӮбһҖбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғ cosine similarityбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҠбҹӮбһӣбһҹбҹ’бһҡбһҠбҹҖбһ„бһ”бҹҶбһ•бһ»бһҸбһ“бһ№бһ„бһҹбҹҶбһҺбһҪбһҡбҹ”

бһ–бһёбһ‘бһҹбҹ’бһҹбһ“бһңбһ·бһҮбҹ’бһҮбһ¶бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ cosine similarity бһңбһ¶бһҹбҹӢбһӮбҹ’бһҡбҹ„бһ„бһҠбҹӮбһ“бһҳбһ»бҹҶбһҡбһңбһ¶бһ„бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ–бһёбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ  multidimensionalбҹ” бһңбһ¶бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһ–бҹ’бһҡбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡбһ–бһёбһҡбһ‘бҹ…бһҶбҹ’бһ„бһ¶бһҷбһӮбҹ’бһ“бһ¶бһҸбһ¶бһҳбһӣбҹҶбһ  Euclidean бһҠбҹ„бһҷбһҹбһ¶бһҡбһ‘бҹҶбһ бҹҶ бһңбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһҳбһ»бҹҶбһҸбһјбһ…бһҮбһ¶бһ„бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҳбһ¶бһ“ cosine similarity бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбҹғ cosine similarity бһҹбһјбһҳбһҳбһҫбһӣ [Cosine similarity](https://en.wikipedia.org/wiki/Cosine_similarity?WT.mc_id=academic-105485-koreyst)бҹ”

## бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһ“бһ№бһ„бһҡбҹҖбһ“бһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Embeddings бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бҹҒбҹҮбһ“бһ№бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҹбһ·бһҹбҹ’бһҹбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһёбһҠбҹҒбһўбһјбһҠбҹ„бһҷбһңбһ¶бһҷбһҹбҹҶбһҺбһҪбһҡбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһ”бһүбҹ’бһҮбһёбһңбһёбһҠбҹҒбһўбһјбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҹбҹҶбһҺбһҪбһҡбҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһҸбҹҶбһҺбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһҖбҹ’бһ“бһ»бһ„бһңбһёбһҠбҹҒбһўбһј бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҪбһҡбһ“бҹ„бҹҮбҹ”
