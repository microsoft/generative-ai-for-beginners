# бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҹбҹҶбһЎбҹҒбһ„

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҹбҹҶбһЎбҹҒбһ„бһ‘бһ¶бһүбһҷбһҖбһўбһҸбҹ’бһҗбһ”бһ‘бһңбһёбһҠбҹҒбһўбһј YouTube бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһңбһ¶бһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһјбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ Semantic бһҮбһ¶бһҳбһҪбһҷ OpenAI Embeddings бһ“бһ·бһ„ Functions бҹ”

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҹбҹҶбһЎбҹҒбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ“бҹ…бһӣбһҫбһҖбҹҶбһҺбҹӮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ Windows 11, macOS Ventura бһ“бһ·бһ„ Ubuntu 22.04 (бһ“бһ·бһ„бһӣбһҫбһҹ).

## бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Azure OpenAI Service бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

> [!IMPORTANT]
> бһҷбһҫбһ„бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– Azure CLI бһ‘бҹ…бһҖбҹҶбһҺбҹӮбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһ—бһ¶бһ–бһҹбһҳбһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷ OpenAI
> бһҳбһҫбһӣ [бһҜбһҖбһҹбһ¶бһҡбһҺбҹҚ](https://learn.microsoft.com/cli/azure/update-azure-cli?WT.mc_id=academic-105485-koreyst)

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“

> [!NOTE]
> бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ "semantic-video-search" бһ“бҹ… East USбҹ”
> бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһҲбҹ’бһҳбҹ„бҹҮбһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бһ”бһ¶бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹҒбһӣбһҫбһ–бҹҒбһӣбһ”бҹ’бһҠбһјбһҡбһ‘бһёбһҸбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“, 
> бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҠбһҸбһ¶бһҡбһ¶бһ„ [бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ](https://aka.ms/oai/models?WT.mc_id=academic-105485-koreyst)бҹ”

```console
az group create --name semantic-video-search --location eastus
```

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ Azure OpenAI Serviceбҹ”

```console
az cognitiveservices account create --name semantic-video-openai --resource-group semantic-video-search \
    --location eastus --kind OpenAI --sku s0
```

1. бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһ…бһ”бҹӢ бһ“бһ·бһ„бһҖбһјбһ“бһҹбҹ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮ

```console
az cognitiveservices account show --name semantic-video-openai \
   --resource-group  semantic-video-search | jq -r .properties.endpoint
az cognitiveservices account keys list --name semantic-video-openai \
   --resource-group semantic-video-search | jq -r .key1
```

1. бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҠбһјбһ…бһҸбһ‘бҹ…бҹ–
   - `text-embedding-ada-002` бһҖбҹҶбһҺбҹӮ `2` бһ¬бһӣбһҫбһҹ, бһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `text-embedding-ada-002`
   - `gpt-35-turbo` бһҖбҹҶбһҺбҹӮ `0613` бһ¬бһӣбһҫбһҹ, бһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `gpt-35-turbo`

```console
az cognitiveservices account deployment create \
    --name semantic-video-openai \
    --resource-group  semantic-video-search \
    --deployment-name text-embedding-ada-002 \
    --model-name text-embedding-ada-002 \
    --model-version "2"  \
    --model-format OpenAI \
    --scale-settings-scale-type "Standard"
az cognitiveservices account deployment create \
    --name semantic-video-openai \
    --resource-group  semantic-video-search \
    --deployment-name gpt-35-turbo \
    --model-name gpt-35-turbo \
    --model-version "0613"  \
    --model-format OpenAI \
    --sku-capacity 100 \
    --sku-name "Standard"
```

бһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҡбһҡбһ”бҹҖбһ”бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҹбҹҶбһЎбҹҒбһ„ YouTubeбҹ”

### бһ“бҹ…бһӣбһҫ Windows

бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбһҗбҹҒбһҡбһ‘бҹ…бһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ `user` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
`Windows Start` > `Edit the system environment variables` > `Environment Variables` > `User variables` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ [USER] > `New`бҹ”

```text
AZURE_OPENAI_API_KEY  \<your Azure OpenAI Service API key>
AZURE_OPENAI_ENDPOINT \<your Azure OpenAI Service endpoint>
AZURE_OPENAI_MODEL_DEPLOYMENT_NAME \<your Azure OpenAI Service model deployment name>
GOOGLE_DEVELOPER_API_KEY = \<your Google developer API key>
```

## бһҠбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Python бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

1. бһҠбҹҶбһЎбһҫбһ„ [git client](https://git-scm.com/downloads?WT.mc_id=academic-105485-koreyst) бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бҹ”
1. бһ–бһёбһ…бҹҶбһ бҹҖбһ„ `Terminal`, бһ”бһҖбһ…бһҳбҹ’бһӣбһ„бһӮбҹҶбһҡбһјбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҗбһҸ repo бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбҹ”

    ```bash
    git clone https://github.com/gloveboxes/semanic-search-openai-embeddings-functions.git
    ```

1. бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҗбһҸ `data_prep` бҹ”

   ```bash
   cd semanic-search-openai-embeddings-functions/src/data_prep
   ```

1. бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһҡбһ·бһ”бһ‘ Python virtual бҹ”

    бһ“бҹ…бһӣбһҫ Windows:

    ```powershell
    python -m venv .venv
    ```

    бһ“бҹ…бһӣбһҫбһҷ macOS бһ“бһ·бһ„ Linux:

    ```bash
    python3 -m venv .venv
    ```

1. бһ”бһҫбһҖбһ”бһҡбһ·бһ”бһ‘ Python virtual бҹ”

   бһ“бҹ…бһӣбһҫ Windows:

   ```powershell
   .venv\Scripts\activate
   ```

   бһ“бҹ…бһӣбһҫбһҷ macOS бһ“бһ·бһ„ Linux:

   ```bash
   source .venv/bin/activate
   ```

1. бһҠбҹҶбһЎбһҫбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”

   бһ“бҹ…бһӣбһҫ Windows:

   ```powershell
   pip install -r requirements.txt
   ```

   бһ“бҹ…бһӣбһҫбһҷ macOS бһ“бһ·бһ„ Linux:

   ```bash
   pip3 install -r requirements.txt
   ```
