# бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһёбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бһҳбҹ’бһ”бһ„бһҸбһјбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹҠбһё AI бһўбһ„бҹ’бһҖбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ
бһўбҹҠбһёбҹ’бһҡбҹҮбһҡбһ…бһ“бһ¶бһ•бҹ’бһүбһҫбһҮбҹҶбһ“бһ¶бһ“бҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҮбҹҶбһ“бһҝбһ“бһ”бҹ’бһҡбһ¶бһҖбҹӢбһҡбһ”бһҹбҹӢбһ”бһҠбһ·бһңбһҸбҹ’бһҸбһ“бҹҚбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹдәәе·ҘбһҫбһҸ бһўбҹ„бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҗбҹ’бһҳбһёбһ”бһ¶бһ“бҹ” бһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бҹҒбҹҮбһўбһ¶бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ¶бҹҶбһ„бһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ– бһ‘бҹ…бһҠбһӣбҹӢбһҸбһ“бҹ’бһҸбҹ’бһҡбһё бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бһ·бһҳбһҪбһҷбҹ—бһҮбһ¶бһ–бҹ’бһ’бһңбҹҒбһ”бһ“бһ·бһ„бһңбһёбһҠбҹҒбһўбһј бҹ•бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбһҪбһҡбһўбҹ„бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ”бҹҶбһ•бһ»бһҸбһҡбһ”бһҹбҹӢбһўбҹҠбһё AI бһўбһ„бҹ’бһҖбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһӮбһәбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бҹ”

## бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһ”бһ„бһҸбһјбһ… (Small Language Models) бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?

бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бһҳбҹ’бһўбһ„бһҸбһјбһ… (SLM) бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҶбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹғбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶбһ“бһ·бһ„бһ’бҹҶ (LLM) бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ“бһ¶бһ“бһ¶бһҡбһ”бһҹбҹӢ LLM бһҠбҹӮбһӣбһ”бһүбҹ’бһ…бҹҒбһүбһҮбҹҶбһ„бһәбһҖбҹҶбһҺбһҸбҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ–бһёбһҡбһ”бҹҖбһ”бһӮбһҺбһ“бһ¶бҹ”

SLM бһӮбһәбһҮбһ¶вҖӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҖбҹ’бһҡбҹ„бһҳбһ“бҹғбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҳбҹүбһјбһҢбҹӮбһӣбһ’бҹҶбһ“бҹ…бһҠбһјбһ…бһҮбһ¶ GPT-4, SLM бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҸбһјбһ… бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбһ¶бһ„бһ’бҹҶ бһ”бҹҶбһ”бҹӮбһҖбһҳбһ»бһҒбһ„бһ¶бһҡбһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бһҡбһ№бһ„бһҸбһ·бһ…бҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһ‘бҹҶбһ бҹҶбһҸбһјбһ…бһҖбҹҸбһҠбҹ„бһҷ бһӮбҹҒбһ“бҹ…бһҸбҹӮбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ…бһҳбҹ’бһҡбһ„бҹӢбһ”бһ¶бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһҮбһ¶бһ‘бһјбһ‘бҹ… SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҹбһ¶бһ„бһҠбҹ„бһҷбһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ бһ¬бһ”бҹҶбһӣбҹӮбһ„ LLM бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ—бһ¶бһҹбһ¶бһ‘бһёбһ…бҹ’бһҡбһҫбһ“бһҳбһҪбһҷбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҠбһҫбһҳбҹ” бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҢбҹӮбһӣбһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ‘бһјбһ‘бҹ… бһ’бҹ’бһңбһҫбһўбҹ„бһҷ SLM бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүemory бһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ SLM бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҠбҹӮбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ”бһҒбһҺбҹ’бһҢбһҖбһ¶бһҡбһҡбһҪбһҳбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· (NLP)бҹ–

- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһңбһ¶бһӮбҹ’бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ•бҹ’бһҹбһ¶бһҡбһ–бҹҚбһ“бһ·бһ„бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бҹ”
- бһ”бһүбҹ’бһ…бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бҹ– бһҒбһ·бһҸбһҒбҹҶбһўбһ“бһӮбһҸбҹӢбһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһңбһ¶бһӮбҹ’бһҳбһ·бһ“бһҷбҹ„бһ„бһ‘бҹ…бһҸбһ¶бһҳбһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”
- бһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбҹ– бһҖбҹӮбһ”бҹ’бһҡбҹӮбһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһёбһ—бһ¶бһҹбһ¶бһҳбһҪбһҷбһ‘бҹ…бһҳбһҪбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
- бһҹбһ„бҹ’бһҒбҹҒбһ”бҹ– бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһңбҹӮбһ„бһҮбһ¶вҖӢвҖӢбһҹбһ„бҹ’бһҒбҹҒбһ”бһҒбҹ’бһӣбһёбһ“бһ·бһ„бһ„бһ¶бһҷбһҮбҹ’бһҡбһ¶бһ”бҹ”

бһ‘бҹ„бҹҮбһ”бһёбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһ”бһ“бҹ’бһҸбһ·бһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҸбҹӮбһ„бһ¬бһҖбһҳбҹ’бһҡбһ·бһҸбһ“бҹғбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„ бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бҹ…бһ“бһ№бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ’бҹҶбһ‘бҹҖбһҸбҹ”

## бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бһҳбҹ’бһўбһ„бһҸбһјбһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?
SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ“бҹ…бһ–бҹҒбһӣбһ”бһҺбҹ’бһҸбһ»бҹҮ, бһӮбҹҒбһҡбҹҖбһ“бһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ—бһ¶бһҹбһ¶ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһ“бҹӮбһҖбһңбҹҒбһҷбҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ“бһ·бһ„бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ“бҹӮбһҖбһ”бһҡбһ·бһ”бһ‘бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһ”бҹ’бһҡбһҳбһјбһӣбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһ…бҹ’бһҡбһҫбһ“бһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
- бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһӣбһ¶бһ„бһҹбҹҶбһўбһ¶бһҸбһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһІбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ”
- бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ– бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһјбһҡбһёбһ’бһҳбҹҚбһҡбҹҖбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбҹүбһјбһҢбҹӮбһӣбһ“бҹ…бһҷбһӣбҹӢбһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”
- бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ– бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҳбҹүбһјбһҢбҹӮбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбҹ”

бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ SLM бһҹбһҳбҹ’бһҡбһ”бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһҫбһ“бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„бһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҳбһ¶бһ“бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ“бһ№бһ„бһ’бһ“бһ’бһ¶бһ“бһҠбһјбһ…бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһӣбҹҗбһҸ бһ¬бһңбҹҒбһ‘бһ·бһҖбһ¶бһӮбһҺбһ“бһ¶ Edge бһҠбҹӮбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһ’бһ»бҹҶбһўбһ¶бһ…бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ’бҹ’бһ„бһ“бҹӢбҹ” бһҠбҹ„бһҷбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– SLM бһўбһ¶бһ…бһ”бһ“бҹ’бһҹбһӣбҹӢбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһҡбһңбһ¶бһ„бһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҹбһ„бҹ’бһғбһ№бһҳбһҠбҹӮбһӣбһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһўбҹҶбһ–бһё SLM бһ“бһ·бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ Microsoft Phi-3 бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһ–бһёбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһўбһҸбҹ’бһҗбһ”бһ‘, бһ…бһҖбҹ’бһҒбһ»бһңбһ·бһҹбҹҗбһҷ бһ“бһ·бһ„ MoEбҹ”

бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- SLM бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?
- бһ—бһ¶бһ–бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ SLM бһ“бһ·бһ„ LLM бһҮбһ¶бһўбҹ’бһңбһё?
- Microsoft Phi-3 / 3.5 Family бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?
- бһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҠбҹ„бһҷ Microsoft Phi-3 / 3.5 Family бһӮбһәбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?

бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһ бһҫбһҷ? бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ…бһ»бҹҮбҹ”

## бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLMs) бһ“бһ·бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһ”бһ„бһҸбһјбһ… (SLMs)

бһ‘бһ¶бҹҶбһ„ LLMs бһ“бһ·бһ„ SLMs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҹбһ¶бһ„бһЎбһҫбһ„бһӣбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҖбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһӣбҹ’бһўбһ“бһ·бһҷбһҳ бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбҹүбһ¶бһ„бһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶бһ“бҹ…бһӣбһҫбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһјбһҢбҹӮбһӣ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбҹҶбһӣбҹғбһҳбҹүбһјбһҢбҹӮбһӣбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶бһҖбҹҸбһҠбҹ„бһҷ бһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһ”бҹҶбһ”бҹӮбһҖбһҳбҹүбһјбһҢбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮбҹ”

## бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹғбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһ”бһ„бһҸбһјбһ…

SLMs бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһҮбһҮбҹӮбһҖбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢ (Chatbots): бһ•бҹ’бһҸбһӣбҹӢбһҮбҹҶбһ“бһҪбһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһ”бҹҖбһ”бһ“бһ·бһҷбһ¶бһҷбҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶: бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’бһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһ“бһ·бһҸ бһ¬бһҡбҹҖбһ”бһ…бҹҶбһўбһҸбҹ’бһҗбһ”бһ‘бһ–бҹҒбһүбһҳбһҪбһҷбҹ”
- бһўбһ”бҹӢбһҡбҹҶ: бһҮбһҪбһҷбһҹбһ·бһҹбҹ’бһҹбһҹбһ¶бһӣбһ¶бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ¬бһҡбҹҖбһ“бһ—бһ¶бһҹбһ¶бһҗбҹ’бһҳбһёбҹ”
- бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһҳбһ¶бһ“бһҮбҹҶбһ„бһәбһ–бһ·бһҖбһ¶бһҡбһ—бһ¶бһ– бһҠбһјбһ…бһҮбһ¶ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹҶбһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ‘бҹ…бһҹбҹҶбһЎбҹҒбһ„бҹ”

**бһ‘бҹҶбһ бҹҶ**

бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҡбһңбһ¶бһ„ LLMs бһ“бһ·бһ„ SLMs бһӮбһәбһҸбҹ’бһҡбһјбһңбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҳбһ¶бһҸбҹ’бһҡбһ“бҹғбһҳбҹүбһјбһҢбҹӮбһӣбҹ” LLMs бһҠбһјбһ…бһҮбһ¶ ChatGPT (GPT-4) бһўбһ¶бһ…бһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶ 1.76 бһ‘бҹ’бһҡбһёбһӣбҹ’бһҷбһјбһ“ бһҒбһҺбҹҲ SLM бһҠбһјбһ…бһҮбһ¶ Mistral 7B бһӮбһәбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһ·бһ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ”бҹ’бһҡбһ бҹӮбһӣ 7 бһ–бҹ’бһҡбһ·бһӣбһ“бһ¶бһҖбҹӢбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҳбһҖбһ–бһёбһҒбһ»бһҹбһ”бҹ’бһӣбҹӮбһҖбһ“бҹғбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһјбһҢбҹӮбһӣ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ ChatGPT бһ”бҹ’бһҡбһҫбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡ self-attention бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶ encoder-decoder бһҒбһҺбҹҲ Mistral 7B бһ”бҹ’бһҡбһҫ sliding window atenГ§ГЈo бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбҹӮбһӣ decoder бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ‘бҹ…бһӣбһҫбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһҸбҹӮбһ„бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„**

SLMs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҖбҹҶбһҺбһҸбҹӢ бһҮбһ¶бһ’бҹҶбһ’бҹҒбһ„ бһҸбҹӮбһўбһ¶бһ…бһҳбһ¶бһ“бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ‘бһјбһ‘бҹ…бһӣбһҫбһҮбһҪбһҡбһҠбҹӮбһ“бһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹ„бҹҮ LLMs бһ•бҹ’бһҹбһ¶бһҷбһӣбҹ’бһ”бһ„бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһҸбһ¶бһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһүбҹ” LLMs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһҡбһ»бҹҮбһ’бҹҶ бһ‘бһҫбһ”бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“ бһ“бһ·бһ„бһҶбһ”бһӮбҹ’бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ LLMs бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ–бһ бһ»бһңбһ¶бһӣбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһјбһҠбҹ”

**бһӮбһҺбһ“бһ¶бһҡбҹүбһ№бһ„**

бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ LLMs бһҮбһ¶бһҖбһ¶бһҡбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһ’бһ“бһ’бһ¶бһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷ cluster GPU бһ‘бҹҶбһ бҹҶбһ’бҹҶбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҢбҹӮбһӣбһҠбһјбһ…бһҮбһ¶ ChatGPT бһ–бһёбһҠбһҫбһҳ бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ бһ» GPUs бһӣбһҫбһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹ„бҹҮ SLMs бһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһјбһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҒбҹ’бһӣбһҪбһ“бһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҮбһ¶бһ„бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһҠбһјбһ…бһҮбһ¶ Mistral 7B бһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“ GPU бһҳбһ’бҹ’бһҷбһҳ бһҠбҹ„бһҷбһ‘бҹ„бҹҮбһ”бһёбһ”бһҺбҹ’бһҸбһ»бҹҮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҳбҹүбҹ„бһ„бһӣбһҫ GPU бһ…бҹ’бһҡбһҫбһ“бһҖбҹҸбһҠбҹ„бһҷбҹ”

**бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ (Bias)**

бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбҹҶбһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһӮбһәбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ LLMs бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ“бһ№бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбҹ” бһҳбҹүбһјбһҢбҹӮбһӣбһ‘бҹҶбһ“бһҫбһ”бһ‘бһ¶бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһҫбһҳбһ–бһёбһўбҹҠбһ·бһ“бһ’бһәбһҺбҹӮбһҸбһҠбҹӮбһӣбһўбһ¶бһ…бһҒбҹ’бһңбҹҮбһҒбһ¶бһҸ бһ¬бһҸбһҪбһ”бҹ’бһ”бһүбҹ’бһүбһҸбҹ’бһҸбһ·бһҒбһ»бһҹбһ‘бһ·бһҹбһ–бһёбһҖбҹ’бһҡбһ»бһҳбһҒбҹ’бһӣбҹҮбҹ— бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҹбҹ’бһӣбһ¶бһҖбһҮбҹҶбһ бһ¶бһ“бһҒбһ»бһҹ бһ¬бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ—бһ¶бһҹбһ¶бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһ—бһ¶бһҹбһ¶бһ“бһ·бһ„бһҖбҹ’бһ”бһ¶бһ…бҹӢбһңбҹҒбһҷбҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһ“бҹҒбҹҮ бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ LLM бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбҹҶбһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһ“бһ·бһ…бҹ’бһ… бһ бһҫбһҷбһўбһ¶бһ…бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһүбһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһ·бһ“бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹ„бҹҮ SLMs бһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢ бһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҮбһ¶бһҸбһ· бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҸбһ·бһ…бһҸбһҪбһ…бһ…бҹҶбһ–бҹ„бҹҮбһ…бҹ„бһӣбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбҹҶбһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбһҸбҹӢбһ”бһ¶бһ“бһҸбҹ’бһҡбһЎбһ”бҹӢбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһӮбҹ’бһҳбһ¶бһ“бһ‘бҹҒбҹ”

**бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһ•бһӣбһӣбһ‘бҹ’бһ’бһ•бһӣ (Inference)**

бһ‘бҹҶбһ бҹҶбһҸбһјбһ…бһҡбһ”бһҹбҹӢ SLM бһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ“бһјбһңбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹғбһӣбҹ’бһ”бһҝбһ“бһ—бһ¶бһ–бһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһҠбҹ„бһҷбһўбһ¶бһ…бһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ…бҹ’бһҡбһҫбһ“бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹҒбҹҮ LLM бһӮбһәбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбҹҶбһ”бһ“бҹӢбһӮбһҺбһ“бһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһҷбһҖбһ”бһ¶бһ“бҹ” бһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„бһҳбҹүбһјбһҢбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһҖбҹҸбһ“бһ¶бҹҶбһІбҹ’бһҷбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҡбһ”бһҹбҹӢ LLM бһҷбһәбһҸбһҮбһ¶бһ„бһӮбҹҒбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

бһҹбһ„бҹ’бһҒбҹҒбһ”бһӮбһә бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶ LLM бһ“бһ·бһ„ SLM бһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһҫбһҖбһ¶бһҡбһҡбҹҖбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҖбҹҸбһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ‘бҹ…бһ“бһ№бһ„бһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҢбҹӮбһӣ, бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ’бһ“бһ’бһ¶бһ“, бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’, бһ—бһ¶бһ–бһ„бһ¶бһҷбһҡбһ„бһ”бҹ’бһҡбһ бҹӮбһӣбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһӣбҹ’бһҳбһҫбһҹ бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһ”бҹ’бһӣбҹӮбһҖбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһҹбһҳбһҡбһҳбҹ’бһҷбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶ бһҠбҹ„бһҷ LLM бһҳбһ¶бһ“бһ—бһ¶бһ–бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ’бһ“бһ’бһ¶бһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҒбһҺбҹҲбһҠбҹӮбһӣ SLM бһ•бҹ’бһҸбһӣбҹӢбһ—бһ¶бһ–бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҸбһ·бһ…бһҸбһҪбһ…бҹ”

***бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶ SLM бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Phi-3 / 3.5 бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ”***

## бһҺбҹӮбһ“бһ¶бҹҶвҖӢбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ Phi-3 / Phi-3.5

бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ Phi-3 / 3.5 бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһ“бҹӮбһҖбһўбһҸбҹ’бһҗбһ”бһ‘, бһ…бһҖбҹ’бһҒбһ»бһңбһ·бһҹбҹҗбһҷ бһ“бһ·бһ„ Agent (MoE)бҹ–

бһ•бҹ’бһҸбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӣбһҫбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘, бһ”бһүбҹ’бһ…бһ”бҹӢбһҮбһҮбҹӮбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”

**Phi-3-mini**

бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶ 3.8B бһҳбһ¶бһ“бһҖбһ¶бһҡбһҒбҹ’бһҮбһ¶бһ”бҹӢбһҒбҹ’бһҮбһ¶бһҷбһ“бҹ… Microsoft Azure AI Studio, Hugging Face бһ“бһ·бһ„ Ollamaбҹ” бһҳбҹүбһјбһҢбҹӮбһӣ Phi-3 бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһӣбҹ’бһўбһҮбһ¶бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ’бҹҶбһҮбһ¶бһ„ бһ“бҹ…бһӣбһҫбһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһҹбһҳбҹ’бһҸбҹӮбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— (бһҳбһҫбһӣбһӣбҹҒбһҒбһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ, бһӣбҹҒбһҒбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһӣбҹ’бһўбһҮбһ¶бһ„)бҹ” Phi-3-mini бһӣбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҠбһ„бһ–бһёбһҡбһҡбһ”бһҹбҹӢбһңбһ¶ бһҒбһҺбҹҲ Phi-3-small бһ“бһ·бһ„ Phi-3-medium бһӣбһҫбһҹбһ–бһёбһҳбҹүбһјбһҢбҹӮбһӣбһ’бҹҶбҹ—бһҠбһјбһ…бһҮбһ¶ GPT-3.5бҹ”

**Phi-3-small & medium**

бһҳбһ¶бһ“бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ 7B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ Phi-3-small бһҲбҹ’бһ“бҹҮ GPT-3.5T бһ“бҹ…бһӣбһҫбһңбһ¶бһӣ бһ—бһ¶бһҹбһ¶, бһҖбһ¶бһҡбһңбһ·бһ…бһ¶бһҡ, бһҖбһјбһҠ бһ“бһ·бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ”

Phi-3-medium бһҮбһ¶бһҳбһҪбһҷ 14B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ”бһ“бҹ’бһҸбһҸбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһ“бһ·бһ„бһӣбһҫбһҹ Gemini 1.0 Proбҹ”

**Phi-3.5-mini**

бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһңбһ¶бһҠбһјбһ…бһҮбһ¶бһҗбҹ’бһ“бһ¶бһӣбһҫбһҖбһҖбҹҶбһҡбһ·бһҸбһҡбһ”бһҹбҹӢ Phi-3-miniбҹ” бһ‘бҹ„бҹҮбһ”бһёбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹҒ бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“ (бһӮбһ¶бҹҶбһ‘бҹ’бһҡ 20+ бһ—бһ¶бһҹбһ¶бҹ– бһўбһ¶бһҡбҹүбһ¶бһ”бҹӢ, бһ…бһ·бһ“, бһҲбһәбһҖ, бһҠбһ¶бһҺбһёбһҹ, бһҠбһ»бһ…, бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹ, бһ бҹ’бһңбһ¶бҹҶбһ„бһЎбһ„бҹӢ, бһ”бһ¶бһҡбһ¶бҹҶбһ„, бһўбһ¶бһӣбҹ’бһӣбһәбһҳбҹүбһ„бҹӢ, бһ бҹҒбһ”бҹ’бһҡбҹҠбһј, бһ бһ»бһ„бһӮбҹ’бһҡбһё, бһўбҹҠбһёбһҸбһ¶бһӣбһё, бһҮбһ”бҹүбһ»бһ“, бһҖбһјбһҡбҹүбҹҒ, бһ“бҹҗбһҡбһңбҹӮбһҹ, бһ”бҹүбһјбһЎбһ„бҹӢ, бһ–бҹҗбһҡбһ‘бһ»бһҷбһ бҹ’бһӮбһ¶бһӣбҹӢ, бһҡбһ»бһҹбҹ’бһҹбҹҠбһё, бһҹбҹ’бһ”бҹүбһ¶бһү, бһҹбҹ’бһңбҹҒбһҢбһ·бһ“, бһҗбҹғ, бһ‘бһҪбһҡбһӮбһё, бһўбҹҠбһ»бһҷбһҖбҹ’бһҡбҹӮбһ“) бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҡбһ·бһ”бһ‘бһңбҹӮбһ„бҹ”

Phi-3.5-mini бһҮбһ¶бһҳбһҪбһҷ 3.8B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһӣбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҠбһјбһ…бһ‘бҹҶбһ бҹҶбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҹбһҳбһҠбҹ„бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҠбһ„бһ–бһёбһҡбҹ”

### Phi-3 / 3.5 Vision

бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһҳбҹүбһјбһҢбҹӮбһӣ Instruct бһҡбһ”бһҹбҹӢ Phi-3/3.5 бһҮбһ¶бһҮбҹҶбһ“бһ¶бһүбһҡбһ”бһҹбҹӢ Phi бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһ“бһ·бһ„ Vision бһӮбһәбһҮбһ¶бһҗбҹ’бһ“бһ¶ бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ—бҹ’бһ“бҹӮбһҖбһІбҹ’бһҷ Phi бһҷбһӣбҹӢбһ–бһёбһӣбҹ„бһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ”

**Phi-3-Vision**

Phi-3-vision бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ 4.2B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ”бһ“бҹ’бһҸбһҸбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһ“бһ·бһ„бһӣбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣбһ’бҹҶбҹ—бһҠбһјбһ…бһҮбһ¶ Claude-3 Haiku бһ“бһ·бһ„ Gemini 1.0 Pro V бһӣбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһңбһ·бһ…бһ¶бһҡбһҺбҹҲбһ‘бһјбһ‘бҹ…бһ—бҹ’бһ“бҹӮбһҖ, OCR, бһ“бһ·бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһ“бһ·бһ„бһҢбһёбһҷбҹүбһ¶бһҖбҹ’бһҡбһ¶бһҳбҹ”

**Phi-3.5-Vision**

Phi-3.5-Vision бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бҹҶбһ“бһҫбһ”бһҡбһ”бһҹбҹӢ Phi-3-Vision бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһјбһ”бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбһ·бһҸбһңбһ¶бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ…бһҖбҹ’бһҒбһ»бһңбһ·бһҹбҹҗбһҷ бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ“бҹ…бһІбҹ’бһҷбһҳбһҫбһӣбһҡбһјбһ”бһ—бһ¶бһ– бһҷбҹүбһ¶бһ„бһҸбҹӮбһңбһёбһҠбҹҒбһўбһјбһ…бҹ’бһҡбһҫбһ“бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҖбҹҸбһҠбҹ„бһҷбҹ”

Phi-3.5-vision бһӣбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣбһ’бҹҶбҹ—бһҠбһјбһ…бһҮбһ¶ Claude-3.5 Sonnet бһ“бһ·бһ„ Gemini 1.5 Flash бһ“бҹ…бһӣбһҫбһ…бҹҶбһҺбһ»бһ… OCR, бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бһёбһҸбһ¶бһҡбһ¶бһ„бһ“бһ·бһ„бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһ… бһ“бһ·бһ„бһҳбһ¶бһ“бһ‘бҹҶбһ„бһ“бҹӢбһҹбҹ’бһҳбһҫбһ“бҹ…бһӣбһҫбһҖбһ¶бһҡбһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бһҖбҹ’бһҒбһ»бһңбһ·бһҹбҹҗбһҷбһ‘бһјбһ‘бҹ…бҹ” бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ“бһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһўбһ”бҹӢ бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ”

### Phi-3.5-MoE

***Mixture of Experts (MoE)*** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҠбҹ„бһҷбһҖбһ¶бһҡбһ…бҹҶбһҺбһ¶бһҷбһӮбһҺбһ“бһ¶бһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“бһҸбһ·бһ… бһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҢбҹӮбһӣбһ¬бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһҳбһҪбһҷбһҗбһңбһ·бһҖбһ¶бһӮбһҺбһ“бһ¶вҖӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһ“бһ№бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһҠбһ„бҹӢбһҹбҹҠбһёбҹ” бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһҳбҹүбһјбһҢбҹӮбһӣ MoE бһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҠбһ„бҹӢбһҹбҹҠбһёбһӮбҹ’бһӣбһёбһ„бһҷбҹүбһ¶бһ„бһӣбһҝбһ“бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ”

Phi-3.5-MoE бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһ„ 16x3.8B modulesбҹ” Phi-3.5-MoE бһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ 6.6B бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҮбҹҶбһ“бһ¶бһүбһңбһ·бһ…бһ¶бһҡбһҺбһ¶, бһҷбһӣбҹӢбһҖбһ¶бһҡбһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ’бҹҶбһҮбһ¶бһ„бһҖбҹҶбһ–бһјбһӣбҹ”

бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣ Phi-3 / 3.5 Family бһҳбһҪбһҷбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһё LLM бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„ Phi-3/3.5-mini бһ¬ Phi-3/3.5-Vision бһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ Edgeбҹ”

**NVIDIA NIM**

бһҖбҹ’бһҡбҹ…бһ–бһёбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Model Catalog бһҖбҹ’бһ“бһ»бһ„бһ–бһ–бһҖбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҠбҹ„бһҷ Azure бһ“бһ·бһ„ GitHub бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ [NVIDIA NIM](https://developer.nvidia.com/nim?WT.mc_id=academic-105485-koreyst) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһ‘бһ¶бһҖбҹӢбһ‘бһ„бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ‘бҹ…бһ‘бһҹбҹ’бһҹбһ“бһ¶ NVIDIA NIM бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ бҹ… API бһҡбһ”бһҹбҹӢбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ Phi-3/3.5 Familyбҹ” NVIDIA NIM (NVIDIA Inference Microservices) бһӮбһәбһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҳбҹүбһёбһҖбҹ’бһҡбһјбһҶбҹ’бһӣбһ¶бһҸбһҠбҹӮбһӣбһ”бһҳбҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҡбҹҖбһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣ AI бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ— бһҡбһҪбһҳбһҳбһ¶бһ“бһ–бһ–бһҖ, бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһҹбһ¶бһ„бҹ”

бһ“бҹҒбҹҮбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғ NVIDIA NIMбҹ–

- **бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„бҹ–** NIM бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҸбһҳбҹ’бһӣбһҫбһ„бһҳбҹүбһјбһҢбҹӮбһӣ AI бһҸбһ¶бһҳбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһҪбһҷ бһІбҹ’бһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
- **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһ“бҹ–** бһңбһ¶бһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҹбһҫбһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһүбҹ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бһӢбһҳбһҡбһ”бһҹбҹӢ NVIDIA бһҠбһјбһ…бһҮбһ¶ TensorRT бһ“бһ·бһ„ TensorRT-LLM бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷ latency бһ‘бһ¶бһ” бһ“бһ·бһ„ throughput бһҒбҹ’бһ–бһҹбҹӢбҹ”
- **бһҖбһ¶бһҡбһҖбһҫбһ“бһ‘бһёбһҸбһ¶бҹҶбһ„бһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ–** NIM бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲ autoscaling бһ“бҹ… Kubernetes бһІбҹ’бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

- **API бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ–** NIM бһ•бҹ’бһҸбһӣбҹӢ API бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһўбҹ’бһ“бһҖбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбһјбһ…бһҮбһ¶ chatbot, бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһўбҹҠбһёбһ“бһ’бһёбһӣбһёбһ бҹ’бһҹбһ„бҹӢ бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

**Hugging Face Transformer**
Hugging Face Transformers бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһ“бһ·бһҷбһҳбһ—бһ¶бһҹбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’(NLP) бһ“бһ·бһ„бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһўбҹҶбһ–бһёбһңбһ¶бҹ–

1. **бһҳбҹүбһјбһҢбҹӮбһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһҪбһ…бһ бһҫбһҷ**бҹ– бһңбһ¶бһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһјбһҢбҹӮбһӣбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһҪбһ…бһ бһҫбһҷбһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһўбһҸбҹ’бһҗбһ”бһ‘ бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҲбҹ’бһҳбҹ„бҹҮ бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бһҹбһ¶бһҡ бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”

2. **бһҖбһ¶бһҡбһўбһ¶бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳбһ”бҹ’бһҡбһ№бһҖбҹ’бһҹбһ¶бһҹбһ“бҹ’бһ‘бһҹбҹ’бһҹбһ“бҹҚбһ“бһ¶бһ“бһ¶**бҹ– бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһҮбҹ’бһҡбҹ…бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҡбһҪбһҳбһҳбһ¶бһ“ PyTorch, TensorFlow бһ“бһ·бһ„ JAXбҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҢбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһҪбһҷ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

3. **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһўбһ“бһ»бһӮбһҺбһ“бһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—**бҹ– бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„ NLP бһҳбһҪбһҷбһ‘бҹҖбһҸ Hugging Face Transformers бһ…бһјбһӣбһҡбһҪбһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡвҖӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ’бҹҗбһҡ (бһ§. бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҡбһјбһ”бһ—бһ¶бһ– бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһңбһҸбҹ’бһҗбһ») бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһӣбҹҒбһ„ (бһ§. бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҹбҹҶбһӣбҹҒбһ„ бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҹбһҳбҹ’бһӣбҹҒбһ„)бҹ”

5. **бһҹбһ бһӮбһҳбһ“бҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“**бҹ– Hugging Face бһҳбһ¶бһ“бһҹбһ бһӮбһҳбһ“бҹҚбһҠбҹҸбһҹбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһү бһҳбҹҒбһҡбҹҖбһ“ бһ“бһ·бһ„бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҷбҹүбһ¶бһ„бһ–бҹҒбһүбһӣбҹҒбһүбҹ”

[ONNX Runtime](https://github.com/microsoft/onnxruntime-genai?WT.mc_id=academic-105485-koreyst) бһӮбһәбһҮбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Generative AI (GENAI) бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһҠбҹӮбһӣбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһҳбҹүбһјбһҢбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһёбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҸбһ¶бһҳбһ”бһ»бһӮбҹ’бһӮбһӣбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбһҫбһҡбһ”бҹҖбһ”бһңбҹҒбһ‘бһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

ONNX Runtime бһӮбһәбһҮбһ¶бһӮбһҳбҹ’бһҡбҹ„бһ„бһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ” бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡвҖӢбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ Open Neural Network Exchange (ONNX) бһҠбҹӮбһӣбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһҺбһ¶бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бҹ” бһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷ ONNX Runtime бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһӣбһҝбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҗбҹ’бһӣбҹғбһҠбһҫбһҳ бһҮбҹҶбһ“бһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣбһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҮбһҳбҹ’бһҡбҹ…бһ“бһ·бһ„ classic бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһҠбһјбһ…бһҮбһ¶ PyTorch, TensorFlow/Keras бһ“бһ·бһ„ бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһҠбһјбһ…бһҮбһ¶ scikit-learn, LightGBM, XGBoostбҹ” ONNX Runtime бһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбһҳбһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷбһҡбһ№бһҳбһҒ hardware бһ”бһ¶бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһүбһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһӣбһҫ hardware accelerators бһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ”бһјбһҖбһҡбһҪбһҳбһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹ’бһҡбһ¶бһҹ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбҹ’бһҡбһ¶бһ•бһ·бһҖбҹ”

## Generative AI бһҮбһ¶бһўбҹ’бһңбһё?
Generative AI бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҗбҹ’бһҳбһёбҹ—бһҠбһјбһ…бһҮбһ¶ бһўбһҸбҹ’бһҗбһ”бһ‘ бһҡбһјбһ”бһ—бһ¶бһ– бһ¬бһҸбһ“бҹ’бһҸбҹ’бһҡбһё бһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһңбһ¶бһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҳбһ¶бһ“бһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶вҖӢбһҠбһјбһ…бһҮбһ¶ GPT-3 бһ“бһ·бһ„бһҳбҹүбһјбһҢбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҠбһјбһ…бһҮбһ¶ Stable Diffusionбҹ” бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ ONNX Runtime бһҮбһ¶бһҳбһҪбһҷ GenAI бһ•бҹ’бһҸбһӣбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣ ONNX бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһҠбҹ„бһҷ ONNX Runtime бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ logits бһҖбһ¶бһҡвҖӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһӮбһ»бҹҶбһҹбһҡбһҹбҹҒбһҡ KV cacheбҹ”

## ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI
ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI бһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ ONNX Runtime бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҢбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸбҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бҹ–

- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһңбҹҒбһ‘бһ·бһҖбһ¶бһ…бҹ’бһҡбһҫбһ“**бҹ– бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҡбһҪбһҳбһҳбһ¶бһ“ Windows, Linux, macOS, Android бһ“бһ·бһ„ iOSбҹ”
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҢбҹӮбһӣ**бҹ– бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҢбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ LLaMA, GPT-Neo, BLOOM бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”
- **бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**бҹ– бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ hardware accelerators бһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһјбһ…бһҮбһ¶ NVIDIA GPUs, AMD GPUs бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
- **бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫ**бҹ– бһ•бҹ’бһҸбһӣбҹӢ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘ бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹ„бһҷбһҖбһјбһҠбһҸбһ·бһ…бҹ”
- бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ¶бһ…бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ generate() бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҖбҹҶбһ–бһјбһӣ бһ¬бһҡбһҸбҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҳбһҪбһҷбһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһүбҹ’бһүбһ¶бһўбһҖбҹ’бһҹбһҡбһҳбһҪбһҷбҹ— бһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһӣбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣбҹ”
- ONNX Runtime бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ greedy/beam бһ“бһ·бһ„бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖ TopP, TopK бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбһ“бҹғбһҹбһүбҹ’бһүбһ¶ бһ“бһ·бһ„бһҳбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһҳ logits бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ‘бһҺбҹ’бһҢбһҖбһҳбҹ’бһҳбһҖбһҡбһҺбһёбһ‘бҹ’бһңбҹҒбһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

```Python
import onnxruntime_genai as og

```
### бһ–бһ“бҹ’бһӣбһәбһ”бһ„бҹ’бһ бһ¶бһүбҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ ONNX Runtime GenAI бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ… Phi-3.5-Vision


```python

import onnxruntime_genai as og

model_path = './Your Phi-3.5-vision-instruct ONNX Path'

img_path = './Your Image Path'

model = og.Model(model_path)

processor = model.create_multimodal_processor()

tokenizer_stream = processor.create_stream()

text = "Your Prompt"

prompt = "<|user|>\n"

prompt += "<|image_1|>\n"

prompt += f"{text}<|end|>\n"

prompt += "<|assistant|>\n"

image = og.Images.open(img_path)

inputs = processor(prompt, images=image)

params = og.GeneratorParams(model)

params.set_inputs(inputs)

params.set_search_options(max_length=3072)

generator = og.Generator(model, params)

while not generator.is_done():

    generator.compute_logits()
    
    generator.generate_next_token()

    new_token = generator.get_next_tokens()[0]
    
    output = tokenizer_stream.decode(new_token)
    
    print(tokenizer_stream.decode(new_token), end='', flush=True)

```
