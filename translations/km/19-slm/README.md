# бһ”бҹҶбһҺбһҫбһҸбһҳбһҪбһҷбһ‘бҹ…бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ  
AI бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһәбһҮбһ¶бһңбһ·бһҹбҹҗбһҷбһҳбһҪбһҷбһҠбҹҸбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹғбһ”бһүбҹ’бһүбһ¶бһ’бһ·бһ”бҹ’бһ”бһ¶бһҷ бһҠбҹӮбһӣбһ•бҹ’бһҠбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҗбҹ’бһҳбһёбҹ” бһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бҹҒбҹҮбһўбһ¶бһ…бһҮбһ¶бһҡбһ”бһҹбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ– бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҸбһ“бҹ’бһҸбҹ’бһҡбһё бһ“бһ·бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһңбһёбһҡбһ»бһҹбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹҸбһӮбһҪбһҡбһұбҹ’бһҷбһҡбҹҶбһ—бһҫбһ”бһ”бҹҶбһ•бһ»бһҸбһ“бҹғ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһәбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бҹ”  

## бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ…бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ?  
бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ… (SLM) бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһ–бһёбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLM) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ–бһё LLM бһҒбһҺбҹҲбһ–бҹҒбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶбһӮбһҺбһ“бһ¶бһҳбһҪбһҷбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бҹ”  

SLMs бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹ…бһҹбҹ’бһўбһ¶бһҸбһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһҳбһ·бһ“бһҠбһјбһ…бһӮбҹҶбһҡбһјбһ’бҹҶбһ“бҹ…бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбһҠбһјбһ…бһҮбһ¶бһҡ GPT-4, SLM бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҸбһјбһ…бһҮбһ¶бһ„ бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бҹ” бһ‘бҹ„бҹҮбһ”бһёбһҹбһјбһ…бһҳбһҖбһҳбһ·бһ“бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ–бҹҒбһү бһҖбҹҸбһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бҹ’бһҡбһҫбһ“бһ”бһ¶бһ“бҹ” бһҮбһ¶бһ‘бһјбһ‘бҹ… SLMs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢ бһ¬бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһҸбһ¶бһ…бҹӢ LLM бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ—бһ¶бһҹбһ¶бһҠбһҫбһҳбҹ” бһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһӮбҹҶбһҡбһјбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҗбһҷбһҹбҹҶбһҺбһ»бҹҶбһ“бҹғбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ’бҹ’бһңбһҫбһұбҹ’бһҷ SLM бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ¶бҹҶбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһӮбһҺбһ“бһ¶бҹ” бһ‘бҹ„бҹҮбһ”бһёбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ SLMs бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһўбһ—бһ·бһ”бһ¶бһӣбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· (NLP)бҹ–  

- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҷбҹ„бһӮ бһ¬бһңбһ„бҹӢбһҖбһҗбһ¶бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бһ¶бһҹбһ»бһҖбһ—бһ¶бһ–бһ“бһ·бһ„бһҹбһҳбһҹбҹ’бһҡбһ”бһҡбһ”бҹҖбһ”бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”  
- бһ”бһүбҹ’бһ…бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бҹ– бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ”бҹ’бһҡбһҷбҹ„бһӮбһҳбһҪбһҷбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбҹ”  
- бһ”бһҖбһ”бҹ’бһҡбҹӮбҹ– бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһёбһ—бһ¶бһҹбһ¶бһҳбһҪбһҷбһ‘бҹ…бһ—бһ¶бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ”  
- бһҹбһ„бҹ’бһҒбҹҒбһ”бҹ– бһҸбһҪбһўбһҸбҹ’бһҗбһ”бһ‘бһңбҹӮбһ„бһҮбһ¶бһҹбһ„бҹ’бһҒбҹҒбһ”бһҒбҹ’бһӣбһёбҹ—бһҹбҹ’бһҡбһҪбһӣбһҷбһӣбҹӢбҹ”  

бһ‘бҹ„бҹҮбһҮбһ¶бһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһҒбҹ’бһӣбҹҮбһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ¬бһҮбҹ’бһҡбҹ…бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„ бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ“бһ№бһ„бһӮбҹҶбһҡбһјбһ’бҹҶбҹ—бҹ”  

## бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?  
SLMs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бҹ” бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ бһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғбһ–бһҪбһҖбһңбһ¶бһҹбһ·бһҖбҹ’бһҹбһ¶бһӮбҹҶбһҡбһјбһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ—бһ¶бһҹбһ¶ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ–бһҪбһҖбһңбһ¶бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣ бһӣбҹҶбһҠбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһ“бһ·бһ„бһҹбһҳбһҹбҹ’бһҡбһ”бһ…бҹҶбһ–бҹ„бҹҮбһ”бһҡбһ·бһ”бһ‘бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–  

- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһ’бҹҶбҹ—бһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”  
- бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһҹбҹҶбһўбһ¶бһҸбһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһұбҹ’бһҷбһҹ бһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ”  
- бһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ– бһ”бҹ’бһҡбһҫбһҹбһјбһҳбҹ’бһ”бһёбһ¶бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҡбҹҖбһ“бһӮбҹҶбһҡбһјбһ–бһёбһҡбһ”бҹҖбһ”бһІбҹ’бһҷбһҷбһӣбҹӢбһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”  
- бһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбҹҖбһ”бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ– бһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһӮбҹҶбһҡбһјбһұбҹ’бһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бһӣбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”  

бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ SLM бһҹбҹ’бһҡбһ”бһ‘бҹ…бһ“бһ№бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ…бҹҶбһ–бҹ„бҹҮбһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“ бһҠбһјбһ…бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһӣбҹҗбһҸ бһ¬бһңбҹҒбһ‘бһ·бһҖбһ¶бһӮбһҺбһ“бһ¶бһҡбһ„бҹ’бһңбһ·бһӣ бһҠбҹӮбһӣ LLM бһ‘бҹҶбһ бҹҶбһ–бҹҒбһүбһӮбһҪбһҡбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһ–бҹ’бһҡбҹ„бҹҮбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҗбһ¶бһҳбһ–бһӣбһҒбҹ’бһ–бһҹбҹӢбҹ” бһҠбҹ„бһҷбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– SLM бһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһҡбһңбһ¶бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бһ·бһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ”бһ„бҹ’бһҖбһҫбһҸбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶  

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҹбһ„бҹ’бһғбһ№бһҳбһҺбҹӮбһ“бһ¶бҹҶбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһўбҹҶбһ–бһё SLM бһ“бһ·бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ Microsoft Phi-3 бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһ–бһёбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶ бһўбһҸбҹ’бһҗбһ”бһ‘ бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶ бһ“бһ·бһ„ MoEбҹ”  

бһ“бҹ…бһ…бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһўбһ¶бһ…бһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ¶бһ“бҹ–  

- бһҸбһҫ SLM бһҮбһ¶бһўбҹ’бһңбһё?  
- бһҸбһҫбһҒбҹ’бһӣбһјбһ“бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһўбҹ’бһңбһёбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳ SLM бһ“бһ·бһ„ LLM?  
- бһҸбһҫ Microsoft Phi-3 / 3.5 бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?  
- бһҸбһҫбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹғ Microsoft Phi-3 / 3.5?  

бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһ бһҫбһҷ? бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ•бҹ’бһӣбһјбһңбҹ”  

## бһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһңбһ¶бһ„бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLMs) бһ“бһ·бһ„ бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ… (SLMs)  

бһ‘бһ¶бҹҶбһ„ LLM бһ“бһ·бһ„ SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҹбһ¶бһ„бһӣбһҫбһӮбҹ’бһҡбһ№бҹҮбһӮбҹҶбһ“бһ·бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹүбҹ’бһҡбһјбһ”бһ¶бһ”бһ¶бһҠбһ·бһ… бһҮбһ¶бһҳбһҪбһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ”бһҡбһ·бһҷбһ¶бһҷбһӮбҹҶбһҡбһјбҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһҳбһ¶бһ“бһҖбһҸбҹ’бһҸбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһ”бҹҶбһ”бҹӮбһҖбһ–бһёбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбҹ”  

## бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҡбһ”бһҹбҹӢбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ…  

SLMs бһҳбһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶бҹ–  

- Chatbotsбҹ– бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҹбһ“бҹ’бһ‘бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҮбһ¶бһҹбһ“бҹ’бһ‘бһ“бһ¶бҹ”  
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бҹ– бһҮбҹҶбһ“бһҪбһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһ“бһ·бһҸбһ¬бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһ–бҹҒбһүбһҳбһҪбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”  
- бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶбҹ– бһҮбһҪбһҷбһҹбһ·бһҹбҹ’бһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ”бһ‘бһ”бһүбҹ’бһҮбһ¶ бһ¬бһҡбҹҖбһ“бһ—бһ¶бһҹбһ¶бһҗбҹ’бһҳбһёбҹ”  
- бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ…бһјбһӣбһ”бҹ’бһҡбһҫбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһ–бһ·бһҖбһ¶бһҡбһҮбһ¶бһҠбһјбһ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ‘бҹ…бһҹбҹҶбһЎбҹҒбһ„бҹ”  

**бһ‘бҹҶбһ бҹҶ**  

бһҖбһҸбҹ’бһҸбһ¶бһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ…бһҳбҹ’бһ”бһ„бһҠбҹӮбһӣбһ”бҹҶбһ”бҹӮбһҖ LLM бһ“бһ·бһ„ SLM бһӮбһәбһ‘бҹҶбһ бҹҶбһӮбҹҶбһҡбһјбҹ” LLM бһҠбһјбһ…бһҮбһ¶ ChatGPT (GPT-4) бһҳбһ¶бһ“бһ”бҹ’бһҡбһ бҹӮбһӣ 1.76 бһҸбҹ’бһҡбһёбһӣбҹҖбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһҒбһҺбҹҲбһҠбҹӮбһӣ SLM бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһҫбһҖбһ…бҹҶбһ бһҠбһјбһ…бһҮбһ¶ Mistral 7B бһҸбҹ’бһҡбһјбһңбһҡбһ…бһ“бһ¶бһ‘бҹҶбһ бҹҶбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһ·бһ…бһҮбһ¶бһ„бһҒбҹ’бһӣбҹҮ вҖ” бһ”бҹ’бһҡбһ бҹӮбһӣ 7 бһ”бҹҠбһёбһӣбҹҖбһ“бҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһӮбһәбһҠбҹ„бһҷбһҹбһ¶бһҡбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ ChatGPT бһ”бҹ’бһҡбһҫбһҳбҹҒбһҖбһ¶бһ“бһ·бһ…бһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһҒбҹ’бһӣбһҪбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹҠбһ»бһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһјбһҢбҹӮбһӣ encoder-decoder бһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҒбһҺбҹҲ Mistral 7B бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһҫбһҖбһ”бҹ’бһҡбһҫбһ”бһ„бҹ’бһ бһ¶бһ”бҹӢбһҠбҹӮбһӣбһўбһ¶бһ…бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҹбҹҠбһ»бһҳбһҳбҹүбһјбһҢбҹӮбһӣ decoder бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһҒбҹ’бһӣбһ¶бҹҶбһ„бһӣбһҫбһҹбҹҶбһҺбһ»бҹҶбһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҡбһ”бһҹбҹӢбһӮбҹҶбһҡбһјбҹ”  

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„**  

SLMs бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ— бһ”бһ„бҹ’бһҖбһҫбһҸбһІбҹ’бһҷбһҮбһ¶ бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҒбҹ’бһ–бһҹбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ•бҹ’бһ‘бҹғбһҖбҹҶбһ бһ·бһҸбһӣбһҫбһҠбҹӮбһ“бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ•бҹ’бһ‘бҹғбһҳбһ»бһҒбһңбһ·бһү LLMs бһ”бҹ’бһҡбһ¶бһҗбҹ’бһ“бһ¶бһ…бһ„бҹӢбһ…бһҳбҹ’бһӣбһ„бһ”бһүбҹ’бһүбһ¶бһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһ“бҹ…бһӣбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ” бһ”бһ¶бһ“бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ’бҹҶ бһ“бһ·бһ„бһ…бһҳбҹ’бһҡбһ»бҹҮ LLM бһҸбҹ’бһҡбһјбһңбһҡбһ…бһ“бһ¶бһІбҹ’бһҷбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶ бһ•бҹ’бһҸбһӣбҹӢбһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“бһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ LLM бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бҹ”  

**бһӮбһҺбһ“бһ¶**  

бһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһү LLM бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бһҒбҹ’бһ–бһҹбҹӢ бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бһ¶бһҹбһҹantaged GPU бһ‘бҹҶбһ бҹҶбһ’бҹҶбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖ ChatGPT бһ–бһёбһҠбһҫбһҳбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GPU бһ–бһёбһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһҒбҹ’бһҹбҹӮбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҷбһјбһҡбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ‘бҹ…бһңбһ·бһү SLM, бһҠбҹ„бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһ·бһ…бһҮбһ¶бһ„ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһ·бһүбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бҹ” бһӮбҹҶбһҡбһјбһҠбһјбһ…бһҮбһ¶ Mistral 7B бһўбһ¶бһ…бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– GPU бһҳбһ’бҹ’бһҷбһҳ бһ”бһҫбһ‘бҹ„бҹҮбһ”бһёбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҖбҹҸбһ“бҹ…бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҳбҹүбҹ„бһ„бһӣбһҫ GPU бһ…бҹ’бһҡбһҫбһ“бҹ”  

**бһ–бһ·бһ“бҹҗбһҷбһҮбһҳбҹ’бһҡбһ»бһү**  

бһ–бһ·бһ“бҹҗбһҷбһҮбһҳбҹ’бһҡбһ»бһүбһӮбһәбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһҡбһ”бһҹбҹӢбһўбҹҶбһ–бһҫ LLM бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбһҫбһҸбһЎбһҫбһ„бһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ” бһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҳбһ¶бһ“бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ бһ“бһ·бһ„бһҳбһ¶бһ“бһ‘бһёбһҸбһ¶бҹҶбһ„бһӣбһҫбһўбҹҠбһ·бһ“бһ’бһәбһҺбһ·бһҸбһҠбҹӮбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹҶбһҺбһ¶бһ„бһ¬бһҸбҹҶбһҺбһ¶бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӣбһҫбһҖбҹ’бһҡбһ»бһҳбһҳбһ“бһ»бһҹбҹ’бһҹбһҒбҹ’бһӣбҹҮ бһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҒбһ»бһҹ бһ¬бһ”бһүбҹ’бһ…бһјбһӣбһ–бһ·бһ“бҹҗбһҷбһ—бһ¶бһҹбһ¶бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ—бһ¶бһҹбһ¶бһҮбһ¶бһҸбһ· бһ—бһјбһҳбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бһӣбҹ’бһӣбһ¶бһҷ бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһ”бҹӢбһңбҹҒбһҷбҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбҹ” бһӣбһҫбһҹбһ–бһёбһ“бҹҒбҹҮ бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбҹҸбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҡбһ”бһҹбҹӢ LLM бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ·бһ“бҹҗбһҷбһҮбһҳбҹ’бһҡбһ»бһүбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҠбҹӮбһӣбһ”бҹ’бһҡбһ бҹӮбһӣбһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹ’бһ”бһ¶бһҹбҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбҹҖбһ”бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ‘бҹ…бһңбһ·бһү SLM, бһҠбҹӮбһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһ“бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ’бһҳбҹ’бһҳбһҸбһ¶бһӮбһәбһ—бһ¶бһӮбһҸбһ·бһ…бһ“бҹғбһҖбһ¶бһҡбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ–бһёбһ–бһ·бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһ‘бҹ„бҹҮбһ”бһёбһҳбһ·бһ“бһ”бһ¶бһ“бһ”бҹ’бһҡбһ¶бһҖбһҠбһ‘бһ¶бҹҶбһ„бһҹбҹ„бҹҮбҹ”  

**бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“**  

бһ‘бҹҶбһ бҹҶбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҡбһ”бһҹбҹӢ SLM бһ•бҹ’бһҸбһӣбҹӢбһўбһ¶бһ‘бһ·бһ—бһ¶бһ–бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹ…бһӣбһҫбһӣбҹ’бһ”бһҝбһ“бһ”бһүбҹ’бһ…бҹҒбһүбһҹбҹҒбһңбһ¶, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбһҫбһҡбһ№бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһҡбһ·бһҳбһ¶бһҺбһӮбһҺбһ“бһ¶бһҠбҹӮбһӣбһ…бҹ’бһҡбһҫбһ“бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹҒбҹҮ LLM бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶ бһ“бһ·бһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһҡбһ·бһҳбһ¶бһҺбһӮбһҺбһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҹбҹ’бһҡбһ”бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбҹ’бһ”бһҝбһ“бһ”бһүбҹ’бһ…бҹҒбһүбһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһҷбһҖбһ”бһ¶бһ“бҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ¶бһ”бҹӢбһ“бһ¶бһҖбҹӢбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҹбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹ„бһҷ LLM бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһӣбһҝбһ“бһҷбһәбһҸбһ…бһ»бҹҮ бһ–бһ·бһҹбҹҒбһҹбһ“бҹ…бһ–бҹҒбһӣбһ“бһ¶бҹҶбһ…бҹҒбһүбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”  

бһҠбҹ„бһҷбһҹбһ„бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһ‘бҹ„бҹҮбһ”бһё LLM бһ“бһ·бһ„ SLM бһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ”бҹӮбһ”бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бһҖбҹҸбһ–бһҪбһҖбһңбһ¶бһҖбҹҸбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ‘бҹ…бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹ…бһӣбһҫбһ‘бҹҶбһ бҹҶбһӮбҹҶбһҡбһј бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„ бһ—бһ¶бһ–бһ„бһ¶бһҷбһ‘бһ‘бһҪбһӣбһҡбһ„бһ–бһ·бһ“бҹҗбһҷ бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһҗбһҷбһ…бҹҒбһүбҹ” бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһҹбһҳбһҡбһҳбҹ’бһҷбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶ бһҮбһ¶бһҳбһҪбһҷ LLM бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҗбһ¶бһҳбһ–бһӣбһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„ SLM бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ—бһ¶бһ–бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҠбҹ„бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҸбһ·бһ…бҹ”  

***бһ…бҹҶбһҺбһ¶бҹҶбҹ– бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶ SLM бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Phi-3 / 3.5 бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ”***  

## бһ”бһҺбҹ’бһҠбһ¶бһ—бһ¶бһ– Phi-3 / Phi-3.5  

Phi-3 / 3.5 Family бһ•бҹ’бһҸбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘, бһңбһ·бһ…бһ·бһҸбҹ’бһҡ, бһ“бһ·бһ„бһ—бһ¶бһӮбһёбһҮбҹҶбһ“бһҪбһҷ (MoE)бҹ–  

### Phi-3 / 3.5 Instruct  

бһ•бҹ’бһҸбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘ бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһ“бҹ’бһ‘бһ“бһ¶ бһ“бһ·бһ„бһҠбһҖбһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”  

**Phi-3-mini**  

бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶ 3.8B бһҳбһ¶бһ“бһҮбҹҶбһ“бһҪбһҷбһ“бҹ… Microsoft Azure AI Studio, Hugging Face, бһ“бһ·бһ„ Ollamaбҹ” бһӮбҹҶбһҡбһј Phi-3 бһҳбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбҹ’бһҡбһјбһңбһ‘бҹҶбһ бҹҶбһҹбҹ’бһҳбһҫбһҡбһ¬бһ’бҹҶбһҮбһ¶бһ„ бһ“бҹ…бһӣбһҫбһңбһ·бһҳбһ¶бһҸбҹ’бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢ (бһҳбһҫбһӣбһӣбҹҒбһҒбһңбһ·бһҳбһҸбҹ’бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ, бһӣбҹҒбһҒбһҒбҹ’бһ–бһҹбҹӢбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡ)бҹ” Phi-3-mini бһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ“бһ·бһ„бһӮбһ»бһҺбһ—бһ¶бһ–бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ–бҹҒбһӣбһҠбҹӮбһӣ Phi-3-small бһ“бһ·бһ„ Phi-3-medium бһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҶбһҡбһјбһ’бҹҶбҹ— бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ GPT-3.5бҹ”  

**Phi-3-small & medium**  

бһҳбһ¶бһ“бһҸбҹӮ 7B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ Phi-3-small бһҮбҹҗбһҷбһҮбҹҶбһ“бҹҮбһӣбһҫ GPT-3.5T бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ—бһ¶бһҹбһ¶ бһҹбһјбһҸбҹ’бһҡбһ§бһҹбҹ’бһҹбһ¶бһ бҹҚ бһҖбһјбһҠ бһ“бһ·бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ”  

Phi-3-medium бһҳбһ¶бһ“ 14B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ”бһ“бҹ’бһҸбһ—бһ¶бһ–бһҮбҹ„бһӮбһҮбҹҗбһҷбһ“бҹҒбҹҮ бһ“бһ·бһ„бһӣбҹ’бһўбһҮбһ¶бһ„ Gemini 1.0 Proбҹ”  

**Phi-3.5-mini**  

бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһҗбһ¶бһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһҖбҹӮбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҡбһ”бһҹбҹӢ Phi-3-miniбҹ” бһ”бһ“бҹ’бһҸбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбһјбһ…а№Җаё”аёҙаёЎ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“ (бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһҮбһ¶бһ„ 20 бһ—бһ¶бһҹбһ¶бҹ– бһўбһ¶бһҡбҹүбһ¶бһ”бҹӢ, бһ…бһ·бһ“, бһ…бҹҒбһҖ, бһҠбһ¶бһҺбһәбһҳбҹүбһ¶бһҖбҹӢ, бһҠбһ¶бһ…, бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹ, бһ бҹ’бһңбһ¶бҹҶбһ„бһЎбһ„бҹӢ, бһ”бһ¶бһҡбһ¶бҹҶбһ„, бһўбһ¶бһӣбҹ’бһӣбһәбһҳбҹүбһ„бҹӢ, бһ бҹҒбһ”бҹ’бһҡбҹҠбһјбһҳ, бһ бһ»бһ„бһӮбҹ’бһҡбһё, мқҙнғҲлҰ¬м•„, бһҮбһ”бҹүбһ»бһ“, бһҖбһ»бһҡбҹүбҹҒ, бһ“бҹҗбһҡбһңбҹӮбһҹ, бһ”бҹүбһјбһЎбһјбһү, бһ–бҹҗбһҡбһ‘бһ»бһҷбһ бҹ’бһӮбһ¶бһӣбҹӢ, бһҡбһ»бһҹбҹ’бһҹбҹҠбһё, бһҹбҹ’бһ”бҹүбһ¶бһү, бһҹбҹҠбһ»бһҷбһўбҹӮбһҠ, бһҗбҹғ, бһ‘бһҪаёЈаёҒбһё, бһўбҹҠбһ»бһҷбһҖбҹ’бһҡбҹӮбһ“)  бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбһҫбһ”бһҡбһ·бһ”бһ‘бһңбҹӮбһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһ„бҹ”  

Phi-3.5-mini бһҳбһ¶бһ“ 3.8B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ…бһ‘бҹҶбһ бҹҶбһҹбҹ’бһҳбһҫ бһ“бһ·бһ„бһҹбҹ’бһҳбһҫбһӮбҹҶбһҡбһјбһ‘бҹҶбһ бҹҶбһ’бҹҶбһ–бһёбһҡбҹ”  

### Phi-3 / 3.5 Vision  

бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһҗбһ¶бһӮбҹҶбһҡбһј Instruct бһҡбһ”бһҹбҹӢ Phi-3/3.5 бһӮбһәбһҮбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҷбһӣбҹӢбһҹбҹ’бһҸбһёбһ–бһё Phi бһ бһҫбһҷ Vision бһӮбһәбһҮбһ¶бһҳбҹүбһјбһҢбҹӮбһӣбһ•бҹ’бһҠбһӣбҹӢбһ—бҹ’бһ“бҹӮбһҖбһІбҹ’бһҷ Phi бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбҹ”  

**Phi-3-Vision**  

Phi-3-Vision бһҳбһ¶бһ“бһҸбҹӮ 4.2B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ”бһ“бҹ’бһҸбһӣбһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбҹ’бһў бһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҶбһҡбһјбһ’бҹҶбһҠбһјбһ…бһҮбһ¶ Claude-3 Haiku бһ“бһ·бһ„ Gemini 1.0 Pro V бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһ‘бһјбһ‘бҹ… бһңбһ·бһ—бһ¶бһӮбһўбһҖбҹ’бһҹбһҡбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ OCR бһ“бһ·бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҸбһ¶бһҡбһ¶бһ„бһ“бһ·бһ„бһ•бҹӮбһ“бһ‘бһёбҹ”  

**Phi-3.5-Vision**  

Phi-3.5-Vision бһҮбһ¶бһҖбһ¶бһҡбһңбһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӣбһҫ Phi-3-Vision бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбһ·бһҸбһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһңбһ·бһҹбҹҗбһҷ бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһғбһҫбһүбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹҒ бһҸбҹӮбһҳбһ¶бһ“бһңбһ·бһҠбҹҒбһўбһјбһ•бһ„бһҠбҹӮбһҡбҹ”  

Phi-3.5-Vision бһӣбҹ’бһўбһҮбһ¶бһ„бһӮбҹҶбһҡбһјбһ’бҹҶбһҠбһјбһ…бһҮбһ¶ Claude-3.5 Sonnet бһ“бһ·бһ„ Gemini 1.5 Flash бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… OCR, бһҸбһ¶бһҡбһ¶бһ„, бһ“бһ·бһ„бһҖбһ¶бһ•бҹӮбһ“бһ‘бһё бһ“бһ·бһ„бһҹбҹ’бһҳбһҫбһӮбҹ’бһ“бһ¶бһ“бҹ…бһӣбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһҠбҹҶбһҺбһ№бһ„бһ‘бһјбһ‘бҹ…бҹ” бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ…бһјбһӣбһҒбҹ’бһӣбһӣбҹӢбһ–бһёбһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ вҖ” бһ’бҹ’бһңбһҫбһңбһ·бһ…бһ¶бһҡбһҺбһ¶бһ–бһёбһҡбһјбһ”бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бҹ”  

### Phi-3.5-MoE  

***Mixture of Experts (MoE)*** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһӮбҹҶбһҡбһјбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҠбҹ„бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҸбһ·бһ…бһҮбһ¶бһ„бһ’бһҳбҹ’бһҳбһҸбһ¶ бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ‘бҹҶбһ бҹҶбһӮбҹҶбһҡбһј бһ¬бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһҺбһ¶бһҷбһӮбһҺбһ“бһ¶вҖӢбһҠбһҠбҹӮбһӣбһ“бһ№бһ„вҖӢбһӮбҹҶбһҡбһјбһҠбһ„бҹӢбһҹбҹҚбҹ” бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹ бһӮбҹҶбһҡбһј MoE бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбһ»бһҺбһ—бһ¶бһ–бһҠбһјбһ…бһӮбҹҶбһҡбһјбһҠбһ„бҹӢбһҹбҹҚбһ“бҹ…бһ–бҹҒбһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҝбһ“бһҮбһ¶бһ„бҹ”  

Phi-3.5-MoE бһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣ 16x3.8B бһҳбҹүбһјбһҢбһ»бһӣбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһүбҹ” Phi-3.5-MoE бһҳбһ¶бһ“бһҸбҹӮ 6.6B бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһҹбһҖбһҳбҹ’бһҳ бһ”бҹүбһ»бһ“бҹ’бһҸбҹҒбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһңбһ·бһ…бһ¶бһҡбһҺбһ¶ бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһҠбһјбһ…бһӮбҹҶбһҡбһјбһ’бҹҶбҹ—бҹ”  

бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһӮбҹҶбһҡбһј Phi-3/3.5 Family бһ“бҹ…бһӣбһҫбһҹбһҳбһ¶бһҹбһ—бһ¶бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһҒбһ»бһҹбһ–бһё LLM бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫ Phi-3/3.5-mini бһ¬ Phi-3/3.5-Vision бһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһӮбҹҶбһҡбһјбҹ”

бһҖбһ¶бһҡбһўбһ“бһ»бһүбһ¶бһҸбһ“бҹҚ (Inference) бһҹбһ“бҹ’бһ’бһ·бһҹбһ“бҹ’бһ’бһ¶бһӮбҹ’бһҡбһ”бҹӢбһ‘бһ¶бҹҶбһ„ Phi-3 бһ¬бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҺбһ¶бһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ GPT-3 бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ бһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷ бһ¬бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһўбһүбҹ’бһүбһҫбһүбһҠбҹӮбһӣбһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бҹ” бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһҹбҹҶбһҺбһҪбһҡ бһ¬бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҠбҹ’бһӢбһ¶бһ“бһ‘бҹ… Phi-3 бһңбһ¶бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһ”бҹ’бһҡбһҹбһ¶бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһҡбһҪбһ…бһҮбһҳбҹ’бһҡбһ¶бһ”бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣбһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһҹбһҳбһҡбһҳбҹ’бһҷбһҠбҹ„бһҷбһҹбһ·бһҖбҹ’бһҹбһ¶бһ–бһёбһӣбҹҶбһ“бһ¶бҹҶ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ”

**Hugging Face Transformer**
Hugging Face Transformers бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһ бһ¶бһ—бһ¶бһҹбһ¶бһҹбһҳбҹ’бһҳбһүбҹ’бһү (NLP) бһ“бһ·бһ„бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһҡбҹүбһјбһ”бһјбһҸбһ·бһ…бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” бһӮбһәбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. **бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ‘бһ»бһҖбһҮбһ¶бһҳбһ»бһ“**: бһңбһ¶бһҮбһҪбһҹбһҮбһ»бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҠбһ»бҹҮбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡ бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһўбһҸбҹ’бһҗбһ”бһ‘ бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡ бһҖбһ¶бһҡбһҹбһ„бҹ’бһҒбҹҒбһ” бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”

2. **бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„**: бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбҹҖбһ“бһҮбҹ’бһҡбҹ…бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ PyTorch, TensorFlow бһ“бһ·бһ„ JAXбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһҪбһҷ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ¶бһ“бҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ”бһ¶бһ“бҹ”

3. **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ–бһ бһ»бһҳбҹүбһјбһҠбһ»бһӣ**: бһӣбһҫбһҹбһ–бһё NLP Hugging Face Transformers бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҗбһҸбһҡбһјбһ” (vision) бһҮбһ¶бһҠбһҫбһҳ бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҡбһјбһ”бһҗбһҸ бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһңбһҸбҹ’бһҗбһ» бһ“бһ·бһ„бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҹбҹҶбһЎбҹҒбһ„бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҹбҹҶбһЎбҹҒбһ„ бһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһҹбҹҶбһЎбҹҒбһ„бҹ”

4. **бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ**: бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ•бҹ’бһҸбһӣбҹӢ API бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһ”бһ„бҹ’бһҡбҹҖбһ“бһҹбҹ’бһҳбһ¶бһҡбһҸбһёбһңбһ·бһү бһ бһҫбһҷбһ„бһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһјбһ…бһҮбһ¶бһҗбҹ’бһҳбһёбһ“бһ·бһ„бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢбҹ”

5. **бһҹбһ бһӮбһҳбһ“бҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“**: Hugging Face бһҳбһ¶бһ“бһҹбһ бһӮбһҳбһ“бҹҚбһҠбҹҸбһҡбһҹбһҫбһ” бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһү бһ‘бһјбһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбҹ”

**ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GenAI**

[ONNX Runtime](https://github.com/microsoft/onnxruntime-genai?WT.mc_id=academic-105485-koreyst) бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹүбҹӮбһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ“бһ бҹ’бһҹбһ·бһ“бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖ бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Generative AI (GENAI) бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹҸбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһҠбҹӮбһӣбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸ efficiently бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

## ONNX Runtime бһҮбһ¶бһўбҹ’бһңбһё?
ONNX Runtime бһӮбһәбһҮбһ¶бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҳбһҪбһҷбһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бһүбҹ’бһ…бҹҒбһүбһҖбҹҶбһ бһ·бһҸбһҶбһ“бҹ’бһ‘бҹҲбһҒбҹ’бһ–бһҹбҹӢбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бҹ” бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ Open Neural Network Exchange (ONNX) бһҠбҹӮбһӣбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһҺбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бҹ” ONNX Runtime бһўбһ¶бһ…бһҮбһҪбһҷбһІбҹ’бһҷбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҡбһ бҹҗбһҹ бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһҖбҹ’бһҡбҹ„бһҳбһҠбҹ„бһҷбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбҹҖбһ“бһҮбҹ’бһҡбҹ…бһҠбһјбһ…бһҮбһ¶ PyTorch бһ“бһ·бһ„ TensorFlow/Keras бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһ”бһ»бһҡбһ¶бһҺбһҠбһјбһ…бһҮбһ¶ scikit-learn, LightGBM, XGBoostбҹ” ONNX Runtime бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҳбһҪбһҷбһҡбҹҖбһ„бҹ—бһ§бһ”бһҖбһҡбһҺбҹҚ бһ”бһҺбҹ’бһҠбһ¶бһүбһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„ бһ“бһ·бһ„бһҮбһҪбһҹбһҮбһ»бһӣбһ бҹ’бһҡбҹ’бһңбҹҒбһҳбһңбҹҗбһҡбһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ¶бһ”бҹ”

## Generative AI бһҮбһ¶бһўбҹ’бһңбһё?
Generative AI бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҗбҹ’бһҳбһёбҹ—бһҠбһјбһ…бһҮбһ¶ бһўбһҸбҹ’бһҗбһ”бһ‘ бһҡбһјбһ”бһ—бһ¶бһ– бһ¬бһҸбһ“бҹ’бһҸбҹ’бһҡбһё бһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһңбһ¶бһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ¶бһ“ бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ GPT-3 бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҠбһјбһ…бһҮбһ¶ Stable Diffusionбҹ” бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GenAI бһ•бҹ’бһҸбһӣбҹӢбһҒбҹ’бһҹбҹӮбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҮбһ¶бһҳбһҪбһҷ ONNX Runtime бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„ logits бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһҗбһҸ бһ бһҫбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ KV cacheбҹ”

## ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI
ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI бһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ“бҹғ ONNX Runtime бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ generative AI бҹ” бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бҹ–

- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһңбҹҒбһ‘бһ·бһҖбһ¶бһ…бҹ’бһҡбһҫбһ“**бҹ– бһңбһ¶бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ Windows, Linux, macOS, Android, бһ“бһ·бһ„ iOSбҹ”
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ**бҹ– бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ generative AI бһҖбҹҶбһ–бһјбһӣбҹ— бһҠбһјбһ…бһҮбһ¶ LLaMA, GPT-Neo, BLOOM бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
- **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ**бҹ– бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҮбһҪбһҹбһҮбһ»бһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶ NVIDIA GPUs, AMD GPUs бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
- **бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ**бҹ– бһ•бҹ’бһҸбһӣбҹӢ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘ бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠбһҸбһ·бһ…бһҸбһҪбһ…бҹ”
- бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ¶бһ…бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ generate() бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ бһ¬бһҡбһҸбҹӢбһӮбҹ’бһҡбһ”бҹӢбһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ—бһ¶бһӮбһҳбһҪбһҷ бһ–бһёбһҡ бһҡбһәбһ”бһё бһ“бһ·бһ„бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһӣбҹ’бһ”бһҝбһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣбҹ”
- ONNX Runtime бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ greedy/beam бһ“бһ·бһ„ TopP, TopK бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҸбҹҶбһҺбһ—бһ¶бһӮ бһ бһҫбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„ logits Build-in бһҠбһјбһ…бһҮбһ¶бһ‘бһҺбҹ’бһҢбһҖбһҳбҹ’бһҳбһ’бҹ’бһңбһҫбһҹбҹ’бһ‘бһҪбһ“бһҡбһҠбһјбһңбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ”бһ¶бһ“бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

### бһҠбҹҶбһЎбһҫбһ„бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳ Generative AIбҹ–

### бһҡбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбҹ– бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһ¶бһҳбһүбҹ’бһүбһҖбҹ’бһ“бһ»бһ„ Pythonбҹ–
```Python
import onnxruntime_genai as og

model = og.Model('path_to_your_model.onnx')

tokenizer = og.Tokenizer(model)

input_text = "Hello, how are you?"

input_tokens = tokenizer.encode(input_text)

output_tokens = model.generate(input_tokens)

output_text = tokenizer.decode(output_tokens)

print(output_text) 
```
### бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ– бһ”бҹ’бһҡбһҫ ONNX Runtime GenAI бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ… Phi-3.5-Vision


```python

import onnxruntime_genai as og

model_path = './Your Phi-3.5-vision-instruct ONNX Path'

img_path = './Your Image Path'

model = og.Model(model_path)

processor = model.create_multimodal_processor()

tokenizer_stream = processor.create_stream()

text = "Your Prompt"

prompt = "<|user|>\n"

prompt += "<|image_1|>\n"

prompt += f"{text}<|end|>\n"

prompt += "<|assistant|>\n"

image = og.Images.open(img_path)

inputs = processor(prompt, images=image)

params = og.GeneratorParams(model)

params.set_inputs(inputs)

params.set_search_options(max_length=3072)

generator = og.Generator(model, params)

while not generator.is_done():

    generator.compute_logits()
    
    generator.generate_next_token()

    new_token = generator.get_next_tokens()[0]
    
    output = tokenizer_stream.decode(new_token)
    
    print(tokenizer_stream.decode(new_token), end='', flush=True)

```
