# бһҳбҹҒбһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһўбҹҶбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ
AI бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһәбһҮбһ¶бһңбһ·бһҹбҹҗбһҷбһҳбһҪбһҷбһҠбҹӮбһӣбһӮбһҪбһҡбһұбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹғбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҗбҹ’бһҳбһёбҹ” бһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бҹҒбҹҮбһўбһ¶бһ…бһҳбһ¶бһ“бһ…бһ¶бһ”бҹӢбһ–бһёбһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ– бһҠбһӣбҹӢбһ—бҹ’бһӣбҹҒбһ„ бһ“бһ·бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһңбҹҒбһҡбһҮбһ¶бһҹбһ»бһ‘бҹ’бһ’бһҹбҹ’бһ‘бһҫбһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ” бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ“бҹҗбһҷбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹӮбһӣбһӮбһҪбһҡбһұбҹ’бһҷбһҡбҹҶбһ—бһҫбһ”бһ”бҹҶбһ•бһ»бһҸ бһӮбһәбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бҹ”

## бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ…бһҮбһ¶бһўбҹ’бһңбһё?

бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ… (SLM) бһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ“бҹ’бһҸбһ·бһ…бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLM) бһҳбһҪбһҷ бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ–бһё LLM бһҒбһҺбҹҲбһ–бҹҒбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһүбҹ’бһүбһ¶бһ‘бҹҶбһ бҹҶбһӮбһҺбһ“бһ¶бһ”бһ“бҹ’бһҗбһҷбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ”

SLMs бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶбҹ—бһҠбһјбһ…бһҮбһ¶ GPT-4, SLM бһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҸбһјбһ… бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһ бһҫбһҷбһҹбһ¶бһҖбһҹбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶ бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” бһ‘бҹ„бҹҮбһ”бһёбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҸбһјбһ… бһ–бһҪбһҖбһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бһ¶бһ“бҹ” бһ’бһҳбҹ’бһҳбһҸбһ¶ SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһ”бҹӢ бһ¬бһ”бһ„бҹ’бһ бһ¶бһҷ LLM бһҠбҹ„бһҷбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ—бһ¶бһҹбһ¶ бһ…бҹ’бһҡбһҫбһ“бһ бһҫбһҷбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҠбһҫбһҳбҹ” бһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ‘бһјбһ‘бҹ… бһ’бҹ’бһңбһҫбһІбҹ’бһҷ SLM бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбһ¶бһ„бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ бһ“бһ·бһ„бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҖбһ¶бһҡбһҖбһҺбҹ’бһҸбһ¶бһӣбһӮбһҺбһ“бһ¶бҹ” бһ”бһ“бҹ’бһ‘бһ»бҹҮбһҗбһ¶бһҹбһ бһ—бһ¶бһ– бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ SLM бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҠбҹӮбһ“бһҠбһёбһҡбҹҖбһ”бһ…бҹҶбһ—бһ¶бһҹбһ¶бһҹбһ»бһ‘бҹ’бһ’ бһ”бҹӮбһ” NLP:

- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҷбҹ„бһӮ бһ¬бһңбһӮбҹ’бһӮбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһјбһ”бһҳбһ“бҹ’бһҸ бһ“бһ·бһ„бһҹбһҳбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһ”бһҡбһ·бһ”бһ‘бҹ”
- бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бҹ– бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ‘бһ¶бһҷ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ”бҹ’бһҡбһҷбҹ„бһӮбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҫбһҖбһ”бһҡбҹ”
- бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбҹ– бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһёбһ—бһ¶бһҹбһ¶бһҳбһҪбһҷбһ‘бҹ…бһ—бһ¶бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
- бһҖбһ¶бһҡбһҹбһ„бҹ’бһҒбҹҒбһ”бҹ– бһҹбһ»бһ бҹ’бһңбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһңбҹӮбһ„бһІбҹ’бһҷбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҷбһӣбҹӢбҹ”

бһ‘бҹ„бҹҮбһҮбһ¶бһҳбһ¶бһ“бһҖбһ¶бһҡбһҮбһҪбһүбһҠбһјбһҡбһҒбҹ’бһӣбҹҮбһҖбҹ’бһ“бһ»бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ¬бһҮбҹҶбһҡбҹ…бһ“бҹғбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„ бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһ‘бҹ…бһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶбҹ”

## бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?
SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбҹ’бһҗбһ”бһ‘бһ…бҹ’бһҡбһҫбһ“бһ‘бһ№бһҖбҹ” бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ бһ–бһҪбһҖбһңбһ¶бһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бһ·бһ„бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ”бһҹбҹӢбһ—бһ¶бһҹбһ¶ бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘ бһҠбҹӮбһӣбһҳбһ¶бһ“бһңбҹҒбһҷбҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҹбһҳбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһ”бһҡбһ·бһ”бһ‘бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһ”бҹ’бһҡбһҳбһјбһӣбһ”бһҺбҹ’бһҠбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҡбһҹбҹҒбһҡбһҠбҹҸбһ’бҹҶбһ…бҹ’бһҡбһҫбһ“бһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһ…бһ¶бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ– бһҹбҹҶбһўбһ¶бһҸ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ”бҹӮбһ”бһ”бһ‘бһІбҹ’бһҷбһҹбһҳбһ“бһ№бһ„бһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ”
- бһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ– бһ”бҹ’бһҡбһҫбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҡбҹҖбһ“бһҳбҹүбһјбһҠбҹӮбһӣ бһҡбһ”бҹҖбһ”бһҷбһӣбҹӢ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҡбҹҖбһ“

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҳбһ¶бһ“бһ”бҹҶбһҺбһ„бһҺбҹӮбһ“бһ¶бҹҶбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһўбҹҶбһ–бһё SLM бһ бһҫбһҷбһ•бҹ’бһӮбһ»бҹҶбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ Microsoft Phi-3 бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҹбһ·бһ“бһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһўбһҸбҹ’бһҗбһ”бһ‘, бһңбһёбһҹбҹ’бһҷбһјбһ“бһ·бһ„ MoEбҹ”

бһ“бҹ…бһ…бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҶбҹ’бһӣбһҫбһҷбһ”бһ¶бһ“бһ…бҹҶбһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- SLM бһҮбһ¶бһўбҹ’бһңбһё?
- бһ•бҹ’бһ“бҹӮбһҖбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ SLM бһ“бһ·бһ„ LLM бһҮбһ¶бһўбҹ’бһңбһё?
- Microsoft Phi-3/3.5 бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?
- бһҸбһҫбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҸбҹӢбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҮбһ¶бһҳбһҪбһҷ Microsoft Phi-3/3.5?

бһҸбһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҡбһҪбһ…бһ бһҫбһҷ? бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҷбһҫбһ„бһ”бҹ’бһҡбһ¶бһ”бҹӢбҹ”

## бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLMs) бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ… (SLMs)

бһ‘бһ¶бҹҶбһ„ LLM бһ“бһ·бһ„ SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһҫбһӮбҹ’бһҡбһ№бҹҮбһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶бһҠбһӣбҹӢбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹ’бһҗбһ·бһҸбһ· бһҹбһҳбҹ’бһ”бһҸбҹ’бһҠбһ·бһ‘бһ¶бҹҶбһ„бһЎбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶, бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ, бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһ‘бҹ„бҹҮбһҷбҹүбһ¶бһ„бһ“бҹ„бҹҮ бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹҒбҹҮ бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹүбһ·бһ“бһ”бҹ’бһҡбһҹбһ”бҹӢбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҒбҹ’бһӣбҹҮбҹ”

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ…бҹҶбһҺбһ»бһ…бһҸбһјбһ…

SLM бһҳбһ¶бһ“бһҖбһҸбҹ’бһҸбһ¶бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶бҹ–

- бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҖбһ”бҹ’бһҡбҹӮбҹ– бһ•бҹ’бһҸбһӣбҹӢбһҮбҹҶбһ“бһҪбһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһ”бҹ’бһҡбһ–бҹғбһҺбһёбҹ”
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бҹ– бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’бһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһ“бһ·бһҸ бһ¬бһҖбҹҸбһҹбһҡбһҹбҹҒбһҡбһўбһҸбҹ’бһҗбһ”бһ‘бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ”
- бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶбҹ– бһҮбһҪбһҷбһҹбһ·бһҹбҹ’бһҹбһІбҹ’бһҷбһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ¬бһҡбҹҖбһ“бһ—бһ¶бһҹбһ¶бһҗбҹ’бһҳбһёбҹ”
- бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҒбҹ’бһңбһ¶бһҖбҹӢ бһҠбһјбһ…бһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһ”бһјбһҡбһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹ…бһҹбһҳбҹ’бһӣбҹҒбһ„бҹ”

**бһ‘бҹҶбһ бҹҶ**
 
бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ…бһҳбҹ’бһ”бһ„бһҡбһңбһ¶бһ„ LLM бһ“бһ·бһ„ SLM бһӮбһәбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҠбҹӮбһӣбҹ” LLMs бһҠбһјбһ…бһҮбһ¶ ChatGPT (GPT-4) бһҳбһ¶бһ“бһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ бһ”бҹ’бһҡбһ бҹӮбһӣ 1.76 бһҸбҹ’бһҡбһёбһӣбҹ’бһӣбһ¶бһ“ бһҒбһҺбҹҲбһҠбҹӮбһӣ SLM бһ”бһҫбһҖбһ…бҹҶбһ  Mistral 7B бһҳбһ¶бһ“бһ–бһ¶бһҡбҹүбһ¶бһӮбһә бҹ§бһ–бһ¶бһ“бҹӢбһӣбһ¶бһ“ бһҸбһ·бһ…бһҮбһ¶бһ„бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҹбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ” бһ§. ChatGPT бһ”бҹ’бһҡбһҫбһҖбҹҶбһ…бһ¶бһҸбҹӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (self-attention mechanism) бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘ encoder-decoder бһҒбһҺбҹҲ Mistral 7B бһ”бҹ’бһҡбһҫбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһ”бһҫбһҖбһ”бһүбҹ’бһҮбҹҶбһ‘бҹ’бһңбһ¶бһҡ attention бһҠбҹӮбһӣбһўбһ¶бһ…бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„ бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҠбҹӮбһӣ decoder бһҹбҹ’бһҸбһ¶бһ…бҹӢбҹ” бһҖбһ¶бһҡбһ”бҹ’бһҠбһјбһҡбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһӣбһҫбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бҹ”

**бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„**

SLM бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҳбҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһҹбҹҗбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ‘бһјбһӣбһ¶бһҷбһ“бҹ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһ“бһҮбҹҶбһ“бһ¶бһүбҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹҒбҹҮ LLM бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһўбҹҠбһёбһ“бһ’бһ·бһӣбһёбһ бҹ’бһҹбһ„бҹӢбһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ” бһҖбһ¶бһҡбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһ…бҹҶбһҡбһјбһ„ бһ…бһ·бһҸбҹ’бһҸ LLM бһўбһ¶бһ…бһ”бҹҶбһ–бҹҒбһүбһ”бһ¶бһ“бһӣбҹ’бһўбһ“бҹ…бһңбһ·бһҹбҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһҸбҹӢбһ”бҹӮбһ“ бһ“бһ·бһ„бһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ LLM бһҹбһ¶бһҖбһҹбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ NLP бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ—бһ¶бһҹбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ”

**бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶**

бһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ LLM бһҸбһҳбҹ’бһҡбһјбһңбһңбһ·бһ—бһ¶бһӮбһ“бҹҚбһ’бһ“бһ’бһ¶бһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚбһӮбһҺбһ“бһ¶ бһ§.бһҖбҹ’бһҡбһ»бһҳ GPU бһ‘бҹҶбһ бҹҶбһ’бҹҶбҹ” бһ§. бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҠбһјбһ…бһҮбһ¶ ChatGPT бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһӮбҹҒбһӣ бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢбһ“бҹғ GPU бһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бҹ” бһ•бҹ’бһ‘бһ»бһҷбһӮбҹ’бһ“бһ¶ SLM бһҮбһ¶бһҳбһҪбһҷбһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһјбһ…бһҮбһ¶бһ„ бһӮбһәбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбҹ„бһҷбһ’бһ“бһ’бһ¶бһ“бһӮбһҺбһ“бһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһҠбһјбһ…бһҮбһ¶ Mistral 7B бһўбһ¶бһ…бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбһҪбһҷбҹ—бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– GPU бһҳбһ’бҹ’бһҷбһҳ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹ’бһҡбһҫбһ“бһҳбҹүбҹ„бһ„бһ–бһё GPU бһ–бһ бһ»бҹ”

**бһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһ“бһ№бһ„**

бһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһ“бһ№бһ„бһӮбһәбһҮбһ¶бһ”бһүбҹ’бһ бһ¶бһҠбҹӮбһӣбһӮбҹҒбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ LLM бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢбһңбһ¶ бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бһ–бһ–бһёбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„ бһ¬бһ”бһ„бҹ’бһ бһ¶бһүбһҒбһ»бһҹбһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһ’бһ¶бһҸбһ»бһӮбҹ„бһӣбһ“бҹ…бһҳбһ“бһ»бһҹбҹ’бһҹбһҮбһҪбһӣ бһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһңбһ¶бһҷбһ–бһ·бһ“бҹ’бһ‘бһ»бһҒбһ»бһҹбһҶбҹ’бһӮбһ„ бһҠбҹ„бһҷбһҹбһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһҹбһ¶ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбҹҶбһ”бһ“бҹӢ бһ¬бһ…бҹ’бһ”бһ¶бһ”бҹӢбһңбҹҒбһҷбҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһ“бҹҒбҹҮ бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ’бҹҶбһҡбһ”бһҹбҹӢ LLM бһўбһ¶бһ…бһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһ“бһ№бһ„бһҠбҹ„бһҷбһҘбһҸбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһӣбһ‘бҹ’бһ’бһ•бһӣбһўбһ¶бһ…бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢбҹ” бһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ SLM бһҠбҹӮбһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҮбһ¶бһ„ бһҸбһ·бһ…бһҮбһ¶бһ„бһ”бҹ’бһҡбһҲбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҡбһ”бһҹбҹӢбһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһ“бһ№бһ„бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһүбһ‘бҹҒбҹ”

**бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮ**

бһ‘бҹҶбһ бҹҶбһҸбһјбһ…бһҡбһ”бһҹбҹӢ SLM бһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҒбһүбһӣбһҝбһ“ бһӣбһҫбһҡбһ№бһҺбҹҲбһҗбҹҒбһҡ бһ бһҫбһҷбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ лі‘пҝҪпҝҪбһҖбһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһёбһ“бҹҒбҹҮ LLM, бһҠбҹ„бһҷбһҹбһ¶бһҡбһ‘бҹҶбһ бҹҶ бһ“бһ·бһ„бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһ…бҹ’бһҡбһҫбһ“ бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӮбһҺбһ“бһ¶ лі‘пҝҪпҝҪбһҖбһӮбҹ’бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһңбһ·бһ—бһ¶бһӮбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһҖбҹҸбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ–бһёбһӣбһҫ LLM бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бһ»бҹҮ бһҠбҹ„бһҷбһӮбҹҒбһӣбһҫбһ•бҹ’бһ‘бҹғбҹ”

бһҹбһ„бҹ’бһҒбҹҒбһ”бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ‘бҹ„бҹҮбһ”бһё LLM бһ“бһ·бһ„ SLM бһҳбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһҖбҹҸбһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ“бҹ…бһӣбһҫбһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҠбҹӮбһӣ, бһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбһ’бһ“бһ’бһ¶бһ“, бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһҸбһ¶бһҳбһ”бһҡбһ·бһ”бһ‘, бһ—бһ¶бһ–бһ„бһ¶бһҷбһ”бҹ’бһҡбһҲбһҳбһ“бһ№бһ„бһ—бһ¶бһ–бһҳбһ·бһ“бһҹбһҳбһ“бһ№бһ„ бһ“бһ·бһ„бһ—бһ¶бһ–бһҡбһ бҹҗбһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹӮбһӣбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹғбһңбһ¶ бһҠбҹӮбһӣ LLM бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбҹҶбһҡбһ¶бһ”бҹӢбһҮбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһ’бһ“бһ’бһ¶бһ“бһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„ SLM бһ•бҹ’бһҸбһӣбҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҠбҹ„бһҷбһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһ’бһ“бһ’бһ¶бһ“бһҸбһ·бһ…бҹ”

***бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶ SLM бһҸбһ¶бһҳбһҡбһҷбҹҲ Microsoft Phi-3 / 3.5 бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ”***

## бһҳбһёбһ“бһҳбһҳбһҫбһӣбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ—бһё-3 / бһ—бһё-3.5

бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ—бһё-3 / 3.5 бһҳбһ¶бһ“бһӮбҹ„бһӣбһҠбҹ…бһ‘бҹ…бһӣбһҫбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹӮбһ”бһўбһҸбҹ’бһҗбһ”бһ‘, бһңбһёбһҹбҹ’бһҷбһјбһ“бһ·бһ„бһ—бһ¶бһӮбһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (MoE) :

### бһ—бһё-3 / 3.5бһўбһ“бһ»бһҹбҹ’бһҹбһҡбһёбһҖбһҡбһ¶бһҳ

бһҮбһ¶бһ…бһҳбҹ’бһ”бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘, бһ”бһүбҹ’бһ…бһ”бҹӢбһҮбһҮбҹӮбһҖ, бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”

**бһ—бһё-3-бһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶ**

бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ Microsoft Foundry, Hugging Face, бһ“бһ·бһ„ Ollamaбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһё-3 бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢбһӣбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһ’бҹҶ бһҮбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— (бһҮбҹҶбһ“бҹ„бһҡбһ‘бҹ…бһӣбҹҒбһҒбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ бһӣбҹҒбһҒбһҒбҹ’бһ–бһҹбҹӢбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһӣбҹ’бһўбһҮбһ¶бһ„)бҹ” бһ—бһё-3-бһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶ бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһ–бһёбһҡбһҮбһҫбһ„бһ‘бҹҒ бһҒбһҺбҹҲбһ—бһё-3-бһҸбһјбһ… бһ“бһ·бһ„бһ—бһё-3-бһҳбһ’бҹ’бһҷбһҳбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶ бһҡбһҪбһҳбһҳбһ¶бһ“ GPT-3.5бҹ”

**бһ—бһё-3-бһҸбһјбһ… бһ“бһ·бһ„ бһҳбһ’бҹ’бһҷбһҳ**

бһҮбһ¶бһҳбһҪбһҷбһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ 7B, бһ—бһё-3-бһҸбһјбһ… бһҲбҹ’бһ“бҹҮбһӣбһҫ GPT-3.5T бһ“бҹ…бһӣбһҫбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ—бһ¶бһҹбһ¶, бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ, бһҖбһјбһҠ бһ“бһ·бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бҹ”

бһ—бһё-3-бһҳбһ’бҹ’бһҷбһҳ бһҳбһ¶бһ“бһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ 14B бһ”бһ“бҹ’бһҸбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһ бһҫбһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„ Gemini 1.0 Proбҹ”

**бһ—бһё-3.5-бһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶ**

бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһҗбһ¶ бһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ—бһё-3-бһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбҹ” бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҳбһ·бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“ (бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһ„ 20 бһ—бһ¶бһҹбһ¶ бһҠбһјбһ…бһҮбһ¶ бһўбһ¶бһҡбҹүбһ¶бһ”бҹҠбһё, бһ…бһ·бһ“, бһ…бҹҒбһҖ, бһҠбһ¶бһҺбһ»бһҹ, бһҠбһ¶бһ…бҹӢ, бһўбһ„бҹӢбһӮбҹ’бһӣбҹҒбһҹ, бһ бҹ’бһңбһ¶бҹҶбһ„бһЎбһ„бҹӢ, бһ”бһ¶бһҡбһ¶бҹҶбһ„, бһўбһ¶бһӣбҹ’бһӣбһәбһҳбҹүбһ„бҹӢ, бһ бҹҒбһ”бҹ’бһҡбҹҠбһј, бһ бһ»бһ„бһӮбҹ’бһҡбһё, бһўбҹҠбһёбһҸбһ¶бһӣбһё, бһҮбһ”бҹүбһ»бһ“, бһҖбһјбһҡбҹүбҹҒ, бһ“бҹҗбһҡбһңбҹӮбһҹ, бһ”бҹүбһјбһЎбһјбһү, бһ–бҹҗбһҡбһ‘бһ»бһҷбһ бҹ’бһӮбһ¶бһӣбҹӢ, бһҡбһ»бһҹбҹ’бһҹбҹҠбһё, бһҹбҹ’бһ”бһ¶бһү, бһҹбҹҠбһ»бһҷбһўбҹӮбһҠ бһ’бҹғ, бһ‘бһҪбһҡбһӮбһё, бһўбҹҠбһ»бһҷбһҖбҹ’бһҡбҹӮбһ“) бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ…бҹҶбһ–бҹ„бҹҮбһ”бһҡбһ·бһ”бһ‘бһңбҹӮбһ„бҹ”

бһ—бһё-3.5-бһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бһ»бҹҶ бһҮбһ¶бһҳбһҪбһҷбһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ 3.8B бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ бҹҶбһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһҹбҹ’бһҳбһҫбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһ бҹӮбһӣбһ–бһёбһҡбһҠбһ„бһ‘бҹҶбһ бҹҶбһҮбһ¶бһ„бһңбһ¶бҹ”

### бһ—бһё-3 / 3.5 бһңбһёбһҹбҹ’бһҷбһјбһ“

бһҷбһҫбһ„бһўбһ¶бһ…бһӮбһ·бһҸбһҗбһ¶ бһҳбҹүбһјбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹҗбһҸбҹ’бһҸбһ—бһё-3/3.5 бһӮбһәбһҮбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһ—бһёбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢ, бһ бһҫбһҷбһңбһ·бһҹбҹ’бһҷбһјбһ“бһӮбһәбһҠбһјбһ…бһ—бһёбһҳбһ¶бһ“бһ—бҹ’бһ“бҹӮбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбһӣбҹӢбһ–бһ·бһ—бһ–бһӣбҹ„бһҖбҹ”


**бһ—бһё-3-бһңбһ·бһҹбҹ’бһҷбһјбһ“**

бһ—бһё-3-бһңбһ·бһҹбҹ’бһҷбһјбһ“ бһҳбһ¶бһ“бһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ 4.2B бһ”бһ“бҹ’бһҸбһ”бҹӮбһ”бһ“бҹҒбҹҮ бһ бһҫбһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶ бһ”бҹ’бһҡбһ бҹӮбһӣ Claude-3 Haiku бһ“бһ·бһ„ Gemini 1.0 Pro V бһӣбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҷбҹ„бһ”бһӣбҹӢбһ‘бһјбһ‘бҹ… бһҠбһјбһ…бһҮбһ¶ OCR, бһ“бһ·бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бһёбһҸбһ¶бһҡбһ¶бһ„ бһ“бһ·бһ„бһҡбҹҖбһ”аёҒаёЈбҹҒбһёбһ“бҹ”


**бһ—бһё-3.5-бһңбһ·бһҹбҹ’бһҷбһјбһ“**

бһ—бһё-3.5-бһңбһ·бһҹбҹ’бһҷбһјбһ“ бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ“бҹғбһ—бһё-3-бһңбһ·бһҹбҹ’бһҷбһјбһ“ бһҠбҹӮбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбһ·бһҸбһҗбһ¶бһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ·бһҹбҹ’бһҷбһјбһ“бһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһғбһҫбһүбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹҒ бһҸбҹӮбһҖбҹҸбһғбһҫбһүбһңбһёбһҠбҹҒбһўбһјбһ•бһ„бһҠбҹӮбһҡбҹ”

бһ—бһё-3.5-бһңбһ·бһҹбҹ’бһҷбһјбһ“ бһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶбҹ—бһҠбһјбһ…бһҮбһ¶ Claude-3.5 Sonnet бһ“бһ·бһ„ Gemini 1.5 Flash бһӣбһҫбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„ OCR, бһҸбһ¶бһҡбһ¶бһ„ бһ“бһ·бһ„бһҸбһ¶бһҡбһ¶бһ„бһ”бһҪбһ“бһ·бһ…бһ¶бһҡ бһ“бһ·бһ„бһҹбҹ’бһҳбһҫбһҸбһҳбҹ’бһӣбҹғбһ“бһ№бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ‘бһјбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһңбһ·бһҹбҹҗбһҷбһңбһ·бһҹбҹ’бһҷбһјбһ“бҹ” бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҡбһјбһ”бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһ·бһҸбһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бҹ”


### бһ—бһё-3.5-MoE

***бһҖбһ¶бһҡбһўбһҳбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһү (MoE)*** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҳбһ–бһӣбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ бһҸбһҫбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһ“бһ‘бҹҶбһ бҹҶбһҳбҹүбһјбһҠбҹӮбһӣ бһ¬бһӮбҹҶбһҡбһјбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹ„бһҷбһҗбһңбһ·бһҖбһ¶бһӮбһҺбһ“бһ¶ бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бҹ” бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһҳбҹүбһјбһҠбҹӮбһӣ MoE бһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҠбһјбһ…бһҳбҹүбһјбһҠбҹӮбһӣбһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҶбһ¶бһ”бҹӢбһҡбһ бҹҗбһҹбһҮбһ¶бһ„бһ“бҹ…бһ–бҹҒбһӣбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбҹ”

бһ—бһё-3.5-MoE бһҳбһ¶бһ“ 16x3.8B бһҳбҹүбһјбһҢбһ»бһӣбһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһүбҹ” бһ—бһё-3.5-MoE бһҳбһ¶бһ“бһҸбҹӮ 6.6B бһ–бһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбҹӮбһӣбһҹбһҖбһҳбҹ’бһҳ бһӮбһәбһўбһ¶бһ…бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһ·бһҸ бһўбһҸбҹ’бһҸбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ—бһ¶бһҹбһ¶ бһ“бһ·бһ„бһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ’бҹҶбһҮбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ”

бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ—бһё-3/3.5 бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ•бҹ’бһ‘бһ»бһҷбһ–бһё LLM бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„ Phi-3/3.5-mini бһ¬ Phi-3/3.5-бһңбһ·бһҹбҹ’бһҷбһјбһ“бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹҶбһҺбһҸбҹӢбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбҹ”


## бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ—бһё-3/3.5

бһҷбһҫбһ„бһҳбһ¶бһ“бһ”бҹҶбһҺбһ„бһ”бҹ’бһҡбһҫ Phi-3/3.5 бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһҫ Phi-3/3.5 бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

**NVIDIA NIM**

бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry бһ“бҹ…бһӣбһҫбһӮбҹ’бһӣбҹ„бһҠ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ [NVIDIA NIM](https://developer.nvidia.com/nim?WT.mc_id=academic-105485-koreyst) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһ‘бҹ… NVIDIA NIM бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ бҹ… API бһҡбһ”бһҹбҹӢбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡбһ—бһё-3/3.5бҹ” NVIDIA NIM (NVIDIA Inference Microservices) бһӮбһәбһҮбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҳбҹҒбһҳбҹүбҹғбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһңбҹҒбһ…бһҒбҹ’бһ…бһ”бҹӢ бһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ”бһ„бҹ’бһ бҹ„бҹҮбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҠбҹ„бһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһҸбһ¶бһҳбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһјбһ…бһҮбһ¶ бһӮбҹ’бһӣбҹ„бһҠ, бһҖбһҺбҹ’бһҸбһ¶бһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡбҹ”

бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢ NVIDIA NIM бһҳбһ¶бһ“бһҠбһјбһ…бһҮбһ¶бҹ–

- **бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бҹ–** NIM бһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„бһҳбҹүбһјбһҠбҹӮбһӣ AI бһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶вҖӢбһҳбһҪбһҷбһ‘бһ¶бһҸбҹӢ бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бҹ”

- **бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–:** бһңбһ¶бһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһ„бҹ’бһҡбҹҖбһ“бһҖбҹҶбһҺбһҸбҹӢбһӣбҹ’бһ”бһҝбһ“бһҳбһ»бһ“бһҡбһ”бһҹбҹӢ NVIDIA бһҠбһјбһ…бһҮбһ¶ TensorRT бһ“бһ·бһ„ TensorRT-LLM бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ”бһ¶бһ“ latency бһ‘бһ¶бһ” бһ“бһ·бһ„ throughput бһҒбҹ’бһ–бһҹбҹӢбҹ”
- **бһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“:** NIM бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһҖбҹӮбһӣбһўбһјбһҸбһјбһҖбҹ’бһ“бһ»бһ„ Kubernetes бһҠбҹӮбһӣбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹҶбһ–бҹҒбһүбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”
- **бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ:** бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ NIM microservices бһ“бҹ…бһӣбһҫбһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бҹ”
- **API бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ:** NIM бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“ API бһҠбҹӮбһӣбһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳ бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбһјбһ…бһҮбһ¶бһ…бһ¶бһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһҡбһҮбһҮбҹӮбһҖ, бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ AI бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”

NIM бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғ NVIDIA AI Enterprise бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһӣбҹ’бһ”бһҝбһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹҶбһҡбһј AI бһҠбҹ„бһҷбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһңбһ¶бһҡбһҸбҹӢбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһӣбһҫ GPU бһҡбһ”бһҹбҹӢ NVIDIAбҹ”

бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“ бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„ Phi-3 бһ¬бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҺбһ¶бһҳбһҪбһҷбһҠбһјбһ…бһҮбһ¶ GPT-3 бһӮбһәбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷ бһ¬бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹӮбһӣбһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бҹ” бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһ“бһјбһңбһҹбҹҶбһҺбһҪбһҡбһ¬бһ”бһҡбһ·бһ”бһ‘бһ‘бҹ… Phi-3 бһңбһ¶бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһҹбһҡбһҹбҹғбһ”бҹ’бһҡбһҹбһ¶бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһ”бһҹбҹӢбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ”бҹҶбһ•бһ»бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһӣбҹҶбһ“бһ¶бҹҶ бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ”

**Hugging Face Transformer**
Hugging Face Transformers бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹ’бһҡбһҳбһҹбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· (NLP) бһ“бһ·бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһўбҹҶбһ–бһёбһңбһ¶бһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ:

1. **бһӮбҹҶбһҡбһјбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбһ»бһ“**: бһңбһ¶бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһӮбҹҶбһҡбһјбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбһ»бһ“бһҡбһ¶бһ”бҹӢбһ–бһ¶бһ“бҹӢбһӮбҹҶбһҡбһј бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘, бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ“бҹғбһўбһ„бҹ’бһӮбһ—бһ¶бһ–, бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡ, бһҖбһ¶бһҡбһҹбһ„бҹ’бһҒбҹҒбһ”, бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ”

2. **бһҖбһ¶бһҡбһўбһ“бһ»бһӣбҹ„бһҳбһҹбһ¶бһҖбһӣбһҖбһҳбҹ’бһҳ Framework**: бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ framework бһ…бҹ’бһҡбһҫбһ“бһҠбһјбһ…бһҮбһ¶ PyTorch, TensorFlow, бһ“бһ·бһ„ JAXбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӮбҹҶбһҡбһјбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ framework бһҳбһҪбһҷ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ framework бһҳбһҪбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

3. **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһёбһӣбҹ’бһӮбҹ„бһЎбһјбһҠ (Multimodal)**: бһӣбһҫбһҹбһ–бһё NLP, Hugging Face Transformers бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбҹҒбһҡбҹ„бһӮбһңбһ·бһ‘бҹ’бһҷбһ¶ (бһ§. бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–, бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһңбһҸбҹ’бһҗбһ») бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹҶбһӣбҹҒбһ„ (бһ§. бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҹбҹҶбһЎбҹҒбһ„, бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбҹҶбһЎбҹҒбһ„) бһ•бһ„бһҠбҹӮбһҡбҹ”

4. **бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ**: бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһң API бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһ”бһ„бҹ’бһҡбҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһӮбҹҶбһҡбһјбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҹбҹ’бһҡбһҪбһӣ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһүбҹ”

5. **бһҹбһ бһӮбһҳбһ“бҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“**: Hugging Face бһҳбһ¶бһ“бһҹбһ бһӮбһҳбһ“бҹҚбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһү, бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһ¶бһ“бһ–бҹҒбһүбһӣбҹҒбһүбҹ”

**ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GenAI**

[ONNX Runtime](https://github.com/microsoft/onnxruntime-genai?WT.mc_id=academic-105485-koreyst) бһӮбһәбһҮбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһҶбҹ’бһ„бһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бҹ” ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI бһ”бһ„бҹ’бһҖбһҫбһҸ (GENAI) бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ…бһҠбҹӮбһӣбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҡбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

## ONNX Runtime бһҮбһ¶бһўбҹ’бһңбһё?
ONNX Runtime бһӮбһәбһҮбһ¶вҖӢбһӮбһҳбҹ’бһҡбҹ„бһ„вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёвҖӢбһҳбһҪбһҷвҖӢбһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸвҖӢбһІбҹ’бһҷвҖӢбһҳбһ¶бһ“вҖӢбһҖбһ¶бһҡвҖӢбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“вҖӢбһҳбһ¶бһ“вҖӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–вҖӢбһҒбҹ’бһ–бһҹбҹӢвҖӢбһҶбҹ’бһ„бһ¶бһҷвҖӢбһӣбһҫвҖӢбһӮбҹҶбһҡбһј Machine Learningбҹ” бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ Open Neural Network Exchange (ONNX) бһҠбҹӮбһӣбһҮбһ¶вҖӢбһҹбҹ’бһҠбһ„бҹӢбһҠбһ¶бһҡвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһ“бһ·бһҳбһ·бһҸбҹ’бһҸвҖӢбһҡбһ”бһҹбҹӢвҖӢбһӮбҹҶбһҡбһј Machine Learningбҹ” ONNX Runtime бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһ”бһ¶бһ“бһӣбһҝбһ“бһҠбһӣбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷ бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһ–бһё framework бһҡбҹҖбһ“бһҮбҹ’бһҡбҹ…бһҠбһјбһ…бһҮбһ¶ PyTorch бһ“бһ·бһ„ TensorFlow/Keras бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Machine Learning бһ…бһҳбҹ’бһҡбһ„бҹӢбһҠбһјбһ…бһҮбһ¶ scikit-learn, LightGBM, XGBoost бҹ” ONNX Runtime бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһҮбҹҶбһ бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ бһ•бҹ’бһ“бҹӮбһҖбһ”бһҫбһҖбһ”бһҡ бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹҶбһ–бһ„бҹӢбһҖбһҡбһҺбҹҚбһ‘бһ“бҹӢбһ бҹ„бҹҮбһҺбҹҚ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһ”бҹҶбһӣбҹӮбһ„бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ“бҹ…бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бҹ”

## AI бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһәбһҮбһ¶бһўбҹ’бһңбһё?
AI бһ”бһ„бҹ’бһҖбһҫбһҸ (Generative AI) бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҗбҹ’бһҳбһёбҹ— бһҠбһјбһ…бһҮбһ¶ бһўбһҸбҹ’бһҗбһ”бһ‘ бһҡбһјбһ”бһ—бһ¶бһ– бһ¬бһҸбһ“бҹ’бһҸбҹ’бһҡбһё бһҸбһ¶бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҡбһҪбһҳбһҳбһ¶бһ“бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҠбһјбһ…бһҮбһ¶ GPT-3 бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҠбһјбһ…бһҮбһ¶ Stable Diffusionбҹ” бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GenAI бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһңбһҠбҹ’бһҸ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹүбһ¶бһ“бҹӢбһҹбҹ’бһҳбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ ONNX Runtime, бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһ“бҹ’бһ‘бһ» logits, бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹҶбһҡбһј бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ¶бһҡбһ–бһ·бһҸбҹ’бһҸ KV cacheбҹ”

## ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI
ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI бһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбҹ’бһҗбһҸбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ ONNX Runtime бһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸбҹ” бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһңбҹҒбһ‘бһ·бһҖбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ–** бһңбһ¶бһҡбһҸбҹӢбһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҡбһҪбһҳбһҳбһ¶бһ“ Windows, Linux, macOS, Android бһ“бһ·бһ„ iOSбҹ”
- **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ:** бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹ’бһ”бһёбҹ—бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ LLaMA, GPT-Neo, BLOOM бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”
- **бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–:** бһңбһ¶бһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹҶбһ–бһ„бҹӢбһҖбһҡбһҺбҹҚбһ‘бһ“бҹӢбһ бҹ„бҹҮбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶ NVIDIA GPUs, AMD GPUs бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”
- **бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫ:** бһңбһ¶бһ•бҹ’бһҠбһӣбҹӢ API бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘ бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹ„бһҷбһҖбһјбһҠбһҸбһ·бһ…бһҸбһҪбһ…бҹ”
- бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ¶бһ…бһ бҹ…бһҳбһ’бҹ’бһҷбҹ„бһ”бһ¶бһҷ generate() бһ“бҹ…бһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбҹ’бһ–бһҹбҹӢ бһ¬бһҡбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбһҳбһҪбһҖбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһјбһҡ бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһҪбһўбһҖбҹ’бһҹбһҡбһҳбҹ’бһҠбһ„бһҳбһҪбһҷ бһ“бһ·бһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһјбһҡбһ“бҹ„бҹҮбһҖбҹҸбһ”бһ¶бһ“бҹ”
- ONNX runtime бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ greedy/beam бһ“бһ·бһ„ TopP, TopK бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҸбһҪбһўбһҖбҹ’бһҹбһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһ“бҹ’бһ‘бһ» logits бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһҠбһјбһ…бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһ–бһ·бһ“бҹ’бһ‘бһ»бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸбҹ” бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

## бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ
бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ ONNX Runtime бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ GENAI бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

### бһҠбҹҶбһЎбһҫбһ„ ONNX Runtime:
```Python
pip install onnxruntime
```
### бһҠбҹҶбһЎбһҫбһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ AI бһ”бһ„бҹ’бһҖбһҫбһҸ:
```Python
pip install onnxruntime-genai
```

### бһҡбһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбҹ– бһ“бҹҒбҹҮбһҮбһ¶бһӮбҹҶбһҡбһјбһӮбһ“бҹ’бһӣбһ„бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ Pythonбҹ–
```Python
import onnxruntime_genai as og

model = og.Model('path_to_your_model.onnx')

tokenizer = og.Tokenizer(model)

input_text = "Hello, how are you?"

input_tokens = tokenizer.encode(input_text)

output_tokens = model.generate(input_tokens)

output_text = tokenizer.decode(output_tokens)

print(output_text) 
```
### бһӮбҹҶбһҡбһјбһ”бһ„бҹ’бһ бһ¶бһүбҹ– бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ ONNX Runtime GenAI бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ… Phi-3.5-Vision


```python

import onnxruntime_genai as og

model_path = './Your Phi-3.5-vision-instruct ONNX Path'

img_path = './Your Image Path'

model = og.Model(model_path)

processor = model.create_multimodal_processor()

tokenizer_stream = processor.create_stream()

text = "Your Prompt"

prompt = "<|user|>\n"

prompt += "<|image_1|>\n"

prompt += f"{text}<|end|>\n"

prompt += "<|assistant|>\n"

image = og.Images.open(img_path)

inputs = processor(prompt, images=image)

params = og.GeneratorParams(model)

params.set_inputs(inputs)

params.set_search_options(max_length=3072)

generator = og.Generator(model, params)

while not generator.is_done():

    generator.compute_logits()
    
    generator.generate_next_token()

    new_token = generator.get_next_tokens()[0]
    
    output = tokenizer_stream.decode(new_token)
    
    print(tokenizer_stream.decode(new_token), end='', flush=True)

```
