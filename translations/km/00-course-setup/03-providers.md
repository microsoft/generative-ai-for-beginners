- `hf` - бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһүбҹ’бһүбһ¶ Hugging Face
- `githubmodels` - бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бҹҶбһҺбһ»бһ…бһҷбҹ„бһ„бһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry бһ“бһ·бһ„бһҹбҹ„бһҡ (GitHub Models бһ“бһ№бһ„бһ•бһ”бҹӢбһ“бҹ…бһ…бһ»бһ„бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бҹўбҹ бҹўбҹҰ)

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһҪбһҷ, бһӮбҹ’бһҳбһ¶бһ“, бһ¬бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбҹ” бһҖбһ¶бһҡбһ„бһ¶бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ”бҹ’бһҡбһ¶бһ”бҹӢбһҒбһ»бһҹбһҗбһ¶ бһ“бҹ…бһ–бҹҒбһӣбһҒбҹ’бһңбҹҮбһҹбһүбҹ’бһүбһ¶бһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбҹ”

## бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ `.env`

бһҷбһҫбһ„бһҹбһ“бҹ’бһҷбһ¶бһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ¶бһ“бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҒбһ¶бһ„бһӣбһҫ бһ бһҫбһҷбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҹбһүбҹ’бһүбһ¶бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң (API_KEY бһ¬ token) бһҡбһҪбһ…бһ бһҫбһҷбҹ” бһҠбһӣбҹӢAzure OpenAI бһҷбһҫбһ„бһҹбһ“бҹ’бһҷбһ¶бһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбһҪбһҷбһ бһҫбһҷбһ“бҹғбһҹбҹҒбһңбһ¶бһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳ Azure OpenAI (endpoint) бһҠбҹ„бһҷбһҳбһ¶бһ“бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ GPT бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶бҹ”

бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһәбһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ **бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ** бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҠбһҫбһҳбһҜбһҖбһҹбһ¶бһҡ `.env.copy` бһҳбһҪбһҷбһҠбҹӮбһӣбһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

   ```bash
   # бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢ OpenAI
   OPENAI_API_KEY='<add your OpenAI API key here>'

   ## Azure OpenAI бһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry
   ## (бһҹбҹҒбһңбһ¶ Azure OpenAI бһҘбһЎбһјбһңбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғ Microsoft Foundry: https://ai.azure.com)
   AZURE_OPENAI_API_VERSION='2024-10-21' # бһҖбҹҶбһҺбһҸбҹӢбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҡбһҪбһ…бһ…бҹҶбһ–бҹ„бҹҮ! (бһҖбҹҶбһҺбҹӮ API GA бһҠбҹӮбһӣбһҹбҹ’бһҗбһ·бһҸбһҹбҹ’бһҗбҹҒбһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“)
   AZURE_OPENAI_API_KEY='<add your Foundry resource key here>'
   AZURE_OPENAI_ENDPOINT='<add your Foundry resource endpoint here, e.g. https://<resource-name>.openai.azure.com>'
   AZURE_OPENAI_DEPLOYMENT='<add your chat completion model deployment name here, e.g. gpt-4o-mini>'
   AZURE_OPENAI_EMBEDDINGS_DEPLOYMENT='<add your embeddings model deployment name here, e.g. text-embedding-3-small>'

   ## бһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry (бһҖбһ¶бһҸбһ¶бһЎбһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣбһ…бҹ’бһҡбһҫбһ“бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢ бһҮбҹҶбһ“бһҪбһҹбһҳбҹүбһјбһҠбҹӮбһӣ GitHub бһҠбҹӮбһӣбһ“бһ№бһ„бһ…бһ¶бһҖбһ…бҹҒбһүбһ…бһ»бһ„бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бҹўбҹ бҹўбҹҰ)
   AZURE_INFERENCE_ENDPOINT='<add your Microsoft Foundry project endpoint here>'
   AZURE_INFERENCE_CREDENTIAL='<add your Microsoft Foundry Models API key here>'

   ## Hugging Face
   HUGGING_FACE_API_KEY='<add your HuggingFace API or token here>'
   ```

2. бһ…бһҳбҹ’бһӣбһ„бһҜбһҖбһҹбһ¶бһҡбһ“бҹ„бҹҮ бһ‘бҹ…бһҮбһ¶ `.env` бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ” бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“ _gitignore_ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбҹ”

   ```bash
   cp .env.copy .env
   ```

3. бһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһӣбҹғ (бһ”бҹ’бһҸбһјбһҡбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҠбһ¶бһҖбҹӢбһҸбһҳбҹ’бһӣбҹғбһҒбһ¶бһ„бһҹбҹ’бһҠбһ¶бҹҶбһ“бҹғ `=`) бһҠбһјбһ…бһ”бһ¶бһ“бһ–бһҺбҹҢбһ“бһ¶бһІбҹ’бһҷбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

4. (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GitHub Codespaces бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҮбһ¶ _Codespaces бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ_ бһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһ”бһ¶бһ“бҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹ„бҹҮ бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢ .env бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһ‘бҹҒбҹ” **бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ бһҹбһјбһҳбһ…бҹҶбһҺбһ¶бҹҶбһҗбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ“бҹҒбҹҮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹӮбһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ GitHub Codespaces бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”** бһўбҹ’бһ“бһҖбһ“бҹ…бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһҖбҹҶбһҺбһҸбҹӢ .env бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹ’бһҡбһҫ Docker Desktopбҹ”

бһҸбһҳбҹ’бһӣбҹғбһ…бҹҶбһҺбһ»бһ…вҖӢбһҷбҹ„бһ„ Azure OpenAI бһ“бһ·бһ„бһҹбҹ„бһҡбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңвҖӢбһҡбһҖбһғбһҫбһүвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ [Azure Portal](https://portal.azure.com?WT.mc_id=academic-105485-koreyst) бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮвҖӢбһҳбһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһ‘бһёбһ“бҹ„бҹҮбҹ”

1. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ [Azure Portal](https://portal.azure.com?WT.mc_id=academic-105485-koreyst)
1. бһ…бһ»бһ…бһӣбһҫбһҮбһҳбҹ’бһҡбһҫбһҹ **Keys and Endpoint** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһёбһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„бҹ”
1. бһ…бһ»бһ… **бһ”бһ„бҹ’бһ бһ¶бһүбһҹбҹ„бһҡ** - бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһғбһҫбһү KEY 1, KEY 2 бһ“бһ·бһ„ Endpointбҹ”

1. бһ…бһ»бһ…бһӣбһҫбһҮбһҳбҹ’бһҡбһҫбһҹ **Model deployments** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһёбһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“ Azure OpenAIбҹ”
1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бҹҶбһ–бҹҗбһҡбһӮбҹ„бһӣбһҠбҹ… бһ…бһ»бһ… **бһ‘бҹ…бһҖбһ¶бһ“бҹӢ Microsoft Foundry portal** (бһ¬ **бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ** бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ‘бҹ…бһҸбһ¶бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ’бһ“бһ’бһ¶бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ)

бһ“бҹҒбҹҮбһ“бһ№бһ„бһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбҹҶбһ–бһјбһӣбһ•бһҸбһҗбһӣ Microsoft Foundry бһҠбҹӮбһӣбһ“бҹ…бһ‘бһёбһ“бҹ„бҹҮбһҷбһҫбһ„бһ“бһ№бһ„бһҡбһҖбһғбһҫбһүбһҸбһҳбҹ’бһӣбҹғбһҮбҹҶбһ“бһҪбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһ”бһ¶бһ“бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Azure OpenAI: бһ–бһёбһҖбҹҶбһ–бһјбһӣбһ•бһҸбһҗбһӣ Microsoft Foundry

1. бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ [Microsoft Foundry portal](https://ai.azure.com?WT.mc_id=academic-105485-koreyst) **бһ–бһёбһ’бһ“бһ’бһ¶бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ** бһҠбһјбһ…бһ”бһ¶бһ“бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҒбһ¶бһ„бһӣбһҫбҹ”
1. бһ…бһ»бһ…бһ•бҹ’бһ‘бһ¶бҹҶбһ„ **Deployments** (бһ”бһүбҹ’бһҮбһёбһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„) бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
1. бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ‘бҹҒ бһ”бҹ’бһҡбһҫ **Deploy model** бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһңбһ¶бһ–бһё [бһғбҹ’бһӣбһ¶бҹҶбһ„бһҳбҹүбһјбһҠбҹӮбһӣ](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst)бҹ”
1. бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ _бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘_ вҖ” бһҷбһҫбһ„бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбҹ– **gpt-4o-mini**
1. бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ _бһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ”бһ‘_ вҖ” бһҷбһҫбһ„бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ **text-embedding-3-small**

бһҹбҹ„бһҡ API OpenAI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [бһӮбһҺбһ“бһё OpenAI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](https://platform.openai.com/api-keys?WT.mc_id=academic-105485-koreyst)бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ„бһҡбһҳбһҪбһҷбһ‘бҹҒ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһё бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ„бһҡ API бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбһ¶бһ“бһҹбҹ„бһҡ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹҶбһ–бҹҒбһүбһўбһҗбҹҒбһҡ `OPENAI_API_KEY` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `.env`бҹ”

бһҹбһүбҹ’бһүбһ¶ Hugging Face бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһјбһ”бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһҡбҹ„бһҳ [Access Tokens](https://huggingface.co/settings/tokens?WT.mc_id=academic-105485-koreyst)бҹ” бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ¬бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ–бһҪбһҖбһңбһ¶бһҠбҹ„бһҷбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбҹ” бһҮбһ¶бһҮбҹҶбһ“бһҪбһҹ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһүбҹ’бһүбһ¶бһҗбҹ’бһҳбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮ бһ бһҫбһҷбһ…бһҳбҹ’бһӣбһ„бһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `.env` бһҖбҹ’бһҡбҹ„бһҳбһўбһҗбҹҒбһҡ `HUGGING_FACE_API_KEY` бҹ” _бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ:_ бһңбһ¶бһҮбһ¶бһҹбһүбҹ’бһүбһ¶бһҠбҹӮбһӣбһҳбһ·бһ“бһҳбҹӮбһ“бһҹбҹ„бһҡ API бһҠбҹӮбһҡбһ–бһёбһ–бҹҒбһӣбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһҷбһҫбһ„бһҖбҹҶбһ–бһ»бһ„бһҡбһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбһ¶бһҖбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶бҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry: бһ–бһёбһҖбҹҶбһ–бһјбһӣбһ•бһҸбһҗбһӣ

> **бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ:** GitHub Models бһ“бһ№бһ„бһ•бһ”бҹӢбһ“бҹ…бһ…бһ»бһ„бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бҹўбҹ бҹўбҹҰбҹ” бһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Foundry бһҮбһ¶бһ…бһҳбҹ’бһӣбһҫбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ, бһ•бҹ’бһҸбһӣбҹӢбһұбһҖбһ¶бһҹбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ Azure AI Inference SDK / OpenAI SDK бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

1. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ [Microsoft Foundry](https://ai.azure.com?WT.mc_id=academic-105485-koreyst) бһ бһҫбһҷбһ”бһ„бҹ’бһҖбһҫбһҸ (бһ¬бһ”бһҫбһҖ) бһӮбһҳбҹ’бһҡбҹ„бһ„ Foundryбҹ”
1. бһҡбһ»бһҖбһҡбһҖ [бһғбҹ’бһӣбһ¶бҹҶбһ„бһҳбҹүбһјбһҠбҹӮбһӣ](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) бһ бһҫбһҷбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷ, бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ `gpt-4o-mini`бҹ”
1. бһ“бҹ…бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡ **бһҹбһ„бҹ’бһҒбҹҒбһ”** бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„ бһ…бһҳбҹ’бһӣбһ„ **бһ…бҹҶбһҺбһ»бһ…бһҷбҹ„бһ„** бһ“бһ·бһ„ **бһҹбҹ„бһҡ API**бҹ”
1. бһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһӣбҹғбһ…бҹҶбһҺбһ»бһ…бһҷбҹ„бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `AZURE_INFERENCE_ENDPOINT` бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһҹбҹ„бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `AZURE_INFERENCE_CREDENTIAL` бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `.env` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
