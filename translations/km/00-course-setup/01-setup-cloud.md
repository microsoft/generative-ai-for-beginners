# ការតំឡើងភពមេ ☁️ – GitHub Codespaces

**ប្រើមគ្គុទេសក៍នេះ ប្រសិនបើអ្នកមិនចង់ដំឡើងអ្វីមួយនៅក្នុងតំបន់របស់អ្នក។**  
Codespaces ផ្ដល់ជូនអ្នកនូវ VS Code ដែលដំណើរការតាមកម្មវិធីរុករកដោយឥតគិតថ្លៃ ជាមួយនឹងការតំឡើងប្លែកៗទាំងអស់រួចជាស្រេច។

---

|---------|----------------------|
| ✅ មិនចាំបាច់ដំឡើងអ្វីទេ | ដំណើរការលើ Chromebook, iPad, កុំព្យូទ័រពិភពविद्यालय... |

> (មើល ▸ Command Palette ▸ *Codespaces: បញ្ឈប់ Codespace*)។

## 2. បង្កើត Codespace (ចុចមួយដង)

1. **Fork** ចម្លងបណ្ណាល័យនេះ (ប៊ូតុង **Fork** ខាងលើប៉ុន្មានខាងស្តាំ)។  
2. នៅក្នុង fork របស់អ្នក ចុច **Code ▸ Codespaces ▸ បង្កើត codespace នៅលើ main**។  
   ![ialog showing buttons to create a codespace](../../../translated_images/km/who-will-pay.4c0609b1c7780f44.webp)

✅ បង្អួច VS Code នៅក្នុងកម្មវិធីរុករកបើកឡើង ហើយធុង dev ចាប់ផ្តើមកសាង។  
វាចំណាយពេលប្រហែល **~2 នាទី** នៅពេលដំបូង។

## 3. បន្ថែមកូនសោ API របស់អ្នក (វិធីសុវត្ថិភាព)

### ជម្រើស A Codespaces Secrets — សូមណែនាំ

1. ⚙️ រូបតំណាងកង់ -> Command Pallete -> Codespaces : គ្រប់គ្រងឯកជនថ្មី -> បន្ថែមឯកជនថ្មីមួយ។  
2. ឈ្មោះ: OPENAI_API_KEY  
3. តម្លៃ: វាយបញ្ចូលកូនសោរបស់អ្នក → បន្ថែមឯកជន

នេះគឺគ្រប់គ្រាន់—កូដរបស់យើងនឹងទាញយកវាដោយស្វ័យប្រវត្តិ។

### ជម្រើស B ឯកសារ .env (ប្រសិនបើអ្នកត្រូវការពិតប្រាកដមួយ)

```bash
cp .env.copy .env
code .env         # បញ្ចូល OPENAI_API_KEY=សោររបស់អ្នកនៅទីនេះ
```
