Codespaces бһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһўбҹ’бһ“бһҖбһ“бһјбһңVS Code бһ“бҹ…бһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡбһҠбҹ„бһҷбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҡбһҪбһ…бһ бһҫбһҷбҹ”

## 1.  бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ Codespaces?

| бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ | бһҳбһ¶бһ“бһ“бҹҗбһҷбһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ… |
|---------|----------------------|
| вң… бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҠбҹҶбһЎбһҫбһ„ | бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫ Chromebook, iPad, бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ–бһ·бһҹбһ¶бһӣбһёбһҹбһ¶бһӣбһ¶вҖҰ |
| вң… бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ | Python 3, Node.js, .NET, Java бһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ“бҹҒбҹҮбһҡбһҪбһ…бһ бһҫбһҷ |
| вң… бһ•бҹ’бһҸбһӣбҹӢбһўбһҸбҹ’бһҡбһ¶бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ | бһӮбһҺбһ“бһёбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“ **120 бһҳбҹүбҹ„бһ„бһҖбһјбһҡ/ 60 GB-бһҳбҹүбҹ„бһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҒбҹӮ** |

> бһҡбһ¶бһҡбһ¶бҹҶбһ„бһўбһҸбҹ’бһҡбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбһ»бһҒбһ—бһ¶бһ–бһӣбҹ’бһўбһҠбҹ„бһҷ **бһ”бһүбҹ’бһҲбһ”бҹӢ** бһ¬ **бһӣбһ»бһ”** codespaces бһҠбҹӮбһӣбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ  
> (бһҳбһҫбһӣ в–ё бһ”бһүбҹ’бһҮбһёбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ в–ё *Codespaces: Stop Codespace*).

## 2. бһ”бһ„бҹ’бһҖбһҫбһҸ Codespace (бһ”бһ¶бһӣбҹӢбһӮбҹ’бһҡбһ”бҹӢ)

1. **бһ…бһҳбҹ’бһӣбһ„** бһҡбһ»бһҖбһҡбһҖбһ“бҹҒбҹҮ (бһ”бҹҠбһјбһҸбһ»бһ„ **Fork** бһ“бҹ…бһӣбһҫбһҹбҹ’бһҸбһ¶бҹҶбһҒбһ¶бһ„бһӣбһҫ).  
2. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бһҳбҹ’бһӣбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ…бһ»бһ… **Code в–ё Codespaces в–ё Create codespace on main**.  
   ![Dialog showing buttons to create a codespace](../../../translated_images/km/who-will-pay.4c0609b1c7780f44.webp)

вң… бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹ’бһ‘бһ¶бҹҶбһ„ VS Code бһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбҹҶбһүбҹҗбһҡ бһ“бһ·бһ„бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбҹҶбһ–бһ»бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһ¶бһ„бһҹбһ„бҹӢбҹ”
бһ“бҹҒбҹҮбһўбһ¶бһ…бһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣ **~2 бһ“бһ¶бһ‘бһё** бһӣбһҫбһҠбһ„бһҠбҹҶбһ”бһјбһ„бҹ”

## 3. бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһјбһ“бһҹбҹ„ API бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһңбһ·бһ’бһёбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–)

### бһҮбһҳбҹ’бһҡбһҫбһҹ A Codespaces Secrets вҖ” бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ

1. вҡҷпёҸ бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһҖбһ„бҹӢ -> бһ”бһүбҹ’бһҮбһёбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶-> Codespaces : бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ user secret -> бһ”бһ“бҹ’бһҗбҹӮбһҳ secret бһҗбҹ’бһҳбһёбҹ”
2. бһҲбҹ’бһҳбҹ„бҹҮ: OPENAI_API_KEY
3. бһҸбһҳбҹ’бһӣбҹғ: бһ”бһ·бһ‘бһ”бһүбҹ’бһ…бһјбһӣбһҖбһјбһ“бһҹбҹ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҶ’ бһ”бһ“бҹ’бһҗбҹӮбһҳ secret

бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮвҖ”code бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ“бһ№бһ„бһҷбһҖбһңбһ¶бһҳбһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

### бһҮбһҳбҹ’бһҡбһҫбһҹ B бһҖбһ¶бһҡ .env file (бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ·бһҸ)

```bash
cp .env.copy .env
code .env         # бһ”бһүбҹ’бһ…бһјбһӣ OPENAI_API_KEY=бһҖбһјбһ“бһҹбҹ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ
```
