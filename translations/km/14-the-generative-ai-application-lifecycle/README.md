# бһҮбһёбһңбһ…бһҡбһҺбҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ”бһ„бҹ’бһҖбһҫбһҸ

бһҹбҹҶбһҺбһҪбһҡбһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбһәбһ—бһ¶бһ–бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бҹғбһҳбһ»бһҒбһ„бһ¶бһҡ AI бһҠбҹ’бһ”бһ·бһҸ AI бһӮбһәбһҮбһ¶бһ”бһҡбһ·бһ”бһ‘бһҖбҹҶбһ–бһ»бһ„бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһҷбҹүбһ¶бһ„бһҶбһ¶бһ”бҹӢбһҡбһ бҹҗбһҹ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҸбҹӮбһҳбһ¶бһ“бһ—бһ¶бһ–бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһӮбһҪбһҡбһҸбҹӮбһўбһ¶бһ…бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ“бһ·бһ„бһҡбһ№бһ„бһҳбһ¶бҹҶ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ“бһ№бһ„бһҸбһ¶бһҳбһҠбһ¶бһ“ бһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһңбһ¶бһ‘бҹ…бһҮбһ¶бһ“бһ·бһҡбһ“бҹ’бһҸбһҡбһ—бһ¶бһ–бҹ” бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһҮбһёбһңбһ…бһҡбһҺбҹҚ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһјбһӣбһҳбһҖбҹ”

бһҮбһёбһңбһ…бһҡбһҺбҹҚ AI бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбһәбһҮбһ¶бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһҺбҹҚбһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҠбһ¶бһ“бһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ”бһүбҹ’бһ…бҹҒбһү бһ“бһ·бһ„бһҗбҹӮбһ‘бһ¶бҹҶбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ”бһ„бҹ’бһҖбһҫбһҸбҹ” бһңбһ¶бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҖбҹҶбһҺбһҸбҹӢбһӮбҹ„бһӣбһҠбҹ…бһңбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһүбҹ’бһ бһ¶ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҖбҹҸбһҮбһҪбһҷбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһҹбҹ’бһҠбһ„бҹӢбһҠбһ¶бһҡбһўбҹҒбһҸбһ·бһҖ бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷ бһ“бһ·бһ„бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ—бһ¶бһӮбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҠбҹ„бһҷбһ’бҹ’бһңбһҫбһҸбһ¶бһҳбһҮбһёбһңбһ…бһҡбһҺбҹҚ AI бһ”бһ„бҹ’бһҖбһҫбһҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бһ¶бҹҶбһӮбһ»бһҺбһҸбһҳбҹ’бһӣбҹғ бһ“бһ·бһ„бһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ“бһ·бһҡбһ“бҹ’бһҸбһҡбһ—бһ¶бһ–бҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„:

- бһҷбһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„ Paradigm бһ–бһё MLOps бһ‘бҹ… LLMOps
- бһҮбһёбһңбһ…бһҡбһҺбҹҚ LLM
- бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһёбһңбһ…бһҡбһҺбҹҚ
- бһҖбһ¶бһҡбһңбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ¶бһҖбҹӢбһ”бҹӮбһҖбһҮбһёбһңбһ…бһҡбһҺбҹҚ

## бһҷбһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„ Paradigm бһ–бһё MLOps бһ‘бҹ… LLMOps

LLM бһӮбһәбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹғбһҸбһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһңбһ¶бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ бһҫбһҷбһ•бһӣбһ·бһҸбһҡбһҠбҹ’бһӢбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбҹҶбһҺбһ¶бһ…бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһү бһ“бһ·бһ„бһ…бҹ’бһҡбһҖбһ…бҹҒбһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ“бһ·бһ„ Machine Learning бһ…бһ¶бһҹбҹӢбҹ—бҹ”

бһҠбҹ„бһҷбһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ•бһӣбһ·бһҸ Paradigm бһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһўбҹ„бһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҶбһ¶бһ”бҹӢбһҡбһ бҹҗбһҹ бһ бһҫбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” бһҷбһҫбһ„бһўбһ¶бһ…бһ”бҹӮбһ„бһ…бҹӮбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһ…бһ¶бһҹбҹӢбҹ—бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё "ML Apps" бһ бһҫбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҗбҹ’бһҳбһёбҹ—бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё "GenAI Apps" бһ¬бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮ "AI Apps" бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһ“бҹ„бҹҮбҹ” бһ“бҹҒбҹҮбһ”бҹҶбһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ…бҹ’бһҡбһҫбһ“бһңбһ·бһ’бһё бһҹбһјбһҳбһҳбһҫбһӣбһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

![бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ” LLMOps бһ“бһ·бһ„ MLOps](../../../translated_images/km/01-llmops-shift.29bc933cb3bb0080.webp)

бһҹбһјбһҳбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҗбһ¶ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ LLMOps бһҷбһҫбһ„бһ•бҹ’бһҸбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ‘бҹ…бһӣбһҫбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶вҖӢбһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ”бҹ’бһҡбһҫ "Models-as-a-Service" бһ бһҫбһҷбһӮбһ·бһҸбһӮбһјбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһӣбһҫбһҖбһҺбҹ’бһҸбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ”бһ…бҹҶбһңбһ·бһ’бһ¶бһ“бһҳбһ¶бһҸбҹ’бһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

- бһӮбһ»бһҺбһ—бһ¶бһ–бҹ– бһӮбһ»бһҺбһ—бһ¶бһ–бһ…бһҳбҹ’бһӣбһҫбһҷ
- бһўбҹҶбһ–бһҫбһўбһ¶бһҖбҹ’бһҡбһҖбҹӢбҹ– AI бһҳбһ¶бһ“бһҶбһ“бҹ’бһ‘бҹҮбһ”бһҡбһ·бһ…бҹ’бһ…бҹҗбһҷ
- бһ—бһ¶бһ–бһҹбҹ’бһҳбҹ„бҹҮбһҸбҹ’бһҡбһ„бҹӢбҹ– бһ–бҹ’бһҡбһҳбһ¶бһ“бһӣбһҫбһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһҫбһҷ (бһҳбһ¶бһ“бһ“бҹҗбһҷ? бһҸбһҫбһңбһ¶бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң?)
- бһҗбҹ’бһӣбҹғбҹ– бһҗбһңбһ·бһҖбһ¶бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ
- бһӣбҹ’бһ”бһҝбһ“бҹ– бһ–бҹҒбһӣбһҳбһ’бҹ’бһҷбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҸбһҪбһўбһҖбҹ’бһҹбһҡ

## бһҮбһёбһңбһ…бһҡбһҺбҹҚ LLM

бһҮбһ¶бһӣбһҳбҹ’бһўбһ·бһҸбһҠбҹҶбһ”бһјбһ„ бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһӣбҹӢбһ–бһёбһҮбһёбһңбһ…бһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ бһҹбһјбһҳбһҳбһҫбһӣбһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһ¶бһҡбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

[Azure AI Platform](https://azure.microsoft.com/solutions/ai/?WT.mc_id=academic-105485-koreyst) бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ [Microsoft Foundry](https://ai.azure.com/?WT.mc_id=academic-105485-koreyst)бҹ” Microsoft Foundry (бһҲбҹ’бһҳбҹ„бҹҮбһ…бһ¶бһҹбҹӢ Azure AI Studio) бһӮбһәбһҮбһ¶бһ•бҹ’бһ‘бҹҮбһ”бһҺбҹ’бһҠбһ¶бһүбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбҹүбһјбһҠбҹӮбһӣ бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһҪбһҳ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ”бҹ’бһҡбһҫбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ UI бһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ бһ“бһ·бһ„бһҮбһҳбҹ’бһҡбһҫбһҹ SDK/CLI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Code-Firstбҹ”

![бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣ Azure AI](../../../translated_images/km/04-azure-ai-platform.80203baf03a12fa8.webp)

Azure AI бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ бһӮбһҳбҹ’бһҡбҹ„бһ„ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбҹүбһ·бһ…бһ‘бҹҗбһҡ бһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

![LLMOps бһҮбһ¶бһҳбһҪбһҷ Azure AI](../../../translated_images/km/05-llm-azure-ai-prompt.a5ce85cdbb494bdf.webp)

бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһё Proof-of-Concept (POC) бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹҶбһ бҹҶбһ’бҹҶбһҮбһ¶бһҳбһҪбһҷ PromptFlow:

- бһҡбһ…бһ“бһ¶ бһ“бһ·бһ„бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһё VS Code бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ„бҹ’бһ бһ¶бһү бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһ»бһҺбһ—бһ¶бһ– AI бһҠбҹ„бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”
- бһ”бҹ’бһҡбһҫ Microsoft Foundry бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷ cloud, бһ”бһүбҹ’бһ…бһјбһ“ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӣбһҝбһ“бҹ”
