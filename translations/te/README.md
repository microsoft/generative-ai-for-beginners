# ప్రారంభికుల కోసం జనరేటివ్ AI (సంస్కరణ 3) - ఒక కోర్సు

మైక్రోసాఫ్ట్ క్లౌడ్ అడ్వొకేట్స్ అందిస్తున్న 21 పాఠాల సమగ్ర కోర్సుతో జనరేటివ్ AI అప్లికేషన్లు నిర్మించడంలో ప్రాథమికాలు నేర్చుకోండి.

## 🌱 ప్రారంభం

ఈ కోర్సులో 21 పాఠాలు ఉన్నాయి. ప్రతి పాఠం దాని అంశాన్ని కవర్ చేస్తుంది కాబట్టి మీరు మీరు ఇష్టపడే ఎక్కడైనా ప్రారంభించవచ్చు!

పాఠాలు "Learn" లెసన్లు (జనరేటివ్ AI కాన్సెప్ట్‌ను వివరిస్తాయి) లేదా "Build" లెసన్లు (ఒక కాన్సెప్ట్ మరియు కోడ్ ఉదాహరణలు Python మరియు TypeScript లో చాలావరకు)గా గుర్తింపబడతాయి.

.NET డెవలపర్లు కొరకు [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) చూడండి!

ప్రతి పాఠం "Keep Learning" విభాగాన్ని కలిగి ఉంటుంది, అదనపు అధ్యయన సాధనాలతో.

## మీకు కావలసినవి
### ఈ కోర్సు కోడ్ నడిపించడానికి, మీరు ఈ ఒకటిని లేదా అంతకన్నా ఎక్కువ ఉపయోగించవచ్చు:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **పాఠాలు:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **పాఠాలు:** "githubmodels" (GitHub Models 2026 జూలై చివరన రిటైర్ అవుతుంది - దాని స్థానంలో Microsoft Foundry Models ఉపయోగించండి)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **పాఠాలు:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - మీ స్వంత పరికరంలో మోడల్స్‌ను పూర్తిగా ఆఫ్‌లైన్ నడపండి, క్లౌడ్ సబ్‌స్క్రిప్షన్ అవసరం లేదు
   
- Python లేదా TypeScript యొక్క ప్రాథమిక జ్ఞానం సహాయకం - \*పూర్తిగా కొత్తవారికి ఈ [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) మరియు [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) కోర్సులను చూడండి
- [GitHub ఖాతా](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) ఈ రిపోను మీ GitHub ఖాతాకు ఫోర్క్ చేసుకోవడానికి

మేము మీ అభివృద్ధి వాతావరణాన్ని సెట్ చేసుకోవడానికి సహాయం చేసే **[కోర్సు సెటప్](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** పాఠం రూపొందించాము.
