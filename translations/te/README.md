# అరంభ దశలో ఉన్న వారి కోసం జెనరేటివ్ AI (సంస్కరణ 3) - ఒక కోర్స్

మైక్రోసాఫ్ట్ క్లౌడ్ అడ్వకేట్స్ చేత రూపొందించిన మా 21 పాఠాల సమగ్ర కోర్సుతో జెనరేటివ్ AI అప్లికేషన్ల నిర్మాణంలోని మౌలికాంశాలను నేర్చుకోండి.

## 🌱 ప్రారంభించండి

ఈ కోర్సులో 21 పాఠాలు ఉన్నాయి. ప్రతి పాఠం తమ సొంత విషయాన్ని కవర్ చేస్తుంది కాబట్టి మీరు ఇష్టమైన దాని నుండి ప్రారంభించండి!

పాఠాలు "Learn" అనే పాఠాలు జెనరేటివ్ AI కాన్సెప్ట్ వివరించాయి లేదా "Build" పాఠాలు కాన్సెప్ట్ మరియు కోడ్ ఉదాహరణలను **Python** మరియు **TypeScript** లో (సాధ్యమైతే) వివరించాయి.

.NET డెవలపర్లకు [Generative AI for Beginners (.NET సంచిక)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) చూడండి!

ప్రతి పాఠంలో అదనపు అభ్యాస సాధనాలతో "Keep Learning" సెక్షన్ కూడా ఉంటుంది.

## మీకు అవసరం
### ఈ కోర్స్ కోడ్ నడిపించేందుకు, మీరు ఈ అవకాశాలు ఉపయోగించవచ్చు: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **పాఠాలు:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **పాఠాలు:** "githubmodels" (GitHub Models జూలై 2026 చివరలో రిటైర్ అవుతుంది - దానికి బదులు Microsoft Foundry Models ఉపయోగించండి)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **పాఠాలు:** "oai-assignment"

- Python లేదా TypeScript యొక్క ప్రాథమిక పరిజ్ఞానం ఉపయోగకరం - *మొత్తం కొత్తవారికి ఈ [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) మరియు [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) కోర్సులు చూడండి*
- మీ స్వంత GitHub అకౌంట్‌కు [ఇటీవల రిపోజిటరీని ఫోర్క్ చేయడానికి GitHub అకౌంట్](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) అవసరం
