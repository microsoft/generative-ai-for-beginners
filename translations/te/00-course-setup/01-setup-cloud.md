## 2.  ఒక కోడ్స్పేస్ సృష్టించండి (ఒక క్లిక్)  

1. ఈ రిపోను **Fork** చేసుకోండి (గది-కుడి వైపు ఉన్న **Fork** బటన్).  
2. మీ ఫోర్క్ లో, **Code ▸ Codespaces ▸ Create codespace on main** క్లిక్ చేయండి.  
   ![Dialog showing buttons to create a codespace](../../../translated_images/te/who-will-pay.4c0609b1c7780f44.webp)  

✅ ఒక బ్రౌజర్ VS కోడ్ విండో తెరుచుకుంటుంది మరియు డెవ్ కంటెయినర్ నిర్మాణం ప్రారంభమవుతుంది.  
ఇది మొదటి సారి **~2 నిమిషాలు** పడుతుంది.  

## 3. మీ API కీని జోడించండి (సురక్షితమైన విధానం)  

### ఎంపిక A కోడ్స్పేసెస్ సీక్రెట్స్ — సిఫారసు చేయబడింది  

1. ⚙️ గేర్ చిహ్నం -> కమాండ్ ప్యాలెట్ -> Codespaces : Manage user secret -> కొత్త సీక్రెట్ జోడించండి.  
2. పేరు: OPENAI_API_KEY  
3. విలువ: మీ కీను పేస్ట్ చేసి → Add secret  

అంతే—మన కోడ్ ఇది ఆటోమేటిక్‌గా పిక్ చేస్తుంది.  

### ఎంపిక B .env ఫైల్ (మీకు నిజంగా అవసరమైతే)  

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=మీ_కీ_ఇక్కడ_పూరించండి
```
