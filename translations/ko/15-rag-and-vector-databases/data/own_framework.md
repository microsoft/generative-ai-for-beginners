# 신경망 소개. 다층 퍼셉트론

이전 섹션에서는 가장 간단한 신경망 모델인 단층 퍼셉트론, 즉 선형 이진 분류 모델에 대해 배웠습니다.

이번 섹션에서는 이 모델을 더 유연한 프레임워크로 확장하여 다음을 수행할 수 있도록 합니다:

* 이진 분류 외에 **다중 클래스 분류** 수행
* 분류 외에 **회귀 문제** 해결
* 선형으로 분리할 수 없는 클래스 분리

또한 다양한 신경망 아키텍처를 구축할 수 있도록 파이썬에서 자체 모듈형 프레임워크를 개발할 것입니다.

## 기계 학습의 공식화

기계 학습 문제를 공식화하는 것부터 시작해 봅시다. 훈련 데이터셋 **X**와 레이블 **Y**가 있다고 가정하고, 가장 정확한 예측을 할 수 있는 모델 *f*를 구축해야 합니다. 예측의 품질은 **손실 함수** ℒ로 측정됩니다. 다음과 같은 손실 함수가 자주 사용됩니다:

* 숫자를 예측해야 하는 회귀 문제의 경우, **절대 오차** ∑<sub>i</sub>|f(x<sup>(i)</sup>)-y<sup>(i)</sup>| 또는 **제곱 오차** ∑<sub>i</sub>(f(x<sup>(i)</sup>)-y<sup>(i)</sup>)<sup>2</sup>를 사용할 수 있습니다.
* 분류의 경우, **0-1 손실** (모델의 **정확도**와 본질적으로 동일) 또는 **로지스틱 손실**을 사용합니다.

단층 퍼셉트론의 경우, 함수 *f*는 선형 함수 *f(x)=wx+b*로 정의되었습니다 (여기서 *w*는 가중치 행렬, *x*는 입력 특성 벡터, *b*는 바이어스 벡터입니다). 다양한 신경망 아키텍처에 따라 이 함수는 더 복잡한 형태를 취할 수 있습니다.

> 분류의 경우, 네트워크 출력으로 해당 클래스의 확률을 얻는 것이 바람직한 경우가 많습니다. 임의의 숫자를 확률로 변환하기 위해 (즉, 출력을 정규화하기 위해) 종종 **소프트맥스** 함수 σ를 사용하며, 함수 *f*는 *f(x)=σ(wx+b)*가 됩니다.

위의 *f* 정의에서 *w*와 *b*는 **매개변수** θ=⟨*w,b*⟩라고 불립니다. 데이터셋 ⟨**X**,**Y**⟩가 주어지면, 매개변수 θ의 함수로 전체 데이터셋에 대한 전체 오류를 계산할 수 있습니다.

> ✅ **신경망 훈련의 목표는 매개변수 θ를 변경하여 오류를 최소화하는 것입니다**

## 경사 하강 최적화

함수 최적화의 잘 알려진 방법으로 **경사 하강법**이 있습니다. 이 방법은 매개변수에 대한 손실 함수의 도함수(다차원 경우 **그래디언트**라고 함)를 계산하고, 오류가 감소하도록 매개변수를 변경하는 것입니다. 이는 다음과 같이 공식화될 수 있습니다:

* 초기 매개변수를 임의의 값 w<sup>(0)</sup>, b<sup>(0)</sup>로 설정합니다.
* 다음 단계를 여러 번 반복합니다:
    - w<sup>(i+1)</sup> = w<sup>(i)</sup>-η∂ℒ/∂w
    - b<sup>(i+1)</sup> = b<sup>(i)</sup>-η∂ℒ/∂b

훈련 중에는 최적화 단계가 전체 데이터셋을 고려하여 계산되어야 합니다 (손실은 모든 훈련 샘플을 통해 합산하여 계산됨을 기억하세요). 그러나 실제로는 **미니배치**라고 불리는 데이터셋의 작은 부분을 가져와 데이터의 하위 집합을 기반으로 그래디언트를 계산합니다. 매번 무작위로 하위 집합을 가져오기 때문에 이러한 방법을 **확률적 경사 하강법** (SGD)이라고 합니다.

## 다층 퍼셉트론과 역전파

위에서 본 바와 같이, 단층 네트워크는 선형으로 분리 가능한 클래스를 분류할 수 있습니다. 더 풍부한 모델을 구축하기 위해 네트워크의 여러 층을 결합할 수 있습니다. 수학적으로 이는 함수 *f*가 더 복잡한 형태를 가지며 여러 단계로 계산된다는 것을 의미합니다:
* z<sub>1</sub>=w<sub>1</sub>x+b<sub>1</sub>
* z<sub>2</sub>=w<sub>2</sub>α(z<sub>1</sub>)+b<sub>2</sub>
* f = σ(z<sub>2</sub>)

여기서 α는 **비선형 활성화 함수**, σ는 소프트맥스 함수이며, 매개변수 θ=<*w<sub>1</sub>,b<sub>1</sub>,w<sub>2</sub>,b<sub>2</sub>*>입니다.

경사 하강 알고리즘은 동일하게 유지되지만, 그래디언트를 계산하는 것이 더 어려워집니다. 체인 미분 규칙을 사용하여 다음과 같이 도함수를 계산할 수 있습니다:

* ∂ℒ/∂w<sub>2</sub> = (∂ℒ/∂σ)(∂σ/∂z<sub>2</sub>)(∂z<sub>2</sub>/∂w<sub>2</sub>)
* ∂ℒ/∂w<sub>1</sub> = (∂ℒ/∂σ)(∂σ/∂z<sub>2</sub>)(∂z<sub>2</sub>/∂α)(∂α/∂z<sub>1</sub>)(∂z<sub>1</sub>/∂w<sub>1</sub>)

> ✅ 체인 미분 규칙은 매개변수에 대한 손실 함수의 도함수를 계산하는 데 사용됩니다.

이 모든 표현식의 가장 왼쪽 부분은 동일하므로, 손실 함수에서 시작하여 계산 그래프를 "역방향"으로 따라가며 도함수를 효과적으로 계산할 수 있습니다. 따라서 다층 퍼셉트론을 훈련하는 방법을 **역전파** 또는 '백프로프'라고 합니다.

> ✅ 우리는 노트북 예제에서 역전파에 대해 훨씬 더 자세히 다룰 것입니다.
