# Подешавање у облаку ☁️ – GitHub Codespaces

**Користите овај водич ако не желите да ништа инсталирате локално.**  
Codespaces вам пружа бесплатну VS Code инстанцу у прегледачу са свим унапред инсталираним зависностима.

---

## 1.  Зашто Codespaces?

| Предност | Шта то значи за вас |
|---------|----------------------|
| ✅ Није потребна инсталација | Ради на Chromebook-у, iPad-у, рачунарима у школи… |
| ✅ Унапред припремљени дев контејнер | Python 3, Node.js, .NET, Java су већ унутра |
| ✅ Бесплатна квота | Лични налози добијају **120 цоре-часова / 60 GB-часова месечно** |

> 💡 **Савет**  
> Одржавајте квоту здравом тако што ћете **зауставити** или **обрисати** неактивне codespaces  
> (Погледајте ▸ Command Palette ▸ *Codespaces: Stop Codespace*).

---

## 2.  Направите Codespace (једним кликом)

1. **Форк**-ујте овај репозиторијум (дугме **Fork** горе десно).  
2. У вашем форку кликните **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Дијалог који приказује дугмад за креирање codespace-а](../../../translated_images/sr/who-will-pay.4c0609b1c7780f44.webp)

✅ Отвориће се прозор VS Code у прегледачу и дев контејнер ће почети да се прави.
Ово први пут траје **око 2 минута**.

## 3. Додајте свој API кључ (безбедан начин)

### Опција А Codespaces Secrets — Препоручено

1. ⚙️ Икона зупчаника -> Command Palette -> Codespaces : Manage user secret -> Додај нови тајни податак.
2. Назив: OPENAI_API_KEY
3. Вредност: налепите ваш кључ → Додај тајни податак

То је то — наш код ће аутоматски преузети тај податак.

### Опција Б .env фајл (ако вам је заиста потребан)

```bash
cp .env.copy .env
code .env         # попуните OPENAI_API_KEY=ваш_кључ_овде
```
