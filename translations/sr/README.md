# Генеративна вештачка интелигенција за почетнике (Верзија 3) - Курс

Научите основе прављења генеративних AI апликација са нашим свеобухватним курсом од 21 лекције који су припремили Microsoft Cloud Advocates.

## 🌱 Почетак

Овај курс садржи 21 лекцију. Свакa лекцијa обрађује своју тему па можете почети било где!

Лекције су означене као "Learn" лекције које објашњавају генеративни AI концепт или "Build" лекције које објашњавају концепт и примере кода у оба језика, **Python** и **TypeScript** када је могуће.

За .NET програмере погледајте [Генеративни AI за почетнике (.NET издање)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Свaka лекцијa такође укључује секцију "Keep Learning" са додатним алатима за учење.

## Шта вам треба
### За покретање кода овог курса, можете користити или:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Лекције:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Лекције:** "githubmodels" (GitHub Models престаје са радом крајем јула 2026 - користите Microsoft Foundry Models уместо тога)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Лекције:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Покрените моделе потпуно офлајн на свом уређају, није потребно претплатно на облак
   
- Основно знање Питхона или ТајпСкрипта је корисно - \*За апсолутне почетнике погледајте ове [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) и [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) курсеве
- GitHub налог за [форковање целог овог репозиторијума](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) на свој GitHub налог

Не заборавите да [означите (🌟) овај репозиторијум](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) да бисте га касније лакше нашли.
