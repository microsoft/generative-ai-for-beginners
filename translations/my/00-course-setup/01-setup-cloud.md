# မိုးကောင်းကင်ပြင် အဆင့်ဆင့် ☁️ – GitHub Codespaces

**သင့်မှာ မည်သည့် software ကိုမှ ကိုယ်ပိုင်စက်မှာ ထည့်သွင်းချင်ခြင်းမရှိပါက ဒီလမ်းညွှန်ကို အသုံးပြုပါ။**  
Codespaces သည် အခမဲ့ဖြစ်ပြီး ဘရောက်ဇာပေါ်မှာ နေရာယူထားသည့် VS Code ကိရိယာတစ်ခုဖြစ်ပြီး လိုအပ်တဲ့ dependency များ စုံလင်စွာ ထည့်သွင်းပေးထားသည်။

---

## 1. Codespaces ဘာကြောင့်လဲ?

| အားသာချက် | သင့်အတွက် အဓိပ္ပါယ် |
|---------|----------------------|
| ✅ စက်ထဲတွင် install လုပ်ရန် မလိုအပ်ခြင်း | Chromebook, iPad, စာသင်ကျောင်းဂီယူစက်များစသဖြင့် အဆင်ပြေသည် |
| ✅ ကြိုတင်ထည့်ထားသော dev container | Python 3, Node.js, .NET, Java များစွာကို အပါအဝင်ထားသည် |
| ✅ အခမဲ့ အချိန်ခွင့် | ကိုယ်ပိုင်အကောင့်များအတွက် **တစ်လလျှင် 120 core-hours / 60 GB-hours** ရရှိနိုင်သည် |

> 💡 **အကြံပြုချက်**  
> မသုံးနေသော codespaces များကို **ရပ်တန့်ခြင်း** သို့မဟုတ် **ဖျက်ခြင်း** ဖြင့် quota ကို ပြန်လည်ကျန်းမာအောင် ထိန်းသိမ်းပါ  
> (ကြည့်ပါ ▸ Command Palette ▸ *Codespaces: Stop Codespace*)။

---

## 2. Codespace အသစ်တစ်ခု ဖန်တီးခြင်း (တစ်ချက်နှိပ်ခြင်း)

1. ဒီ repo ကို **Fork** လုပ်ပါ (ညာထောင့်အပေါ် **Fork** ခလုတ်)။  
2. သင့် Fork ထဲတွင် **Code ▸ Codespaces ▸ Create codespace on main** ကို နှိပ်ပါ။  
   ![Dialog showing buttons to create a codespace](../../../translated_images/my/who-will-pay.4c0609b1c7780f44.webp)

✅ ဘရောက်ဇာတွင် VS Code ပြတင်းပေါ်ပွင့်ပြီး dev container တည်ဆောက်မှု စတင်သည်။
ဒါကို မိနစ် ၂ ခန့် ကြာသည်။

## 3. သင့် API key ကို ထည့်သွင်းခြင်း (လုံခြုံသောနည်း)

### နည်းလမ်း A Codespaces ရဲ့ Secrets — အကြံပြုသည်

1. ⚙️ ဂီယာအိုင်ကွန် -> Command Pallete-> Codespaces : Manage user secret -> Add a new secret။
2. အမည်: OPENAI_API_KEY
3. တန်ဖိုး: သင့် key ကို ကူးထည့်ပြီး → Add secret ကိုနှိပ်ပါ

ဒါဖြင့် သင့်ကုဒ်သည် အလိုအလျှောက် ယူဆောင်မည်။

### နည်းလမ်း B .env ဖိုင် (လိုအပ်ပါကသာ)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=သင့် key ကို ဒီနေရာတွင် ဖြည့်ပါ
```
