# Microsoft ဖွင့်လှစ်အရင်းအမြစ်အပြုသဘောကျင့်ဝတ်

ဒီပရောဂျက်က [Microsoft ဖွင့်လှစ်အရင်းအမြစ်အပြုသဘောကျင့်ဝတ်](https://opensource.microsoft.com/codeofconduct/) ကို လက်ခံထားပါတယ်။

အရင်းအမြစ်များ:

- [Microsoft ဖွင့်လှစ်အရင်းအမြစ်အပြုသဘောကျင့်ဝတ်](https://opensource.microsoft.com/codeofconduct/)
- [Microsoft ကျင့်ဝတ်ဆိုင်ရာ မေးခွန်းများ](https://opensource.microsoft.com/codeofconduct/faq/)
- မေးခွန်းများ သို့မဟုတ် စိုးရိမ်မှုများရှိပါက [opencode@microsoft.com](mailto:opencode@microsoft.com) သို့ ဆက်သွယ်ပါ

**ပယ်ဖျက်ချက်**:  
ဤစာရွက်စာတမ်းကို AI ဘာသာပြန်ဆိုမှုဝန်ဆောင်မှု [Co-op Translator](https://github.com/Azure/co-op-translator) ကို အသုံးပြု၍ ဘာသာပြန်ဆိုထားပါသည်။ ကျွန်ုပ်တို့သည် တိကျမှန်ကန်မှုအတွက် ကြိုးစားသော်လည်း အလိုအလျောက် ဘာသာပြန်ဆိုမှုများတွင် အမှားများ သို့မဟုတ် မမှန်ကန်မှုများ ပါဝင်နိုင်ကြောင်း ကျေးဇူးပြု၍ သတိပြုပါ။ မူရင်းစာရွက်စာတမ်းကို ၎င်း၏ ဒေသိယဘာသာစကားဖြင့် အာဏာတော်အရင်းအမြစ်အဖြစ် သတ်မှတ်သင့်ပါသည်။ အရေးကြီးသော အချက်အလက်များအတွက် ပရော်ဖက်ရှင်နယ် လူ့ဘာသာပြန်ဆိုမှုကို အကြံပြုပါသည်။ ဤဘာသာပြန်ဆိုမှုကို အသုံးပြုခြင်းမှ ဖြစ်ပေါ်လာသော အလွဲသဘောပေါက်မှုများ သို့မဟုတ် အနားလည်မှားမှုများအတွက် ကျွန်ုပ်တို့သည် တာဝန်မယူပါ။
