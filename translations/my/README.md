# Generative AI စမတ်ကျငယ်သူများအတွက် (ဗားရှင်း ၃) - သင်ခန်းစာ

Microsoft Cloud Advocates မှ ၂၁ မျိုးသင်ခန်းစာပါဝင်သည့် Generative AI အပလီကေးရှင်း ဖန်တီးခြင်း အခြေခံတွေကို သင်ယူပါ။

## 🌱 စပြီးလုပ်ဆောင်ခြင်း

ဒီသင်ခန်းစာမှာ ၂၁ မျိုးသင်ခန်းစာတွေ ရှိသည်။ သင်လိုချင်သလို စတင်လေ့လာနိုင်ပါသည်။

သင်ခန်းစာတွေဟာ "သင်ယူ" သင်ခန်းစာများဖြစ်ပြီး Generative AI မှတ်ချက်များကိုရှင်းပြသည် ၊ ဒါမှမဟုတ် "ဖန်တီး" သင်ခန်းစာများဖြစ်ပြီး အကြောင်းအရာနဲ့အတူ Python နဲ့ TypeScript နှစ်မျိုးလုံးဖြင့် ကုဒ်နမူနာများ ပါဝင်သော်လည်း အရင်ဆုံးဖြစ်ပါသည်။

.NET Developer များအတွက် [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ကိုကြည့်ပါ။

သင်ခန်းစာတိုင်းတွင် "သင်ကြားခြင်းကို ဆက်လက်ပါဝင်ရန်" အပိုင်းနဲ့ အပိုဆောင်းသင်ကြားမှု ပစ္စည်းများပါဝင်သည်။

## လိုအပ်သောအရာများ
### ဤသင်ခန်းစာ၏ ကုဒ်ကို စမ်းသပ်ရန် သို့မဟုတ် အသုံးပြုရန် အောက်ပါများကို အသုံးပြုနိုင်သည်။
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "githubmodels" (GitHub Models သည် ၂၀၂၆ ဇူလိုင်လကုန်တွင် သက်တမ်းသတ်မှတ်ပြီးဖြစ်သည် - Microsoft Foundry Models ကို ရွေးချယ်အသုံးပြုပါ)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - မိမိစက်တွင် လုံးဝအော့ဖ်လိုင်းဖြင့် မော်ဒယ်များကို အပြည့်အစုံပြေးဆွဲနိုင်သည်၊ cloud subscription မလိုပါ။
   
- Python သို့မဟုတ် TypeScript အခြေခံနားလည်မှု ရှိခြင်းက အထောက်အကူဖြစ်သည် - \*စတင်သင်ယူသူများအတွက် [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) နှင့် [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) သင်တန်းများကို ကြည့်ပါ
- GitHub အကောင့်ရှိရန် သင့်ရဲ့ GitHub အကောင့်သို့ [repo ဒီတာအားလုံးကို fork လုပ်ရန်](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst)

ကျွန်ုပ်တို့က **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** သင်ခန်းစာကို သင့်ရဲ့ တီထွင်မှုပတ်ဝန်းကျင် ပြင်ဆင်ခြင်းအတွက် ဖန်တီးပေးထားပါသည်။

ဒီ repo ကို မမေ့ဘဲ [ကြယ်ပေးရန် (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) မှတ်သားပေးပါ။

## 🧠 တင်သွင်းအသုံးပြုရန် အသင့်ဖြစ်ပါပြီလား?

ပိုမိုတိုးတက်ပြီး ကုဒ်နမူနာများ ရှာဖွေနေပါက ကျွန်ုပ်တို့ရဲ့ [Generative AI Code Samples စာမျက်နှာ](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) မှ Python နှင့် TypeScript နှစ်မျိုးလုံးဖြင့် ကြည့်ပါ။
