# Generative AI for Beginners (ဗားရှင်း ၃) - သင်ခန်းစာတစ်ခု

Microsoft Cloud Advocates မှ တင်ဆက်တဲ့ ၂၁ သင်ခန်းစာပါရှိတဲ့ ကျယ်ပြန့်စွာ သင်ကြားပေးတဲ့ Generative AI အပလီကေးရှင်းတွေ တည်ဆောက်ခြင်း အခြေခံများကို သင်ယူပါ။

## 🌱 စတင်သင်ယူခြင်း

ဒီသင်တန်းမှာ သင်ခန်းစာ ၂၁ ခု ပါဝင်ပါတယ်။ ကိုယ့်လိုချင်သမျှ သင်ခန်းစာကနေ စတင်လေ့လာနိုင်ပါတယ်။

သင်ခန်းစာများကို "သင်ယူ" သင်ခန်းစာများနဲ့ Generative AI အယူအဆကို ရှင်းပြသည့် သင်ခန်းစာများ၊ သဘောထားနဲ့ ပိုပြီး code အကြောင်းအရာပါသော "တည်ဆောက်" သင်ခန်းစာများအဖြစ် တိုက်ရိုက် ပေးထားပြီး **Python** နှင့် **TypeScript** ဘာသာနှစ်မျိုးဖြင့် သင်ကြားပေးသည်။

.NET Developer များအတွက် [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ကို ကြည့်ပါ။

သင်ခန်းစာတစ်ခုစီမှာ "ဆက်လက်သင်ယူရန်" အပိုင်းတစ်ခုပါရှိပြီး ထိုနေရာတွင် နောက်ထပ်သင်ယူရန် ကိရိယာများ ပါဝင်သည်။

## လိုအပ်ချက်များ
### ဒီသင်တန်းရဲ့ code များကို ပြေးဆွဲဖို့ အောက်ပါ များအနက် လုပ်နိုင်ပါတယ်
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "githubmodels" (GitHub Models ကို ၂၀၂၆ ခုနှစ် ဇူလိုင်လကုန်တွင် ရပ်ဆိုင်းမည် - Microsoft Foundry Models ကို စတင်သုံးပါ)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - မိမိစက်ပေါ်မှာ အပြည်ပြည်ဆိုင်ရာ အင်တာနက်မလိုဘဲ လုံးဝ offline ဖြစ်စေသော မော်ဒယ်များအသုံးပြုနိုင်သည်။
   
- Python သို့မဟုတ် TypeScript အခြေခံ နားလည်မှု ရှိခြင်းက အထောက်အကူဖြစ်သည် - \*အခြေခံစတင်သူများအတွက် [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) နှင့် [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) သင်တန်းများပါရှိသည်။
- အပြည့်အစုံနားလည်မှုအတွက် GitHub အကောင့်တစ်ခုထားပြီး [repo အားလုံးကို fork လုပ်ပါ](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst)

ကျွန်ုပ်တို့က သင့်ရဲ့ ဖွံ့ဖြိုးတိုးတက်မှု ပတ်ဝန်းကျင် စတင်ဖို့ အကူအညီပေးရန် **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** သင်ခန်းစာ တစ်ခု ဖန်တီးထားပါတယ်။

နောက်တစ်ကြိမ် မခက်ခဲစေဖို့ [repo ကို (🌟) စသည့်နှိပ်ကာ star လုပ်ပါ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)။

## 🧠 ပြင်ဆင်ပြီး ပို့ရန် အသင့်ဖြစ်ပြီလား?

ပိုမိုတိုးတက်သော code နမူနာများကို လိုချင်ရင်၊ ကျွန်ုပ်တို့ရဲ့ [Generative AI Code Samples စုစည်းမှု](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) ကို **Python** နှင့် **TypeScript** ဘာသာနှစ်မျိုးဖြင့် ကြည့်ပါ။

## 🗣️ အခြားသင်ကြားနေသူများနှင့် တွေ့ဆုံချင်လား၊ ထောက်ခံမှုလိုအပ်လား

ဒီသင်တန်းကို လေ့လာနေသူများ နဲ့ ဆက်သွယ်ရန် နှင့် သင်ကြားမှု ထောက်ခံမှုရရန် [Microsoft Foundry Discord server တရားဝင်](https://aka.ms/genai-discord?WT.mc_id=academic-105485-koreyst) ကို ဝင်ရောက်ပါ။

မေးခွန်းများ မေးခွန်းမေးရန် သို့မဟုတ် ထုတ်ကုန် တုံ့ပြန်ချက် မျှဝေရန် [Microsoft Foundry Developer Forum](https://aka.ms/azureaifoundry/forum) မှာ Github တွင် ပါဝင်ပါ။

## 🚀 စတင်တည်ထောင်နေပါသလား?

အကြံပြုချက်ရှိပါသလား သို့မဟုတ် စာလုံးပေါင်း အမှား သို့မဟုတ် code အမှားတွေတွေ့တယ်ဆိုရင်?
