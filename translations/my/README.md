# Generative AI for Beginners (Version 3) - သင်တန်း

Microsoft Cloud Advocates မှ ၂၁ သင်ခန်းစာများပါဝင်သည့် ကျယ်ကျယ်ပြန့်ပြန့် သင်တန်းဖြင့် Generative AI အက်ပလီကေးရှင်းများ တည်ဆောက်ခြင်း၏ အခြေခံကို သင်ယူပါ။

## 🌱 စတင်ရန်

ဒီသင်တန်းမှာ ၂၁ သင်ခန်းစာပါဝင်ပါတယ်။ သင်ခန်းစာတိုင်းမှာ ကိုယ်ပိုင်အကြောင်းအရာကို ဖော်ပြထားပြီး သင်စိတ်ဝင်စားရာနေရာက စတင်နိုင်ပါတယ်။

သင်ခန်းစာများကို "Learn" သင်ခန်းစာများအဖြစ် Generative AI အကြောင်းအရာကို ရှင်းလင်းဖော်ပြထားပြီး "Build" သင်ခန်းစာများမှာ အကြောင်းအရာနှင့် **Python** နှင့် **TypeScript** တို့ဖြင့် ကုဒ်နမူနာများကို ရှင်းလင်းဖော်ပြထားပါတယ်။

.NET Developer များအတွက် [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ကို ကြည့်ပါ!

သင်ခန်းစာတိုင်းမှာ "Keep Learning" အပိုင်းပါဝင်ပြီး အပိုသင်ယူမှုအရင်းအမြစ်များကို ထည့်သွင်းထားပါတယ်။

## လိုအပ်ချက်များ
### ဒီသင်တန်းရဲ့ ကုဒ်ကို အလုပ်လုပ်စေဖို့ အောက်ပါအရာများကို အသုံးပြုနိုင်ပါတယ်။
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "oai-assignment" 

- Python သို့မဟုတ် TypeScript အခြေခံအသိရှိခြင်း အထောက်အကူဖြစ်ပါတယ် - \*အခြေခံအသိမရှိသူများအတွက် [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) နှင့် [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) သင်တန်းများကို ကြည့်ပါ
- GitHub အကောင့်တစ်ခုရှိပြီး [ဒီ repo အားလုံးကို fork](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) လုပ်ရန်လိုအပ်ပါတယ်

ကျွန်တော်တို့မှာ **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** သင်ခန်းစာတစ်ခုရှိပြီး သင့်ရဲ့ development environment ကို စတင်တည်ဆောက်ရန် အကူအညီပေးပါမယ်။

ဒီ repo ကို [star (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) လုပ်ထားပြီး နောက်ပိုင်းမှာ လွယ်ကူစွာ ရှာဖွေပါ။

## 🧠 Deploy လုပ်ရန် အသင့်ဖြစ်ပါသလား?

ပိုမိုအဆင့်မြင့်ကုဒ်နမူနာများကို ရှာဖွေချင်ပါက **Python** နှင့် **TypeScript** တို့ဖြင့် [Generative AI Code Samples](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) စုစည်းမှုကို ကြည့်ပါ။

## 🗣️ အခြားသော သင်ယူသူများနှင့် တွေ့ဆုံပြီး အထောက်အပံ့ရယူပါ

ဒီသင်တန်းကို လေ့လာနေသော အခြားသော သင်ယူသူများနှင့် တွေ့ဆုံပြီး အထောက်အပံ့ရယူရန် [Azure AI Foundry Discord server](https://aka.ms/genai-discord?WT.mc_id=academic-105485-koreyst) ကို ဝင်ပါ။

GitHub မှာ [Azure AI Foundry Developer Forum](https://aka.ms/azureaifoundry/forum) တွင် မေးခွန်းများမေးပါ သို့မဟုတ် ထုတ်ကုန်အကြံပြုချက်များကို မျှဝေပါ။

## 🚀 စတင်တည်ဆောက်နေပါသလား?

ယနေ့ Azure credits ဖြင့် စတင်တည်ဆောက်ရန် [Microsoft for Startups](https://www.microsoft.com/startups) ကို သွားရောက်ကြည့်ရှုပါ။

## 🙏 အကူအညီပေးချင်ပါသလား?

အကြံပြုချက်များရှိပါသလား သို့မဟုတ် စာလုံးပေါင်းအမှားများ သို့မဟုတ် ကုဒ်အမှားများကို ရှာဖွေတွေ့ရှိပါသလား? [Issue တစ်ခုတင်ပါ](https://github.com/microsoft/generative-ai-for-beginners/issues?WT.mc_id=academic-105485-koreyst) သို့မဟုတ် [Pull request တစ်ခုဖန်တီးပါ](https://github.com/microsoft/generative-ai-for-beginners/pulls?WT.mc_id=academic-105485-koreyst)

## 📂 သင်ခန်းစာတိုင်းတွင် ပါဝင်သည်။

- အကြောင်းအရာကို ရှင်းလင်းဖော်ပြထားသော အတိုချုံးဗီဒီယို
- README တွင် ရေးသားထားသော သင်ခန်းစာ
- Azure OpenAI နှင့် OpenAI API ကို ထောက်ပံ့သော Python နှင့် TypeScript ကုဒ်နမူနာများ
- သင့်ရဲ့ သင်ယူမှုကို ဆက်လက်တိုးတက်စေမည့် အပိုအရင်းအမြစ်များ
