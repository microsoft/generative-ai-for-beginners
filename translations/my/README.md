# မူလ အိုင်အိုအေတီ စတင်သူများအတွက် (ဗားရှင်း ၃) - သင်တန်း

Microsoft Cloud Advocates မှ ၂၁-သင်ခန်းစာ အပြည့်အစုံ သင်တန်းဖြင့် မူလ အိုင်အိုအေတီ လျှောက်လွှာများ ဖွဲ့စည်းခြင်း၏ အခြေခံအချက်များကို လေ့လာပါ။

## 🌱 စတင်ရန်

ဒီသင်တန်းတွင် ၂၁ သင်ခန်းစာများ ပါဝင်သည်။ သင်ခန်းစာတစ်ခုစီသည် သူ၏ မူရင်းခေါင်းစဉ်ကို ဖုံးလွှမ်းထားသောကြောင့် သင်နှစ်သက်ရာနေရာမှ စတင်ပါ!

သင်ခန်းစာများကို "Learn" သင်ခန်းစာများဖြင့် မူလ အိုင်အိုအေတီ အကြောင်းအရာကို ရှင်းပြသည်။ "Build" သင်ခန်းစာများဖြင့် အကြောင်းအရာကို ရှင်းပြပြီး **Python** နှင့် **TypeScript** နှစ်ခုလုံးတွင် နမူနာကုဒ်များကို ရှင်းပြသည်။

.NET Developer များအတွက် [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ကို ကြည့်ပါ!

သင်ခန်းစာတိုင်းတွင် "Keep Learning" အပိုင်းကို အပိုသင်ယူရေးကိရိယာများနှင့် အတူ ပါဝင်သည်။

## သင့်အတွက် လိုအပ်သော အရာများ
### ဒီသင်တန်း၏ ကုဒ်ကို အလုပ်လည်စေရန် သင့်အား အောက်ပါအရာများ တစ်ခုခုကို အသုံးပြုနိုင်ပါသည်: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **သင်ခန်းစာများ:** "oai-assignment" 
   
- Python သို့မဟုတ် TypeScript အခြေခံအကြောင်းအရာ သိရှိခြင်း - \* အစအဆုံးစတင်သူများအတွက် [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) နှင့် [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) သင်တန်းများကို ကြည့်ပါ
- GitHub အကောင့်တစ်ခု [ဒီ repo အပြည့်ကို fork လုပ်ရန်](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) သင့် GitHub အကောင့်သို့

သင်၏ ဖွံ့ဖြိုးရေး ပတ်ဝန်းကျင်ကို စတင်ရန် အကူအညီပေးရန် **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** သင်ခန်းစာကို ဖန်တီးထားပါသည်။

နောက်ပိုင်းတွင် အလွယ်တကူ ရှာဖွေရန် [ဒီ repo ကို star (🌟) လုပ်ရန်](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) မမေ့ပါနှင့်။

## 🧠 Deploy ပြင်ဆင်နေပါသလား?

ပိုမို ကြီးမားသော ကုဒ်နမူနာများကို ရှာနေပါက **Python** နှင့် **TypeScript** နှစ်ခုလုံးတွင် [Generative AI Code Samples](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) စုစည်းမှုကို ကြည့်ပါ။

## 🗣️ အခြားသော သင်ယူသူများနှင့် တွေ့ဆုံပါ၊ အထောက်အပံ့ရယူပါ

GitHub တွင်ရှိသော [Azure AI Foundry Developer Forum](https://aka.ms/azureaifoundry/forum) တွင် မေးခွန်းမေးပါ သို့မဟုတ် ထုတ်ကုန်အကြောင်းအရာများကို မျှဝေပါ။
