# ကိုယ်တိုင်လေ့လာမှုအတွက် အရင်းအမြစ်များ

OpenAI နှင့် Microsoft Foundry မှ အဓိကအရင်းအမြစ်များကို အသုံးပြု၍ သင်ခန်းစာကို တည်ဆောက်ထားပြီး အကြောင်းအရာများနှင့် သင်ခန်းစာများအတွက် ကိုးကားထားသည်။ သင်၏ ကိုယ်ပိုင် ကိုယ်တိုင်လေ့လာမှုခရီးစဉ်များအတွက် အပြည့်အစုံမဟုတ်သော ဖော်ပြချက်စာရင်းတစ်ခု ဖြစ်ပါသည်။ အောက်ပါလင့်ခ်များသည် လက်ရှိ ပံ့ပိုးမှုရှိသော ပစ္စည်းများသို့ ဦးတည်ပါသည်။

## 2. ဒုတိယအဆင့် အရင်းအမြစ်များ

ဒီအပိုင်းမှာ သင်ခန်းစာထဲမှာ မဖော်ပြနိုင်ခဲ့သော လေ့လာသင့်သော အပိုရင်းအမြစ်များ ပါဝင်သည်။ ဒီအရင်းအမြစ်များကို အသုံးပြုပြီး ဒီခေါင်းစဉ်အကြောင်း ကိုယ်တိုင် ကျွမ်းကျင်မှု ဖွံ့ဖြိုးနိုင်ပါစေ။
