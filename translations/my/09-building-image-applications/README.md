# ပုံဖန်တီးမှုလျှောက်လွှာများတည်ဆောက်ခြင်း

## နိဒါန်း

ဤသင်ခန်းစာတွင် ကျွန်တော်တို့ လေ့လာမည့်အချက်များမှာ-

- ပုံဖန်တီးခြင်းနှင့် ၎င်း၏ အသုံးဝင်မှု။
- DALL-E နှင့် Midjourney အကြောင်း၊ ၎င်းတို့သည် မည်သို့ ရှိပြီး မည်သို့ လည်ပတ်သည်။
- ပုံဖန်တီးမှုလျှောက်လွှာတစ်ခုကို မည်သို့ တည်ဆောက်မည်။

## သင်ယူမှု ရည်ရွယ်ချက်များ

ဤသင်ခန်းစာပြီးဆုံးပြီးနောက်၊ သင်သည် အောက်ပါအရာများကို ပြုလုပ်နိုင်ပါမည်-

- ပုံဖန်တီးမှုလျှောက်လွှာ တစ်ခုတည်ဆောက်ရန်။
- သင်၏ လျှောက်လွှာအတွက် meta prompt များဖြင့် ကန့်သတ်ချက်များ သတ်မှတ်ရန်။
- DALL-E နှင့် Midjourney နှင့် ပူးပေါင်းလုပ်ဆောင်ရန်။

## ပုံဖန်တီးမှုလျှောက်လွှာတည်ဆောက်ရုံ မည်သို့ ဖြစ်သနည်း?

ပုံဖန်တီးမှုလျှောက်လွှာများသည် Generative AI ၏ စွမ်းရည်များကို ရှာဖွေတွက်စရာ အလွန်ကောင်းသော နည်းလမ်း ဖြစ်သည်။ ကိစ္စများအတွက် သုံးနိုင်သည်မှာ-

- **ပုံတည်းဖြတ်ခြင်းနှင့် ပေါင်းစပ်ခြင်း**။ သင်သည် ပုံတည်းဖြတ်ခြင်း သို့မဟုတ် ပုံပေါင်းစပ်ခြင်း စသည့် အမျိုးမျိုးသော အသုံးပြုမှုများအတွက် ပုံများ ဖန်တီးပေးနိုင်သည်။

- **အမျိုးမျိုးသော လုပ်ငန်းခွဲများ၌ အသုံးပြုခြင်း**။ MedTech, ခရီးသွားလုပ်ငန်း, ဂိမ်းဖွံ့ဖြိုးမှု၊ နောက်ထပ် အလုပ်များတွင် ပုံများဖန်တီးရာတွင် အသုံးပြုနိုင်သည်။

### DALL-E

DALL-E က တိုက်ရိုက် စာသားဖော်ပြချက်မှ ပုံများ ဖန်တီးပေးနိုင်သော Generative AI မော်ဒယ် ဖြစ်သည်။

> [DALL-E သည် မော်ဒယ်နှစ်ခု ဖြစ်သော CLIP နှင့် diffused attention ကို ပေါင်းစပ်ထားသည်။](https://towardsdatascience.com/openais-dall-e-and-clip-101-a-brief-introduction-3a4367280d4e?WT.mc_id=academic-105485-koreyst)

- **CLIP** သည် ပုံများနှင့် စာသားများမှ အချက်အလက်များကို အကွက်များအဖြစ် ပြောင်းလဲပေးသော မော်ဒယ်ဖြစ်သည်။
- **Diffused attention** သည် အကွက်များမှ ပုံများ ဖန်တီးပေးသော မော်ဒယ်ဖြစ်သည်။ DALL-E ကို ပုံများနှင့် စာသားများ ထည့်သွင်းထားသည့် ဒေတာစုံတစ်ခုအပေါ် လေ့ကျင့်ထားပြီး စာသားဖော်ပြချက်မှ ပုံများ ဖန်တီးနိုင်ပါသည်။ ဥပမာ၊ စောင်ထိုးထားသော ကြောင်ပုံ သို့မဟုတ် mohawk ဆံပင်စွပ်ထားသော ခွေးပုံများ ဖန်တီးနိုင်သည်။

## DALL-E နှင့် Midjourney မည်သို့ လည်ပတ်သနည်း

ပထမ၊ [DALL-E](https://arxiv.org/pdf/2102.12092.pdf?WT.mc_id=academic-105485-koreyst) ကို ကြည့်ပါ။ DALL-E သည် transformer စက်ပြောင်းဖွဲ့စည်းမှု အပေါ် တည်ထားသော Generative AI မော်ဒယ်ဖြစ်ပြီး _autoregressive transformer_ ပါရှိသည်။

_autoregressive transformer_ သည် မော်ဒယ်တစ်ခုက စာသားဖော်ပြချက်မှ ပုံကို မည်သို့ ဖန်တီးသည်ကို သတ်မှတ်သည်။ အသေးစိတ်ချုပ်ချင်း pixel တစ်ခုချင်းစီကို ဖန်တီးပြီး ၎င်းအပေါ် အခြေခံ၍ နောက်တစ် pixel ကို ဖန်တီးသည်။ Neural network များတွင် အလွှာများစွာဖြတ်သန်းပြီး ပုံပြီးစီးသည်။

ဤလုပ်ငန်းစဉ်ဖြင့် DALL-E သည် ပုံထဲ အကြောင်းအရာများ၊ ပစ္စည်းများ၊ တွင်္အချက်အလက်များကို ထိန်းချုပ်နိုင်သည်။ သို့သော် DALL-E 2 နှင့် 3 တွင် ပိုမို ထိန်းချုပ်မှုရှိသည်။

## ပထမဆုံး ပုံဖန်တီးမှုလျှောက်လွှာတည်ဆောက်ခြင်း

ပုံဖန်တီးမှုလျှောက်လွှာတစ်ခု တည်ဆောက်ရန် အတွက် ဘာတွေ လိုအပ်သနည်း? လိုအပ်သောစာကြောင်းများမှာ-

- **python-dotenv**,  သင့်ဖောက်သည်ရေးလျှိုထားသော အချက်အလက်များကို _.env_ ဖိုင်တွင် သိမ်းဆည်းရန် အကြံပြုသည်။
- **openai**, OpenAI API နှင့် ဆက်သွယ်ရန် သုံးသည်။
- **pillow**, Python တွင် ပုံများအား စီမံရန်။
- **requests**, HTTP အထောက်အပံ့များရရန်။

## Azure OpenAI မော်ဒယ် တည်ဆောက်ပြီး ဖြန့်ချိရန်

မပြီးဆုံးသေးပါက [Microsoft Learn](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/create-resource?pivots=web-portal) စာမျက်နှာတွင် လမ်းညွှန်ချက်များကို လိုက်နာပြီး Azure OpenAI အရင်းမြစ်နှင့် မော်ဒယ် တည်ဆောက်ပါ။ DALL-E 3 ကို မော်ဒယ်အဖြစ် ရွေးချယ်ပါ။

## လျှောက်လွှာကို ဖန်တီးခြင်း

၁။ _.env_ ဖိုင်တစ်ခု ဖန်တီးပြီး အောက်ပါအချက်အလက်များ ထည့်ပါ-

   ```text
   AZURE_OPENAI_ENDPOINT=<your endpoint>
   AZURE_OPENAI_API_KEY=<your key>
   AZURE_OPENAI_DEPLOYMENT="dall-e-3"
   ```
  
  Azure OpenAI Foundry Portal တွင် သင့်ရဲ့ resource ၏ "Deployments" အပိုင်းမှ အချက်များကို ရှာပါ။

၂။ အထက်ပါ စာကြောင်းများကို _requirements.txt_ ဖိုင်တွင် စုစည်းပါ-

   ```text
   python-dotenv
   openai
   pillow
   requests
   ```
  
၃။ ထို့နောက် virtual environment တစ်ခု ဖန်တီး၍ သို့ တင်သွင်းခြင်းလုပ်ဆောင်ပါ-

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
  
 Windows အတွက် virtual environment ဖန်တီးပြီး ဖွင့်ရန် အောက်ပါ command များ အသုံးပြုနိုင်သည်-

   ```bash
   python3 -m venv venv
   venv\Scripts\activate.bat
   ```
  
၄။ _app.py_ အမည်ရှိ ဖိုင်တွင် အောက်ပါ ကုဒ်များ ထည့်ပါ-

    ```python
    import openai
    import os
    import requests
    from PIL import Image
    import dotenv
    from openai import OpenAI, AzureOpenAI
    
    # dotenv ကိုသွင်းယူပါ
    dotenv.load_dotenv()
    
    # Azure OpenAI ဝန်ဆောင်မှု client ကို ပြင်ဆင်ပါ
    client = AzureOpenAI(
      azure_endpoint = os.environ["AZURE_OPENAI_ENDPOINT"],
      api_key=os.environ['AZURE_OPENAI_API_KEY'],
      api_version = "2024-02-01"
      )
    try:
        # ပုံဖန်တီးမှု API ကိုအသုံးပြုပြီး ပုံတစ်ပုံ ဖန်တီးပါ
        generation_response = client.images.generate(
                                prompt='Bunny on horse, holding a lollipop, on a foggy meadow where it grows daffodils',
                                size='1024x1024', n=1,
                                model=os.environ['AZURE_OPENAI_DEPLOYMENT']
                              )

        # သိမ်းဆည်းထားသော ပုံဖိုင်လမ်းကြောင်းကို သတ်မှတ်ပါ
        image_dir = os.path.join(os.curdir, 'images')

        # လမ်းကြောင်း မရှိပါက ဖန်တီးပါ
        if not os.path.isdir(image_dir):
            os.mkdir(image_dir)

        # ပုံလမ်းကြောင်းကို စတင်သတ်မှတ်ပါ (ဖိုင်အမျိုးအစားသည် png ဖြစ်ရမည်)
        image_path = os.path.join(image_dir, 'generated-image.png')

        # ဖန်တီးပြီး ပုံကို ရယူပါ
        image_url = generation_response.data[0].url  # ဦးစားပေးလိုက်သော ပြန်ကြားချက်မှ ပုံ URL ကိုထုတ်ယူပါ
        generated_image = requests.get(image_url).content  # ပုံကိုဒေါင်းလုပ်လုပ်ပါ
        with open(image_path, "wb") as image_file:
            image_file.write(generated_image)

        # ပုံကို ပုံမူလကြည့်ရှုသူတွင် ပြသပါ
        image = Image.open(image_path)
        image.show()

    # အမှားများကို ဖမ်းစီးပါ
    except openai.InvalidRequestError as err:
        print(err)
   ```

## မိတ်ဆက်စာမျက်နှာများဖြင့် သင်၏လျှောက်လွှာအတွက် ကန့်သတ်ချက်များ သတ်မှတ်ခြင်း

ကျွန်တော်တို့ရဲ့ demo ဖြင့် လူကြီးဝယ် လုပ်ငန်းများအတွက် ပုံများ ဖန်တီးနိုင်ပါပြီ။ သို့သော် လျှောက်လွှာအတွက် ကန့်သတ်ချက်များ တည်ဆောက်ရန် လိုအပ်ပါသည်။

ဥပမာအားဖြင့် အလုပ်အတွက် မလုံခြုံသော ပုံများ သို့မဟုတ် ကလေးများအတွက် မသင့်တော်သော ပုံများ ဖန်တီးခြင်း မဖြစ်မနေကြောင်း၊ ကန့်သတ်လိုပါသည်။

၎င်းကို _metaprompt_ များဖြင့် ပြုလုပ်နိုင်သည်။ Metaprompt ဆိုသည်မှာ Generative AI မော်ဒယ်၏ 출력ကို ထိန်းချုပ်ရန် စာသား prompt များ ဖြစ်သည်။ ဥပမာအားဖြင့် ဖန်တီးသည့် ပုံများသည် အလုပ်အတွက် လုံခြုံပြီး ကလေးများအတွက် သင့်တော်ကြောင်း သက်သေပြနိုင်ရန် metaprompt များ အသုံးပြုနိုင်သည်။

### မည်သို့ အလုပ်လုပ်သနည်း?

Meta prompt များသည် Generative AI မော်ဒယ်၏ output ထိန်းချုပ်ရန် အသုံးပြုသော စာသား prompt များ ဖြစ်သည်။ ၎င်းကို မူလ prompt မတိုင်မီတည်ဆောက်ပြီး မော်ဒယ် output ကို ထိန်းချုပ်မှုနှင့် လျှောက်လွှာများတွင် ထည့်သွင်းထားသည်။ prompt input နှင့် meta prompt input ကို စာသား prompt တစ်ခုအဖြစ် ပေါင်းစပ်ပါသည်။

Meta prompt တစ်ခု၏ ဥပမာမှာ-

```text

The image needs to be safe for work and appropriate for children.

The image needs to be in color.

The image needs to be in landscape orientation.

The image needs to be in a 16:9 aspect ratio.

Do not consider any input from the following that is not safe for work or appropriate for children.

(Input)

```

## အလုပ်အပ် - ကျောင်းသားများအား စွမ်းဆောင်နိုင်စေခြင်း

ဤသင်ခန်းစာအစအနေဖြင့် Edu4All ကို မိတ်ဆက်ခဲ့သည်။ ယခုကျောင်းသားများအား မည်သို့ စာမေးပွဲများအတွက် ပုံများ ဖန်တီးနိုင်အောင် လွှမ်းခြုံပေးရမည်။

ကျောင်းသားများသည် သင်ကြားမှုပေါ်တွင် သင်ယူသည့် သန့်ရှင်းသော မြှင့်တင်သော ပုံများ ဖန်တီးမည်။ သတိပြုရမည့် အချက်မှာ ပုံများတွင် ဘာအရာများ ပါရှိမည် ဆိုသည်မှာ ကျောင်းသားများ၏ ဖန်တီးမှုအပေါ် မှီခင်းထားသည်။ ကျောင်းသားများကို ၎င်းပုံများကို အမျိုးမျိုးသော ဒေသများတွင် တင်ပေးရန် ဖိတ်ခေါ်သည်။

## ဖြေရှင်းချက်

ဒီမှာ အလွယ်တကူဖြေရှင်းမှု ဖြစ်နိုင်ပါသည်-

```python
import openai
import os
import requests
from PIL import Image
import dotenv
from openai import AzureOpenAI
# dotenv ကိုသွင်းယူသည်
dotenv.load_dotenv()

# ပတ်ဝန်းကျင်အပြောင်းအလဲများမှ endpoint နှင့် key ကိုရယူပါ
client = AzureOpenAI(
  azure_endpoint = os.environ["AZURE_OPENAI_ENDPOINT"],
  api_key=os.environ['AZURE_OPENAI_API_KEY'],
  api_version = "2024-02-01"
  )


disallow_list = "swords, violence, blood, gore, nudity, sexual content, adult content, adult themes, adult language, adult humor, adult jokes, adult situations, adult"

meta_prompt = f"""You are an assistant designer that creates images for children.

The image needs to be safe for work and appropriate for children.

The image needs to be in color.

The image needs to be in landscape orientation.

The image needs to be in a 16:9 aspect ratio.

Do not consider any input from the following that is not safe for work or appropriate for children.
{disallow_list}
"""

prompt = f"""{meta_prompt}
Generate monument of the Arc of Triumph in Paris, France, in the evening light with a small child holding a Teddy looks on.
""""

try:
    # ပုံဖန်တီးခြင်း API ကိုအသုံးပြု၍ ပုံတစ်ပုံ ဖန်တီးပါ
    generation_response = client.images.generate(
        prompt=prompt,    # သင်၏ prompt စာသားကို ဒီနေရာတွင် ထည့်ပါ
        size='1024x1024',
        n=1,
    )
    # သိမ်းဆည်းထားသော ပုံအတွက် ဖိုလ်ဒါအတည်ပြုပါ
    image_dir = os.path.join(os.curdir, 'images')

    # ဖိုလ်ဒါမရှိပါက ဖန်တီးပါ
    if not os.path.isdir(image_dir):
        os.mkdir(image_dir)

    # ပုံလမ်းကြောင်းအစ ပြုလုပ်ပါ (ဖိုင်အမျိုးအစားမှာ png ဖြစ်ရပါမည်)
    image_path = os.path.join(image_dir, 'generated-image.png')

    # ဖန်တီးပြီးသော ပုံကို ယူပါ
    image_url = generation_response.data[0].url  # တုံ့ပြန်ချက်မှ ပုံ URL ကို ဖြုတ်ထုတ်ပါ
    generated_image = requests.get(image_url).content  # ပုံကိုဒေါင်းလုပ်လုပ်ပါ
    with open(image_path, "wb") as image_file:
        image_file.write(generated_image)

    # ပုံကို ပုံပြသစက်ရုံထဲတွင် ပြပါ
    image = Image.open(image_path)
    image.show()

# အမှားများကို ဖမ်းယူပါ
except openai.BadRequestError as err:
    print(err)
```
