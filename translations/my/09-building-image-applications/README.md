# ပုံပျိုးထွန်းမှု အက်ပလီကေးရှင်းများ တည်ဆောက်ခြင်း

LLM များတွင် စာသားထုတ်လုပ်မှုထက် ပိုသော အရာများရှိသည်။ စာသားဖေါ်ပြချက်များမှ ပုံများထုတ်လုပ်နိုင်သည်။ ပုံများကို modality အဖြစ် အသုံးပြုခြင်းသည် MedTech, ဆောက်လုပ်ရေး, ခရီးသွားလုပ်ငန်း, ဂိမ်းဖွံ့ဖြိုးရေးနှင့် အခြား အကွာအဝေးများတွင် အလွန်အသုံးဝင်နိုင်သည်။ ဒီအခန်းမှာတော့ DALL-E နှင့် Midjourney ဆိုသော အထူးလူကြိုက်များသော ပုံပျိုးထွန်းမှု မော်ဒယ်နှစ်ခုကို လေ့လာပါမည်။

## နိဒါန်း

ဒီသင်ခန်းစာမှာတော့-

- ပုံပျိုးထွန်းမှုနှင့် ၎င်း၏ အသုံးဝင်မှု။
- DALL-E နှင့် Midjourney, ၎င်းတို့ဘာလဲ၊ ဘယ်လိုအလုပ်လုပ်လဲ။
- ပုံပျိုးထွန်းမှုအက်ပလီကေးရှင်းတစ်ခုကို ဘယ်လိုတည်ဆောက်မလဲ။

## သင်ယူရမည့် အချက်များ

ဒီသင်ခန်းစာပြီးဆုံးပြီးနောက်-

- ပုံပျိုးထွန်းမှုအက်ပလီကေးရှင်း တစ်ခုကို တည်ဆောက်နိုင်မည်။
- သင်၏အက်ပလီကေးရှင်းအတွက် meta prompts ဖြင့် နယ်နိမိတ်များ သတ်မှတ်နိုင်မည်။
- DALL-E နှင့် Midjourney နှင့် အလုပ်လုပ်နိုင်မည်။

## ဘာကြောင့် ပုံပျိုးထွန်းမှုအက်ပလီကေးရှင်း တည်ဆောက်မလဲ?

ပုံပျိုးထွန်းမှုအက်ပလီကေးရှင်းများသည် Generative AI ၏စွမ်းရည်များကို ရှာဖွေဖို့ အလွန်ကောင်းမွန်သောနည်းလမ်းဖြစ်သည်။ ၎င်းတို့ကို ဥပမာအားဖြင့်-

- **ပုံတည်းဖြတ်ခြင်းနှင့် ပုံစပ်ခြင်း**။ သင်သည် ပုံတည်းဖြတ်ခြင်းနှင့် ပုံစပ်ခြင်း စသည့် အသုံးပြုမှုအမျိုးမျိုးအတွက် ပုံများကို ထုတ်လုပ်နိုင်သည်။

- **စက်မှုလုပ်ငန်း အမျိုးမျိုးတွင် အသုံးပြုခြင်း**။ Medtech, ခရီးသွားလုပ်ငန်း, ဂိမ်းဖွံ့ဖြိုးရေး စသည့် စက်မှုလုပ်ငန်း အမျိုးမျိုးအတွက် ပုံများကို ထုတ်လုပ်နိုင်သည်။

## DALL-E နှင့် Midjourney ဘာလဲ?

[DALL-E](https://openai.com/dall-e-2?WT.mc_id=academic-105485-koreyst) နှင့် [Midjourney](https://www.midjourney.com/?WT.mc_id=academic-105485-koreyst) သည် အထူးလူကြိုက်များသော ပုံပျိုးထွန်းမှု မော်ဒယ်နှစ်ခုဖြစ်ပြီး၊ ၎င်းတို့ကို prompt များအသုံးပြုပြီး ပုံများကို ဖန်တီးနိုင်သည်။

### DALL-E

DALL-E နှင့် စတင်ပါမည်၊ ၎င်းသည် Generative AI မော်ဒယ်တစ်ခုဖြစ်ပြီး၊ စာသားဖေါ်ပြချက်များမှ ပုံများကို ဖန်တီးသည်။

> [DALL-E သည် CLIP နှင့် diffused attention ဆိုသော မော်ဒယ်နှစ်ခု၏ ပေါင်းစပ်မှု](https://towardsdatascience.com/openais-dall-e-and-clip-101-a-brief-introduction-3a4367280d4e?WT.mc_id=academic-105485-koreyst) ဖြစ်သည်။

- **CLIP**, သည် ပုံများနှင့် စာသားမှ ဒေတာ၏ ကိန်းဂဏန်း ဖျော်ဖြေရေးကို ဖန်တီးသော မော်ဒယ်ဖြစ်သည်။

- **Diffused attention**, သည် ဖျော်ဖြေရေးများမှ ပုံများကို ဖန်တီးသော မော်ဒယ်ဖြစ်သည်။ DALL-E သည် ပုံများနှင့် စာသားများ၏ dataset ပေါ်တွင် လေ့ကျင့်ထားပြီး၊ စာသားဖေါ်ပြချက်များမှ ပုံများကို ဖန်တီးရန် အသုံးပြုနိုင်သည်။ ဥပမာအားဖြင့် DALL-E သည် ဦးထုပ်တပ်ထားသော ကြောင်တစ်ကောင် သို့မဟုတ် mohawk သွားဖြင့် ခွေးတစ်ကောင်၏ ပုံများကို ဖန်တီးရန် အသုံးပြုနိုင်သည်။

### Midjourney

Midjourney သည် DALL-E နှင့် ဆင်တူသော နည်းလမ်းဖြင့် အလုပ်လုပ်ပြီး၊ စာသား prompt များမှ ပုံများကို ဖန်တီးသည်။ Midjourney သည် "ဦးထုပ်တပ်ထားသော ကြောင်" သို့မဟုတ် "mohawk သွားဖြင့် ခွေး" စသည့် prompt များကို အသုံးပြုပြီး ပုံများကို ဖန်တီးရန် အသုံးပြုနိုင်သည်။

## DALL-E နှင့် Midjourney ဘယ်လိုအလုပ်လုပ်လဲ

ပထမဆုံး [DALL-E](https://arxiv.org/pdf/2102.12092.pdf?WT.mc_id=academic-105485-koreyst)။ DALL-E သည် transformer architecture ကို အခြေခံသော Generative AI မော်ဒယ်ဖြစ်ပြီး _autoregressive transformer_ ပါဝင်သည်။

_autoregressive transformer_ သည် စာသားဖေါ်ပြချက်များမှ ပုံများကို မော်ဒယ်ဘယ်လိုဖန်တီးသလဲဆိုတာ သတ်မှတ်ပြီး၊ တစ် pixel ချင်းစီ ဖန်တီးပြီး၊ ဖန်တီးထားသော pixel များကို နောက် pixel ဖန်တီးရန် အသုံးပြုသည်။ နယူးရယ်ကွန်ယက်တွင် အလွှာများစွာဖြတ်သန်းပြီး၊ ပုံပြီးစီးသည်အထိ။

ဒီလုပ်ငန်းစဉ်ဖြင့် DALL-E သည် ဖန်တီးထားသော ပုံ၏ အင်္ဂါရပ်များ၊ အရာဝတ္ထုများ၊ လက္ခဏာများနှင့် ပိုမိုထိန်းချုပ်နိုင်သည်။ သို့သော် DALL-E 2 နှင့် 3 သည် ဖန်တီးထားသော ပုံကို ပိုမိုထိန်းချုပ်နိုင်သည်။

## ပထမဆုံး ပုံပျိုးထွန်းမှု အက်ပလီကေးရှင်းကို တည်ဆောက်ခြင်း

ဒါဆိုရင် ပုံပျိုးထွန်းမှု အက်ပလီကေးရှင်းတစ်ခုကို တည်ဆောက်ရန် ဘာများလိုအပ်မလဲ? အောက်ပါ library များလိုအပ်သည်-

- **python-dotenv**, သင်၏လျို့ဝှက်ချက်များကို _.env_ ဖိုင်တွင် ကုဒ်မှဝေး၍ ထားရန် ဒီ library ကို အသုံးပြုရန် အလွန်အကြံပြုပါသည်။
- **openai**, ဒီ library သည် OpenAI API နှင့် ဆက်သွယ်ရန် သင်အသုံးပြုမည့် library ဖြစ်သည်။
- **pillow**, Python တွင် ပုံများနှင့် အလုပ်လုပ်ရန်။
- **requests**, HTTP requests များလုပ်ရန် ကူညီရန်။

1. အောက်ပါအကြောင်းအရာနှင့် _env_ ဖိုင်ကို ဖန်တီးပါ-

   ```text
   AZURE_OPENAI_ENDPOINT=<your endpoint>
   AZURE_OPENAI_API_KEY=<your key>
   ```

   Azure Portal တွင် သင်၏ resource အတွက် "Keys and Endpoint" အပိုင်းတွင် ဤအချက်အလက်ကို ရှာပါ။

1. အထက်ပါ library များကို _requirements.txt_ ဖိုင်တွင် အောက်ပါအတိုင်းစုဆောင်းပါ-

   ```text
   python-dotenv
   openai
   pillow
   requests
   ```

1. နောက်တစ်ဆင့် virtual environment ကို ဖန်တီးပြီး library များကို install လုပ်ပါ-

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

   Windows တွင် သင်၏ virtual environment ကို ဖန်တီးပြီး အသက်ဝင်ရန် အောက်ပါ command များကို အသုံးပြုပါ-

   ```bash
   python3 -m venv venv
   venv\Scripts\activate.bat
   ```

1. _app.py_ ဆိုသော ဖိုင်တွင် အောက်ပါကုဒ်ကို ထည့်ပါ-

   ```python
   import openai
   import os
   import requests
   from PIL import Image
   import dotenv

   # import dotenv
   dotenv.load_dotenv()

   # Get endpoint and key from environment variables
   openai.api_base = os.environ['AZURE_OPENAI_ENDPOINT']
   openai.api_key = os.environ['AZURE_OPENAI_API_KEY']

   # Assign the API version (DALL-E is currently supported for the 2023-06-01-preview API version only)
   openai.api_version = '2023-06-01-preview'
   openai.api_type = 'azure'


   try:
       # Create an image by using the image generation API
       generation_response = openai.Image.create(
           prompt='Bunny on horse, holding a lollipop, on a foggy meadow where it grows daffodils',    # Enter your prompt text here
           size='1024x1024',
           n=2,
           temperature=0,
       )
       # Set the directory for the stored image
       image_dir = os.path.join(os.curdir, 'images')

       # If the directory doesn't exist, create it
       if not os.path.isdir(image_dir):
           os.mkdir(image_dir)

       # Initialize the image path (note the filetype should be png)
       image_path = os.path.join(image_dir, 'generated-image.png')

       # Retrieve the generated image
       image_url = generation_response["data"][0]["url"]  # extract image URL from response
       generated_image = requests.get(image_url).content  # download the image
       with open(image_path, "wb") as image_file:
           image_file.write(generated_image)

       # Display the image in the default image viewer
       image = Image.open(image_path)
       image.show()

   # catch exceptions
   except openai.InvalidRequestError as err:
       print(err)

   ```

ဒီကုဒ်ကို ရှင်းပြပါ-

- ပထမဆုံး၊ OpenAI library, dotenv library, requests library, Pillow library စသည့် library များကို import လုပ်ပါ။

  ```python
  import openai
  import os
  import requests
  from PIL import Image
  import dotenv
  ```

- နောက်တစ်ဆင့်၊ _.env_ ဖိုင်မှ environment variables များကို load လုပ်ပါ။

  ```python
  # import dotenv
  dotenv.load_dotenv()
  ```

- ၎င်းနောက် OpenAI API အတွက် endpoint, key, version နှင့် type ကို သတ်မှတ်ပါ။

  ```python
  # Get endpoint and key from environment variables
  openai.api_base = os.environ['AZURE_OPENAI_ENDPOINT']
  openai.api_key = os.environ['AZURE_OPENAI_API_KEY']

  # add version and type, Azure specific
  openai.api_version = '2023-06-01-preview'
  openai.api_type = 'azure'
  ```

- နောက်တစ်ဆင့် ပုံကို ဖန်တီးပါ-

  ```python
  # Create an image by using the image generation API
  generation_response = openai.Image.create(
      prompt='Bunny on horse, holding a lollipop, on a foggy meadow where it grows daffodils',    # Enter your prompt text here
      size='1024x1024',
      n=2,
      temperature=0,
  )
  ```

  အထက်ပါကုဒ်သည် ဖန်တီးထားသော ပုံ၏ URL ပါဝင်သော JSON object ဖြင့် တုံ့ပြန်သည်။ URL ကို အသုံးပြုပြီး ပုံကို download လုပ်ပြီး ဖိုင်အဖြစ် save လုပ်နိုင်သည်။

- နောက်ဆုံး၊ ပုံကို ဖွင့်ပြီး စံပုံကြည့်ရှုကိရိယာကို အသုံးပြုပြီး ပြသပါ-

  ```python
  image = Image.open(image_path)
  image.show()
  ```

### ပုံကို ဖန်တီးခြင်းအပိုင်းများကို ပိုမိုသေချာလေ့လာခြင်း

ပုံကို ဖန်တီးသော ကုဒ်ကို ပိုမိုသေချာလေ့လာပါ-

```python
generation_response = openai.Image.create(
        prompt='Bunny on horse, holding a lollipop, on a foggy meadow where it grows daffodils',    # Enter your prompt text here
        size='1024x1024',
        n=2,
        temperature=0,
    )
```

- **prompt**, သည် ပုံကို ဖန်တီးရန် အသုံးပြုသော စာသား prompt ဖြစ်သည်။ ဤအခါတွင် "Bunny on horse, holding a lollipop, on a foggy meadow where it grows daffodils" prompt ကို အသုံးပြုပါသည်။
- **size**, သည် ဖန်တီးသော ပုံ၏ အရွယ်အစားဖြစ်သည်။ ဤအခါတွင် 1024x1024 pixels ပုံကို ဖန်တီးပါသည်။
- **n**, သည် ဖန်တီးသော ပုံများ၏ အရေအတွက်ဖြစ်သည်။ ဤအခါတွင် ပုံနှစ်ခုကို ဖန်တီးပါသည်။
- **temperature**, သည် Generative AI မော်ဒယ်၏ output ၏ မတည်ငြိမ်မှုကို ထိန်းချုပ်သော parameter ဖြစ်သည်။ temperature သည် 0 မှ 1 အထိတန်ဖိုးဖြစ်ပြီး၊ 0 ဆိုသည်မှာ output ကို deterministic ဖြစ်စေပြီး၊ 1 ဆိုသည်မှာ output ကို random ဖြစ်စေသည်။ အလိုလျောက်တန်ဖိုးသည် 0.7 ဖြစ်သည်။

ပုံများနှင့်လုပ်ဆောင်နိုင်သည့် အခြားအရာများကို နောက်အပိုင်းတွင် ဖော်ပြပါမည်။

## ပုံပျိုးထွန်းမှု၏ အပိုဆောင်းစွမ်းရည်များ

Python တွင် အချို့သောကြောင်းချောင်းများဖြင့် ပုံကို ဖန်တီးနိုင်သည့်ပုံကို သင်မြင်ခဲ့ပါပြီ။ သို့သော် ပုံများနှင့် လုပ်ဆောင်နိုင်သည့် အခြားအရာများရှိသည်။

သင်သည် အောက်ပါအရာများကိုလည်း လုပ်နိုင်သည်-

- **တည်းဖြတ်မှုများ လုပ်ဆောင်ခြင်း**။ ရှိပြီးသား ပုံကို mask နှင့် prompt ပေးခြင်းဖြင့် ပုံကို ပြောင်းလဲနိုင်သည်။ ဥပမာအားဖြင့် ပုံ၏ အပိုင်းတစ်ခုကို အရာဝတ္ထုတစ်ခု ထည့်နိုင်သည်။ ကျွန်ုပ်တို့၏ ကြွက်ပုံကို စဉ်းစားပါ၊ ကြွက်ကို ဦးထုပ်ထည့်နိုင်သည်။ ၎င်းကို လုပ်ဆောင်ရန် ပုံကို၊ mask (ပြောင်းလဲရန် အပိုင်းကို သတ်မှတ်ခြင်း) နှင့် ပြောင်းလဲရမည့် အရာကို စာသား prompt ဖြင့် ပေးသည်။

  ```python
  response = openai.Image.create_edit(
    image=open("base_image.png", "rb"),
    mask=open("mask.png", "rb"),
    prompt="An image of a rabbit with a hat on its head.",
    n=1,
    size="1024x1024"
  )
  image_url = response['data'][0]['url']
  ```

  အခြေခံပုံတွင် ကြွက်သာပါဝင်ပြီး၊ အဆုံးအဖြတ်ပုံတွင် ကြွက်၏ ဦးထုပ်ပါဝင်သည်။

- **မျိုးစုံဖန်တီးခြင်း**။ အဆိုပါ အတွေးသည် ရှိပြီးသား ပုံကို ယူပြီး မျိုးစုံဖန်တီးရန် မျှော်မှန်းချက်ကို ပြုလုပ်ခြင်းဖြစ်သည်။ မျိုးစုံဖန်တီးရန် ပုံနှင့် စာသား prompt ကို ပေးပြီး ကုဒ်ကို အောက်ပါအတိုင်း လုပ်ဆောင်ပါ-

  ```python
  response = openai.Image.create_variation(
    image=open("bunny-lollipop.png", "rb"),
    n=1,
    size="1024x1024"
  )
  image_url = response['data'][0]['url']
  ```

  > မှတ်ချက်၊ ဤသည် OpenAI မှသာ ပံ့ပိုးထားသည်
