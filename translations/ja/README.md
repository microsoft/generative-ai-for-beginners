# 初心者向け生成AI（バージョン3） - コース

Microsoft Cloud Advocatesによる21のレッスンで構成された包括的なコースで、生成AIアプリケーションの基礎を学びましょう。

## 🌱 はじめに

このコースは21のレッスンで構成されています。各レッスンはそれぞれのトピックを扱っているので、好きなところから始めてください！

レッスンは、「Learn」レッスンとして生成AIの概念を説明するものと、「Build」レッスンとして概念とコード例を**Python**と**TypeScript**で説明するものに分類されています。

.NET開発者の方は[初心者向け生成AI（.NET版）](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)をご覧ください！

各レッスンには追加の学習ツールを含む「Keep Learning」セクションもあります。

## 必要なもの
### このコースのコードを実行するには、以下のいずれかを使用できます：
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **レッスン:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **レッスン:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **レッスン:** "oai-assignment" 

- PythonまたはTypeScriptの基本的な知識があると役立ちます - \*初心者の方はこれらの[Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst)と[TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)コースをご覧ください
- [このリポジトリ全体をフォークする](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst)ためのGitHubアカウント

開発環境の設定を支援するために、**[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**レッスンを作成しました。

後で見つけやすくするために[このリポジトリをスター（🌟）する](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)ことを忘れないでください。

## 🧠 デプロイ準備完了？

より高度なコードサンプルを探している場合は、**Python**と**TypeScript**の両方で提供されている[生成AIコードサンプルのコレクション](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst)をご覧ください。

## 🚀 スタートアップを立ち上げる？
