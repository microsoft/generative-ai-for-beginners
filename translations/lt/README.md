# Generatyvioji AI pradedantiesiems (3 versija) - kursas

Sužinokite pagrindus, kaip kurti generatyviosios AI programas su Microsoft Cloud Advocates parengtu 21 pamokos išsamiu kursu.

## 🌱 Pradžia

Šiame kurse yra 21 pamoka. Kiekviena pamoka apima savo temą, tad pradėkite nuo bet kurios!

Pamokos paženklintos kaip "Išmokite" - paaiškinančios generatyviosios AI koncepciją arba "Kurti" - paaiškinančios koncepciją ir pateikiančios kodų pavyzdžius tiek **Python**, tiek **TypeScript**, kai įmanoma.

.NET programuotojams rekomenduojame [Generatyvioji AI pradedantiesiems (.NET leidimas)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Kiekviena pamoka taip pat turi skiltį "Toliau mokytis" su papildomomis mokymosi priemonėmis.

## Ko reikia
### Norėdami paleisti šio kurso kodą, galite naudoti:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Pamokos:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Pamokos:** "githubmodels" (GitHub Models pasitrauks 2026 liepos pabaigoje - naudokite vietoje Microsoft Foundry Models)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Pamokos:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - paleiskite modelius visiškai neprisijungę savo įrenginyje, nereikia debesijos prenumeratos
   
- Pagrindinės Python arba TypeScript žinios naudingos - *Pradedantiesiems rekomenduojame šiuos [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ir [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) kursus
- GitHub paskyra, kad galėtumėte [forkinti šią saugyklą](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) į savo GitHub paskyrą

Sukūrėme **[Kurso sąranka](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** pamoką, kuri padės jums nustatyti kūrimo aplinką.

Nepamirškite [pažymėti žvaigždute (🌟) šią saugyklą](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), kad vėliau ją būtų lengviau rasti.
