# Generatyvusis DI pradedantiesiems (3 versija) – Kursas

Išmokite generatyviųjų DI programų kūrimo pagrindus su 21 pamokos visapusišku Microsoft Cloud Advocates parengtu kursu.

## 🌱 Pradžia

Šis kursas turi 21 pamoką. Kiekviena pamoka nagrinėja savo temą, tad pradėkite bet kur!

Pamokos yra pažymėtos kaip "Learn" (mokytis) – paaiškinančios generatyviojo DI konceptą arba "Build" (kurti) – paaiškinančios konceptą ir pateikiančios kodo pavyzdžius tiek **Python**, tiek **TypeScript** kalbomis, kai tai įmanoma.

.NET kūrėjams siūlome [Generatyvusis DI pradedantiesiems (.NET leidimas)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Kiekvienoje pamokoje taip pat yra skyrius „Toliau mokytis“ su papildomomis mokymosi priemonėmis.

## Ko jums reikia
### Norint vykdyti šio kurso kodą, galite naudoti:
 - [Azure OpenAI paslaugą](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Pamokos:** „aoai-assignment“
 - [GitHub Marketplace modelių katalogą](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **Pamokos:** „githubmodels“
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Pamokos:** „oai-assignment“ 
   
- Pagrindinės Python ar TypeScript žinios yra naudingos – \*Visiems absoliučiams pradedantiesiems rekomenduojame šiuos [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ir [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) kursus
- GitHub paskyrą, kad galėtumėte [padaryti forką šiam repozitoriumui](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) į savo GitHub paskyrą

Mes parengėme **[Kurso pradžios nustatymo](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** pamoką, kuri padės jums susikurti vystymosi aplinką.

Neužmirškite [pažymėti (🌟) šį repozitoriumą](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), kad vėliau jį lengviau rastumėte.
