# IA Generativa per Principianti (Versione 3) - Un corso

Impara le basi per costruire applicazioni di IA generativa con il nostro corso completo di 21 lezioni tenuto dai Microsoft Cloud Advocates.

## üå± Per iniziare

Questo corso comprende 21 lezioni. Ogni lezione tratta un argomento diverso, quindi inizia dove preferisci!

Le lezioni sono etichettate come lezioni "Learn" che spiegano un concetto di IA generativa o lezioni "Build" che spiegano un concetto e forniscono esempi di codice in **Python** e **TypeScript** quando possibile.

Per gli sviluppatori .NET consulta [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Ogni lezione include anche una sezione "Continua a imparare" con strumenti di apprendimento aggiuntivi.
