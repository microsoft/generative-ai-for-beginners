# IA Generativa per Principianti (Versione 3) - Un Corso

Impara le basi della creazione di applicazioni di IA Generativa con il nostro corso completo di 21 lezioni realizzato dai Microsoft Cloud Advocates.

## 🌱 Per Iniziare

Questo corso ha 21 lezioni. Ogni lezione tratta un argomento specifico, quindi inizia da dove vuoi!

Le lezioni sono etichettate come "Learn" per imparare un concetto di IA Generativa o "Build" che spiegano un concetto con esempi di codice sia in **Python** che in **TypeScript**, quando possibile.

Per sviluppatori .NET, dai un'occhiata a [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Ogni lezione include anche una sezione "Keep Learning" con ulteriori strumenti di apprendimento.

## Cosa Ti Serve
### Per eseguire il codice di questo corso, puoi usare uno tra: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Lezioni:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Lezioni:** "githubmodels" (GitHub Models sarà ritirato alla fine di luglio 2026 - usa invece Microsoft Foundry Models)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Lezioni:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Esegui i modelli completamente offline sul tuo dispositivo, senza bisogno di abbonamento cloud
   
- Conoscenze di base di Python o TypeScript sono utili - \*Per assoluti principianti guarda questi corsi [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) e [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Un account GitHub per [fare fork di questo intero repo](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) sul tuo account GitHub

Abbiamo creato una lezione **[Installazione del Corso](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** per aiutarti a configurare il tuo ambiente di sviluppo.

Non dimenticare di [mettere una stella (🌟) a questo repo](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) per trovarlo più facilmente in seguito.
