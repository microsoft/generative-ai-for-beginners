# Generatiivinen tekoäly aloittelijoille (Versio 3) - Kurssi

Opettele generatiivisten tekoälysovellusten rakentamisen perusteet Microsoft Cloud Advocatesin 21 oppitunnin kattavalla kurssilla.

## 🌱 Aloittaminen

Tässä kurssissa on 21 oppituntia. Jokainen oppitunti käsittelee omaa aihettaan, joten aloita mistä haluat!

Oppitunnit on merkitty joko "Learn"-oppitunneiksi, joissa selitetään generatiivisen tekoälyn käsitteitä, tai "Build"-oppitunneiksi, joissa käsitellään sekä käsitettä että tarjotaan koodiesimerkkejä **Python**illa ja **TypeScript**illä kun mahdollista.

.NET-kehittäjille on tarjolla [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Jokaisessa oppitunnissa on myös "Keep Learning" -osio, joka sisältää lisäoppimateriaaleja.

## Mitä tarvitset
### Tämä kurssi toimii koodin ajoon seuraavilla työkaluilla: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Oppitunnit:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **Oppitunnit:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Oppitunnit:** "oai-assignment"
   
- Perustiedot Pythonista tai TypeScriptistä ovat hyödyllisiä - \*Täysin aloittelijoille suosittelemme näitä [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ja [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) -kursseja
- GitHub-tili voidaksesi [tähdätä (fork) tämän koko repositorion](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) omalle GitHub-tilillesi

Olemme luoneet **[Kurssin asetukset](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** -oppitunnin auttamaan kehitysympäristön pystyttämisessä.

Älä unohda [tähdätä (🌟) tätä repositoriota](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) löytääksesi sen helpommin myöhemmin.
