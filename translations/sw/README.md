# Generative AI kwa Komunyo (Toleo la 3) - Kozi

Jifunze misingi ya kujenga programu za Generative AI kwa kozi yetu ya kina yenye masomo 21 kutoka kwa Microsoft Cloud Advocates.

## üå± Kuanzia

Kozi hii ina masomo 21. Kila somo lina mada yake binafsi, hivyo anza popote unapotaka!

Masomo yamewekwa lebo kama "Jifunze" yanayofafanua dhana ya Generative AI au "Jenga" yanayoelezea dhana na mifano ya msimbo katika **Python** na **TypeScript** inapowezekana.

Kwa Waendelezaji wa .NET angalia [Generative AI kwa Komunyo (Toleo la .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Kila somo pia lina sehemu ya "Endelea Kujifunza" yenye zana za ziada za kujifunza.

## Unachohitaji 
### Kuendesha msimbo wa kozi hii, unaweza kutumia yoyote kati ya: 
 - [Huduma ya Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Masomo:** "aoai-assignment"
 - [Katalogi ya Modeli ya Soko la GitHub](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **Masomo:** "githubmodels"
 - [API ya OpenAI](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Masomo:** "oai-assignment" 
   
- Ujuzi wa msingi wa Python au TypeScript ni msaada - \*Kwa wanaoanza kabisa angalia hizi [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) na [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) kozi
- Akaunti ya GitHub ili [kuhaia hii repo yote](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) kwa akaunti yako ya GitHub

Tumetengeneza somo la **[Kuweka Kozi](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** kusaidia kuweka mazingira yako ya maendeleo.

Usisahau [kuipenda (üåü) repo hii](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ili kuihifadhi kwa urahisi baadaye.
