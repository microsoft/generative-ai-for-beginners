# Generative AI kwa Waanzilishi (Toleo la 3) - Kozi

Jifunze misingi ya kujenga programu za Generative AI na masomo yetu kamili ya masomo 21 kutoka kwa Microsoft Cloud Advocates.

## üå± Kuanzia

Kozi hii ina masomo 21. Kila somo linashughulikia mada yake mwenyewe hivyo anza popote unapopendelea!

Masomo yamewekwa lebo kama "Jifunze" yanayofafanua dhana ya Generative AI au masomo ya "Jenga" yanayofafanua dhana na mifano ya msimbo kwa **Python** na **TypeScript** pale inapowezekana.

Kwa Waendelezaji wa .NET angalia [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Kila somo pia linajumuisha sehemu ya "Endelea Kujifunza" na zana za ziada za kujifunzia.
