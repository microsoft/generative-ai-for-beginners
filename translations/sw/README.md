# Generative AI kwa Waanzaji (Toleo la 3) - Kozi

Jifunze misingi ya kujenga programu za Generative AI na masomo yetu 21-lesson yaliyojumuisha yaliyotengenezwa na Microsoft Cloud Advocates.

## 🌱 Kuanzia

Kozi hii ina masomo 21. Kila somo linahusu mada yake, hivyo anza mahali popote unapotaka!

Masomo yamewekwa alama kuwa ni "Jifunze" yanayoelezea dhana ya Generative AI au "Jenga" yanayoelezea dhana na mifano ya msimbo kwa **Python** na **TypeScript** inapowezekana.

Kwa waendelezaji wa .NET angalia [Generative AI kwa Waanzaji (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Kila somo pia lina sehemu ya "Endelea Kujifunza" na zana za ziada za kujifunza.

## Unachohitaji
### Kuendesha msimbo wa kozi hii, unaweza kutumia:
 - [Huduma ya Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Masomo:** "aoai-assignment"
 - [Katalogi ya Mfano ya Soko la GitHub](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **Masomo:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Masomo:** "oai-assignment"
   
- Maarifa ya msingi ya Python au TypeScript ni msaada - \*Kwa waanzaji kabisa angalia kozi hizi za [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) na [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Akaunti ya GitHub kwa [kuchoma nakala ya hifadhidata hii yote](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) kwenye akaunti yako ya GitHub

Tumeunda somo la **[Kuweka Kozi](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** kusaidia kuanzisha mazingira yako ya maendeleo.

Usisahau [kuipa nyota (🌟) hifadhidata hii](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ili kuipata kwa urahisi baadaye.
