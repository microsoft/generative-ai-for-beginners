# AI Inayotengeneza kwa Waanzilishi (Toleo la 3) - Kozi

Jifunze misingi ya kujenga programu za AI Inayotengeneza kupitia kozi yetu kamili ya masomo 21 kutoka kwa Wakili wa Wingu wa Microsoft.

## 🌱 Kuanzia

Kozi hii ina masomo 21. Kila somo linaelezea mada yake hivyo anza mahali popote unapotaka!

Masomo yameandikwa kama "Jifunze" yanayoelezea dhana za AI Inayotengeneza au "Jenga" masomo yanayoelezea dhana na mifano ya nambari kwa **Python** na **TypeScript** pale inapowezekana.

Kwa Waendelezaji wa .NET angalia [AI Inayotengeneza kwa Waanzilishi (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Kila somo pia lina sehemu ya "Endelea Kujifunza" yenye zana za kujifunzia zaidi.

## Unachohitaji
### Kuendesha nambari za kozi hii, unaweza kutumia moja kati ya:
 - [Huduma ya Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Masomo:** "aoai-assignment"
 - [Modeli za Microsoft Foundry](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Masomo:** "githubmodels" (GitHub Models inakwisha mwisho wa Julai 2026 - tumia Microsoft Foundry Models badala yake)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Masomo:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Endesha modeli kikamilifu offiline kwenye kifaa chako binafsi, hakuna usajili wa wingu unahitajika
   
- Ujuzi wa msingi wa Python au TypeScript ni msaada - \*Kwa wanaoanza kabisa angalia kozi hizi za [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) na [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Akaunti ya GitHub ili [kufork hii repo yote](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) hadi akaunti yako ya GitHub

Tumeunda somo la **[Ufungaji wa Kozi](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** kusaidia na usanidi wa mazingira yako ya maendeleo.

Usisahau [kutoa nyota (🌟) kwenye repo hii](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ili kuipata kwa urahisi baadaye.
