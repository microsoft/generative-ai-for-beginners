# Εισαγωγή στα Νευρωνικά Δίκτυα. Πολυστρωματικός Αντιληπτικός

Στην προηγούμενη ενότητα, μάθατε για το πιο απλό μοντέλο νευρωνικού δικτύου - τον μονοστρωματικό αντιληπτικό, ένα γραμμικό μοντέλο ταξινόμησης δύο κλάσεων.

Σε αυτή την ενότητα θα επεκτείνουμε αυτό το μοντέλο σε ένα πιο ευέλικτο πλαίσιο, που μας επιτρέπει να:

* εκτελούμε **ταξινόμηση πολλαπλών κλάσεων** επιπλέον της ταξινόμησης δύο κλάσεων
* λύνουμε **προβλήματα παλινδρόμησης** επιπλέον της ταξινόμησης
* διαχωρίζουμε κλάσεις που δεν είναι γραμμικά διαχωρίσιμες

Θα αναπτύξουμε επίσης το δικό μας αρθρωτό πλαίσιο σε Python που θα μας επιτρέπει να κατασκευάζουμε διαφορετικές αρχιτεκτονικές νευρωνικών δικτύων.

## Τυποποίηση της Μηχανικής Μάθησης

Ας ξεκινήσουμε με την τυποποίηση του προβλήματος της Μηχανικής Μάθησης. Ας υποθέσουμε ότι έχουμε ένα σύνολο εκπαίδευσης **X** με ετικέτες **Y**, και πρέπει να κατασκευάσουμε ένα μοντέλο *f* που θα κάνει τις πιο ακριβείς προβλέψεις. Η ποιότητα των προβλέψεων μετράται από τη **Συνάρτηση Απώλειας** ℒ. Οι ακόλουθες συναρτήσεις απώλειας χρησιμοποιούνται συχνά:

* Για πρόβλημα παλινδρόμησης, όταν πρέπει να προβλέψουμε έναν αριθμό, μπορούμε να χρησιμοποιήσουμε **απόλυτο σφάλμα** ∑<sub>i</sub>|f(x<sup>(i)</sup>)-y<sup>(i)</sup>|, ή **τετραγωνικό σφάλμα** ∑<sub>i</sub>(f(x<sup>(i)</sup>)-y<sup>(i)</sup>)<sup>2</sup>
* Για ταξινόμηση, χρησιμοποιούμε **0-1 απώλεια** (που ουσιαστικά είναι το ίδιο με την **ακρίβεια** του μοντέλου), ή **λογιστική απώλεια**.

Για τον μονοστρωματικό αντιληπτικό, η συνάρτηση *f* οριζόταν ως γραμμική συνάρτηση *f(x)=wx+b* (εδώ *w* είναι ο πίνακας βαρών, *x* είναι το διάνυσμα χαρακτηριστικών εισόδου, και *b* είναι το διάνυσμα μετατόπισης). Για διαφορετικές αρχιτεκτονικές νευρωνικών δικτύων, αυτή η συνάρτηση μπορεί να πάρει πιο σύνθετη μορφή.

> Στην περίπτωση της ταξινόμησης, είναι συχνά επιθυμητό να λαμβάνουμε πιθανότητες των αντίστοιχων κλάσεων ως έξοδο του δικτύου. Για να μετατρέψουμε αυθαίρετους αριθμούς σε πιθανότητες (π.χ. να κανονικοποιήσουμε την έξοδο), συχνά χρησιμοποιούμε τη συνάρτηση **softmax** σ, και η συνάρτηση *f* γίνεται *f(x)=σ(wx+b)*

Στον ορισμό της *f* παραπάνω, τα *w* και *b* ονομάζονται **παράμετροι** θ=⟨*w,b*⟩. Δεδομένου του συνόλου δεδομένων ⟨**X**,**Y**⟩, μπορούμε να υπολογίσουμε το συνολικό σφάλμα στο σύνολο δεδομένων ως συνάρτηση των παραμέτρων θ.

> ✅ **Ο στόχος της εκπαίδευσης νευρωνικού δικτύου είναι να ελαχιστοποιηθεί το σφάλμα μεταβάλλοντας τις παραμέτρους θ**

## Βελτιστοποίηση με Κατάβαση Κλίσης

Υπάρχει μια γνωστή μέθοδος βελτιστοποίησης συναρτήσεων που ονομάζεται **κατάβαση κλίσης**. Η ιδέα είναι ότι μπορούμε να υπολογίσουμε την παράγωγο (στην πολυδιάστατη περίπτωση καλούμενη **κλίση**) της συνάρτησης απώλειας ως προς τις παραμέτρους και να μεταβάλουμε τις παραμέτρους με τέτοιο τρόπο ώστε το σφάλμα να μειωθεί. Αυτό μπορεί να τυποποιηθεί ως εξής:

* Αρχικοποιήστε τις παραμέτρους με κάποιες τυχαίες τιμές w<sup>(0)</sup>, b<sup>(0)</sup>
* Επαναλάβετε το παρακάτω βήμα πολλές φορές:
    - w<sup>(i+1)</sup> = w<sup>(i)</sup>-η∂ℒ/∂w
    - b<sup>(i+1)</sup> = b<sup>(i)</sup>-η∂ℒ/∂b

Κατά την εκπαίδευση, τα βήματα βελτιστοποίησης υποτίθεται ότι υπολογίζονται λαμβάνοντας υπόψη ολόκληρο το σύνολο δεδομένων (θυμηθείτε ότι η απώλεια υπολογίζεται ως άθροισμα όλων των δειγμάτων εκπαίδευσης). Ωστόσο, στην πραγματική ζωή παίρνουμε μικρά μέρη του συνόλου δεδομένων που ονομάζονται **minibatches**, και υπολογίζουμε τις κλίσεις με βάση ένα υποσύνολο δεδομένων. Επειδή το υποσύνολο λαμβάνεται τυχαία κάθε φορά, αυτή η μέθοδος ονομάζεται **στοχαστική κατάβαση κλίσης** (SGD).

## Πολυστρωματικοί Αντιληπτικοί και Ανάδραση

Ένα μονοστρωματικό δίκτυο, όπως έχουμε δει παραπάνω, είναι ικανό να ταξινομεί γραμμικά διαχωρίσιμες κλάσεις. Για να κατασκευάσουμε ένα πλουσιότερο μοντέλο, μπορούμε να συνδυάσουμε πολλά στρώματα του δικτύου. Μαθηματικά, αυτό θα σημαίνει ότι η συνάρτηση *f* θα έχει πιο σύνθετη μορφή και θα υπολογίζεται σε διάφορα βήματα:
* z<sub>1</sub>=w<sub>1</sub>x+b<sub>1</sub>
* z<sub>2</sub>=w<sub>2</sub>α(z<sub>1</sub>)+b<sub>2</sub>
* f = σ(z<sub>2</sub>)

Εδώ, α είναι μια **μη γραμμική συνάρτηση ενεργοποίησης**, σ είναι η συνάρτηση softmax, και οι παράμετροι θ=<*w<sub>1</sub>,b<sub>1</sub>,w<sub>2</sub>,b<sub>2</sub>*>.

Ο αλγόριθμος κατάβασης κλίσης θα παραμείνει ο ίδιος, αλλά θα είναι πιο δύσκολο να υπολογιστούν οι κλίσεις. Δεδομένου του κανόνα διαφορικής αλυσίδας, μπορούμε να υπολογίσουμε τις παραγώγους ως:

* ∂ℒ/∂w<sub>2</sub> = (∂ℒ/∂σ)(∂σ/∂z<sub>2</sub>)(∂z<sub>2</sub>/∂w<sub>2</sub>)
* ∂ℒ/∂w<sub>1</sub> = (∂ℒ/∂σ)(∂σ/∂z<sub>2</sub>)(∂z<sub>2</sub>/∂α)(∂α/∂z<sub>1</sub>)(∂z<sub>1</sub>/∂w<sub>1</sub>)

> ✅ Ο κανόνας διαφορικής αλυσίδας χρησιμοποιείται για τον υπολογισμό των παραγώγων της συνάρτησης απώλειας ως προς τις παραμέτρους.

Σημειώστε ότι το αριστερό μέρος όλων αυτών των εκφράσεων είναι το ίδιο, και έτσι μπορούμε να υπολογίσουμε αποτελεσματικά τις παραγώγους ξεκινώντας από τη συνάρτηση απώλειας και πηγαίνοντας "προς τα πίσω" μέσω του γραφήματος υπολογισμού. Έτσι, η μέθοδος εκπαίδευσης ενός πολυστρωματικού αντιληπτικού ονομάζεται **ανάδραση**, ή 'backprop'.
