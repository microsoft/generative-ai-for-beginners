# Πόροι για Αυτοκαθοδηγούμενη Μάθηση

Το μάθημα δημιουργήθηκε χρησιμοποιώντας βασικούς πόρους από τις OpenAI και Microsoft Foundry ως αναφορές για τη ορολογία και τα σεμινάρια. Ακολουθεί μια μη εξαντλητική λίστα για τα δικά σας ταξίδια αυτοκαθοδηγούμενης μάθησης. Κάθε σύνδεσμος παρακάτω οδηγεί σε τρέχον, υποστηριζόμενο υλικό.

## 1. Κύριοι Πόροι

| Τίτλος/Σύνδεσμος | Περιγραφή |
| :--- | :--- |
| [Fine-tuning with OpenAI Models](https://platform.openai.com/docs/guides/fine-tuning?WT.mc_id=academic-105485-koreyst) | Η βελτιστοποίηση βελτιώνει την εκμάθηση με λίγα παραδείγματα (few-shot learning) εκπαιδεύοντας σε πολύ περισσότερα παραδείγματα από αυτά που χωρούν στην πρόταση – μειώνοντας το κόστος, βελτιώνοντας την ποιότητα απόκρισης και επιτρέποντας αιτήματα με χαμηλότερη καθυστέρηση. **Πάρτε μια επισκόπηση της βελτιστοποίησης από την OpenAI.** |
| [When to use Microsoft Foundry fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/concepts/fine-tuning-considerations?WT.mc_id=academic-105485-koreyst) | Κατανοήστε **τι είναι η βελτιστοποίηση (έννοια)**, γιατί πρέπει να την εξετάσετε, ποια δεδομένα να χρησιμοποιήσετε και πώς να μετρήσετε την ποιότητα – καθώς και πότε το SFT, DPO ή RFT είναι η σωστή επιλογή. |
| [Customize a model with fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?WT.mc_id=academic-105485-koreyst) | Η πλήρης **οδηγία βήμα προς βήμα (διαδικασία)** για βελτιστοποίηση στο Microsoft Foundry χρησιμοποιώντας το portal, το OpenAI / Foundry Python SDK ή το REST API – καλύπτοντας την προετοιμασία δεδομένων, εκπαίδευση, σημεία ελέγχου και ανάπτυξη. |
| [Continuous fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?WT.mc_id=academic-105485-koreyst#perform-continuous-fine-tuning) | Η επαναληπτική διαδικασία επιλογής ενός ήδη βελτιστοποιημένου μοντέλου ως βασικού και η **περαιτέρω βελτιστοποίησή του** σε νέους συνόλους παραδειγμάτων εκπαίδευσης. |
| [Fine-tuning with tool (function) calling](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning-functions?WT.mc_id=academic-105485-koreyst) | Η βελτιστοποίηση του μοντέλου σας **με παραδείγματα κλήσης εργαλείων** βελτιώνει την έξοδο – πιο ακριβείς, συνεπείς, ομοιόμορφα μορφοποιημένες απαντήσεις χρησιμοποιώντας λιγότερους τοκέν στην πρόταση. |
| [Fine-tuning models: Microsoft Foundry guidance](https://learn.microsoft.com/azure/ai-foundry/foundry-models/concepts/models-sold-directly-by-azure?WT.mc_id=academic-105485-koreyst#fine-tuning-models) | Ανατρέξτε **ποια μοντέλα μπορούν να βελτιστοποιηθούν**, τις μεθόδους που υποστηρίζουν (SFT / DPO / RFT) και τις περιοχές όπου είναι διαθέσιμα. |
| [Fine-tuning overview: techniques and modalities](https://learn.microsoft.com/azure/ai-foundry/concepts/fine-tuning-overview?WT.mc_id=academic-105485-koreyst) | Συγκρίνετε τις τρεις τεχνικές εκπαίδευσης (SFT, DPO, RFT) και τις δύο μορφές (serverless έναντι managed compute), με καθοδήγηση για την επιλογή βασικού μοντέλου και την έναρξη. |
| **Tutorial**: [Fine-tune a model in Microsoft Foundry](https://learn.microsoft.com/azure/ai-foundry/openai/tutorials/fine-tune?WT.mc_id=academic-105485-koreyst) | Δημιουργήστε ένα δείγμα δεδομένων, προετοιμαστείτε για βελτιστοποίηση, εκτελέστε μια εργασία βελτιστοποίησης σε ένα επίκαιρα υποστηριζόμενο μοντέλο όπως το `gpt-4.1-mini` και αναπτύξτε το βελτιστοποιημένο μοντέλο στο Azure. |
| **Tutorial**: [Fine-tune models with serverless API deployments](https://learn.microsoft.com/azure/ai-foundry/how-to/fine-tune-serverless?WT.mc_id=academic-105485-koreyst) | Προσαρμόστε τα ανοιχτά και συνεργατικά μοντέλα (Phi, Llama, Mistral και άλλα) στα σύνολα δεδομένων σας _χρησιμοποιώντας μια ροή εργασίας με χαμηλή κωδικοποίηση, βασισμένη σε UI_ στο Microsoft Foundry. |
| **Tutorial**: [Fine-tune Hugging Face models on Azure Databricks](https://learn.microsoft.com/azure/databricks/machine-learning/train-model/huggingface/fine-tune-model?WT.mc_id=academic-105485-koreyst) | Βελτιστοποιήστε ένα μοντέλο Hugging Face με τη βιβλιοθήκη `transformers` σε μία GPU χρησιμοποιώντας το Azure Databricks και τον Hugging Face Trainer. |
| **Training**: [Fine-tune a foundation model with Azure Machine Learning](https://learn.microsoft.com/training/modules/finetune-foundation-model-with-azure-machine-learning/?WT.mc_id=academic-105485-koreyst) | Ο κατάλογος μοντέλων του Azure Machine Learning προσφέρει πολλά ανοιχτού κώδικα μοντέλα που μπορείτε να βελτιστοποιήσετε. Μέρος της [διαδρομής μάθησης Azure ML Generative AI](https://learn.microsoft.com/training/paths/work-with-generative-models-azure-machine-learning/?WT.mc_id=academic-105485-koreyst). |
| **Tutorial**: [Azure OpenAI fine-tuning with Weights & Biases](https://docs.wandb.ai/guides/integrations/azure-openai-fine-tuning?WT.mc_id=academic-105485-koreyst) | Παρακολουθήστε και αναλύστε τις εκτελέσεις βελτιστοποίησης στο Azure με W&B. Επέκτεινει τον οδηγό fine-tuning της OpenAI με βήματα ειδικά για Azure και παρακολούθηση πειραμάτων. |

## 2. Δευτερεύοντες Πόροι

Αυτή η ενότητα περιλαμβάνει επιπλέον πόρους που αξίζει να εξερευνήσετε και δεν προλάβαμε να καλύψουμε στο μάθημα. Χρησιμοποιήστε τους για να χτίσετε τη δική σας εξειδίκευση γύρω από αυτό το θέμα.

Συνδυάζεται με τον [οδηγό fine-tuning της OpenAI](https://platform.openai.com/docs/guides/fine-tuning?WT.mc_id=academic-105485-koreyst). |
| **OpenAI Cookbook**: [Fine-tuning για Retrieval Augmented Generation (RAG) με Qdrant](https://cookbook.openai.com/examples/fine-tuned_qa/ft_retrieval_augmented_generation_qdrant?WT.mc_id=academic-105485-koreyst) | Ένα ολοκληρωμένο παράδειγμα βελτιστοποίησης μοντέλων OpenAI για RAG, συνδυάζοντας Qdrant και few-shot learning για να ενισχύσετε την απόδοση και να μειώσετε τις παραποιήσεις. |
| **OpenAI Cookbook**: [Fine-tuning GPT με Weights & Biases](https://cookbook.openai.com/examples/third_party/gpt_finetuning_with_wandb?WT.mc_id=academic-105485-koreyst) | Χρησιμοποιήστε το W&B για να παρακολουθείτε την εκπαίδευση και βελτιστοποίηση μοντέλου. Διαβάστε πρώτα τον [οδηγό OpenAI Fine-Tuning](https://docs.wandb.ai/guides/integrations/openai-fine-tuning/?WT.mc_id=academic-105485-koreyst), μετά δοκιμάστε την άσκηση Cookbook. |
| **Hugging Face Tutorial**: [Πώς να βελτιστοποιήσετε LLMs με Hugging Face TRL](https://www.philschmid.de/fine-tune-llms-in-2024-with-trl?WT.mc_id=academic-105485-koreyst) | Βελτιστοποιήστε ανοιχτά LLMs χρησιμοποιώντας Hugging Face TRL, Transformers και datasets: ορίστε χρήση, στήστε περιβάλλον ανάπτυξης, προετοιμάστε σύνολο δεδομένων, βελτιστοποιήστε, αξιολογήστε και αναπτύξτε. |
| **Hugging Face**: [AutoTrain Advanced](https://github.com/huggingface/autotrain-advanced?WT.mc_id=academic-105485-koreyst) | Μια βιβλιοθήκη χωρίς κωδικοποίηση / με χαμηλή κωδικοποίηση από την Hugging Face για βελτιστοποίηση πολλών τύπων μοντέλων. Τρέξτε την στο δικό σας cloud, σε Hugging Face Spaces ή τοπικά μέσω GUI, CLI ή YAML config. |
| **Unsloth**: [Οδηγός Βελτιστοποίησης LLMs](https://docs.unsloth.ai/get-started/fine-tuning-llms-guide) | Ένα ανοικτό πλαίσιο που απλοποιεί την τοπική βελτιστοποίηση LLM και την ενίσχυση μάθησης, με έτοιμα προς χρήση [notebooks](https://github.com/unslothai/notebooks?WT.mc_id=academic-105485-koreyst). |
