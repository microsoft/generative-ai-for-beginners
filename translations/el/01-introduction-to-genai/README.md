# Εισαγωγή στην Παραγωγική Τεχνητή Νοημοσύνη και τα Μεγάλα Γλωσσικά Μοντέλα

[![Εισαγωγή στην Παραγωγική Τεχνητή Νοημοσύνη και τα Μεγάλα Γλωσσικά Μοντέλα](../../../translated_images/el/01-lesson-banner.2424cfd092f43366.webp)](https://youtu.be/lFXQkBvEe0o?si=6ZBcQTwLJJDpnX0K)

_(Κάντε κλικ στην εικόνα παραπάνω για να δείτε το βίντεο αυτού του μαθήματος)_

Η Παραγωγική Τεχνητή Νοημοσύνη είναι η τεχνητή νοημοσύνη που είναι ικανή να δημιουργεί κείμενο, εικόνες και άλλους τύπους περιεχομένου. Αυτό που την καθιστά μια φανταστική τεχνολογία είναι ότι εκδημοκρατίζει την Τεχνητή Νοημοσύνη, ο καθένας μπορεί να τη χρησιμοποιήσει με όσο το δυνατόν πιο απλό τρόπο, με την απλή εισαγωγή κειμένου, μια πρόταση γραμμένη σε φυσική γλώσσα. Δεν χρειάζεται να μάθετε μια γλώσσα προγραμματισμού όπως Java ή SQL για να πραγματοποιήσετε κάτι χρήσιμο, το μόνο που χρειάζεστε είναι να χρησιμοποιήσετε τη γλώσσα σας, να δηλώσετε τι θέλετε και να προκύψει μια πρόταση από ένα μοντέλο τεχνητής νοημοσύνης. Οι εφαρμογές και ο αντίκτυπος είναι μεγάλες, μπορείτε να γράφετε ή να κατανοείτε αναφορές, να γράφετε εφαρμογές και πολλά άλλα, όλα σε δευτερόλεπτα.

Σε αυτή την εκπαιδευτική ύλη, θα εξερευνήσουμε πώς η startup μας αξιοποιεί την παραγωγική τεχνητή νοημοσύνη για να ξεκλειδώσει νέα σενάρια στον κόσμο της εκπαίδευσης και πώς αντιμετωπίζουμε τις αναπόφευκτες προκλήσεις που σχετίζονται με τις κοινωνικές επιπτώσεις της εφαρμογής της και τους περιορισμούς της τεχνολογίας.

## Εισαγωγή

Αυτό το μάθημα θα καλύψει:

- Εισαγωγή στο επιχειρηματικό σενάριο: η ιδέα και η αποστολή της startup μας.
- Παραγωγική Τεχνητή Νοημοσύνη και πώς φτάσαμε στο τρέχον τεχνολογικό τοπίο.
- Εσωτερική λειτουργία ενός μεγάλου γλωσσικού μοντέλου.
- Κύριες δυνατότητες και πρακτικές χρήσεις των Μεγάλων Γλωσσικών Μοντέλων.

## Στόχοι Μάθησης

Μετά την ολοκλήρωση αυτού του μαθήματος, θα κατανοήσετε:

- Τι είναι η παραγωγική τεχνητή νοημοσύνη και πώς λειτουργούν τα Μεγάλα Γλωσσικά Μοντέλα.
- Πώς μπορείτε να αξιοποιήσετε τα μεγάλα γλωσσικά μοντέλα για διάφορες χρήσεις, με έμφαση σε σενάρια εκπαίδευσης.

## Σενάριο: η εκπαιδευτική startup μας

Η Παραγωγική Τεχνητή Νοημοσύνη (AI) αντιπροσωπεύει την κορυφή της τεχνολογίας AI, ωθώντας τα όρια αυτού που μέχρι πρότινος θεωρούνταν αδύνατο. Τα παραγωγικά μοντέλα AI έχουν πολλές δυνατότητες και εφαρμογές, αλλά σε αυτή την εκπαιδευτική ύλη θα εξερευνήσουμε πώς καινοτομεί την εκπαίδευση μέσω μιας φανταστικής startup. Θα αναφερόμαστε σε αυτή την startup ως _η startup μας_. Η startup μας δραστηριοποιείται στον τομέα της εκπαίδευσης με τον φιλόδοξο στόχο

> _βελτίωση της προσβασιμότητας στη μάθηση, σε παγκόσμια κλίμακα, διασφαλίζοντας ίση πρόσβαση στην εκπαίδευση και παρέχοντας εξατομικευμένες μαθησιακές εμπειρίες σε κάθε μαθητευόμενο, ανάλογα με τις ανάγκες του_.

Η ομάδα της startup μας γνωρίζει ότι δεν θα καταφέρει να πετύχει αυτόν τον στόχο χωρίς να αξιοποιήσει ένα από τα πιο ισχυρά εργαλεία της σύγχρονης εποχής – τα Μεγάλα Γλωσσικά Μοντέλα (LLMs).

Η Παραγωγική Τεχνητή Νοημοσύνη αναμένεται να φέρει επανάσταση στον τρόπο που μαθαίνουμε και διδάσκουμε σήμερα, με τους μαθητές να έχουν στη διάθεσή τους εικονικούς δασκάλους 24 ώρες το 24ωρο που παρέχουν τεράστιες ποσότητες πληροφοριών και παραδειγμάτων, και τους εκπαιδευτικούς να μπορούν να αξιοποιήσουν καινοτόμα εργαλεία για να αξιολογήσουν τους μαθητές τους και να δώσουν ανατροφοδότηση.

![Πέντε νέοι μαθητές κοιτάζουν μια οθόνη - εικόνα από DALLE2](../../../translated_images/el/students-by-DALLE2.b70fddaced1042ee.webp)

Για αρχή, ας ορίσουμε μερικές βασικές έννοιες και ορολογία που θα χρησιμοποιήσουμε σε όλη την εκπαιδευτική ύλη.

## Πώς φτάσαμε στην Παραγωγική ΤΝ;

Παρά τον εξαιρετικό _θόρυβο_ που δημιουργήθηκε πρόσφατα με την ανακοίνωση των παραγωγικών μοντέλων AI, αυτή η τεχνολογία έχει αναπτυχθεί για δεκαετίες, με τις πρώτες ερευνητικές προσπάθειες να χρονολογούνται από τη δεκαετία του ’60. Βρισκόμαστε τώρα σε ένα σημείο όπου η ΤΝ έχει ανθρώπινες γνωστικές ικανότητες, όπως η συνομιλία, όπως δείχνουν για παράδειγμα το [OpenAI ChatGPT](https://openai.com/chatgpt) ή το [Bing Chat](https://www.microsoft.com/edge/features/bing-chat?WT.mc_id=academic-105485-koreyst), το οποίο επίσης χρησιμοποιεί μοντέλο GPT για τις διαδικτυακές αναζητήσεις του Bing.

Πίσω λίγο, τα πρώτα πρωτότυπα ΤΝ αποτελούνταν από πληκτρολογημένα chatbots που βασίζονταν σε μία βάση γνώσεων εξαγόμενη από μια ομάδα εμπειρογνωμόνων και αποθηκευμένη σε υπολογιστή. Οι απαντήσεις στη βάση γνώσεων ενεργοποιούνταν από λέξεις-κλειδιά που εμφανίζονταν στο εισαγόμενο κείμενο.
Ωστόσο, σύντομα έγινε σαφές ότι μια τέτοια προσέγγιση με πληκτρολογημένα chatbots δεν ήταν επεκτάσιμη.

### Μια στατιστική προσέγγιση στην ΤΝ: Μηχανική Μάθηση

Η αλλαγή ήρθε τη δεκαετία του ’90, με την εφαρμογή μιας στατιστικής προσέγγισης στην ανάλυση κειμένου. Αυτό οδήγησε στην ανάπτυξη νέων αλγορίθμων – γνωστών ως μηχανικής μάθησης – ικανοτήτων που διδάσκονται από δεδομένα χωρίς να προγραμματίζονται ρητά. Αυτή η προσέγγιση επιτρέπει στις μηχανές να προσομοιώνουν την κατανόηση της ανθρώπινης γλώσσας: ένα στατιστικό μοντέλο εκπαιδεύεται σε ζεύγη κειμένου-ετικέτας, που επιτρέπει στο μοντέλο να ταξινομεί άγνωστο εισαγόμενο κείμενο με μια προκαθορισμένη ετικέτα που αντιπροσωπεύει την πρόθεση του μηνύματος.

### Νευρωνικά δίκτυα και σύγχρονοι εικονικοί βοηθοί

Τα τελευταία χρόνια, η εξέλιξη του υλικού, ικανό να χειρίζεται μεγαλύτερους όγκους δεδομένων και πιο σύνθετους υπολογισμούς, ενθάρρυνε την έρευνα στην ΤΝ, οδηγώντας στην ανάπτυξη προηγμένων αλγορίθμων μηχανικής μάθησης γνωστών ως νευρωνικά δίκτυα ή αλγόριθμοι βαθιάς μάθησης.

Τα νευρωνικά δίκτυα (και ιδιαίτερα τα Επαναλαμβανόμενα Νευρωνικά Δίκτυα – RNNs) βελτίωσαν σημαντικά την επεξεργασία φυσικής γλώσσας, επιτρέποντας την αναπαράσταση του νοήματος του κειμένου με πιο ουσιαστικό τρόπο, λαμβάνοντας υπόψη το πλαίσιο μια λέξης μέσα σε μια πρόταση.

Αυτή είναι η τεχνολογία που υποστήριξε τους εικονικούς βοηθούς που εμφανίστηκαν στην πρώτη δεκαετία του 21ου αιώνα, πολύ ικανούς στην ερμηνεία της ανθρώπινης γλώσσας, στην αναγνώριση μιας ανάγκης, και στην εκτέλεση μιας δράσης για την κάλυψή της – όπως η απάντηση με προεπιλεγμένα σενάρια ή η χρήση υπηρεσίας τρίτου μέρους.

### Η σύγχρονη εποχή, Παραγωγική ΤΝ

Έτσι φτάσαμε στην Παραγωγική Τεχνητή Νοημοσύνη σήμερα, που μπορεί να θεωρηθεί ως υποσύνολο της βαθιάς μάθησης.

![AI, ML, DL και Παραγωγική ΤΝ](../../../translated_images/el/AI-diagram.c391fa518451a40d.webp)

Μετά από δεκαετίες έρευνας στον τομέα της ΤΝ, μια νέα αρχιτεκτονική μοντέλου – που ονομάζεται _Transformer_ – ξεπέρασε τους περιορισμούς των RNN, ικανή να επεξεργάζεται πολύ μεγαλύτερες ακολουθίες κειμένου ως είσοδο. Οι Transformers βασίζονται στον μηχανισμό προσοχής, που επιτρέπει στο μοντέλο να δίνει διαφορετικό βάρος στις εισόδους που λαμβάνει, «δίνoντας περισσότερη προσοχή» εκεί που συγκεντρώνεται η πιο σημαντική πληροφορία, ανεξάρτητα από τη σειρά που έχουν στην ακολουθία κειμένου.

Τα περισσότερα πρόσφατα παραγωγικά μοντέλα ΤΝ – γνωστά και ως Μεγάλα Γλωσσικά Μοντέλα (LLMs), επειδή δουλεύουν με κειμενικές εισόδους και εξόδους – βασίζονται όντως σε αυτή την αρχιτεκτονική. Το ενδιαφέρον με αυτά τα μοντέλα – που εκπαιδεύονται σε τεράστιους όγκους μη επισημασμένων δεδομένων από διάφορες πηγές όπως βιβλία, άρθρα και ιστότοπους – είναι ότι μπορούν να προσαρμοστούν σε πλήθος εργασιών και να δημιουργήσουν γραμματικώς ορθό κείμενο με μια αίσθηση δημιουργικότητας. Έτσι, όχι μόνο βελτίωσαν πολύ την ικανότητα μιας μηχανής να «κατανοεί» ένα εισαγόμενο κείμενο, αλλά της έδωσαν και την ικανότητα να παράγει μια πρωτότυπη απάντηση σε ανθρώπινη γλώσσα.

## Πώς λειτουργούν τα μεγάλα γλωσσικά μοντέλα;

Στο επόμενο κεφάλαιο θα εξερευνήσουμε διαφορετικούς τύπους παραγωγικών μοντέλων ΤΝ, αλλά προς το παρόν ας δούμε πώς λειτουργούν τα μεγάλα γλωσσικά μοντέλα, με έμφαση στα μοντέλα OpenAI GPT (Generative Pre-trained Transformer).

- **Tokenizer, κείμενο σε αριθμούς**: Τα Μεγάλα Γλωσσικά Μοντέλα λαμβάνουν ως είσοδο ένα κείμενο και παράγουν ως έξοδο ένα κείμενο. Ωστόσο, επειδή είναι στατιστικά μοντέλα, δουλεύουν πολύ καλύτερα με αριθμούς παρά με ακολουθίες κειμένου. Γι’ αυτό κάθε είσοδος επεξεργάζεται από έναν tokenizer, πριν χρησιμοποιηθεί από τον πυρήνα του μοντέλου. Ένα token είναι ένα κομμάτι κειμένου – που αποτελείται από μεταβλητό αριθμό χαρακτήρων, οπότε ο βασικός στόχος του tokenizer είναι να σπάσει την είσοδο σε ένα πίνακα tokens. Στη συνέχεια, κάθε token αντιστοιχίζεται με έναν δείκτη token, που είναι η ακέραια κωδικοποίηση του αρχικού κομματιού κειμένου.

![Παράδειγμα tokenization](../../../translated_images/el/tokenizer-example.80a5c151ee7d1bd4.webp)

- **Πρόβλεψη εξόδου tokens**: Δίνοντας n tokens ως είσοδο (με το μέγιστο n να διαφέρει από μοντέλο σε μοντέλο), το μοντέλο μπορεί να προβλέψει ένα token ως έξοδο. Αυτό το token ενσωματώνεται στη συνέχεια στην είσοδο της επόμενης επανάληψης, σε ένα μοτίβο που επεκτείνει το παράθυρο, επιτρέποντας καλύτερη εμπειρία χρήστη με την παραλαβή μίας (ή πολλαπλών) προτάσεων ως απάντηση. Αυτό εξηγεί γιατί, αν έχετε παίξει με το ChatGPT, ίσως παρατηρήσατε ότι μερικές φορές φαίνεται να σταματάει στη μέση μιας πρότασης.

- **Διαδικασία επιλογής, κατανομή πιθανοτήτων**: Το token εξόδου επιλέγεται από το μοντέλο σύμφωνα με την πιθανότητά του να εμφανιστεί μετά την τρέχουσα ακολουθία κειμένου. Αυτό συμβαίνει επειδή το μοντέλο προβλέπει μια κατανομή πιθανοτήτων για όλα τα πιθανά «επόμενα tokens», υπολογισμένη βάσει της εκπαίδευσής του. Ωστόσο, δεν επιλέγεται πάντα το token με τη μεγαλύτερη πιθανότητα από την κατανομή που παρέχεται. Προστίθεται ένα στοιχείο τυχαιότητας σε αυτή την επιλογή, με τέτοιο τρόπο ώστε το μοντέλο να λειτουργεί με μη-ντετερμινιστικό τρόπο – δεν λαμβάνουμε την ίδια ακριβώς έξοδο για την ίδια ακριβώς είσοδο. Αυτό το στοιχείο τυχαιότητας προστίθεται για να προσομοιωθεί η διαδικασία της δημιουργικής σκέψης και μπορεί να προσαρμοστεί μέσω μιας παραμέτρου του μοντέλου που ονομάζεται θερμοκρασία.

## Πώς μπορεί η startup μας να αξιοποιήσει τα Μεγάλα Γλωσσικά Μοντέλα;

Τώρα που έχουμε καλύτερη κατανόηση της εσωτερικής λειτουργίας ενός μεγάλου γλωσσικού μοντέλου, ας δούμε μερικά πρακτικά παραδείγματα από τις πιο κοινές εργασίες που μπορούν να εκτελέσουν αρκετά καλά, με έμφαση στο επιχειρηματικό μας σενάριο.
Είπαμε ότι η κύρια δυνατότητα ενός Μεγάλου Γλωσσικού Μοντέλου είναι _να παράγει κείμενο από την αρχή, ξεκινώντας από μια κειμενική είσοδο, γραμμένη σε φυσική γλώσσα_.

Αλλά τι είδος κειμένου εισόδου και εξόδου;
Η είσοδος ενός μεγάλου γλωσσικού μοντέλου είναι γνωστή ως prompt, ενώ η έξοδος αναφέρεται ως completion, ένας όρος που περιγράφει τον μηχανισμό του μοντέλου να παράγει το επόμενο token για να ολοκληρώσει την τρέχουσα είσοδο. Θα εμβαθύνουμε στο τι είναι ένα prompt και πώς να το σχεδιάζουμε ώστε να αξιοποιούμε στο μέγιστο τις δυνατότητες του μοντέλου μας. Προς το παρόν, ας πούμε ότι ένα prompt μπορεί να περιλαμβάνει:

- Μια **εντολή** που καθορίζει τον τύπο της εξόδου που αναμένουμε από το μοντέλο. Αυτή η εντολή μερικές φορές μπορεί να ενσωματώνει παραδείγματα ή επιπρόσθετα δεδομένα.

  1. Περίληψη ενός άρθρου, βιβλίου, αξιολογήσεων προϊόντων και άλλα, μαζί με εξαγωγή πληροφοριών από μη δομημένα δεδομένα.
    
    ![Παράδειγμα περίληψης](../../../translated_images/el/summarization-example.7b7ff97147b3d790.webp)
  
  2. Δημιουργική ιδέα και σχεδιασμός ενός άρθρου, δοκιμίου, εργασίας ή περισσότερων.
      
     ![Παράδειγμα δημιουργικής γραφής](../../../translated_images/el/creative-writing-example.e24a685b5a543ad1.webp)

- Μια **ερώτηση**, που τίθεται με τη μορφή συνομιλίας με έναν πράκτορα.
  
  ![Παράδειγμα συνομιλίας](../../../translated_images/el/conversation-example.60c2afc0f595fa59.webp)

- Ένα κομμάτι **κειμένου για ολοκλήρωση**, που υποδηλώνει ζήτηση βοήθειας στην γραφή.
  
  ![Παράδειγμα ολοκλήρωσης κειμένου](../../../translated_images/el/text-completion-example.cbb0f28403d42752.webp)

- Ένα κομμάτι **κώδικα** μαζί με ζήτηση εξήγησης και τεκμηρίωσής του, ή ένα σχόλιο που ζητά τη δημιουργία ενός κώδικα που εκτελεί μια συγκεκριμένη εργασία.
  
  ![Παράδειγμα κωδικοποίησης](../../../translated_images/el/coding-example.50ebabe8a6afff20.webp)

Τα παραπάνω παραδείγματα είναι αρκετά απλά και δεν αποσκοπούν να είναι μια εξαντλητική επίδειξη των δυνατοτήτων των Μεγάλων Γλωσσικών Μοντέλων. Σκοπεύουν να δείξουν το δυναμικό χρήσης της παραγωγικής τεχνητής νοημοσύνης, ειδικά αλλά όχι αποκλειστικά σε εκπαιδευτικά περιβάλλοντα.

Επίσης, η έξοδος ενός παραγωγικού μοντέλου τεχνητής νοημοσύνης δεν είναι τέλεια και μερικές φορές η δημιουργικότητα του μοντέλου μπορεί να λειτουργήσει εναντίον του, παράγοντας μια έξοδο που είναι ένας συνδυασμός λέξεων που ο χρήστης μπορεί να ερμηνεύσει ως μυστηριοποίηση της πραγματικότητας ή να είναι προσβλητική. Η Παραγωγική Τεχνητή Νοημοσύνη δεν είναι έξυπνη – τουλάχιστον σύμφωνα με τον ευρύτερο ορισμό της νοημοσύνης που περιλαμβάνει κριτική και δημιουργική σκέψη ή συναισθηματική νοημοσύνη. Δεν είναι ντετερμινιστική και δεν είναι αξιόπιστη, αφού εφευρέσεις, όπως λανθασμένες αναφορές, περιεχόμενο και δηλώσεις, μπορεί να συνδυαστούν με σωστές πληροφορίες και να παρουσιαστούν με πειστικό και βέβαιο τρόπο. Στα επόμενα μαθήματα θα ασχοληθούμε με όλους αυτούς τους περιορισμούς και θα δούμε τι μπορούμε να κάνουμε για να τους μετριάσουμε.

## Έλεγχος γνώσεων

Τι ισχύει για τα μεγάλα γλωσσικά μοντέλα;

1. Λαμβάνεις την ίδια ακριβώς απάντηση κάθε φορά.
1. Κάνει τα πράγματα τέλεια, είναι εξαιρετικό στο να προσθέτει αριθμούς, να παράγει λειτουργικό κώδικα κλπ.
1. Η απάντηση μπορεί να διαφέρει ακόμα και αν χρησιμοποιήσεις το ίδιο prompt. Είναι επίσης εξαιρετικό στο να σου δίνει ένα πρώτο προσχέδιο, είτε κείμενο είτε κώδικα. Αλλά χρειάζεται να βελτιώσεις τα αποτελέσματα.

Α: 3, ένα LLM είναι μη-ντετερμινιστικό, η απάντηση διαφέρει, ωστόσο μπορείτε να ελέγξετε αυτή τη διαφορά μέσω της παραμέτρου θερμοκρασίας. Επίσης, δεν πρέπει να περιμένετε να κάνει τα πράγματα τέλεια, είναι εδώ για να κάνει τον βαριόκοπο βασικό φόρτο για εσάς που συχνά σημαίνει ότι παίρνετε μια καλή πρώτη προσπάθεια που πρέπει να βελτιώσετε σταδιακά.
