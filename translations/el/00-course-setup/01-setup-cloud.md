# Ρύθμιση στο Cloud ☁️ – GitHub Codespaces

**Χρησιμοποιήστε αυτόν τον οδηγό αν δεν θέλετε να εγκαταστήσετε τίποτα τοπικά.**  
Το Codespaces σας δίνει μια δωρεάν, βασιζόμενη σε browser, παρουσία VS Code με όλες τις εξαρτήσεις προεγκατεστημένες.

---

## 1.  Γιατί Codespaces;

| Όφελος | Τι σημαίνει για εσάς |
|---------|----------------------|
| ✅ Μηδενικές εγκαταστάσεις | Λειτουργεί σε Chromebook, iPad, υπολογιστές εργαστηρίου σχολείου… |
| ✅ Προδημιουργημένο περιβάλλον ανάπτυξης | Python 3, Node.js, .NET, Java ήδη μέσα |
| ✅ Δωρεάν όριο | Τα προσωπικά προφίλ λαμβάνουν **120 ώρες πυρήνα / 60 ώρες GB ανά μήνα** |

> 💡 **Συμβουλή**  
> Διατηρήστε το όριό σας υγιές με το **να σταματάτε** ή **να διαγράφετε** αδρανή codespaces  
> (Προβολή ▸ Command Palette ▸ *Codespaces: Stop Codespace*).

---

## 2.  Δημιουργία ενός Codespace (με ένα κλικ)

1. **Fork** αυτό το αποθετήριο (πάνω δεξιά κουμπί **Fork**).  
2. Στο fork σας, κάντε κλικ **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Διάλογος που δείχνει κουμπιά για δημιουργία codespace](../../../translated_images/el/who-will-pay.4c0609b1c7780f44.webp)

✅ Ανοίγει ένα παράθυρο VS Code στον browser και ξεκινά η κατασκευή του περιβάλλοντος ανάπτυξης.
Αυτό παίρνει **περίπου 2 λεπτά** την πρώτη φορά.

## 3. Προσθήκη του API κλειδιού σας (με τον ασφαλή τρόπο)

### Επιλογή Α Secrets των Codespaces — Συνιστάται

1. ⚙️ Εικονίδιο γραναζιού -> Command Pallete -> Codespaces : Manage user secret -> Προσθήκη νέου κλειδιού.
2. Όνομα: OPENAI_API_KEY
3. Τιμή: επικολλήστε το κλειδί σας → Προσθήκη κλειδιού

Αυτό ήταν—ο κώδικάς μας θα το αναγνωρίσει αυτόματα.

### Επιλογή Β Αρχείο .env (αν πραγματικά χρειάζεστε)

```bash
cp .env.copy .env
code .env         # συμπληρώστε OPENAI_API_KEY=your_key_here
```
