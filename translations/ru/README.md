# Генеративный ИИ для начинающих (Версия 3) - Курс

Изучите основы создания приложений на базе генеративного ИИ с помощью нашего комплексного курса из 21 урока от Microsoft Cloud Advocates.

## 🌱 Начинаем

В курсе 21 урок. Каждый урок посвящен своей теме, так что начинайте с любого!

Уроки отмечены как "Учимся" — объясняют концепцию генеративного ИИ, либо "Создаем" — объясняют концепцию и содержат примеры кода на **Python** и **TypeScript**, когда возможно.

Для разработчиков .NET смотрите [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Каждый урок также включает раздел "Продолжайте обучение" с дополнительными инструментами для изучения.

## Что вам нужно
### Для запуска кода из курса можно использовать:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Уроки:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Уроки:** "githubmodels" (GitHub Models устареет в конце июля 2026 — используйте Microsoft Foundry Models)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Уроки:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) — запуск моделей полностью оффлайн на вашем устройстве, подписка на облако не требуется
   
- Базовые знания Python или TypeScript помогут — \*Для абсолютных новичков рекомендуем эти курсы [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) и [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Аккаунт GitHub для [форка всего этого репозитория](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) в свой собственный аккаунт

Мы создали урок **[Настройка курса](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**, чтобы помочь вам с настройкой среды разработки.

Не забудьте [поставить звезду (🌟) этому репозиторию](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), чтобы потом легко его найти.
