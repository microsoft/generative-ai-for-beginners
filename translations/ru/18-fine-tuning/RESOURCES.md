# Ресурсы для самостоятельного обучения

Урок был построен с использованием основных ресурсов OpenAI и Microsoft Foundry в качестве ориентиров для терминологии и учебных материалов. Вот неполный список для ваших собственных самостоятельных обучающих путешествий. Каждая ссылка ниже ведет к актуальному, поддерживаемому материалу.

## 1. Основные ресурсы

| Название/Ссылка | Описание |
| :--- | :--- |
| [Тонкая настройка моделей OpenAI](https://platform.openai.com/docs/guides/fine-tuning?WT.mc_id=academic-105485-koreyst) | Тонкая настройка улучшает обучение с несколькими примерами (few-shot learning), обучаясь на гораздо большем количестве примеров, чем помещается в подсказку — это снижает затраты, улучшает качество ответов и позволяет делать запросы с меньшей задержкой. **Получите обзор тонкой настройки от OpenAI.** |
| [Когда использовать тонкую настройку Microsoft Foundry](https://learn.microsoft.com/azure/ai-foundry/openai/concepts/fine-tuning-considerations?WT.mc_id=academic-105485-koreyst) | Узнайте, **что такое тонкая настройка (концепция)**, почему стоит её рассматривать, какие данные использовать и как измерять качество — плюс когда подходят SFT, DPO или RFT. |
| [Настройка модели с помощью тонкой настройки](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?WT.mc_id=academic-105485-koreyst) | Полный **пошаговый процесс (how-to)** тонкой настройки в Microsoft Foundry через портал, OpenAI / Foundry Python SDK или REST API — включая подготовку данных, обучение, контрольные точки и развертывание. |
| [Непрерывная тонкая настройка](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?WT.mc_id=academic-105485-koreyst#perform-continuous-fine-tuning) | Итеративный процесс выбора уже тонко настроенной модели в качестве базовой и **дальнейшей тонкой настройки** на новых наборах обучающих примеров. |
| [Тонкая настройка с вызовом функций (tool calling)](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning-functions?WT.mc_id=academic-105485-koreyst) | Тонкая настройка вашей модели **с примерами вызова инструментов** улучшает выходные данные — ответы становятся более точными, последовательными и заданного формата, используя меньше токенов в подсказке. |
| [Тонкая настройка моделей: руководство Microsoft Foundry](https://learn.microsoft.com/azure/ai-foundry/foundry-models/concepts/models-sold-directly-by-azure?WT.mc_id=academic-105485-koreyst#fine-tuning-models) | Узнайте, **какие модели можно тонко настраивать**, какие методы они поддерживают (SFT / DPO / RFT) и в каких регионах доступны. |
| [Обзор тонкой настройки: техники и модальности](https://learn.microsoft.com/azure/ai-foundry/concepts/fine-tuning-overview?WT.mc_id=academic-105485-koreyst) | Сравните три техники обучения (SFT, DPO, RFT) и две модальности (serverless vs управляемый вычислительный ресурс), с рекомендациями по выбору базовой модели и началу работы. |
| **Учебник**: [Тонкая настройка модели в Microsoft Foundry](https://learn.microsoft.com/azure/ai-foundry/openai/tutorials/fine-tune?WT.mc_id=academic-105485-koreyst) | Создайте пример набора данных, подготовьтесь к тонкой настройке, запустите задачу тонкой настройки на поддерживаемой модели, такой как `gpt-4.1-mini`, и разверните тонко настроенную модель в Azure. |
| **Учебник**: [Тонкая настройка моделей с развертываниями serverless API](https://learn.microsoft.com/azure/ai-foundry/how-to/fine-tune-serverless?WT.mc_id=academic-105485-koreyst) | Настройте открытые и партнерские модели (Phi, Llama, Mistral и другие) под свои наборы данных _с помощью low-code, основанного на UI рабочего процесса_ в Microsoft Foundry. |
| **Учебник**: [Тонкая настройка моделей Hugging Face на Azure Databricks](https://learn.microsoft.com/azure/databricks/machine-learning/train-model/huggingface/fine-tune-model?WT.mc_id=academic-105485-koreyst) | Тонко настройте модель Hugging Face с библиотекой `transformers` на одном GPU, используя Azure Databricks и Hugging Face Trainer. |
| **Обучение**: [Тонкая настройка фундаментальной модели с Azure Machine Learning](https://learn.microsoft.com/training/modules/finetune-foundation-model-with-azure-machine-learning/?WT.mc_id=academic-105485-koreyst) | Каталог моделей Azure Machine Learning предлагает множество моделей с открытым кодом для тонкой настройки. Часть [Обучающего пути по генеративному ИИ Azure ML](https://learn.microsoft.com/training/paths/work-with-generative-models-azure-machine-learning/?WT.mc_id=academic-105485-koreyst). |
| **Учебник**: [Тонкая настройка Azure OpenAI с Weights & Biases](https://docs.wandb.ai/guides/integrations/azure-openai-fine-tuning?WT.mc_id=academic-105485-koreyst) | Отслеживайте и анализируйте прогон тонкой настройки в Azure с помощью W&B. Расширяет руководство OpenAI по тонкой настройке, добавляя шаги и трекинг экспериментов, специфичные для Azure. |

В сочетании с [руководством по тонкой настройке OpenAI](https://platform.openai.com/docs/guides/fine-tuning?WT.mc_id=academic-105485-koreyst). |
| **OpenAI Cookbook**: [Тонкая настройка для Retrieval Augmented Generation (RAG) с Qdrant](https://cookbook.openai.com/examples/fine-tuned_qa/ft_retrieval_augmented_generation_qdrant?WT.mc_id=academic-105485-koreyst) | Всеобъемлющий пример тонкой настройки моделей OpenAI для RAG, интегрирующий Qdrant и few-shot обучение для повышения производительности и сокращения выдумок. |
| **OpenAI Cookbook**: [Тонкая настройка GPT с Weights & Biases](https://cookbook.openai.com/examples/third_party/gpt_finetuning_with_wandb?WT.mc_id=academic-105485-koreyst) | Используйте W&B для отслеживания обучения и тонкой настройки модели. Сначала прочитайте их руководство [OpenAI Fine-Tuning](https://docs.wandb.ai/guides/integrations/openai-fine-tuning/?WT.mc_id=academic-105485-koreyst), затем попробуйте упражнение из Cookbook. |
| **Учебник Hugging Face**: [Как тонко настраивать LLM с Hugging Face TRL](https://www.philschmid.de/fine-tune-llms-in-2024-with-trl?WT.mc_id=academic-105485-koreyst) | Тонко настройте открытые LLM, используя Hugging Face TRL, Transformers и датасеты: определите сценарий использования, настройте среду разработки, подготовьте датасет, выполните тонкую настройку, оцените и разверните. |
| **Hugging Face**: [AutoTrain Advanced](https://github.com/huggingface/autotrain-advanced?WT.mc_id=academic-105485-koreyst) | Библиотека без кода / с низким кодом от Hugging Face для тонкой настройки множества типов моделей. Запускается в вашем облаке, на Hugging Face Spaces или локально через GUI, CLI или YAML-конфиг. |
| **Unsloth**: [Руководство по тонкой настройке LLM](https://docs.unsloth.ai/get-started/fine-tuning-llms-guide) | Открытый фреймворк, который упрощает локальную тонкую настройку LLM и обучение с подкреплением, с готовыми к использованию [блокнотами](https://github.com/unslothai/notebooks?WT.mc_id=academic-105485-koreyst). |
