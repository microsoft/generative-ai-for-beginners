# क्लाउड सेटअप ☁️ – GitHub Codespaces

**यदि तपाईंले स्थानीय रूपमा केही इन्स्टल गर्न चाहनुहुन्न भने यो मार्गदर्शक प्रयोग गर्नुहोस्।**  
Codespaces ले तपाईंलाई नि:शुल्क, ब्राउजर-आधारित VS Code इन्स्ट्यान्स दिन्छ जसमा सबै निर्भरताहरू पहिले नै इन्स्टल गरिएको हुन्छ।

---

## 1.  किन Codespaces?

| लाभ | तपाईंको लागि यसको अर्थ के हो |
|---------|----------------------|
| ✅ कुनै इन्स्टलेशन आवश्यक छैन | Chromebook, iPad, विद्यालय ल्याब PCs मा काम गर्छ… |
| ✅ पूर्व-निर्मित विकास कन्टेनर | Python 3, Node.js, .NET, Java पहिले नै भित्र छन् |
| ✅ निःशुल्क कोटा | व्यक्तिगत खाताहरूले पाउँछन् **120 कोर-घंटा / 60 GB-घंटा प्रति महिना** |

> 💡 **टिप**  
> आफ्नो कोटा स्वस्थ राख्नको लागि **रोक्नुहोस्** वा **मेटाउनुहोस्** निष्क्रिय codespaces  
> (हेर्नुहोस् ▸ Command Palette ▸ *Codespaces: Stop Codespace*)।

---

## 2.  एक क्लिकमा Codespace सिर्जना गर्नुहोस्

1. यस रिपोलाई **Fork** गर्नुहोस् (माथि-दायाँको **Fork** बटन)।  
2. तपाईंको fork मा, क्लिक गर्नुहोस् **Code ▸ Codespaces ▸ Create codespace on main**।  
   ![Dialog showing buttons to create a codespace](../../../translated_images/ne/who-will-pay.4c0609b1c7780f44.webp)

✅ एउटा ब्राउजर VS Code विन्डो खुल्छ र विकास कन्टेनर बनिन थाल्छ।
पहिलो पटकमा **~2 मिनेट** लाग्छ।

## 3. तपाईंको API कुञ्जी थप्नुहोस् (सुरक्षित तरिका)

### विकल्प A Codespaces Secrets — सिफारिस गरिएको

1. ⚙️ गियर आइकन -> Command Pallete-> Codespaces : Manage user secret -> नयाँ गुप्त थप्नुहोस्।
2. नाम: OPENAI_API_KEY
3. मान: तपाईंको कुञ्जी टाँस्नुहोस् → Add secret

यत्ति नै—हाम्रो कोडले यसलाई स्वचालित रूपमा उपयोग गर्नेछ।

### विकल्प B .env फाइल (यदि तपाईंलाई साँच्चिकै चाहिन्छ भने)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=तपाईंको कुञ्जी यहाँ भर्नुहोस्
```
