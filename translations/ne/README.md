# जनरेटिभ एआईको लागि सुरुवाती (संस्करण ३) - एक पाठ्यक्रम

माइक्रोसफ्ट क्लाउड एड्भोकेट्सद्वारा प्रदान गरिएको हाम्रो २१ पाठको व्यापक पाठ्यक्रम मार्फत जनरेटिभ एआई अनुप्रयोगहरू निर्माणको आधारभूत कुरा सिक्नुहोस्।

## 🌱 सुरु गर्दै

यस पाठ्यक्रममा २१ पाठहरू छन्। प्रत्येक पाठले आफ्नो विषयलाई समेट्छ, त्यसैले तपाईले जहाँबाट चाहनुहुन्छ त्यहाँबाट सुरु गर्नुहोस्!

पाठहरू "Learn" पाठहरूको रूपमा लेबल गरिएका छन् जसले जनरेटिभ एआईको अवधारणा व्याख्या गर्छन् वा "Build" पाठहरू जसले अवधारणालाई व्याख्या गर्छन् र **Python** र **TypeScript** मा कोड उदाहरणहरू दिन्छन् जब सम्भव छ।

.NET विकासकर्ताहरूका लागि [जनरेटिभ एआईको लागि सुरुवाती (.NET संस्करण)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) हेर्नुहोस्!

प्रत्येक पाठमा थप सिकाइ उपकरणहरूसँग "Keep Learning" खण्ड पनि समावेश गरिएको छ।

## तपाईलाई के चाहिन्छ
### यस पाठ्यक्रमको कोड चलाउनका लागि, तपाईले निम्न मध्ये कुनै पनि प्रयोग गर्न सक्नुहुन्छ: 
 - [Azure OpenAI सेवा](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **पाठहरू:** "aoai-assignment"
 - [GitHub Marketplace मोडेल क्याटलग](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **पाठहरू:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **पाठहरू:** "oai-assignment" 

- Python वा TypeScript को आधारभूत ज्ञान उपयोगी छ - \*नवीनतम सुरुवातीहरूका लागि यी [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) र [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) पाठ्यक्रमहरू हेर्नुहोस्
- तपाईको आफ्नै GitHub खातामा [यस सम्पूर्ण रिपो फोर्क गर्न](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) GitHub खाता

हामीले तपाईको विकास वातावरण सेटअप गर्न मद्दत गर्नका लागि **[पाठ्यक्रम सेटअप](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** पाठ सिर्जना गरेका छौं।

पछिपछि सजिलोसँग फेला पार्नका लागि [यस रिपोमा तार (🌟) लगाउन](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) नबिर्सनुहोस्।
