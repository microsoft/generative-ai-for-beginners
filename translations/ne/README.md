# जेनेरेटिभ AI सुरु गर्नेहरूको लागि (संस्करण ३) - एक कोर्स

Microsoft Cloud Advocates द्वारा तयार पारिएको २१ पाठहरूको व्यापक कोर्समार्फत जेनेरेटिभ AI अनुप्रयोगहरू निर्माण गर्ने आधारभूत कुराहरू सिक्नुहोस्।

## 🌱 सुरु गर्नुहोस्

यो कोर्समा २१ पाठहरू छन्। प्रत्येक पाठले आफ्नो विषयलाई समेट्छ त्यसैले जहाँ मन लाग्छ त्यहाँबाट सुरु गर्नुहोस्!

पाठहरूलाई "सिक्ने" पाठहरू जुन जेनेरेटिभ AI अवधारणा व्याख्या गर्छन् वा "बनाउने" पाठहरू जुन अवधारणा र सम्भव भएमा दुबै **Python** र **TypeScript** मा कोड उदाहरणहरू बताउँछन् भनेर लेबल गरिएको छ।

.NET विकासकर्ताहरूका लागि [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) मा जानुहोस्!

हरेक पाठमा थप सिकाइका उपकरणहरू सहित "अझ सिकिरहनुहोस्" खण्ड पनि हुन्छ।

## तपाईंलाई के चाहिन्छ
### यो कोर्सको कोड चलाउन, तपाईंले यी मध्ये कुनै एउटालाई प्रयोग गर्नसक्नुहुन्छ:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **पाठहरू:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **पाठहरू:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **पाठहरू:** "oai-assignment" 
   
- Python वा TypeScript को आधारभूत ज्ञान उपयोगी छ - \*पूर्ण सुरु गर्नेलाई यी [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) र [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) कोर्सहरू हेर्नुहोस्
- तपाईंको आफ्नै GitHub खातामा यो सम्पूर्ण रिपो [fork](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) गर्न GitHub खाता चाहिन्छ

हामीले तपाईंको विकास वातावरण सेटअपमा सहयोग गर्नको लागि **[कोर्स सेटअप](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** पाठ तयार पारेका छौं।

पछाडि सजिलै फेला पार्न यो रिपोमा [तारा (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) दिन नबिर्सनुहोस्।
