# न्युरल नेटवर्कको परिचय: पर्सेप्ट्रोन

आधुनिक न्युरल नेटवर्कको जस्तै केहि लागू गर्ने पहिलो प्रयासहरू मध्ये एक फ्रान्क रोजेनब्लाट्टले १९५७ मा कोर्नेल एरोनटिकल ल्याबोरेटरीबाट गरेका थिए। यो हार्डवेयर कार्यान्वयन "मार्क-१" भनिन्थ्यो, जसलाई त्रिकोण, वर्ग र वृत्त जस्ता आदिम ज्यामितीय आकृतिहरूलाई चिन्हित गर्न डिजाइन गरिएको थियो।

> विकिपीडियाबाट छविहरू

एक इनपुट छवि २०x२० फोटोसेल एरेले प्रतिनिधित्व गरिएको थियो, त्यसैले न्युरल नेटवर्कसँग ४०० इनपुटहरू र एक बाइनरी आउटपुट थियो। एक साधारण नेटवर्कमा एक न्युरोन समावेश थियो, जसलाई **थ्रेसहोल्ड लजिक युनिट** पनि भनिन्छ। न्युरल नेटवर्कका वजनहरू पोटेन्सियोमिटरहरूको रूपमा काम गर्थे जसलाई तालिम चरणमा म्यानुअल समायोजन आवश्यक थियो।

> ✅ पोटेन्सियोमिटर एक उपकरण हो जसले प्रयोगकर्तालाई सर्किटको प्रतिरोध समायोजन गर्न अनुमति दिन्छ।

> त्यो समयमा न्यूयोर्क टाइम्सले पर्सेप्ट्रोनको बारेमा लेखेको थियो: *एक इलेक्ट्रोनिक कम्प्युटरको भ्रूण जुन [नेवी] ले अपेक्षा गरेको छ कि हिँड्न, बोल्न, देख्न, लेख्न, आफैंलाई पुन: उत्पादन गर्न र यसको अस्तित्वको चेतना गर्न सक्षम हुनेछ।*

## पर्सेप्ट्रोन मोडल

मानौं हाम्रो मोडलमा N विशेषताहरू छन्, जसमा इनपुट भेक्टर N आकारको भेक्टर हुनेछ। पर्सेप्ट्रोन एक **बाइनरी वर्गीकरण** मोडल हो, अर्थात् यसले इनपुट डाटाका दुई वर्गहरू बीच भिन्नता गर्न सक्छ। हामी मान्छौं कि प्रत्येक इनपुट भेक्टर x को लागि हाम्रो पर्सेप्ट्रोनको आउटपुट +1 वा -1 हुनेछ, वर्गको आधारमा। आउटपुट निम्न सूत्र प्रयोग गरेर गणना गरिनेछ:

y(x) = f(w<sup>T</sup>x)

जहाँ f एक स्टेप सक्रियता प्रकार्य हो

## पर्सेप्ट्रोनको तालिम

पर्सेप्ट्रोनलाई तालिम दिन हामीलाई एउटा वजन भेक्टर w फेला पार्न आवश्यक छ जसले अधिकांश मानहरू सही रूपमा वर्गीकृत गर्दछ, अर्थात् सबैभन्दा सानो **त्रुटि**को परिणाम दिन्छ। यो त्रुटि **पर्सेप्ट्रोन मापदण्ड** द्वारा निम्न तरिकामा परिभाषित गरिएको छ:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

जहाँ:

* योग ती प्रशिक्षण डेटा बिन्दुहरूमा लिइन्छ जसले गलत वर्गीकरणको परिणाम दिन्छ
* x<sub>i</sub> इनपुट डेटा हो, र t<sub>i</sub> नकारात्मक र सकारात्मक उदाहरणहरूको लागि क्रमशः -1 वा +1 हो।

यो मापदण्डलाई वजन w को कार्यको रूपमा मानिन्छ, र हामीले यसलाई न्यूनतम गर्न आवश्यक छ। प्रायः, **ग्रेडियेन्ट डिसेन्ट** भनिने विधि प्रयोग गरिन्छ, जसमा हामी केही प्रारम्भिक वजन w<sup>(0)</sup> बाट सुरु गर्छौं, र त्यसपछि प्रत्येक चरणमा निम्न सूत्र अनुसार वजन अद्यावधिक गर्दछौं:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

यहाँ η तथाकथित **शिक्षण दर** हो, र ∇E(w) ले E को **ग्रेडियेन्ट** जनाउँछ। जब हामीले ग्रेडियेन्ट गणना गर्छौं, हामीसँग

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

Python मा एल्गोरिदम यसरी देखिन्छ:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```

## समीक्षा र आत्म अध्ययन

हामी कसरी पर्सेप्ट्रोन प्रयोग गरेर एक सानो समस्या साथै वास्तविक जीवनका समस्याहरू समाधान गर्न सक्छौं भनेर हेर्न, र सिकाइ जारी राख्न - पर्सेप्ट्रोन नोटबुकमा जानुहोस्।

यहाँ पर्सेप्ट्रोनहरूको बारेमा एउटा रोचक लेख पनि छ।

## असाइनमेन्ट
