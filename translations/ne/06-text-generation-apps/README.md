# पाठ उत्पन्न गर्ने अनुप्रयोग निर्माण

> _(यस पाठको भिडियो हेर्न माथिको चित्रमा क्लिक गर्नुहोस्)_

यस पाठ्यक्रमको माध्यमबाट अहिलेसम्म तपाईंले देख्नुभएको छ कि त्यहाँ मुख्य अवधारणाहरू जस्तै प्रॉम्प्टहरू छन् र "प्रॉम्प्ट इन्जिनियरिङ" भनिने पूरै अनुशासन पनि छ। तपाईं जस्तै ChatGPT, Office 365, Microsoft Power Platform र अन्य धेरै उपकरणहरूसँग अन्तरक्रिया गर्न सक्नुहुन्छ, जसले तपाईंलाई केही हासिल गर्न प्रॉम्प्टहरू प्रयोग गर्न समर्थन गर्दछ।

यस प्रकारको अनुभवलाई एपमा थप्नका लागि, तपाईंले प्रॉम्प्टहरू, कम्प्लीशनहरू जस्ता अवधारणाहरू बुझ्न आवश्यक छ र काम गर्न पुस्तकालय छनौट गर्नुपर्छ। यही कुरा तपाईंले यस अध्यायमा सिक्नुहुनेछ।

## परिचय

यस अध्यायमा, तपाईंले:

- openai पुस्तकालय र यसको मुख्य अवधारणाहरूको बारेमा सिक्नुहोस्।
- openai प्रयोग गरेर पाठ उत्पन्न गर्ने अनुप्रयोग निर्माण गर्नुहोस्।
- पाठ उत्पन्न गर्ने अनुप्रयोग निर्माण गर्न प्रॉम्प्ट, तापक्रम, र टोकनहरू जस्ता अवधारणाहरू कसरी प्रयोग गर्ने भनेर बुझ्नुहोस्।

## सिकाइ लक्ष्यहरू

यस पाठको अन्त्यमा, तपाईंले सक्षम हुनुहुनेछ:

- पाठ उत्पन्न गर्ने अनुप्रयोग के हो भनेर व्याख्या गर्नुहोस्।
- openai प्रयोग गरेर पाठ उत्पन्न गर्ने अनुप्रयोग निर्माण गर्नुहोस्।
- तपाईंको अनुप्रयोगलाई कम वा बढी टोकनहरू प्रयोग गर्न र तापक्रम परिवर्तन गर्न कन्फिगर गर्नुहोस्, विविध आउटपुटका लागि।

## पाठ उत्पन्न गर्ने अनुप्रयोग के हो?

सामान्यतया जब तपाईं अनुप्रयोग निर्माण गर्नुहुन्छ, यसमा निम्न जस्तै केही प्रकारको इन्टरफेस हुन्छ:

- कमाण्ड-आधारित। कन्सोल अनुप्रयोगहरू सामान्य अनुप्रयोगहरू हुन् जहाँ तपाईंले एउटा कमाण्ड टाइप गर्नुहुन्छ र यसले कुनै कार्य गर्दछ। उदाहरणका लागि, `git` एक कमाण्ड-आधारित अनुप्रयोग हो।
- प्रयोगकर्ता इन्टरफेस (UI)। केही अनुप्रयोगहरूमा ग्राफिकल प्रयोगकर्ता इन्टरफेसहरू (GUIs) हुन्छन् जहाँ तपाईंले बटन क्लिक गर्नुहुन्छ, पाठ प्रविष्ट गर्नुहुन्छ, विकल्पहरू चयन गर्नुहुन्छ र अन्य धेरै कुरा गर्नुहुन्छ।

### कन्सोल र UI अनुप्रयोगहरू सीमित छन्

तपाईंले टाइप गर्ने कमाण्ड-आधारित अनुप्रयोगसँग तुलना गर्नुहोस्:

- **यो सीमित छ**। तपाईंले कुनै पनि कमाण्ड टाइप गर्न सक्नुहुन्न, केवल ती मात्र जुन अनुप्रयोगले समर्थन गर्दछ।
- **भाषा विशेष**। केही अनुप्रयोगहरूले धेरै भाषाहरू समर्थन गर्दछन्, तर पूर्वनिर्धारित रूपमा अनुप्रयोग एक विशिष्ट भाषाको लागि निर्माण गरिएको हुन्छ, यद्यपि तपाईंले थप भाषा समर्थन थप्न सक्नुहुन्छ।

### पाठ उत्पन्न गर्ने अनुप्रयोगहरूको फाइदाहरू

तपाईंको पाठ उत्पन्न गर्ने अनुप्रयोग कसरी फरक छ?

पाठ उत्पन्न गर्ने अनुप्रयोगमा, तपाईंलाई बढी लचिलोता हुन्छ, तपाईं कमाण्डहरूको सेट वा विशिष्ट इनपुट भाषामा सीमित हुनुहुन्न। बरु, तपाईंले अनुप्रयोगसँग अन्तरक्रिया गर्न प्राकृतिक भाषा प्रयोग गर्न सक्नुहुन्छ। अर्को फाइदा भनेको तपाईं पहिले नै ठूलो जानकारीको संग्रहमा प्रशिक्षित गरिएको डाटास्रोतसँग अन्तरक्रिया गरिरहनुभएको छ, जबकि परम्परागत अनुप्रयोगमा डेटाबेसमा के छ त्यसमा सीमित हुन सक्छ।

### म पाठ उत्पन्न गर्ने अनुप्रयोगसँग के निर्माण गर्न सक्छु?

तपाईं धेरै कुरा निर्माण गर्न सक्नुहुन्छ। उदाहरणका लागि:

- **च्याटबोट**। तपाईंको कम्पनी र यसको उत्पादनहरू जस्ता विषयहरूमा प्रश्नहरूको उत्तर दिने च्याटबोट राम्रो मिलान हुन सक्छ।
- **सहायक**। LLMs पाठलाई संक्षेप गर्न, पाठबाट अन्तर्दृष्टि प्राप्त गर्न, पाठ उत्पादन गर्न जस्तै रिज्युमेहरू र अन्य धेरै चीजहरूमा उत्कृष्ट छन्।
- **कोड सहायक**। तपाईंले प्रयोग गर्ने भाषा मोडेलको आधारमा, तपाईंलाई कोड लेख्न मद्दत गर्ने कोड सहायक निर्माण गर्न सक्नुहुन्छ। उदाहरणका लागि, तपाईंले GitHub Copilot जस्तै उत्पादनहरू साथै ChatGPT लाई कोड लेख्न मद्दतको लागि प्रयोग गर्न सक्नुहुन्छ।

## म कसरी सुरु गर्न सक्छु?

तपाईंले LLM सँग एकीकृत गर्ने तरिका खोज्न आवश्यक छ जुन सामान्यतया निम्न दुई दृष्टिकोणहरू समावेश गर्दछ:

- API प्रयोग गर्नुहोस्। यहाँ तपाईं आफ्नो प्रॉम्प्टको साथ वेब अनुरोधहरू निर्माण गर्दै हुनुहुन्छ र उत्पन्न पाठ फिर्ता प्राप्त गर्नुहुन्छ।
- पुस्तकालय प्रयोग गर्नुहोस्। पुस्तकालयहरूले API कलहरूलाई समेट्न मद्दत गर्दछ र तिनीहरूलाई प्रयोग गर्न सजिलो बनाउँछ।

## पुस्तकालयहरू/SDKs

LLMs सँग काम गर्नका लागि केहि प्रसिद्ध पुस्तकालयहरू छन् जस्तै:

- **openai**, यो पुस्तकालयले तपाईंको मोडेलमा जडान गर्न र प्रॉम्प्टहरू पठाउन सजिलो बनाउँछ।

त्यसपछि त्यहाँ उच्च स्तरमा सञ्चालन गर्ने पुस्तकालयहरू छन् जस्तै:

- **Langchain**। Langchain राम्रोसँग परिचित छ र पायथनलाई समर्थन गर्दछ।
- **Semantic Kernel**। Semantic Kernel माइक्रोसफ्टको पुस्तकालय हो जसले C#, Python, र Java भाषाहरूलाई समर्थन गर्दछ।

## पहिलो अनुप्रयोग openai प्रयोग गर्दै

हेरौं हामी कसरी हाम्रो पहिलो अनुप्रयोग निर्माण गर्न सक्छौं, हामीलाई कुन पुस्तकालयहरू चाहिन्छ, कति आवश्यक छ र यस्तै।

### openai स्थापना गर्नुहोस्

OpenAI वा Azure OpenAI सँग अन्तरक्रिया गर्न धेरै पुस्तकालयहरू छन्। धेरै प्रोग्रामिङ भाषाहरू जस्तै C#, Python, JavaScript, Java र अन्य धेरै प्रयोग गर्न सम्भव छ। हामीले `openai` Python पुस्तकालय प्रयोग गर्ने छनौट गरेका छौं, त्यसैले हामी यसलाई स्थापना गर्न `pip` प्रयोग गर्नेछौं।

### स्रोत सिर्जना गर्नुहोस्

तपाईंले निम्न चरणहरू पूरा गर्न आवश्यक छ:

- Azure मा खाता सिर्जना गर्नुहोस् [https://azure.microsoft.com/free/](https://azure.microsoft.com/free/?WT.mc_id=academic-105485-koreyst)।
- Azure OpenAI मा पहुँच प्राप्त गर्नुहोस्। [https://learn.microsoft.com/azure/ai-services/openai/overview#how-do-i-get-access-to-azure-openai](https://learn.microsoft.com/azure/ai-services/openai/overview#how-do-i-get-access-to-azure-openai?WT.mc_id=academic-105485-koreyst) मा जानुहोस् र पहुँचको लागि अनुरोध गर्नुहोस्।

  > [!NOTE]
  > लेख्दा, तपाईंले Azure OpenAI मा पहुँचको लागि आवेदन दिन आवश्यक छ।

- Python स्थापना गर्नुहोस् <https://www.python.org/>
- Azure OpenAI सेवा स्रोत सिर्जना गर्नुहोस्। [स्रोत सिर्जना गर्नुहोस्](https://learn.microsoft.com/azure/ai-services/openai/how-to/create-resource?pivots=web-portal?WT.mc_id=academic-105485-koreyst) गर्नको लागि यो मार्गदर्शन हेर्नुहोस्।

### API कुञ्जी र अन्त बिन्दु पत्ता लगाउनुहोस्

यस बिन्दुमा, तपाईंले आफ्नो `openai` पुस्तकालयलाई कुन API कुञ्जी प्रयोग गर्ने बताउन आवश्यक छ। तपाईंको API कुञ्जी फेला पार्न, Azure OpenAI स्रोतको "कुञ्जीहरू र अन्त बिन्दु" खण्डमा जानुहोस् र "कुञ्जी 1" मान प्रतिलिपि गर्नुहोस्।

अब तपाईंले यो जानकारी प्रतिलिपि गर्नुभयो, पुस्तकालयहरूलाई यसलाई प्रयोग गर्न निर्देशन दिऔं।

> [!NOTE]
> तपाईंको API कुञ्जीलाई तपाईंको कोडबाट अलग राख्न लायक छ। तपाईंले वातावरणीय भेरिएबलहरू प्रयोग गरेर यसो गर्न सक्नुहुन्छ।
>
> - वातावरणीय भेरिएबल सेट गर्नुहोस् `OPENAI_API_KEY` to your API key.
>   `export OPENAI_API_KEY='sk-...'`

### Azure कन्फिगरेसन सेटअप गर्नुहोस्

यदि तपाईं Azure OpenAI प्रयोग गर्दै हुनुहुन्छ भने, यहाँ कसरी कन्फिगरेसन सेटअप गर्ने:

उपरोक्तमा हामीले निम्न सेट गर्दैछौं:

उपरोक्त कोडमा, हामीले कम्प्लीशन वस्तु सिर्जना गर्दछौं र प्रयोग गर्न चाहेको मोडेल र प्रॉम्प्ट पास गर्दछौं। त्यसपछि हामीले उत्पन्न पाठ मुद्रण गर्दछौं।

### च्याट कम्प्लीशनहरू

अहिलेसम्म, तपाईंले देख्नुभएको छ कि हामीले `Completion` to generate text. But there's another class called `ChatCompletion` प्रयोग गरिरहेका छौं जुन च्याटबोटहरूको लागि बढी उपयुक्त छ। यसलाई प्रयोग गर्ने उदाहरण यहाँ छ:

अर्को अध्यायमा यो कार्यक्षमतामा थप।

## अभ्यास - तपाईंको पहिलो पाठ उत्पन्न गर्ने अनुप्रयोग

अब हामीले openai सेट अप र कन्फिगर कसरी गर्ने भनेर सिक्यौं, तपाईंको पहिलो पाठ उत्पन्न गर्ने अनुप्रयोग निर्माण गर्ने समय हो। तपाईंको अनुप्रयोग निर्माण गर्न, यी चरणहरू पालना गर्नुहोस्:

1. भर्चुअल वातावरण सिर्जना गर्नुहोस् र openai स्थापना गर्नुहोस्:

   > [!NOTE]
   > यदि तपाईं Windows प्रयोग गर्दै हुनुहुन्छ भने `venv\Scripts\activate` instead of `source venv/bin/activate`.

   > [!NOTE]
   > Locate your Azure OpenAI key by going to [https://portal.azure.com/](https://portal.azure.com/?WT.mc_id=academic-105485-koreyst) and search for `Open AI` and select the `Open AI स्रोत` and then select `कुञ्जीहरू र अन्त बिन्दु` and copy the `कुञ्जी 1` मान टाइप गर्नुहोस्।

1. _app.py_ फाइल सिर्जना गर्नुहोस् र यसलाई निम्न कोड दिनुहोस्:

   > [!NOTE]
   > यदि तपाईं Azure OpenAI प्रयोग गर्दै हुनुहुन्छ भने, तपाईंले `api_type` to `azure` and set the `api_key` तपाईंको Azure OpenAI कुञ्जीमा सेट गर्न आवश्यक छ।

## विभिन्न प्रकारका प्रॉम्प्टहरू, विभिन्न कुराहरूका लागि

अब तपाईंले प्रॉम्प्ट प्रयोग गरेर पाठ कसरी उत्पन्न गर्ने देख्नुभयो। तपाईंले विभिन्न प्रकारका पाठ उत्पन्न गर्न संशोधन र परिवर्तन गर्न सक्ने कार्यक्रम पनि चलिरहेको छ।

प्रॉम्प्टहरू सबै प्रकारका कार्यहरूका लागि प्रयोग गर्न सकिन्छ। उदाहरणका लागि:

- **पाठको प्रकार उत्पन्न गर्नुहोस्**। उदाहरणका लागि, तपाईं कविता, क्विजका लागि प्रश्नहरू आदि उत्पन्न गर्न सक्नुहुन्छ।
- **जानकारी खोज्नुहोस्**। तपाईंले जानकारीको लागि खोज गर्न प्रॉम्प्टहरू प्रयोग गर्न सक्नुहुन्छ जस्तै 'वेब विकासमा CORS को अर्थ के हो?'।
- **कोड उत्पन्न गर्नुहोस्**। तपाईंले कोड उत्पन्न गर्न प्रॉम्प्टहरू प्रयोग गर्न सक्नुहुन्छ, उदाहरणका लागि इमेलहरूलाई मान्य बनाउन प्रयोग गरिने नियमित अभिव्यक्ति विकास गर्न वा किन सम्पूर्ण कार्यक्रम, जस्तै वेब अनुप्रयोग उत्पन्न नगर्ने?

## थप व्यावहारिक प्रयोग केस: एक रेसिपी उत्पन्नकर्ता

कल्पना गर्नुहोस् कि तपाईंको घरमा सामग्रीहरू छन् र तपाईंले केही पकाउन चाहनुहुन्छ। त्यसका लागि, तपाईंलाई रेसिपी चाहिन्छ। रेसिपीहरू फेला पार्ने एउटा तरिका खोज इन्जिन प्रयोग गर्नु हो वा तपाईंले LLM प्रयोग गर्न सक्नुहुन्छ।

- किनमेल सूची उत्पादन गर्नुहोस्, यदि मसँग सबै सामग्रीहरू घरमा छैन भने।

अब तपाईंसँग नयाँ परिणाम छ, अर्थात्:
