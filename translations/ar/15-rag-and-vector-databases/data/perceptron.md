# مقدمة في الشبكات العصبية: بيرسيبترون

> صور من ويكيبيديا

تم تمثيل صورة الإدخال بمصفوفة خلايا ضوئية بحجم 20x20، لذا كان لدى الشبكة العصبية 400 مدخل ومخرج ثنائي واحد. كانت الشبكة البسيطة تحتوي على خلية عصبية واحدة، تُعرف أيضًا باسم **وحدة منطق العتبة**. كانت أوزان الشبكة العصبية تعمل مثل المقاومات المتغيرة التي تحتاج إلى تعديل يدوي خلال مرحلة التدريب.

> ✅ المقاومة المتغيرة هي جهاز يسمح للمستخدم بضبط مقاومة الدائرة.

> كتبت صحيفة نيويورك تايمز عن البيرسيبترون في ذلك الوقت: *جنين حاسوب إلكتروني تتوقع [البحرية] أنه سيكون قادرًا على المشي، والتحدث، والرؤية، والكتابة، وإعادة إنتاج نفسه، وأن يكون واعيًا بوجوده.*

## نموذج بيرسيبترون

لنفترض أن لدينا N من الميزات في نموذجنا، وفي هذه الحالة سيكون متجه الإدخال عبارة عن متجه بحجم N. البيرسيبترون هو نموذج **تصنيف ثنائي**، أي يمكنه التمييز بين فئتين من بيانات الإدخال. سنفترض أنه لكل متجه إدخال x سيكون مخرج البيرسيبترون إما +1 أو -1، حسب الفئة. سيتم حساب المخرج باستخدام الصيغة:

y(x) = f(w<sup>T</sup>x)

حيث أن f هي دالة تنشيط خطوة

## تدريب البيرسيبترون

لتدريب البيرسيبترون، نحتاج إلى إيجاد متجه الأوزان w الذي يصنف معظم القيم بشكل صحيح، أي يؤدي إلى أصغر **خطأ**. يُعرف هذا الخطأ بمعيار **البيرسيبترون** بالطريقة التالية:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

حيث:

* يتم أخذ المجموع على نقاط البيانات التدريبية i التي تؤدي إلى التصنيف الخاطئ
* x<sub>i</sub> هو بيانات الإدخال، وt<sub>i</sub> إما -1 أو +1 للأمثلة السلبية والإيجابية على التوالي.

يُعتبر هذا المعيار كدالة للأوزان w، ونحتاج إلى تقليله. غالبًا ما يتم استخدام طريقة تسمى **الانحدار التدرجي**، حيث نبدأ ببعض الأوزان الأولية w<sup>(0)</sup>، ثم نقوم في كل خطوة بتحديث الأوزان وفقًا للصيغة:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

هنا η هو ما يُسمى **معدل التعلم**، و∇E(w) يُشير إلى **تدرج** E. بعد حساب التدرج، نحصل على

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

يبدو الخوارزم في بايثون كالتالي:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
