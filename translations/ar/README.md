# الذكاء الاصطناعي التوليدي للمبتدئين (الإصدار 3) - دورة تدريبية

تعلّم أساسيات بناء تطبيقات الذكاء الاصطناعي التوليدي مع دورتنا الشاملة المكونة من 21 درسًا من Microsoft Cloud Advocates.

## 🌱 البدء

تحتوي هذه الدورة على 21 درسًا. كل درس يغطي موضوعه الخاص لذا ابدأ من حيث تريد!

يتم تصنيف الدروس إما كدروس "تعلم" تشرح مفهوم الذكاء الاصطناعي التوليدي أو دروس "بناء" تشرح مفهومًا مع أمثلة برمجية بلغة **Python** و **TypeScript** عندما يكون ذلك ممكنًا.

لمطوري .NET، اطلع على [الذكاء الاصطناعي التوليدي للمبتدئين (إصدار .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

يشمل كل درس أيضًا قسم "استمر في التعلم" مع أدوات تعلم إضافية.

## ما تحتاجه
### لتشغيل تعليمات هذه الدورة، يمكنك استخدام أي مما يلي: 
 - [خدمة Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **الدروس:** "aoai-assignment"
 - [نماذج Microsoft Foundry](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **الدروس:** "githubmodels" (نماذج GitHub سيتوقف دعمها في نهاية يوليو 2026 - استخدم نماذج Microsoft Foundry بدلاً من ذلك)
 - [واجهة برمجة تطبيقات OpenAI](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **الدروس:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - تشغيل النماذج بالكامل دون الاتصال بالإنترنت على جهازك الشخصي، لا حاجة لاشتراك سحابي
   
- معرفة أساسية بـ Python أو TypeScript مفيدة - *للمبتدئين تمامًا، اطلع على دورات [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) و [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- حساب GitHub لعمل [نسخة مشتقة من هذا المستودع بالكامل](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) لحسابك على GitHub

أنشأنا درس **[إعداد الدورة](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** لمساعدتك في إعداد بيئة التطوير الخاصة بك.

لا تنس أن [تضع نجمة (🌟) على هذا المستودع](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) لتجده بسهولة لاحقًا.
