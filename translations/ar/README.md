# الذكاء الاصطناعي التوليدي للمبتدئين (الإصدار 3) - دورة

تعلّم أساسيات بناء تطبيقات الذكاء الاصطناعي التوليدي مع دورتنا الشاملة التي تتضمن 21 درسًا من قبل دعاة سحابة مايكروسوفت.

## 🌱 البدء

تحتوي هذه الدورة على 21 درسًا. يغطي كل درس موضوعه الخاص لذا ابدأ من حيثما تشاء!

تُوسم الدروس إما كدروس "تعلّم" تشرح مفهومًا من مفاهيم الذكاء الاصطناعي التوليدي أو كدروس "بناء" تشرح المفهوم مع أمثلة على الشيفرة لكل من **Python** و **TypeScript** كلما أمكن ذلك.

لمطوري .NET تفقد [الذكاء الاصطناعي التوليدي للمبتدئين (إصدار .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

يتضمن كل درس أيضًا قسم "استمر في التعلم" مع أدوات تعلم إضافية.

## ما تحتاجه
### لتشغيل شيفرة هذه الدورة، يمكنك استخدام أيًا من: 
 - [خدمة Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **الدروس:** "aoai-assignment"
 - [نماذج Microsoft Foundry](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **الدروس:** "githubmodels" (نماذج GitHub ستتوقف في نهاية يوليو 2026 - استخدم نماذج Microsoft Foundry بدلاً منها)
 - [واجهة OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **الدروس:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - شغّل النماذج بالكامل دون اتصال على جهازك الخاص، لا يتطلب اشتراكًا سحابيًا
   
- معرفة أساسية بـ Python أو TypeScript مفيدة - \*للمبتدئين تمامًا تحقّق من هذه الدورات [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) و[TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- حساب GitHub لعمل [fork لهذا الريبو بالكامل](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) في حساب GitHub الخاص بك

قمنا بإنشاء درس **[إعداد الدورة](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** لمساعدتك في إعداد بيئة التطوير الخاصة بك.

لا تنسى أن [تُعطي نجمة (🌟) لهذا الريبو](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) لتجده بسهولة لاحقًا.
