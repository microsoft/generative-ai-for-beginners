# إعداد السحابة ☁️ – GitHub Codespaces

**استخدم هذا الدليل إذا كنت لا تريد تثبيت أي شيء محليًا.**  
توفر لك Codespaces نسخة مجانية من VS Code تعمل عبر المتصفح مع جميع التبعيات مثبتة مسبقًا.

---

## 1.  لماذا Codespaces؟

| الفائدة | ماذا يعني ذلك بالنسبة لك |
|---------|----------------------|
| ✅ بدون تثبيت | يعمل على Chromebook و iPad وأجهزة مختبرات المدرسة... |
| ✅ حاوية تطوير مبنية مسبقًا | Python 3، Node.js، .NET، Java موجودة بالفعل |
| ✅ حصة مجانية | الحسابات الشخصية تحصل على **120 ساعة نواة / 60 ساعة جيجابايت شهريًا** |

> 💡 **نصيحة**  
> حافظ على صحة حصتك عن طريق **إيقاف** أو **حذف** كود سبيس غير النشط  
> (عرض ▸ لوحة الأوامر ▸ *Codespaces: Stop Codespace*).

---

## 2.  إنشاء Codespace (بنقرة واحدة)

1. **افتح نسخة من** هذا المستودع (زر **Fork** في الأعلى إلى اليمين).  
2. في نسختك، اضغط على **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Dialog showing buttons to create a codespace](../../../translated_images/ar/who-will-pay.4c0609b1c7780f44.webp)

✅ يفتح نافذة VS Code في المتصفح وتبدأ حاوية التطوير في الإنشاء.
تستغرق هذه العملية **حوالي دقيقتين** في المرة الأولى.

## 3. أضف مفتاح API الخاص بك (الطريقة الآمنة)

### الخيار أ أسرار Codespaces — موصى به

1. ⚙️ أيقونة الترس -> لوحة الأوامر -> Codespaces : إدارة سر المستخدم -> إضافة سر جديد.
2. الاسم: OPENAI_API_KEY
3. القيمة: ألصق المفتاح → إضافة السر

هذا كل شيء — سيستخدم رمزنا المفتاح تلقائيًا.

### الخيار ب ملف .env (إذا كنت بحاجة فعلًا إلى واحد)

```bash
cp .env.copy .env
code .env         # قم بملء OPENAI_API_KEY=مفتاحك_هنا
```
