# הקדמה לרשתות עצביות: פרספטון

> תמונות מוויקיפדיה

תמונת הקלט ייוצגה על ידי מערך פוטוצלים בגודל 20x20, כך שלרשת העצבית היו 400 כניסות ופלט בינארי אחד. רשת פשוטה הכילה נוירון אחד, שנקרא גם **יחידת לוגיקה סף**. משקלות הרשת העצבית פעלו כמו פוטנציומטרים שנדרשו לכוונון ידני במהלך שלב האימון.

> ✅ פוטנציומטר הוא מכשיר שמאפשר למשתמש לכוון את ההתנגדות של מעגל.

> הניו יורק טיימס כתב על הפרספטון באותה תקופה: *העובר של מחשב אלקטרוני ש[הצי] מצפה שיהיה מסוגל ללכת, לדבר, לראות, לכתוב, לשכפל את עצמו ולהיות מודע לקיומו.*

## מודל פרספטון

נניח שיש לנו N תכונות במודל שלנו, במקרה זה וקטור הקלט יהיה וקטור בגודל N. פרספטון הוא מודל **סיווג בינארי**, כלומר הוא יכול להבחין בין שתי קבוצות של נתוני קלט. נניח שלכל וקטור קלט x הפלט של הפרספטון שלנו יהיה או +1 או -1, בהתאם לקבוצה. הפלט יחושב באמצעות הנוסחה:

y(x) = f(w<sup>T</sup>x)

כאשר f היא פונקציית הפעלה מדרגתית

## אימון הפרספטון

כדי לאמן פרספטון, עלינו למצוא וקטור משקלות w שמסווג את רוב הערכים בצורה נכונה, כלומר מוביל לשגיאה הקטנה ביותר. שגיאה זו מוגדרת על ידי **קריטריון הפרספטון** באופן הבא:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

כאשר:

* הסכום נלקח על נקודות נתוני האימון i שמובילות לסיווג שגוי
* x<sub>i</sub> הוא נתוני הקלט, ו-t<sub>i</sub> הוא או -1 או +1 לדוגמאות שליליות וחיוביות בהתאמה.

קריטריון זה נחשב כפונקציה של משקלות w, ואנו צריכים למזער אותו. לעיתים קרובות, משתמשים בשיטה שנקראת **ירידת גרדיאנט**, שבה אנו מתחילים עם כמה משקלות ראשוניות w<sup>(0)</sup>, ואז בכל שלב מעדכנים את המשקלות לפי הנוסחה:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

כאן η הוא מה שנקרא **קצב למידה**, ו-∇E(w) מציין את **הגרדיאנט** של E. לאחר חישוב הגרדיאנט, אנו מקבלים

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

האלגוריתם ב-Python נראה כך:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
