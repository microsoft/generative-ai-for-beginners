# הגדרת ענן ☁️ – GitHub Codespaces

**השתמש במדריך זה אם אינך רוצה להתקין שום דבר באופן מקומי.**  
Codespaces נותן לך מופע VS Code מבוסס דפדפן בחינם עם כל התלויות מותקנות מראש.

---

## 1. למה Codespaces?

| יתרון | מה זה אומר עבורך |
|---------|----------------------|
| ✅ אין צורך בהתקנות | עובד על Chromebook, iPad, מחשבים במעבדת בית ספר… |
| ✅ מיכל פיתוח מוכן מראש | Python 3, Node.js, .NET, Java כבר בתוכו |
| ✅ קווטה חינמית | חשבונות אישיים מקבלים **120 שעות ליבה / 60 שעות GB בחודש** |

> 💡 **טיפ**  
> שמור על הקווטה שלך על ידי **עצירה** או **מחיקה** של codespaces שאינם בשימוש  
> (הצג ▸ Command Palette ▸ *Codespaces: Stop Codespace*).

---

## 2. צור Codespace (בלחיצה אחת)

1. **צור Fork** לרפוזיטורי הזה (כפתור **Fork** בפינה הימנית עליונה).  
2. ב-Fork שלך, לחץ על **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![דיאלוג המציג כפתורים ליצירת codespace](../../../translated_images/he/who-will-pay.4c0609b1c7780f44.webp)

✅ נפתחת חלון VS Code בדפדפן ומיכל הפיתוח מתחיל להיבנות.
זה לוקח **~2 דקות** בפעם הראשונה.

## 3. הוסף את מפתח ה-API שלך (הדרך הבטוחה)

### אפשרות א׳ Secrets של Codespaces — מומלץ

1. ⚙️ אייקון ההגדרות -> Command Pallete-> Codespaces : Manage user secret -> הוסף סוד חדש.
2. שם: OPENAI_API_KEY
3. ערך: הדבק את המפתח שלך → הוסף סוד

זהו—הקוד שלנו יקבל אותו אוטומטית.

### אפשרות ב׳ קובץ .env (אם באמת צריך)

```bash
cp .env.copy .env
code .env         # מלא את OPENAI_API_KEY=המפתח_שלך_כאן
```
