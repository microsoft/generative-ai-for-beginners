# בינה מלאכותית יוצרת למתחילים (גרסה 3) - קורס

למדו את היסודות של בניית יישומים של בינה מלאכותית יוצרת עם הקורס המקיף ב-21 שיעורים שנוצר על ידי Microsoft Cloud Advocates.

## 🌱 מתחילים

לקורס זה 21 שיעורים. כל שיעור מכסה נושא משלו, אז התחילו איפה שתרצו!

השיעורים מתויגים כ"שיעורי הלמידה" המסבירים מושג בבינה מלאכותית יוצרת או "שיעורי בנייה" המסבירים מושג ודוגמאות קוד ב**Python** וב**TypeScript** כאשר אפשרי.

למפתחי .NET בדקו את [בינה מלאכותית יוצרת למתחילים (מהדורת .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

כל שיעור כולל גם קטע "המשך ללמוד" עם כלים נוספים ללמידה.

## מה אתם צריכים
### כדי להריץ את קוד הקורס הזה, ניתן להשתמש באחד מהבאים: 
 - [שירות Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **שיעורים:** "aoai-assignment"
 - [דגמים של Microsoft Foundry](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **שיעורים:** "githubmodels" (GitHub Models ייסגר בסוף יולי 2026 - השתמשו במקום זאת בדגמי Microsoft Foundry)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **שיעורים:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - הריצו דגמים באופן מקומי לגמרי על המכשיר שלכם, ללא צורך במנוי ענן
   
- ידע בסיסי ב-Python או TypeScript מועיל - *למתחילים מוחלטים מומלץ לבדוק את הקורסים הבאים ב[Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ו[TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- חשבון GitHub כדי [לפצל את כל המאגר הזה](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) לחשבון GitHub משלכם

יצרנו שיעור **[הגדרת הקורס](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** שיעזור לכם בהגדרת סביבת הפיתוח.

אל תשכחו [להעניק כוכב (🌟) למאגר זה](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) כדי למצוא אותו בקלות בהמשך.
