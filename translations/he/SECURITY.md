## אבטחה

מיקרוסופט מתייחסת ברצינות לאבטחת מוצרי התוכנה והשירותים שלנו, כולל כל מאגרי קוד המקור המנוהלים דרך ארגוני GitHub שלנו, הכוללים את [Microsoft](https://github.com/microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet), [Xamarin](https://github.com/xamarin), ו-[ארגוני GitHub שלנו](https://opensource.microsoft.com/).

אם אתם סבורים שמצאתם פגיעות אבטחה בכל מאגר בבעלות מיקרוסופט שעומדת ב-[הגדרת הפגיעות של מיקרוסופט](https://aka.ms/opensource/security/definition), אנא דווחו לנו כפי שמתואר להלן.

## דיווח על בעיות אבטחה

**אנא אל תדווחו על פגיעות אבטחה דרך בעיות ציבוריות ב-GitHub.**

במקום זאת, דווחו עליהן למרכז תגובת האבטחה של מיקרוסופט (MSRC) ב-[https://msrc.microsoft.com/create-report](https://aka.ms/opensource/security/create-report).

אם אתם מעדיפים להגיש מבלי להיכנס למערכת, שלחו דוא"ל ל-[secure@microsoft.com](mailto:secure@microsoft.com). אם אפשר, הצפינו את ההודעה שלכם עם מפתח ה-PGP שלנו; אנא הורידו אותו מדף [מפתח ה-PGP של מרכז תגובת האבטחה של מיקרוסופט](https://aka.ms/opensource/security/pgpkey).

אתם אמורים לקבל תגובה תוך 24 שעות. אם מסיבה כלשהי לא קיבלתם, אנא עקבו באמצעות דוא"ל כדי לוודא שקיבלנו את ההודעה המקורית שלכם. מידע נוסף ניתן למצוא ב-[microsoft.com/msrc](https://aka.ms/opensource/security/msrc).

אנא כללו את המידע המבוקש המפורט להלן (כמה שיותר תוכלו לספק) כדי לעזור לנו להבין טוב יותר את טבעו והיקפו של הבעיה האפשרית:

  * סוג הבעיה (למשל, גלישת חוצץ, SQL injection, cross-site scripting וכו')
  * נתיבים מלאים של קובץ(י) המקור הקשורים להתגלמות הבעיה
  * מיקום קוד המקור המושפע (תג/ענף/התחייבות או URL ישיר)
  * כל תצורה מיוחדת הנדרשת לשחזור הבעיה
  * הוראות שלב אחר שלב לשחזור הבעיה
  * קוד הוכחת היתכנות או ניצול (אם אפשרי)
  * השפעת הבעיה, כולל כיצד תוקף עשוי לנצל את הבעיה

מידע זה יסייע לנו לסווג את הדוח שלכם במהירות רבה יותר.

אם אתם מדווחים עבור פרס באג, דוחות מלאים יותר יכולים לתרום לפרס גבוה יותר. אנא בקרו בדף [תוכנית פרס באג של מיקרוסופט](https://aka.ms/opensource/security/bounty) שלנו לפרטים נוספים על התוכניות הפעילות שלנו.
