# کلاؤڈ سیٹ اپ ☁️ – GitHub Codespaces

**اگر آپ کچھ بھی لوکل انسٹال نہیں کرنا چاہتے تو اس گائیڈ کا استعمال کریں۔**  
Codespaces آپ کو ایک مفت، براؤزر پر مبنی VS Code کا انسٹانس فراہم کرتا ہے جس میں تمام انحصارات پہلے سے انسٹال ہوتے ہیں۔

---

## 1. کیوں Codespaces؟

| فائدہ | آپ کے لیے اس کا مطلب |
|---------|----------------------|
| ✅ صفر انسٹالز | Chromebook، iPad، اسکول لیب PC پر کام کرتا ہے… |
| ✅ پہلے سے تیار ڈیولپمنٹ کنٹینر | Python 3، Node.js، .NET، Java پہلے سے موجود ہیں |
| ✅ مفت کوٹا | ذاتی اکاؤنٹس کو ہر ماہ **120 کور گھنٹے / 60 جی بی گھنٹے** ملتے ہیں |

> 💡 **مشورہ**  
> اپنا کوٹا صحت مند رکھیں، غیر فعال codespaces کو **روک کر** یا **حذف کر کے**  
> (دیکھیں ▸ کمانڈ پیلیٹ ▸ *Codespaces: Stop Codespace*).

---

## 2. ایک Codespace بنائیں (ایک کلک میں)

1. اس ریپو کو **Fork** کریں (اوپر دائیں جانب **Fork** بٹن)۔  
2. اپنی فورک میں، کلک کریں **Code ▸ Codespaces ▸ Create codespace on main**۔  
   ![ڈائیلاگ جو codespace بنانے کے بٹنز دکھا رہا ہے](../../../translated_images/ur/who-will-pay.4c0609b1c7780f44.webp)

✅ ایک براؤزر VS Code ونڈو کھلتی ہے اور ڈیولپمنٹ کنٹینر بننا شروع ہو جاتا ہے۔
پہلی بار میں یہ عمل **تقریباً 2 منٹ** لیتا ہے۔

## 3. اپنا API کلید شامل کریں (محفوظ طریقہ)

### آپشن A Codespaces Secrets — تجویز کردہ

1. ⚙️ گئر آئیکون -> کمانڈ پیلیٹ-> Codespaces : Manage user secret -> نیا سیکریٹ شامل کریں۔
2. نام: OPENAI_API_KEY
3. ویلیو: اپنی کلید چسپاں کریں → Add secret

بس یہی ہے—ہمارا کوڈ خود بخود اسے اٹھا لے گا۔

### آپشن B .env فائل (اگر واقعی ضرورت ہو)

```bash
cp .env.copy .env
code .env         # یہاں اپنی کلید ڈالیں OPENAI_API_KEY=your_key_here
```
