# ابتدائیوں کے لیے جنریٹو AI (ورژن 3) - ایک کورس

Microsoft Cloud Advocates کے ہمارے 21 اسباق پر مشتمل جامع کورس کے ذریعے جنریٹو AI ایپلیکیشنز بنانے کے بنیادی اصول سیکھیں۔

## 🌱 شروع کرنا

اس کورس میں 21 اسباق ہیں۔ ہر سبق اپنی مخصوص موضوع پر مشتمل ہے، لہٰذا جہاں چاہیں شروع کریں!

اسباق یا تو "سیکھیں" ہوتے ہیں جو جنریٹو AI کے تصور کی وضاحت کرتے ہیں یا "بنائیں" ہوتے ہیں جو تصور اور کوڈ کے نمونے دونوں **Python** اور **TypeScript** میں سمجھاتے ہیں جب ممکن ہو۔

.NET ڈویلپرز کے لیے ملاحظہ کریں [ابتدائیوں کے لیے جنریٹو AI (.NET ایڈیشن)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

ہر سبق میں ایک "مزید سیکھتے رہیں" سیکشن بھی شامل ہے جس میں اضافی تعلیمی وسائل ہوتے ہیں۔

## آپ کو کیا چاہیے
### اس کورس کا کوڈ چلانے کے لیے، آپ استعمال کر سکتے ہیں:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **اسباق:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **اسباق:** "githubmodels" (GitHub Models جولائی 2026 کے آخر میں ریٹائر ہو رہے ہیں - اس کے بجائے Microsoft Foundry Models استعمال کریں)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **اسباق:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - اپنے آلے پر ماڈلز کو مکمل طور پر آف لائن چلائیں، کسی کلاؤڈ سبسکرپشن کی ضرورت نہیں
   
- Python یا TypeScript کا بنیادی علم مفید ہے - \* مکمل نئے مبتدیوں کے لیے یہ [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) اور [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) کورسز ملاحظہ کریں
- ایک GitHub اکاؤنٹ تاکہ آپ اس پورے ریپوزٹری کو اپنے GitHub اکاؤنٹ پر [فورک](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) کر سکیں

ہم نے **[کورس سیٹ اپ](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** کا سبق بنایا ہے جو آپ کو اپنا ڈیولپمنٹ ماحول ترتیب دینے میں مدد دے گا۔

[اس repo کو ⭐ سٹار کرنا](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) نہ بھولیں تاکہ بعد میں اسے آسانی سے تلاش کر سکیں۔

## 🧠 تیار ہیں تعینات کرنے کے لیے؟

اگر آپ مزید جدید کوڈ نمونے تلاش کر رہے ہیں، تو ہمارے [جنریٹو AI کوڈ نمونوں کے مجموعے](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) کو دیکھیں جو دونوں **Python** اور **TypeScript** میں ہیں۔

## 🗣️ دیگر سیکھنے والوں سے ملاقات کریں، مدد حاصل کریں

ہمارے [سرکاری Microsoft Foundry Discord سرور](https://aka.ms/genai-discord?WT.mc_id=academic-105485-koreyst) میں شامل ہوں تاکہ دیگر سیکھنے والوں سے ملیں جو یہ کورس کر رہے ہیں اور مدد حاصل کریں۔

سوالات پوچھیں یا ہماری مصنوعات پر رائے دیں ہمارے [Microsoft Foundry Developer Forum](https://aka.ms/azureaifoundry/forum) پر GitHub پر۔
