# ابتدائی افراد کے لیے جنریٹیو اے آئی (ورژن 3) - ایک کورس

Microsoft Cloud Advocates کے ہمارے 21 اسباق کے جامع کورس کے ساتھ جنریٹیو اے آئی ایپلیکیشنز بنانے کے بنیادی اصول سیکھیں۔

## 🌱 شروعات کرنا

اس کورس میں 21 اسباق ہیں۔ ہر سبق اپنا موضوع ڈھانپتا ہے، تو جہاں چاہیں آغاز کریں!

اسباق کو یا تو "سیکھیں" کے طور پر لیبل کیا گیا ہے جو جنریٹیو اے آئی کے تصور کی وضاحت کرتے ہیں یا "بنائیں" کے طور پر جو ایک تصور اور کوڈ کے مثالیں دونوں **Python** اور **TypeScript** میں ممکن ہونے پر سمجھاتے ہیں۔

.NET ڈیولپرز کے لیے [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) دیکھیں!

ہر سبق میں اضافی سیکھنے کے وسائل کے ساتھ "مزید سیکھتے رہیں" سیکشن بھی شامل ہے۔

## آپ کو کیا چاہیے
### اس کورس کا کوڈ چلانے کے لیے، آپ یہ استعمال کر سکتے ہیں: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **اسباق:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **اسباق:** "githubmodels" (GitHub Models جولائی 2026 کے آخر میں بند ہو رہا ہے - اس کے بجائے Microsoft Foundry Models استعمال کریں)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **اسباق:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - اپنے ڈیوائس پر مکمل طور پر آف لائن ماڈلز چلائیں، کسی کلاؤڈ سبسکرپشن کی ضرورت نہیں
   
- Python یا TypeScript کا بنیادی علم فائدہ مند ہے - \*مکمل مبتدیوں کے لیے یہ [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) اور [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) کورسز دیکھیں
- اپنا گٹ ہب اکاؤنٹ بنائیں تاکہ اس پوری ریپوزیٹری کو اپنے اکاؤنٹ میں [fork](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) کرسکیں

ہم نے آپ کے ڈیولپمنٹ ماحول کی ترتیب میں مدد کے لیے **[کورس سیٹ اپ](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** سبق بنایا ہے۔

اس ریپو کو بعد میں آسانی سے تلاش کرنے کے لیے [اسے اسٹار (🌟) دینا نہ بھولیں](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)۔

## 🧠 تعیناتی کے لیے تیار؟

اگر آپ مزید جدید کوڈ نمونوں کی تلاش میں ہیں، تو ہمارے [جنریٹیو اے آئی کوڈ نمونوں کا مجموعہ](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) دیکھیں جو دونوں **Python** اور **TypeScript** میں موجود ہے۔
