## پرسیپٹرون ماڈل

فرض کریں کہ ہمارے ماڈل میں N خصوصیات ہیں، اس صورت میں ان پٹ ویکٹر N سائز کا ویکٹر ہوگا۔ پرسیپٹرون ایک **بائنری کلاسیفیکیشن** ماڈل ہے، یعنی یہ ان پٹ ڈیٹا کی دو کلاسوں کے درمیان فرق کر سکتا ہے۔ ہم فرض کریں گے کہ ہر ان پٹ ویکٹر x کے لیے ہمارے پرسیپٹرون کا آؤٹ پٹ یا تو +1 یا -1 ہوگا، جو کلاس پر منحصر ہے۔ آؤٹ پٹ درج ذیل فارمولا استعمال کرکے حساب کیا جائے گا:

y(x) = f(w<sup>T</sup>x)

جہاں f ایک سٹیپ ایکٹیویشن فنکشن ہے

## پرسیپٹرون کی تربیت

پرسیپٹرون کی تربیت کے لیے ہمیں ایک وزن ویکٹر w تلاش کرنے کی ضرورت ہے جو زیادہ تر اقدار کو صحیح طریقے سے درجہ بندی کرے، یعنی سب سے چھوٹی **غلطی** کے نتیجے میں ہو۔ یہ غلطی **پرسیپٹرون معیار** کے ذریعے اس طرح سے بیان کی گئی ہے:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

جہاں:

* یہ مجموعہ ان تربیتی ڈیٹا پوائنٹس i پر لیا جاتا ہے جو غلط درجہ بندی کا نتیجہ بنتے ہیں
* x<sub>i</sub> ان پٹ ڈیٹا ہے، اور t<sub>i</sub> منفی اور مثبت مثالوں کے لیے بالترتیب -1 یا +1 ہے۔

اس معیار کو وزن w کی ایک فنکشن کے طور پر سمجھا جاتا ہے، اور ہمیں اسے کم کرنے کی ضرورت ہے۔ اکثر، ایک طریقہ جسے **گریڈینٹ ڈیسنٹ** کہا جاتا ہے استعمال کیا جاتا ہے، جس میں ہم کچھ ابتدائی وزن w<sup>(0)</sup> سے شروع کرتے ہیں، اور پھر ہر قدم پر وزن کو اس فارمولے کے مطابق اپ ڈیٹ کرتے ہیں:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

یہاں η نام نہاد **لرننگ ریٹ** ہے، اور ∇E(w) E کا **گریڈینٹ** ظاہر کرتا ہے۔ جب ہم گریڈینٹ کا حساب لگاتے ہیں، تو ہم حاصل کرتے ہیں

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

پائتھون میں الگورتھم کچھ اس طرح نظر آتا ہے:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
