# क्लाउड सेटअप ☁️ – GitHub Codespaces

**जर तुम्हाला स्थानिक पातळीवर काहीही इंस्टॉल करायचे नसेल तर हा मार्गदर्शक वापरा.**  
Codespaces तुम्हाला एक विनामूल्य, ब्राउझर-आधारित VS Code उदाहरण देते ज्यात सर्व अवलंबित्वे आधीच इंस्टॉल केलेली असतात.

---

## 1.  Codespaces का वापरावे?

| फायदे | तुमच्यासाठी याचा अर्थ काय आहे |
|---------|----------------------|
| ✅ एकाही इंस्टॉलेशनची गरज नाही | Chromebook, iPad, शाळेतील लॅब PC वर काम करते… |
| ✅ पूर्वनिर्मित विकास कंटेनर | Python 3, Node.js, .NET, Java आधीपासूनच आहे |
| ✅ विनामूल्य कोटा | वैयक्तिक खात्यांना **प्रति महिना 120 कोर-तास / 60 GB-तास** मिळतात |

> 💡 **टीप**  
> तुमचा कोटा आरोग्यदायी ठेवण्यासाठी निष्क्रिय codespaces **थांबवा** किंवा **डिलीट करा**  
> (पाहा ▸ Command Palette ▸ *Codespaces: Stop Codespace*).

---

## 2.  एका क्लिकमध्ये Codespace तयार करा

1. या रिपोला **Fork** करा (वरच्या उजव्या बाजूला असलेला **Fork** बटण).  
2. तुमच्या Fork मध्ये क्लिक करा **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Codespace तयार करण्यासाठी बटण दाखवणारा संवाद](../../../translated_images/mr/who-will-pay.4c0609b1c7780f44.webp)

✅ एक ब्राउझर VS Code विंडो उघडेल व dev container बनवायला सुरवात होईल.
हे पहिल्यांदा सुमारे **~2 मिनिटे** लागतात.

## 3. तुमची API की जोडा (सुरक्षित मार्ग)

### पर्याय A Codespaces Secrets — शिफारस केलेला

1. ⚙️ गियर आयकॉन -> Command Palette -> Codespaces : Manage user secret -> नवीन सिक्रेट जोडा.
2. नाव: OPENAI_API_KEY
3. मूल्य: तुमची की पेस्ट करा → Add secret

एवढंच—आमचा कोड तो आपोआप ओळखून चालवेल.

### पर्याय B .env फाईल (जर तुम्हाला खरोखरच आवश्यक असेल तर)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=तुमचा_की_इथे भरा
```
