# मायक्रोसॉफ्ट ओपन सोर्स आचारसंहिता

या प्रकल्पाने [मायक्रोसॉफ्ट ओपन सोर्स आचारसंहिता](https://opensource.microsoft.com/codeofconduct/) स्वीकारली आहे.

स्रोत:

- [मायक्रोसॉफ्ट ओपन सोर्स आचारसंहिता](https://opensource.microsoft.com/codeofconduct/)
- [मायक्रोसॉफ्ट आचारसंहिता FAQ](https://opensource.microsoft.com/codeofconduct/faq/)
- प्रश्न किंवा चिंता असल्यास [opencode@microsoft.com](mailto:opencode@microsoft.com) शी संपर्क साधा

**अस्वीकरण**:  
हा दस्तऐवज AI भाषांतर सेवा [Co-op Translator](https://github.com/Azure/co-op-translator) वापरून भाषांतरित केला आहे. आम्ही अचूकतेसाठी प्रयत्नशील असलो तरी कृपया लक्षात ठेवा की स्वयंचलित भाषांतरे त्रुटी किंवा अचूकतेच्या अभावासह असू शकतात. मूळ भाषेतील दस्तऐवज अधिकृत स्रोत मानला जावा. महत्त्वपूर्ण माहितीसाठी, व्यावसायिक मानवी भाषांतराची शिफारस केली जाते. या भाषांतराचा वापर करून उद्भवणाऱ्या कोणत्याही गैरसमज किंवा चुकीच्या अर्थासाठी आम्ही जबाबदार नाही.
