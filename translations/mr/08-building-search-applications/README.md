# शोध अनुप्रयोग तयार करणे

> > _या धड्याचा व्हिडिओ पाहण्यासाठी वरील प्रतिमेवर क्लिक करा_

LLMs केवळ चॅटबॉट्स आणि मजकूर निर्मितीपेक्षा अधिक आहेत. एम्बेडिंग्ज वापरून शोध अनुप्रयोग तयार करणे देखील शक्य आहे. एम्बेडिंग्ज म्हणजे डेटा ची संख्यात्मक प्रतिनिधित्वे ज्याला वेक्टर म्हणून ओळखले जाते आणि डेटा साठी सेमॅंटिक शोधासाठी वापरले जाऊ शकते.

या धड्यात, तुम्ही आमच्या शैक्षणिक स्टार्टअपसाठी एक शोध अनुप्रयोग तयार करणार आहात. आमचे स्टार्टअप एक ना-नफा संस्था आहे जी विकसनशील देशांतील विद्यार्थ्यांना मोफत शिक्षण प्रदान करते. आमच्या स्टार्टअपकडे मोठ्या प्रमाणात YouTube व्हिडिओ आहेत ज्यांचा वापर विद्यार्थी AI शिकण्यासाठी करू शकतात. आमचे स्टार्टअप एक शोध अनुप्रयोग तयार करू इच्छित आहे जो विद्यार्थ्यांना प्रश्न टाइप करून YouTube व्हिडिओ शोधण्याची परवानगी देतो.

उदाहरणार्थ, एखादा विद्यार्थी 'जुपिटर नोटबुक्स म्हणजे काय?' किंवा 'अझ्युर एमएल म्हणजे काय?' असे टाइप करू शकतो आणि शोध अनुप्रयोग प्रश्नाशी संबंधित YouTube व्हिडिओंची यादी परत करेल, आणि अधिक चांगले म्हणजे, शोध अनुप्रयोग व्हिडिओमध्ये प्रश्नाचे उत्तर ज्या ठिकाणी आहे त्या ठिकाणाचा दुवा परत करेल.

## परिचय

या धड्यात आपण कव्हर करू:

- सेमॅंटिक वि कीवर्ड शोध.
- मजकूर एम्बेडिंग्ज म्हणजे काय.
- मजकूर एम्बेडिंग्ज निर्देशांक तयार करणे.
- मजकूर एम्बेडिंग्ज निर्देशांक शोधणे.

## शिकण्याची उद्दिष्टे

हा धडा पूर्ण केल्यानंतर, तुम्ही सक्षम असाल:

- सेमॅंटिक आणि कीवर्ड शोधातील फरक सांगणे.
- मजकूर एम्बेडिंग्ज म्हणजे काय हे स्पष्ट करणे.
- एम्बेडिंग्ज वापरून डेटा शोधण्यासाठी एक अनुप्रयोग तयार करणे.

## शोध अनुप्रयोग का तयार करावा?

शोध अनुप्रयोग तयार केल्याने तुम्हाला एम्बेडिंग्ज वापरून डेटा शोधण्याची पद्धत समजून घेण्यास मदत होईल. तुम्ही विद्यार्थ्यांना माहिती लवकर शोधण्यासाठी वापरता येईल असा शोध अनुप्रयोग तयार करणे शिकाल.

धड्यात Microsoft [AI शो](https://www.youtube.com/playlist?list=PLlrxD0HtieHi0mwteKBOfEeOYf0LJU4O1) YouTube चॅनेलसाठी YouTube प्रतिलिपींच्या एम्बेडिंग निर्देशांकाचा समावेश आहे. AI शो हा एक YouTube चॅनेल आहे जो तुम्हाला AI आणि मशीन लर्निंगबद्दल शिकवतो. एम्बेडिंग निर्देशांक ऑक्टोबर 2023 पर्यंत प्रत्येक YouTube प्रतिलिपीसाठी एम्बेडिंग्ज समाविष्ट करतो. तुम्ही आमच्या स्टार्टअपसाठी एक शोध अनुप्रयोग तयार करण्यासाठी एम्बेडिंग निर्देशांक वापरणार आहात. शोध अनुप्रयोग प्रश्नाचे उत्तर ज्या ठिकाणी आहे त्या ठिकाणाचा दुवा परत करतो. विद्यार्थ्यांना त्यांना आवश्यक माहिती लवकर शोधण्यासाठी ही एक उत्तम पद्धत आहे.

'तुम्ही रस्टुडिओचा उपयोग अझ्युर एमएलसह करू शकता का?' या प्रश्नासाठी सेमॅंटिक क्वेरीचे हे एक उदाहरण आहे. YouTube URL पहा, तुम्हाला दिसेल की URL मध्ये एक टाइमस्टॅम्प आहे जो तुम्हाला प्रश्नाचे उत्तर ज्या ठिकाणी आहे त्या ठिकाणी नेतो.

## सेमॅंटिक शोध म्हणजे काय?

आता तुम्हाला कदाचित आश्चर्य वाटेल, सेमॅंटिक शोध म्हणजे काय? सेमॅंटिक शोध ही एक शोध तंत्र आहे जी क्वेरीमधील शब्दांचे अर्थ वापरून संबंधित परिणाम परत करते.

येथे सेमॅंटिक शोधाचे एक उदाहरण आहे. समजा तुम्ही कार खरेदी करण्याचा विचार करत आहात, तुम्ही 'माझी स्वप्नातील कार' शोधू शकता, सेमॅंटिक शोध समजतो की तुम्ही कारबद्दल स्वप्न पाहत नाही, तर तुम्ही तुमची स्वप्नातील कार खरेदी करण्याचा विचार करत आहात. सेमॅंटिक शोध तुमचा हेतू समजतो आणि संबंधित परिणाम परत करतो. पर्याय म्हणजे स्वप्नांबद्दलच्या कार शोधणे आणि अनेकदा अप्रासंगिक परिणाम परत करणे.

## मजकूर एम्बेडिंग्ज म्हणजे काय?

[मजकूर एम्बेडिंग्ज](https://en.wikipedia.org/wiki/Word_embedding?WT.mc_id=academic-105485-koreyst) ही [नैसर्गिक भाषा प्रक्रिया](https://en.wikipedia.org/wiki/Natural_language_processing?WT.mc_id=academic-105485-koreyst) मध्ये वापरली जाणारी मजकूर प्रतिनिधित्व तंत्र आहे. मजकूर एम्बेडिंग्ज हे मजकूराचे सेमॅंटिक संख्यात्मक प्रतिनिधित्व आहे. एम्बेडिंग्ज डेटा एका मशीनला समजण्यास सोपे होईल अशा प्रकारे दर्शवण्यासाठी वापरले जातात. मजकूर एम्बेडिंग्ज तयार करण्यासाठी अनेक मॉडेल्स आहेत, या धड्यात, आपण OpenAI एम्बेडिंग मॉडेल वापरून एम्बेडिंग्ज तयार करण्यावर लक्ष केंद्रित करू.

येथे एक उदाहरण आहे, कल्पना करा की पुढील मजकूर AI शो YouTube चॅनेलवरील एका भागाच्या प्रतिलिपीमध्ये आहे:

आम्ही मजकूर OpenAI एम्बेडिंग API कडे पाठवू आणि तो 1536 संख्यांचा समावेश असलेला खालील एम्बेडिंग परत करेल ज्याला वेक्टर म्हणतात. वेक्टरमधील प्रत्येक संख्या मजकूराच्या वेगळ्या पैलूचे प्रतिनिधित्व करते. संक्षेपासाठी, येथे वेक्टरमधील पहिल्या 10 संख्या आहेत.

## एम्बेडिंग निर्देशांक कसा तयार केला जातो?

या धड्यासाठी एम्बेडिंग निर्देशांक Python स्क्रिप्ट्सच्या मालिकेद्वारे तयार केला गेला होता. तुम्हाला या धड्याच्या 'स्क्रिप्ट्स' फोल्डरमधील [README](./scripts/README.md?WT.mc_id=academic-105485-koreyst) मध्ये स्क्रिप्ट्स आणि सूचनांसह सापडतील. तुम्हाला या धड्याची पूर्णता करण्यासाठी या स्क्रिप्ट्स चालवण्याची गरज नाही कारण एम्बेडिंग निर्देशांक तुम्हाला प्रदान केलेला आहे.

स्क्रिप्ट्स खालील ऑपरेशन्स करतात:

1. [AI शो](https://www.youtube.com/playlist?list=PLlrxD0HtieHi0mwteKBOfEeOYf0LJU4O1) प्लेलिस्टमधील प्रत्येक YouTube व्हिडिओची प्रतिलिपी डाउनलोड केली जाते.
2. [OpenAI फंक्शन्स](https://learn.microsoft.com/azure/ai-services/openai/how-to/function-calling?WT.mc_id=academic-105485-koreyst) वापरून, YouTube प्रतिलिपीच्या पहिल्या 3 मिनिटांतून स्पीकरचे नाव काढण्याचा प्रयत्न केला जातो. प्रत्येक व्हिडिओसाठी स्पीकरचे नाव `embedding_index_3m.json` नावाच्या एम्बेडिंग निर्देशांकात संग्रहित केले जाते.
3. नंतर प्रतिलिपी मजकूर **3 मिनिटांच्या मजकूर विभागांमध्ये** विभागला जातो. विभागामध्ये एम्बेडिंग विभाग कापला जाऊ नये आणि चांगल्या शोध संदर्भासाठी जवळपास 20 शब्द पुढील विभागापासून ओलांडले जातात.
4. नंतर प्रत्येक मजकूर विभाग OpenAI चॅट API ला 60 शब्दांमध्ये मजकूर सारांशित करण्यासाठी पाठवला जातो. सारांश देखील एम्बेडिंग निर्देशांक `embedding_index_3m.json` मध्ये संग्रहित केला जातो.
5. शेवटी, विभाग मजकूर OpenAI एम्बेडिंग API ला पाठवला जातो. एम्बेडिंग API विभागाचे सेमॅंटिक अर्थ दर्शवणाऱ्या 1536 संख्यांचा वेक्टर परत करतो. विभाग आणि OpenAI एम्बेडिंग वेक्टर एम्बेडिंग निर्देशांक `embedding_index_3m.json` मध्ये संग्रहित केले जातात.

### वेक्टर डेटाबेस

धड्याच्या साधेपणासाठी, एम्बेडिंग निर्देशांक `embedding_index_3m.json` नावाच्या JSON फाइलमध्ये संग्रहित केला जातो आणि Pandas DataFrame मध्ये लोड केला जातो. तथापि, उत्पादनात, एम्बेडिंग निर्देशांक [Azure Cognitive Search](https://learn.microsoft.com/training/modules/improve-search-results-vector-search?WT.mc_id=academic-105485-koreyst), [Redis](https://cookbook.openai.com/examples/vector_databases/redis/readme?WT.mc_id=academic-105485-koreyst), [Pinecone](https://cookbook.openai.com/examples/vector_databases/pinecone/readme?WT.mc_id=academic-105485-koreyst), [Weaviate](https://cookbook.openai.com/examples/vector_databases/weaviate/readme?WT.mc_id=academic-105485-koreyst) सारख्या वेक्टर डेटाबेसमध्ये संग्रहित केला जाईल.

## कोसाइन साम्य समजून घेणे

आपण मजकूर एम्बेडिंग्जबद्दल शिकलो आहोत, पुढील चरण म्हणजे मजकूर एम्बेडिंग्जचा वापर करून डेटा शोधणे आणि विशेषतः दिलेल्या क्वेरीसाठी सर्वात समान एम्बेडिंग्ज शोधणे शिकणे, कोसाइन साम्य वापरून.

### कोसाइन साम्य म्हणजे काय?

कोसाइन साम्य दोन वेक्टरमधील साम्याचे मापन आहे, तुम्हाला हे `nearest neighbor search` म्हणून देखील ऐकायला मिळेल. कोसाइन साम्य शोध करण्यासाठी तुम्हाला OpenAI एम्बेडिंग API वापरून _क्वेरी_ मजकूरासाठी _वेक्टराइज_ करणे आवश्यक आहे. नंतर क्वेरी वेक्टर आणि एम्बेडिंग निर्देशांकातील प्रत्येक वेक्टरमधील _कोसाइन साम्य_ गणना करा. लक्षात ठेवा, एम्बेडिंग निर्देशांक प्रत्येक YouTube प्रतिलिपी मजकूर विभागासाठी वेक्टर आहे. शेवटी, कोसाइन साम्याने परिणाम क्रमवारी लावा आणि ज्या मजकूर विभागांचा कोसाइन साम्य सर्वाधिक आहे ते क्वेरीसाठी सर्वाधिक समान आहेत.

गणिताच्या दृष्टीकोनातून, कोसाइन साम्य बहुविभागीय जागेत प्रक्षेपित दोन वेक्टरमधील कोनाचे कोसाइन मोजते. हे मापन फायदेशीर आहे, कारण जर दोन दस्तऐवज आकारामुळे युक्लिडियन अंतराने दूर असतील, तरीही त्यांच्यामध्ये कमी कोन असू शकतो आणि त्यामुळे उच्च कोसाइन साम्य असू शकते. कोसाइन साम्य समीकरणांबद्दल अधिक माहितीसाठी, [कोसाइन साम्य](https://en.wikipedia.org/wiki/Cosine_similarity?WT.mc_id=academic-105485-koreyst) पहा.

## तुमचा पहिला शोध अनुप्रयोग तयार करणे

पुढे, आपण एम्बेडिंग्ज वापरून शोध अनुप्रयोग कसा तयार करायचा ते शिकणार आहोत. शोध अनुप्रयोग विद्यार्थ्यांना प्रश्न टाइप करून व्हिडिओ शोधण्याची परवानगी देईल. शोध अनुप्रयोग प्रश्नाशी संबंधित व्हिडिओंची यादी परत करेल. शोध अनुप्रयोग व्हिडिओमध्ये प्रश्नाचे उत्तर ज्या ठिकाणी आहे त्या ठिकाणाचा दुवा देखील परत करेल.

हे समाधान Windows 11, macOS, आणि Ubuntu 22.04 वर Python 3.10 किंवा नंतरच्या आवृत्तीसह तयार आणि चाचणी केले गेले आहे. तुम्ही [python.org](https://www.python.org/downloads/?WT.mc_id=academic-105485-koreyst) वरून Python डाउनलोड करू शकता.

## असाइनमेंट - विद्यार्थ्यांना सक्षम करण्यासाठी शोध अनुप्रयोग तयार करणे

आपण या धड्याच्या सुरुवातीला आमच्या स्टार्टअपची ओळख करून दिली. आता विद्यार्थ्यांना त्यांच्या मूल्यांकनांसाठी शोध अनुप्रयोग तयार करण्यास सक्षम करण्याची वेळ आली आहे.
