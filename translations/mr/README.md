# नवशिक्यांसाठी जनरेटिव एआय (आवृत्ती ३) - एक कोर्स

मायक्रोसॉफ्ट क्लाउड अ‍ॅडव्होकेट्सद्वारे आमच्या २१-धड्यांच्या सर्वसमावेशक कोर्ससह जनरेटिव एआय अनुप्रयोग तयार करण्याच्या मूलभूत गोष्टी शिका.

## 🌱 सुरुवात करताना

या कोर्समध्ये २१ धडे आहेत. प्रत्येक धडा आपला विषय समजावून सांगतो त्यामुळे तुम्हाला जिथून सुरुवात करायची आहे तेथून सुरू करा!

धडे "Learn" धडे म्हणून लेबल केलेले आहेत जे जनरेटिव एआय संकल्पना समजावून सांगतात किंवा "Build" धडे जे संकल्पना आणि **Python** आणि **TypeScript** मध्ये कोड उदाहरणे शक्य असल्यास समजावून सांगतात.

.NET विकासकांसाठी [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) बघा!

प्रत्येक धड्यामध्ये "Keep Learning" विभाग देखील आहे ज्यामध्ये अतिरिक्त शिक्षण साधने आहेत.
