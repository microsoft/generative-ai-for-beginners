# जनरेटिव AI नवशिक्यांसाठी (आवृत्ती 3) - एक कोर्स

मायक्रोसॉफ्ट क्लाउड अॅडव्होकेट्सद्वारे 21 धड्यांच्या व्यापक कोर्ससह जनरेटिव AI अनुप्रयोग तयार करण्याच्या मूलभूत गोष्टी शिकवा.

## 🌱 सुरुवात करणे

या कोर्समध्ये 21 धडे आहेत. प्रत्येक धडा त्याचा स्वतःचा विषय कव्हर करतो त्यामुळे तुम्हाला हवे तेथून सुरुवात करा!

धड्यांना "शिका" किंवा "बांधा" असे लेबल दिले जाते. "शिका" धडे जनरेटिव AI संकल्पना स्पष्ट करतात तर "बांधा" धडे संकल्पना आणि कोड उदाहरणे स्पष्ट करतात, जे शक्य असल्यास **Python** आणि **TypeScript** मध्ये आहेत.

.NET विकसकांसाठी [जनरेटिव AI नवशिक्यांसाठी (.NET संस्करण)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) पाहा!

प्रत्येक धड्यात अतिरिक्त शिकण्याची साधने असलेला "शिकणे सुरू ठेवा" विभाग देखील समाविष्ट आहे.

## तुम्हाला काय आवश्यक आहे
### या कोर्सचा कोड चालवण्यासाठी, तुम्ही खालीलपैकी कोणताही वापरू शकता:
- [Azure OpenAI सेवा](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **धडे:** "aoai-assignment"
- [GitHub मार्केटप्लेस मॉडेल कॅटलॉग](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **धडे:** "githubmodels"
- [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **धडे:** "oai-assignment"

- Python किंवा TypeScript चे मूलभूत ज्ञान उपयुक्त आहे - \*अत्यंत नवशिक्यांसाठी हे [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) आणि [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) कोर्स पहा
- [तुमच्या स्वतःच्या GitHub खात्यावर हा संपूर्ण रेपो फोर्क करण्यासाठी](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) एक GitHub खाते

तुमच्या विकास वातावरणाची सेटअप करण्यात मदत करण्यासाठी आम्ही एक **[कोर्स सेटअप](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** धडा तयार केला आहे.

[या रेपोला स्टार (🌟) द्या](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) म्हणजे नंतर ते सहज सापडेल.

## 🧠 तैनात करण्यास तयार आहात का?

जर तुम्ही अधिक प्रगत कोड नमुन्यांचा शोध घेत असाल, तर आमच्या [जनरेटिव AI कोड नमुन्यांच्या संग्रह](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) मध्ये **Python** आणि **TypeScript** मध्ये पहा.

## 🗣️ इतर शिकणाऱ्यांना भेटा, समर्थन मिळवा

या कोर्समध्ये सहभागी होणाऱ्या इतर शिकणाऱ्यांना भेटण्यासाठी आणि नेटवर्क करण्यासाठी [आमच्या अधिकृत Azure AI Foundry Discord सर्व्हर](https://aka.ms/genai-discord?WT.mc_id=academic-105485-koreyst) मध्ये सामील व्हा आणि समर्थन मिळवा.

GitHub वरील [Azure AI Foundry Developer Forum](https://aka.ms/azureaifoundry/forum) मध्ये प्रश्न विचारा किंवा उत्पादन अभिप्राय शेअर करा.

## 🚀 स्टार्टअप तयार करत आहात का?

**मोफत OpenAI क्रेडिट्स** आणि **Azure OpenAI सेवांच्या माध्यमातून OpenAI मॉडेल्समध्ये प्रवेश करण्यासाठी Azure क्रेडिट्ससाठी $150k पर्यंत** मिळवण्यासाठी [Microsoft for Startups Founders Hub](https://aka.ms/genai-foundershub?WT.mc_id=academic-105485-koreyst) साठी साइन अप करा.
