# प्रारंभिकांसाठी जनरेटिव्ह AI (आवृत्ती 3) - एक कोर्स

Microsoft Cloud Advocates कडून आमच्या 21-धड्यांच्या सर्वसमावेशक कोर्ससह जनरेटिव्ह AI अनुप्रयोग तयार करण्याच्या मूलभूत तत्त्वांचा अभ्यास करा.

## 🌱 सुरुवात करा

हा कोर्स 21 धडे आहे. प्रत्येक धडा स्वतःचा विषय समजावून सांगतो त्यामुळे तुम्हाला जिथून आवडेल तिथून सुरुवात करा!

धडे "Learn" म्हणून लेबल केलेले असतात जे जनरेटिव्ह AI संकल्पना समजवतात किंवा "Build" धडे जे संकल्पना आणि दोन्ही **Python** व **TypeScript** मध्ये कोड उदाहरणे देतात (जर शक्य असेल तर).

.NET विकसकांसाठी [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) पहा!

प्रत्येक धड्यात "Keep Learning" विभाग असतो ज्यात अतिरिक्त अभ्यासासाठी साधने असतात.

## काय आवश्यक आहे
### या कोर्सचा कोड चालवण्यासाठी, तुम्ही कोणतेही वापरू शकता: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **धडे:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **धडे:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **धडे:** "oai-assignment" 
   
- Python किंवा TypeScript चे मूलभूत ज्ञान उपयुक्त आहे - \*पूर्ण नवीन असलेल्यांसाठी या [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) व [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) कोर्स पहा
- तुमच्या स्वतःच्या GitHub खात्याला [हा संपूर्ण रेपो फोर्क करण्यासाठी आवश्यक GitHub खाते](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst)

आम्ही तुमच्या विकास वातावरण सेटअपसाठी **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** हा धडा तयार केला आहे.

नंतर सापडण्यासाठी [हा रेपो स्टार (🌟) करा](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) यास विसरू नका.
