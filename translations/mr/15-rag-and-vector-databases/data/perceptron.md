# न्यूरल नेटवर्कची ओळख: परसेप्ट्रॉन

आधुनिक न्यूरल नेटवर्कसारखे काहीतरी अंमलात आणण्याचे पहिले प्रयत्न 1957 मध्ये कॉर्नेल एरोनॉटिकल लॅबोरेटरीचे फ्रँक रोझनब्लाट यांनी केले. हे एक हार्डवेअर अंमलात आणणे होते ज्याला "मार्क-1" असे म्हटले जाते, ज्याचे उद्दिष्ट त्रिकोण, चौकोन आणि वर्तुळासारख्या मूलभूत भूमितीय आकृत्यांची ओळख पटवणे होते.

> विकिपीडियावरून प्रतिमा

इनपुट इमेज 20x20 फोटोसेल अ‍ॅरेद्वारे दर्शवली जात होती, त्यामुळे न्यूरल नेटवर्कमध्ये 400 इनपुट आणि एक बायनरी आउटपुट होते. एक साधा नेटवर्क एका न्यूरॉनचा समावेश करतो, ज्याला **थ्रेशोल्ड लॉजिक युनिट** असेही म्हणतात. न्यूरल नेटवर्कचे वजन पोटेंशिओमीटरप्रमाणे कार्य करते ज्यासाठी प्रशिक्षणाच्या टप्प्यात मॅन्युअल समायोजन आवश्यक होते.

> ✅ पोटेंशिओमीटर हे एक उपकरण आहे जे वापरकर्त्याला सर्किटच्या प्रतिकार समायोजित करण्यास अनुमती देते.

> त्या वेळी न्यूयॉर्क टाइम्सने परसेप्ट्रॉनबद्दल लिहिले: *इलेक्ट्रॉनिक संगणकाचा भ्रूण ज्याची [नेव्ही] अपेक्षा आहे की चालू शकेल, बोलू शकेल, पाहू शकेल, लिहू शकेल, स्वतःची पुनरुत्पत्ती करू शकेल आणि त्याच्या अस्तित्वाची जाणीव ठेवू शकेल.*

## परसेप्ट्रॉन मॉडेल

समजा आपल्या मॉडेलमध्ये N वैशिष्ट्ये आहेत, अशा परिस्थितीत इनपुट व्हेक्टर हा N आकाराचा व्हेक्टर असेल. परसेप्ट्रॉन हे **बायनरी वर्गीकरण** मॉडेल आहे, म्हणजेच ते इनपुट डेटाच्या दोन वर्गांमध्ये फरक करू शकते. आपण गृहीत धरू की प्रत्येक इनपुट व्हेक्टर x साठी आपल्या परसेप्ट्रॉनचे आउटपुट +1 किंवा -1 असेल, वर्गावर अवलंबून. आउटपुट खालील सूत्र वापरून गणना केली जाईल:

y(x) = f(w<sup>T</sup>x)

जेथे f एक स्टेप अ‍ॅक्टिव्हेशन फंक्शन आहे

## परसेप्ट्रॉनचे प्रशिक्षण

परसेप्ट्रॉन प्रशिक्षण देण्यासाठी, आपल्याला असे वजन व्हेक्टर w शोधावे लागेल जे बहुतेक मूल्ये योग्यरित्या वर्गीकृत करते, म्हणजेच सर्वात कमी **त्रुटी** निर्माण करते. ही त्रुटी **परसेप्ट्रॉन निकष** द्वारे खालीलप्रमाणे परिभाषित केली जाते:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

जेथे:

* बेरीज अशा प्रशिक्षण डेटा बिंदूंवर घेतली जाते जे चुकीच्या वर्गीकरणाचा परिणाम देतात
* x<sub>i</sub> हे इनपुट डेटा आहे आणि t<sub>i</sub> हे नकारात्मक आणि सकारात्मक उदाहरणांसाठी अनुक्रमे -1 किंवा +1 आहे.

हे निकष वजन w च्या फंक्शन म्हणून विचारले जाते आणि आपल्याला ते कमी करणे आवश्यक आहे. अनेकदा, **ग्रेडियंट डिसेंट** नावाची पद्धत वापरली जाते, ज्यामध्ये आपण काही प्रारंभिक वजन w<sup>(0)</sup> सह सुरू करतो आणि नंतर प्रत्येक टप्प्यावर वजन खालील सूत्रानुसार अद्यतनित करतो:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

येथे η हे तथाकथित **लर्निंग रेट** आहे आणि ∇E(w) हे E चे **ग्रेडियंट** दर्शवते. ग्रेडियंटची गणना केल्यानंतर, आपण खालील मिळवतो:

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

पायथनमधील अल्गोरिदम असे दिसते:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```

## निष्कर्ष

या धड्यात, आपण एक परसेप्ट्रॉन, जो एक बायनरी वर्गीकरण मॉडेल आहे, आणि वजन व्हेक्टर वापरून त्याचे प्रशिक्षण कसे द्यावे हे शिकले.

## 🚀 आव्हान

जर तुम्हाला स्वतःचा परसेप्ट्रॉन तयार करण्याचा प्रयत्न करायचा असेल तर मायक्रोसॉफ्ट लर्नवर Azure ML डिझायनर वापरून हे प्रयोगशाळा करून पहा.

## पुनरावलोकन आणि स्व-अभ्यास

परसेप्ट्रॉनचा वापर खेळणी समस्या तसेच वास्तविक जीवनातील समस्यांचे निराकरण करण्यासाठी कसा करता येईल हे पाहण्यासाठी, आणि शिकणे सुरू ठेवण्यासाठी - परसेप्ट्रॉन नोटबुकला जा.

परसेप्ट्रॉनबद्दल एक मनोरंजक लेख येथे आहे.

## असाइनमेंट

या धड्यात, आपण बायनरी वर्गीकरण कार्यासाठी एक परसेप्ट्रॉन लागू केला आहे आणि दोन हस्तलिखित अंकांमध्ये वर्गीकृत करण्यासाठी त्याचा वापर केला आहे. या प्रयोगशाळेत, तुम्हाला अंक वर्गीकरणाची समस्या पूर्णपणे सोडवण्यास सांगितले जाते, म्हणजे दिलेल्या प्रतिमेसाठी कोणता अंक सर्वात जास्त संभाव्य आहे हे ठरवणे.
