# ಶುರುವಾರಿಗಾಗಿ ಜನರೇಟಿವ್ ಎಐ (ಆವೃತ್ತಿ 3) - ಒಂದು ಕೋರ್ಸ್

Microsoft Cloud Advocates ನ 21 ಪಾಠಗಳ ಸಂಪೂರ್ಣ ಕೋರ್ಸಿನೊಂದಿಗೆ ಜನರೇಟಿವ್ ಎಐ ಅಪ್ಲಿಕೇಶನುಗಳನ್ನು ನಿರ್ಮಿಸುವ ಮೂಲಗಳುಗಳನ್ನು ಕಲಿಯಿರಿ.

## 🌱 ಪ್ರಾರಂಭಿಸೋಣ

ಈ ಕೋರ್ಸ್‌ಗೆ 21 ಪಾಠಗಳಿವೆ. ಪ್ರತಿ ಪಾಠವು ತನ್ನದೇ ವಿಷಯವನ್ನು ಹೊಂದಿದೆ ಆದ್ದರಿಂದ ನೀವು ಯಾವಡೆ ಇಚ್ಛಿಸಿದರೂ ಪ್ರಾರಂಭಿಸಿ!

ಪಾಠಗಳನ್ನು "ಕಲಿಯಿರಿ" ಎಂದು ಗುರುತಿಸಲಾಗಿದೆ, ಅದು ಜನರೇಟಿವ್ ಎಐ ಪರಿಕಲ್ಪನೆಯನ್ನು ವಿವರಿಸುವ ಪಾಠ ಅಥವಾ "ನಿರ್ಮಿಸಿ" ಎಂಬುದನ್ನು ವಿವರಿಸುವ ಪಾಠ ಮತ್ತು ನೀವು ಸಾಧ್ಯವಾದಲ್ಲಿ **Python** ಮತ್ತು **TypeScript** ಎರಡರಲ್ಲಿಯೂ ಕೊಂಡೆ ಉದಾಹರಣೆಗಳನ್ನು ನೀಡುತ್ತದೆ.

.NET ಡೆವಲಪರ್‌ಗಳಿಗೆ [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ಪರಿಶೀಲಿಸಿ!

ಪ್ರತಿ ಪಾಠದಲ್ಲೂ "Keep Learning" ವಿಭಾಗವೊಂದಿದೆ, ಅದು ಹೆಚ್ಚುವರಿ ಕಲಿಕೆಯ સાધನಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.
