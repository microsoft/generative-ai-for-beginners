# ಜನರೇಟಿವ್ AI ಆರಂಭಿಕರಿಗೆ (ಆವೃತ್ತಿ 3) - ಒಂದು ಕೋರ್ಸ್

ಮೈಕ್ರೋಸಾಫ್ಟ್ ಕ್ಲೌಡ್ ಅಡ್ವೊಕೇಟ್ಸ್ನ 21 ಪಾಠಗಳ ಸಂಪೂರ್ಣ ಕೋರ್ಸ್ ಮೂಲಕ ಜನರೇಟಿವ್ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ அடಿಸ்தಾಂತಗಳನ್ನು ಕಲಿಯಿರಿ.

## 🌱 ಪ್ರಾರಂಭಿಸುವುದು

ಈ ಕೋರ್ಸ್‌ಗೆ 21 ಪಾಠಗಳಿವೆ. ಪ್ರತಿ ಪಾಠವು ತನ್ನದೇ ವಿಷಯವನ್ನು ಒಳಗೊಂಡಿರುತ್ತದೆ ಹಾಗಾಗಿ ನೀವು ಇಚ್ಛಿಸುವ ಎಲ್ಲಿಯವರೆಗೆ ಪ್ರಾರಂಭಿಸಿ!

ಪಾಠಗಳನ್ನು "ಕಲುಸು" ಪಾಠಗಳು ಜನರೇಟಿವ್ AI ತತ್ತ್ವವನ್ನೂ ವಿವರಿಸುತ್ತವೆ ಅಥವಾ "ನirmaಪಿಸಿ" ಪಾಠಗಳು ತತ್ತ್ವ ಮತ್ತು ಕೋಡ್ ಉದಾಹರಣೆಯನ್ನು **Python** ಮತ್ತು **TypeScript** ಎರಡೂ ಭಾಷೆಗಳಲ್ಲಿ ವಿವರಿಸುತ್ತವೆ.

.NET ಅಭಿವೃದ್ಧಿಪಟುಗಳಿಗೆ [ಜನರೇಟಿವ್ AI ಆರಂಭಿಕರಿಗೆ (.NET ಆವೃತ್ತಿ)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ನೋಡಿ!

ಪ್ರತಿ ಪಾಠದಲ್ಲಿಯೂ "ಇನ್ನಷ್ಟು ಕಲಿಯಿರಿ" ವಿಭಾಗವಿದ್ದು ಹೆಚ್ಚುವರಿ ಕಲಿಕೆ ಸಾಧನಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

## ನೀವು ಬೇಕಾದವು
### ಈ ಕೋರ್ಸ್ ಕೋಡನ್ನು ಚಲಿಸುವುದಕ್ಕೆ, ನೀವು ನಿಮ್ಮ ಆಯ್ಕೆ ಉಪಯೋಗಿಸಬಹುದು: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **ಪಾಠಗಳು:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **ಪಾಠಗಳು:** "githubmodels" (GitHub Models ಜುಲೈ 2026 ಕೊನೆಗೆ ನಿವೃತ್ತರಾಗಲಿದೆ - ಬದಲಾಗಿ Microsoft Foundry Models ಉಪಯೋಗಿಸಿ)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **ಪಾಠಗಳು:** "oai-assignment"

- ಈ ಸಂಪೂರ್ಣ ರೆಪೊವನ್ನು ನಿಮ್ಮದೇ GitHub ಖಾತೆಗೆ [fork ಮಾಡಲು GitHub ಖಾತೆ ಅಗತ್ಯವಿದೆ](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst)
