# ಮೋಘದ ಸೆಟಪ್ ☁️ – GitHub Codespaces

**ನೀವು ಸ್ಥಳೀಯವಾಗಿ ಏನನ್ನೂ ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿಕೊಳ್ಳಬೇಕಾಗಿಲ್ಲವಾದರೆ ಈ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ಬಳಸಿ.**  
Codespaces ನಿಮಗೆ ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳೊಂದಿಗೆ ಪೂರ್ವ-ಸ್ಥಾಪಿತ ಬ್ರೌಸರ್ ಆಧಾರಿತ VS Code ಇನ್ಸ್ಟೆನ್ಸ್ ಅನ್ನು ಉಚಿತವಾಗಿ ನೀಡುತ್ತದೆ.

## 2.  Codespace ರಚಿಸಿ (ಒಂದು ಕ್ಲಿಕ್)

1. **Fork** ಮಾಡಿ ಈ ರೆಪೋವನ್ನು (ಎಡಮೇಲೆ **Fork** ಬಟನ್).  
2. ನಿಮ್ಮ fork ನಲ್ಲಿ, ಕ್ಲಿಕ್ ಮಾಡಿ **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Codespace ರಚಿಸಲು ಬಟನ್ ಗಳನ್ನು ತೋರಿಸುವ ಸಂವಾದಿ ಬಾಕ್ಸ್](../../../translated_images/kn/who-will-pay.4c0609b1c7780f44.webp)

✅ ಬ್ರೌಸರ್ VS Code ವಿಂಡೋ ತೆರೆದ ಮೇಲೆ dev container ನಿರ್ಮಾಣ ಪ್ರಾರಂಭವಾಗುತ್ತದೆ.
ಮೊದಲ ಬಾರಿ ಇದು **~2 ನಿಮಿಷ** ತೆಗೆದುಕೊಳ್ಳಬಹುದು.

## 3. ನಿಮ್ಮ API ಕೀ ಸೇರಿಸಿ (ಭದ್ರ ರೀತಿಯಲ್ಲಿ)

### ಆಯ್ಕೆ A Codespaces ರಹಸ್ಯಗಳು — ಶಿಫಾರಸು ಮಾಡಲ್ಪಟ್ಟಿದೆ

1. ⚙️ ಗಿಯರ್ ಐಕಾನ್ -> ಕಮಾಂಡ್ ಪೆಲೆಟ್-> Codespaces : ಬಳಕೆದಾರ ರಹಸ್ಯ ನಿರ್ವಹಣೆ -> ಹೊಸ ರಹಸ್ಯ ಸೇರಿಸಿ.
2. ಹೆಸರು: OPENAI_API_KEY
3. ಮೌಲ್ಯ: ನಿಮ್ಮ ಕೀ ಪೇಸ್ಟ್ ಮಾಡಿ → ರಹಸ್ಯ ಸೇರಿಸಿ

ಹೀಗೇ — ನಮ್ಮ ಕೋಡ್ ಅದನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ.

### ಆಯ್ಕೆ B .env ಫೈಲ್ (ನಿಜವಾಗಿಯೂ ಬೇಕಾದರೆ)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=ನಿಮ್ಮ_ಕೀ_ಇಲ್ಲಿ_ಪೂರೈಸಿ
```
