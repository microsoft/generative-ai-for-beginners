### 21 บทเรียนที่สอนทุกสิ่งที่คุณต้องรู้เพื่อเริ่มต้นสร้างแอปพลิเคชัน Generative AI

### 🌐 รองรับหลายภาษา

#### รองรับผ่าน GitHub Action (อัตโนมัติและอัพเดทเสมอ)

# AI สร้างสรรค์สำหรับผู้เริ่มต้น (เวอร์ชัน 3) - หลักสูตร

เรียนรู้พื้นฐานการสร้างแอปพลิเคชัน AI สร้างสรรค์กับหลักสูตร 21 บทเรียนที่ครอบคลุมโดย Microsoft Cloud Advocates

## 🌱 เริ่มต้น

หลักสูตรนี้มีทั้งหมด 21 บทเรียน แต่ละบทเรียนครอบคลุมหัวข้อของตัวเอง ดังนั้นเริ่มต้นที่ใดก็ได้ที่คุณชอบ!

บทเรียนถูกจัดเป็นบทเรียน "Learn" ที่อธิบายแนวคิด AI สร้างสรรค์ หรือบทเรียน "Build" ที่อธิบายแนวคิดและตัวอย่างโค้ดในทั้ง **Python** และ **TypeScript** เมื่อเป็นไปได้

สำหรับนักพัฒนา .NET ลองดู [AI สร้างสรรค์สำหรับผู้เริ่มต้น (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

แต่ละบทเรียนยังมีส่วน "Keep Learning" ที่มีเครื่องมือการเรียนรู้เพิ่มเติม

## สิ่งที่คุณต้องมี
### เพื่อรันโค้ดของหลักสูตรนี้ คุณสามารถใช้:
- [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "aoai-assignment"
- [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "githubmodels"
- [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "oai-assignment"

- ความรู้พื้นฐานของ Python หรือ TypeScript จะเป็นประโยชน์ - \*สำหรับผู้เริ่มต้นทั้งหมดลองดูหลักสูตร [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) และ [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- บัญชี GitHub เพื่อ [fork โครงการทั้งหมดนี้](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) ไปยังบัญชี GitHub ของคุณเอง

เราได้สร้างบทเรียน **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** เพื่อช่วยคุณในการตั้งค่าสภาพแวดล้อมการพัฒนา

อย่าลืม [star (🌟) โครงการนี้](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) เพื่อค้นหาได้ง่ายขึ้นในภายหลัง

## 🧠 พร้อมที่จะใช้งาน?

หากคุณกำลังมองหาตัวอย่างโค้ดที่ขั้นสูงขึ้น ลองดู [คอลเลกชันตัวอย่างโค้ด AI สร้างสรรค์](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) ในทั้ง **Python** และ **TypeScript**

ถามคำถามหรือแชร์ความคิดเห็นเกี่ยวกับผลิตภัณฑ์ใน [ฟอรัม Azure AI Foundry Developer](https://aka.ms/azureaifoundry/forum) บน Github

## 🚀 สร้างธุรกิจสตาร์ทอัพ?
