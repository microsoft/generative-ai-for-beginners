# Generative AI สำหรับผู้เริ่มต้น (รุ่น 3) - หลักสูตร

เรียนรู้พื้นฐานการสร้างแอปพลิเคชัน Generative AI กับหลักสูตร 21 บทเรียนโดย Microsoft Cloud Advocates

## 🌱 เริ่มต้น

หลักสูตรนี้มี 21 บทเรียน แต่ละบทเรียนครอบคลุมหัวข้อของตนเอง ดังนั้นเริ่มต้นได้จากที่คุณต้องการ!

บทเรียนจะถูกติดป้ายเป็น "เรียนรู้" ซึ่งอธิบายแนวคิด Generative AI หรือ "สร้าง" ซึ่งอธิบายแนวคิดและตัวอย่างโค้ดใน **Python** และ **TypeScript** เมื่อเป็นไปได้

สำหรับนักพัฒนา .NET โปรดดู [Generative AI สำหรับผู้เริ่มต้น (รุ่น .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

แต่ละบทเรียนยังมีส่วน "เรียนรู้ต่อ" พร้อมเครื่องมือการเรียนรู้เพิ่มเติม

## สิ่งที่คุณต้องมี
### เพื่อรันโค้ดในหลักสูตรนี้ คุณสามารถใช้ได้ทั้ง:
 - [บริการ Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "aoai-assignment"
 - [โมเดล Microsoft Foundry](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "githubmodels" (GitHub Models จะหยุดให้บริการปลายเดือนกรกฎาคม 2026 - กรุณาใช้ Microsoft Foundry Models แทน)
 - [API OpenAI](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - รันโมเดลแบบออฟไลน์เต็มรูปแบบบนอุปกรณ์ของคุณเอง ไม่ต้องสมัครสมาชิกคลาวด์
   
- ความรู้พื้นฐานของ Python หรือ TypeScript จะช่วยได้ - \*สำหรับผู้เริ่มต้นอย่างแท้จริง โปรดดูหลักสูตร [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) และ [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- บัญชี GitHub สำหรับ [fork ที่เก็บทั้งหมดนี้](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) ไปยังบัญชี GitHub ของคุณเอง

เราได้สร้างบทเรียน **[การตั้งค่าหลักสูตร](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** เพื่อช่วยคุณตั้งค่าสภาพแวดล้อมการพัฒนา

อย่าลืม [กดดาว (🌟) ที่ที่เก็บนี้](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) เพื่อหาง่ายขึ้นในภายหลัง
