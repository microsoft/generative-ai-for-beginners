# ปัญญาประดิษฐ์เชิงสร้างสรรค์สำหรับผู้เริ่มต้น (เวอร์ชัน 3) - หลักสูตร

เรียนรู้พื้นฐานการสร้างแอปพลิเคชันปัญญาประดิษฐ์เชิงสร้างสรรค์ผ่านหลักสูตรครอบคลุม 21 บทเรียนโดย Microsoft Cloud Advocates

## 🌱 เริ่มต้น

หลักสูตรนี้มี 21 บทเรียน แต่ละบทเรียนจะครอบคลุมหัวข้อของตัวเอง คุณสามารถเริ่มจากบทเรียนใดก็ได้ตามที่คุณต้องการ!

บทเรียนจะถูกแบ่งเป็นบทเรียนประเภท "เรียนรู้" ที่อธิบายแนวคิดของปัญญาประดิษฐ์เชิงสร้างสรรค์ หรือบทเรียนประเภท "สร้าง" ที่อธิบายแนวคิดและตัวอย่างโค้ดทั้งในรูปแบบ **Python** และ **TypeScript** เมื่อเป็นไปได้

สำหรับนักพัฒนา .NET ลองดู [ปัญญาประดิษฐ์เชิงสร้างสรรค์สำหรับผู้เริ่มต้น (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

แต่ละบทเรียนยังมีส่วน "เรียนรู้ต่อ" ที่มีแหล่งการเรียนรู้อื่น ๆ เพิ่มเติม

## สิ่งที่คุณต้องมี

 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **บทเรียน:** "oai-assignment"

- ความรู้พื้นฐานเกี่ยวกับ Python หรือ TypeScript จะช่วยได้ - \*สำหรับผู้เริ่มต้นอย่างสมบูรณ์ลองดูหลักสูตร [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) และ [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- บัญชี GitHub เพื่อ [fork รีโพสนี้ทั้งหมด](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) ไปยังบัญชี GitHub ของคุณเอง

เราได้สร้างบทเรียน **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** เพื่อช่วยคุณในการตั้งค่าสภาพแวดล้อมการพัฒนา

อย่าลืม [กดดาว (🌟) ที่รีโพของนี้](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) เพื่อค้นหาได้ง่ายในภายหลัง
