# Generatív MI kezdőknek (3. verzió) - Egy tanfolyam

Tanuld meg a generatív MI alkalmazások építésének alapjait 21 leckéből álló átfogó tanfolyamunkkal, amelyet a Microsoft Cloud Advocates készítettek.

## 🌱 Kezdés

Ennek a tanfolyamnak 21 leckéje van. Minden lecke egy-egy témát fed le, szóval kezdj bárhol, ahol szeretnél!

A leckék vagy "Tanulj meg" leckék, amelyek egy generatív MI fogalmat magyaráznak el, vagy "Építs" leckék, amelyek egy fogalmat és kód példákat is tartalmaznak, lehetőség szerint **Python** és **TypeScript** nyelven.

.NET fejlesztőknek nézd meg a [Generatív MI kezdőknek (.NET kiadás)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) projektet!

Minden lecke tartalmaz egy "Tovább tanulás" szekciót is további tanulási eszközökkel.

## Mire lesz szükséged
### A tanfolyam kódjának futtatásához használhatod bármelyiket:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Leckék:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Leckék:** "githubmodels" (A GitHub Models 2026 júliusának végén megszűnik - helyette a Microsoft Foundry Models használata javasolt)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Leckék:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Modellek teljesen offline futtatása saját eszközön, felhő előfizetés nélkül
   
- Alapvető Python vagy TypeScript ismeretek hasznosak - \*Teljesen kezdőknek nézd meg ezeket a [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) és [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) tanfolyamokat
- Egy GitHub fiók a [tárhely teljes fork-olásához](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) a saját GitHub fiókodba

Létrehoztunk egy **[Tanfolyam beállítása](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** leckét, hogy segítsünk a fejlesztői környezeted beállításában.

Ne felejts el [csillagozni (🌟) ezt a tárhelyet](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), hogy később könnyebben megtaláld.
