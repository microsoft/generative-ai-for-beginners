# IA Generativa para Iniciantes (Versão 3) - Um Curso

Aprenda os fundamentos para construir aplicações de IA Generativa com nosso curso abrangente de 21 lições apresentado pelos Microsoft Cloud Advocates.

## 🌱 Começando

Este curso tem 21 lições. Cada lição cobre seu próprio tópico, então comece onde quiser!

As lições são rotuladas como "Aprender", explicando um conceito de IA Generativa, ou "Construir" que explica um conceito e exemplos de código em **Python** e **TypeScript** sempre que possível.

Para desenvolvedores .NET confira [IA Generativa para Iniciantes (Edição .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Cada lição também inclui uma seção "Continue Aprendendo" com ferramentas adicionais de aprendizado.

## O Que Você Precisa
### Para executar o código deste curso, você pode usar:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Lições:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Lições:** "githubmodels" (Modelos GitHub será descontinuado no final de julho de 2026 - use Microsoft Foundry Models em vez disso)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Lições:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Execute modelos totalmente offline no seu próprio dispositivo, sem assinatura na nuvem
   
- Conhecimentos básicos de Python ou TypeScript são úteis - \*Para iniciantes absolutos confira estes cursos de [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) e [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Uma conta GitHub para [fazer fork deste repositório inteiro](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) para sua própria conta GitHub

Criamos uma lição **[Configuração do Curso](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** para ajudar você na configuração do seu ambiente de desenvolvimento.

Não esqueça de [favoritar (🌟) este repositório](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) para encontrá-lo facilmente depois.
