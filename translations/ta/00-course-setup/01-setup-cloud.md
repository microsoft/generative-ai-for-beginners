# கிளவுட் அமைப்பு ☁️ – GitHub Codespaces

**உங்கள் கணினியில் எதையும் நிறுவ விரும்பவில்லையென்றால் இந்த வழிகாட்டியைக் பயன்படுத்துங்கள்.**  
Codespaces உங்களுக்கு அனைத்து சார்புகள் முன்நிறுத்தப்பட்டுள்ள இலவசமான, உலாவி அடிப்படையிலான VS Code நிகழ்வு வழங்குகிறது.

---

## 1.  ஏன் Codespaces?

| பயன் | உங்கள் குறித்த பொருள் |
|---------|----------------------|
| ✅ பொருத்தமில்லா நிறுவல்கள் | Chromebook, iPad, பள்ளி பராமரிப்பு கணினிகள் போன்றவற்றில் வேலை செய்கிறது… |
| ✅ முன்நிறுத்தப்பட்ட dev கொண்டெய்னர் | Python 3, Node.js, .NET, Java ஏற்கனவே உள்ளன |
| ✅ இலவச அளவு | தனிப்பட்ட கணக்குகள் **மாதத்திற்கு 120 கோர்-மணி / 60 GB-மணி** பெறுகின்றன |

> 💡 **உற்சாகக் குறிப்புரை**  
> உங்கள் அளவை ஆரோக்கியமாக வைக்க **நிறுத்தவும்** அல்லது **அழிக்கவும்** தவறான codespaces-ஐ  
> (பார்வையிடவும் ▸ கட்டளை பட்டியல் ▸ *Codespaces: Stop Codespace*).

---

## 2.  Codespace உருவாக்கவும் (ஒரே கிளிக்)

1. இந்த repஐ **Fork** செய்யவும் (மேலே-வலது **Fork** பட்டன்).  
2. உங்கள் forkல், கிளிக் செய்யவும் **Code ▸ Codespaces ▸ Create codespace on main**.  
   ![Codespace உருவாக்கும் பட்டன்களை காட்டும் உரையாடல்](../../../translated_images/ta/who-will-pay.4c0609b1c7780f44.webp)

✅ ஒரு உலாவி VS Code சாளரம் திறக்கப்படுவதுடன் dev கொண்டெய்னர் உருவாக ஆரம்பிக்கிறது.
இது முதன்முறையாக **~2 நிமிடங்கள்** எடுக்கும்.

## 3. உங்கள் API விசையை சேர்க்கவும் (பாதுகாப்பான வழி)

### விருப்பம் A Codespaces ரகசியங்கள் — பரிந்துரைக்கப்பட்டது

1. ⚙️ கியர் ஐகான் -> கட்டளை பட்டியல் -> Codespaces : பயனர் ரகசியத்தை நிர்வகிக்க -> புதிய ரகசியத்தைச் சேர்.
2. பெயர்: OPENAI_API_KEY
3. மதிப்பு: உங்கள் விசையை ஒட்டவும் → ரகசியமாகச் சேர்

அவ்வளவுதான்—எங்கள் குறியீடு தானாகவே அதை எடுத்துக் கொள்வதற்கு தயாராகும்.

### விருப்பம் B .env கோப்பு (உறுதியாக தேவையெனில்)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=உங்கள்_சாவி_இங்கே என்பதை நிரப்புக
```
