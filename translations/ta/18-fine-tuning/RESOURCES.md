# சுய-வழிகாட்டப்பட்ட கற்றலுக்கான வளங்கள்

பாடம் வழிகாட்டுகளுக்கான OpenAI மற்றும் Microsoft Foundry இன் மைய வளங்களை பயன்படுத்தி உருவாக்கப்பட்டது. இது ஒரு முழுமைமிக்க பட்டியல்ல; உங்கள் சுய-வழிகாட்டப்பட்ட கற்றல் பயணங்களுக்கு இது உதவும். கீழே உள்ள அனைத்து இணைப்புகளும் தற்போது ஆதரிக்கப்படும் பொருட்களை குறிக்கின்றன.

## 2. இரண்டாம் நிலை வளங்கள்

இந்த பிரிவு பாடத்தில் காலப்போக்கில் விரிவாகக் கையாள்யாத மேலதிக வளங்களைச் சேகரிக்கிறது. இந்த வளங்களை பயன்படுத்தி இந்த தலைப்பின் சார்பாக உங்கள் திறமையை வளர்க்கவும்.
