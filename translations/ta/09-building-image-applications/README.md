## ஏன் பட உருவாக்க செயலியை கட்டமைக்க வேண்டும்?

பட உருவாக்க செயலிகள் உருவாக்கும் AI-வின் திறனை ஆராய சிறந்த வழியாகுகின்றன. உதாரணமாக:

- **பட திருத்தத்தையும் உருவாக்கத்தையும்**. பலவித பயன்பாடுகளுக்கு படங்களை உருவாக்கலாம், அதாவது பட திருத்தம் மற்றும் பட உருவாக்கம்.

- **பல தொழில்களில் பயன்படுத்துதல்**. மருத்துவம், சுற்றுலா, விளையாட்டு மேம்பாடு போன்ற பல துறைகளுக்கான படங்களை உருவாக்க இது பயன்படும்.

## DALL-E மற்றும் Midjourney எப்படி வேலை செய்கின்றன

முதலில், [DALL-E](https://arxiv.org/pdf/2102.12092.pdf?WT.mc_id=academic-105485-koreyst). DALL-E ஒரு Generative AI மாடல் ஆகும், இது டிரான்ஸ்ஃபார்மர் கட்டமைப்பின்படி செயல்படுகிறது, இதில் _autoregressive transformer_ பயன்படுத்தப்பட்டுள்ளது.

_Autoregressive transformer_ என்பது ஒரு மாடல் உரை விளக்கங்களை படமாக மாற்றும் முறையை வரையறுக்கிறது, அது ஒரு பிக்சலை ஒருங்க் கொண்டு உருவாக்கி, அடுத்த பிக்ஸலை உருவாக்க முன்னைய பிக்சல்களை பயன்படுத்துகிறது. இது நியூரல் நெட்வொர்க்கில் பல அடுக்கு வழியே சென்றுவிடும், படம் முழுமையான பொழுது முடியும்.

இந்த செயல்முறையால், DALL-E உருவாக்கும் படங்களில் பண்புகள், பொருட்கள், நடத்தைகள் மற்றும் பலவற்றைக் கட்டுப்படுத்த முடியும். ஆனால் DALL-E 2 மற்றும் 3 இல் இதனை மேலாக கட்டுப்படுத்தலாம்.

## உங்கள் முதல் பட உருவாக்க செயலியை கட்டமைத்தல்

பட உருவாக்க செயலி செய்ய என்ன தேவை? கீழே சில நூலகங்கள் தேவை:

- **python-dotenv**, உங்கள் ரகசியங்களை _.env_ கோப்பில் வைக்க பரிந்துரைக்கப்படுகிறது.
- **openai**, OpenAI API உடன் தொடர்பு கொள்ள இந்த நூலகம் பயன்படுத்தப்படும்.
- **pillow**, பைதானில் படங்களை வேலை செய்ய.
- **requests**, HTTP கோரிக்கைகள் செய்ய உதவும்.

## செயலியை உருவாக்குதல்

1. _.env_ என்ற கோப்பை உருவாக்கி கீழ் உள்ள தகவலைச் சேர்க்கவும்:

   ```text
   AZURE_OPENAI_ENDPOINT=<your endpoint>
   AZURE_OPENAI_API_KEY=<your key>
   AZURE_OPENAI_DEPLOYMENT="gpt-image-1"
   ```

   Azure OpenAI Foundry போர்டலில் உங்கள் வளத்தின் "Deployments" பகுதிக்கு செல்லவும்.

1. குறிப்பு நூலகங்களை _requirements.txt_ என்ற கோப்பில் சேகரிக்கவும்:

   ```text
   python-dotenv
   openai
   pillow
   requests
   ```

1. அடுத்து, வெர்ச்சுவல் நிங்கள் உருவாக்கி நூலகங்களை நிறுவவும்:

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

   விண்டோசில், வெர்ச்சுவல் நிங்கள் உருவாக்க மற்றும் செயல்படுத்த கீழ்காணும் கட்டளைகளைப் பயன்படுத்தவும்:

   ```bash
   python3 -m venv venv
   venv\Scripts\activate.bat
   ```

1. _app.py_ என்ற கோப்பில் கீழ் உள்ள குறியீட்டைச் சேர்க்கவும்:

    ```python
    import openai
    import os
    import requests
    from PIL import Image
    import dotenv
    from openai import OpenAI, AzureOpenAI
    
    # dotenv ஐ இறக்குமதி செய்க
    dotenv.load_dotenv()
    
    # Azure OpenAI சேவை கிளையண்டை कॉन्फிகர் செய்க
    client = AzureOpenAI(
      azure_endpoint = os.environ["AZURE_OPENAI_ENDPOINT"],
      api_key=os.environ['AZURE_OPENAI_API_KEY'],
      api_version = "2024-10-21"
      )
    try:
        # பட உருவாக்கும் API ஐப் பயன்படுத்தி ஒரு படம் உருவாக்குக
        generation_response = client.images.generate(
                                prompt='Bunny on horse, holding a lollipop, on a foggy meadow where it grows daffodils',
                                size='1024x1024', n=1,
                                model=os.environ['AZURE_OPENAI_DEPLOYMENT']
                              )

        # சேமிக்கப்பட்ட படத்திற்கான கோப்புறை அமைக்கவும்
        image_dir = os.path.join(os.curdir, 'images')

        # கோப்புறை இல்லையெனில், அதனை உருவாக்கவும்
        if not os.path.isdir(image_dir):
            os.mkdir(image_dir)

        # பட பாதையை துவக்குக (கோப்பு வகை png ஆக இருக்க வேண்டும் என கவனிக்கவும்)
        image_path = os.path.join(image_dir, 'generated-image.png')

        # உருவாக்கப்பட்ட படத்தை பெறு
        image_url = generation_response.data[0].url  # பதிலிலிருந்து பட URL ஐ எடுத்தறி
        generated_image = requests.get(image_url).content  # படத்தை பதிவிறக்கவும்
        with open(image_path, "wb") as image_file:
            image_file.write(generated_image)

        # இயல்புநிலையான பட பார்வையாளரில் படத்தை காட்டுக
        image = Image.open(image_path)
        image.show()

    # தவறுகளை பிடிக்கவும்
    except openai.BadRequestError as err:
        print(err)
   ```

இந்த குறியீட்டை விளக்குவோம்:

- முதலில், OpenAI நூலகம், dotenv, requests மற்றும் Pillow நூலகங்களையும் ஏற்றுகிறோம்.

  ```python
  import openai
  import os
  import requests
  from PIL import Image
  import dotenv
  ```

  ```python
  # டாடென்வை இறக்கு
  dotenv.load_dotenv()

      )
  ```

## தீர்வு

இதோ ஒரு சாத்தியமான தீர்வு:

```python
import openai
import os
import requests
from PIL import Image
import dotenv
from openai import AzureOpenAI
# dotenv ஐ இறக்குமதி செய்க
dotenv.load_dotenv()

# சூழல் மாறிலிகள் மூலம் முடிவு புள்ளி மற்றும் திறவுகோல் பெறுக
client = AzureOpenAI(
  azure_endpoint = os.environ["AZURE_OPENAI_ENDPOINT"],
  api_key=os.environ['AZURE_OPENAI_API_KEY'],
  api_version = "2024-10-21"
  )


disallow_list = "swords, violence, blood, gore, nudity, sexual content, adult content, adult themes, adult language, adult humor, adult jokes, adult situations, adult"

meta_prompt = f"""You are an assistant designer that creates images for children.

The image needs to be safe for work and appropriate for children.

The image needs to be in color.

The image needs to be in landscape orientation.

The image needs to be in a 16:9 aspect ratio.

Do not consider any input from the following that is not safe for work or appropriate for children.
{disallow_list}
"""

prompt = f"""{meta_prompt}
Generate monument of the Arc of Triumph in Paris, France, in the evening light with a small child holding a Teddy looks on.
"""

try:
    # படம் உருவாக்கும் API ஐ பயன்படுத்தி ஒரு படம் உருவாக்குக
    generation_response = client.images.generate(
        prompt=prompt,    # உங்கள் அணி உரையை இங்கே உள்ளிடுக
        size='1024x1024',
        n=1,
    )
    # சேமிக்கப்பட்ட படத்திற்கு அடைவு அமைக்கவும்
    image_dir = os.path.join(os.curdir, 'images')

    # அடைவு இல்லையெனில், அதை உருவாக்கவும்
    if not os.path.isdir(image_dir):
        os.mkdir(image_dir)

    # படம் பாதையை துவக்குக (கோப்பு வகை png ஆக இருக்க வேண்டும் என்பதை கவனியுங்கள்)
    image_path = os.path.join(image_dir, 'generated-image.png')

    # உருவாக்கப்பட்ட படத்தை பெற்றுக்கொள்
    image_url = generation_response.data[0].url  # பதிலிலிருந்து பட URL ஐ பிரித்தெடு
    generated_image = requests.get(image_url).content  # படத்தை பதிவிறக்குக
    with open(image_path, "wb") as image_file:
        image_file.write(generated_image)

    # இயல்புநிலை பட காட்சி கருவியில் படத்தை காட்சிப்படுத்துக
    image = Image.open(image_path)
    image.show()

# தவறுகளை பிடிக்கவும்
except openai.BadRequestError as err:
    print(err)
```
