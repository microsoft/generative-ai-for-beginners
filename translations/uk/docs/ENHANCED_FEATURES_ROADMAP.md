## 4. Модернізація API

### 4.1 Застарілі патерни API (Міграція завершена)

Всі зразки Python і TypeScript для **чату** були мігрувані з Chat Completions API до **Responses API** (`client.responses.create(...)` → `response.output_text`).

| Старий патерн | Новий патерн | Статус |
|-------------|-------------|--------|
| `openai.api_type = "azure"` / `AzureOpenAI()` (чат) | `OpenAI(base_url="<endpoint>/openai/v1/")` (Responses API) | Завершено |
| `openai.ChatCompletion.create()` / `client.chat.completions.create()` | `client.responses.create(input=...)` → `response.output_text` | Завершено |
| `@azure/openai` `OpenAIClient.getChatCompletions()` (TypeScript) | Пакет `openai` `client.responses.create()` → `response.output_text` | Завершено |
| `df.append()` (pandas) | `pd.concat()` | Завершено |

> **Примітка:** Зразки Microsoft Foundry Models, які використовують SDK `azure-ai-inference` / `@azure-rest/ai-inference` (`client.complete()`), залишаються на Model Inference API, який не підтримує Responses API. `AzureOpenAI()` свідомо збережено там, де це ще актуально (ембеддінги та генерація зображень).

### 4.2 Нові функції API для демонстрації

1. **Структуровані виходи** (OpenAI)
   - JSON режим
   - Виклик функцій зі строгими схемами

2. **Можливості зору**
   - Аналіз зображень за допомогою GPT-4o (зір)
   - Мультимодальні підказки

3. **Вбудовані інструменти Responses API** (заміна застарілого Assistants API)

## 5. Покращення інфраструктури

### 5.1 Покращення CI/CD

Реалізовано у [`.github/workflows/code-quality.yml`](../../../.github/workflows/code-quality.yml): лінтинг/форматування Python (Ruff + Black) **примусово** на підтримуваному модулі `shared/` та у якості рекомендації для решти курсу, а також рекомендаційний прогін ESLint для JavaScript/TypeScript. Ілюстративна базова лінія була:

```yaml
# .github/workflows/code-quality.yml
name: Code Quality

on: [push, pull_request]

jobs:
  python-lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.10'
      - run: pip install ruff black mypy
      - run: ruff check .
      - run: black --check .

  js-lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm ci
      - run: npx eslint .
```

### 5.2 Сканування безпеки

Реалізовано у [`.github/workflows/security.yml`](../../../.github/workflows/security.yml): аналіз CodeQL для Python та JavaScript/TypeScript (при пушах, пул-реквестах та за тижневим розкладом), а також огляд залежностей у пул-реквестах. Ілюстративна базова лінія була:

```yaml
# .github/workflows/security.yml
name: Security Scan

on: [push, pull_request]

jobs:
  codeql:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: github/codeql-action/init@v3
        with:
          languages: javascript, python
      - uses: github/codeql-action/analyze@v3

  dependency-review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/dependency-review-action@v4
```

Реалізовано в [`.devcontainer/devcontainer.json`](../../../.devcontainer/devcontainer.json) та [`.devcontainer/post-create.sh`](../../../.devcontainer/post-create.sh): тепер контейнер містить розширення Pylance, Black formatter, Ruff, ESLint, Prettier і Copilot, активує форматування при збереженні, підключене до налаштувань Black/Prettier у репозиторії, і встановлює інструменти розробника (`ruff`, `black`, `mypy`, `pytest`), щоб [потік роботи з якістю коду](../../../.github/workflows/code-quality.yml) можна було відтворити локально. Базове зображення `mcr.microsoft.com/devcontainers/universal` вже містить Python та Node, тому додаткові функції не потрібні. Ілюстративна базова версія була:

```json
{
  "name": "Generative AI for Beginners",
  "image": "mcr.microsoft.com/devcontainers/universal:2",
  "features": {
    "ghcr.io/devcontainers/features/python:1": {
      "version": "3.11"
    },
    "ghcr.io/devcontainers/features/node:1": {
      "version": "20"
    }
  },
  "customizations": {
    "vscode": {
      "extensions": [
        "ms-python.python",
        "ms-python.vscode-pylance",
        "ms-toolsai.jupyter",
        "dbaeumer.vscode-eslint",
        "esbenp.prettier-vscode",
        "github.copilot"
      ],
      "settings": {
        "python.formatting.provider": "black",
        "editor.formatOnSave": true
      }
    }
  },
  "postCreateCommand": "pip install -e .[dev] && npm install"
}
```
