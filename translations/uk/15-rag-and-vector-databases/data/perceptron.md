# Вступ до нейронних мереж: Перцептрон

Одна з перших спроб реалізувати щось схоже на сучасну нейронну мережу була здійснена Френком Розенблаттом з Корнельської аеронавтичної лабораторії у 1957 році. Це була апаратна реалізація під назвою "Mark-1", призначена для розпізнавання примітивних геометричних фігур, таких як трикутники, квадрати та кола.

## Модель перцептрона

Припустимо, що у нашій моделі є N ознак, у такому разі вхідний вектор буде вектором розміру N. Перцептрон є моделлю **бінарної класифікації**, тобто він може розрізняти два класи вхідних даних. Ми будемо припускати, що для кожного вхідного вектора x вихід нашого перцептрона буде або +1, або -1, залежно від класу. Вихід буде обчислюватися за формулою:

y(x) = f(w<sup>T</sup>x)

де f — це ступенева функція активації

## Навчання перцептрона

Щоб навчити перцептрон, нам потрібно знайти вектор ваг w, який класифікує більшість значень правильно, тобто дає найменшу **помилку**. Ця помилка визначається **критерієм перцептрона** наступним чином:

E(w) = -∑w<sup>T</sup>x<sub>i</sub>t<sub>i</sub>

де:

* сума береться за тими точками навчальних даних i, які призводять до неправильної класифікації
* x<sub>i</sub> — це вхідні дані, а t<sub>i</sub> — це або -1, або +1 для негативних і позитивних прикладів відповідно.

Цей критерій розглядається як функція ваг w, і ми повинні його мінімізувати. Часто використовується метод, який називається **градієнтним спуском**, у якому ми починаємо з деяких початкових ваг w<sup>(0)</sup>, а потім на кожному кроці оновлюємо ваги за формулою:

w<sup>(t+1)</sup> = w<sup>(t)</sup> - η∇E(w)

Тут η — це так звана **швидкість навчання**, а ∇E(w) позначає **градієнт** E. Після обчислення градієнта ми отримуємо

w<sup>(t+1)</sup> = w<sup>(t)</sup> + ∑ηx<sub>i</sub>t<sub>i</sub>

Алгоритм на Python виглядає так:

```python
def train(positive_examples, negative_examples, num_iterations = 100, eta = 1):

    weights = [0,0,0] # Initialize weights (almost randomly :)
        
    for i in range(num_iterations):
        pos = random.choice(positive_examples)
        neg = random.choice(negative_examples)

        z = np.dot(pos, weights) # compute perceptron output
        if z < 0: # positive example classified as negative
            weights = weights + eta*weights.shape

        z  = np.dot(neg, weights)
        if z >= 0: # negative example classified as positive
            weights = weights - eta*weights.shape

    return weights
```
