# Генеративний ШІ для початківців (Версія 3) - Курс

Вивчайте основи створення додатків Генеративного ШІ за допомогою нашого 21-урочного курсу від Microsoft Cloud Advocates.

## 🌱 Початок роботи

Цей курс складається з 21 уроку. Кожен урок охоплює свою тему, тому починайте з будь-якого, що вам подобається!

Уроки позначені як "Вчимося", які пояснюють концепцію Генеративного ШІ, або "Будуємо", які пояснюють концепцію та наводять приклади коду на **Python** та **TypeScript**, коли це можливо.

Для розробників .NET перегляньте [Генеративний ШІ для початківців (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Кожен урок також включає розділ "Продовжуйте вчитися" з додатковими навчальними інструментами.

## Що вам потрібно
### Щоб запустити код цього курсу, ви можете використовувати або:
- [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Уроки:** "aoai-assignment"
- [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **Уроки:** "githubmodels"
- [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Уроки:** "oai-assignment"

- Базові знання Python або TypeScript будуть корисними - \*Для абсолютних початківців ознайомтесь з цими курсами [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) та [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Обліковий запис GitHub для [форкування цього репозиторію](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) у свій власний обліковий запис GitHub

Ми створили урок **[Налаштування курсу](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**, щоб допомогти вам з налаштуванням вашого середовища розробки.

Не забудьте [позначити зірочкою (🌟) цей репозиторій](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), щоб легше знайти його пізніше.

## 🧠 Готові до розгортання?

Якщо ви шукаєте більш просунуті приклади коду, перегляньте нашу [колекцію прикладів коду Генеративного ШІ](https://aka.ms/genai-beg-code?WT.mc_id=academic-105485-koreyst) як на **Python**, так і на **TypeScript**.

Задавайте питання або діліться відгуками про продукти на нашому [форумі розробників Azure AI Foundry](https://aka.ms/azureaifoundry/forum) на Github.
