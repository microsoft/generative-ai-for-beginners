# Генеративний ШІ для Початківців (Версія 3) - Курс

Вивчайте основи створення додатків на базі Генеративного ШІ з нашим комплексним курсом із 21 уроку від Microsoft Cloud Advocates.

## 🌱 Початок роботи

Цей курс містить 21 урок. Кожен урок присвячений окремій темі — починайте з будь-якого, що забажаєте!

Уроки поділені на "Learn" уроки, які пояснюють концепції Генеративного ШІ, та "Build" уроки, які пояснюють концепції та надають приклади коду на **Python** і **TypeScript**, коли це можливо.

Для розробників .NET перегляньте [Генеративний ШІ для Початківців (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Кожен урок також містить розділ "Keep Learning" із додатковими навчальними матеріалами.

## Що вам потрібно
### Для запуску коду курсу ви можете використовувати:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Уроки:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Уроки:** "githubmodels" (GitHub Models припиняє роботу наприкінці липня 2026 року — використовуйте Microsoft Foundry Models замість нього)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Уроки:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Запускайте моделі повністю офлайн на власному пристрої, підписка на хмару не потрібна
   
- Базові знання Python або TypeScript будуть корисними — \*для повних початківців перегляньте ці курси [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) та [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Обліковий запис GitHub для [форкання цього репозиторію цілком](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) до вашого власного облікового запису на GitHub

Ми створили урок **[Налаштування курсу](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**, щоб допомогти вам налаштувати середовище розробки.

Не забудьте [поставити зірочку (🌟) цьому репозиторію](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), щоб легше було знайти пізніше.
