# ആരംഭക്കാർക്കുള്ള ജനറേറ്റീവ് AI (ആവൃത്തി 3) - ഒരു കോഴ്‌സ്

മൈക്രോസോഫ്റ്റ് ക്ലൗഡ് അഡ്വക്കേറ്റ്സിന്റെ 21 പാഠങ്ങൾ ഉൾക്കൊള്ളുന്ന സമഗ്ര കോഴ്‌സ് ഉപയോഗിച്ച് ജനറേറ്റീവ് AI അപ്ലിക്കേഷനുകൾ നിർമ്മിക്കാനുള്ള അടിസ്ഥാനകൂട്ടങ്ങൾ പഠിക്കുക.

## 🌱 തുടങ്ങുന്നത്

ഈ കോഴ്‌സിൽ 21 പാഠങ്ങളുണ്ട്. ഓരോ പാഠവും അതിന്റെ സ്വതന്ത്ര വിഷയം ഉൾക്കൊള്ളുന്നു, അതിനാൽ നിങ്ങൾക്ക് ഇഷ്ടമുള്ള ഏത് പാഠം കൊണ്ടുമാരംഭിക്കാം!

പാഠങ്ങളിൽ "Learn" എന്ന് അടയാളപ്പെടുത്തി.generator AI ആശയം വിശദീകരിക്കുന്ന പാഠങ്ങളോ, "Build" എന്ന് അടയാളപ്പെടുത്തി generator AI ആശയം കൂടാതെ Python, TypeScript കോഡ് ഉദാഹരണങ്ങളും നൽകുന്ന പാഠങ്ങളോ ആണ്.

.NET ഡവലപ്പർമാർക്ക് [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) പരിശോധിക്കുക!

ഓരോ പാഠത്തിലും "Keep Learning" വിഭാഗം കൂടി ഉൾക്കൊള്ളുന്നു, അധിക പഠന ഉപകരണങ്ങൾക്കായി.

 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **പാഠങ്ങൾ:** "aoai-assignment"
