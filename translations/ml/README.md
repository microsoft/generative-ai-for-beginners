# ആരംഭക്കാർക്ക് വേണ്ടി ജനറേറ്റീവ് AI (പതിപ്പ് 3) - ഒരു കോഴ്‌സ്

നിർമാണം ആരംഭിക്കാൻ നിങ്ങൾക്കു ആവശ്യമായ അടിസ്ഥാനാംശങ്ങൾ എംഎസ് ക്ലൗഡ് അഡ്വക്കേറ്റുകൾ ഒരുക്കിയ 21 പാഠങ്ങളടങ്ങിയ സമഗ്ര കോഴ്‌സിലൂടെ പഠിക്കൂ.

## 🌱 തുടങ്ങുന്നവർക്ക്

ഈ കോഴ്‌സിന് 21 പാഠങ്ങളുണ്ട്. ഓരോ പാഠവും തൻ്റെ വിഷയം ഉൾക്കൊള്ളുന്നു, അതിനാൽ നിങ്ങൾക്ക് ഇഷ്ടപ്പെട്ടിടത്ത് തുടങ്ങാം!

പാഠങ്ങൾ "കഴിവേറും" എന്ന് ലേബൽ ചെയ്തിരിക്കുന്നത് ജനറേറ്റീവ് AI സങ്കൽപ്പങ്ങൾ വിശദീകരിക്കുന്നതാണ്; "നിർമ്മിക്കുന്നു" എന്നു ലേബൽ ചെയ്ത പാഠങ്ങളിൽ സങ്കൽപ്പവും കോഡിന്റെ ഉദാഹരണങ്ങളും **Python** ഉം **TypeScript** ഉം ഉൾക്കൊള്ളുന്നതാണ്.

.NET ഡെവലപ്പർമാർക്ക് [Generative AI for Beginners (.NET പതിപ്പ്)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) പരിശോധിക്കുക!

ഓരോ പാഠത്തോടും കുട്ടിപ്പകർപ്പ് പഠന ഉപകരണം അടങ്ങിയ "കൂടുതൽ പഠനം തുടരുക" എന്ന വിഭാഗവും ഉൾപ്പടുന്നു.

## നിങ്ങള്ക്കു എന്ത് വേണം
### ഈ കോഴ്‌സ് കോഡ് പ്രവർത്തിപ്പിക്കാൻ, നിങ്ങൾക്ക് താഴെപ്പെട്ടവ ഉപയോഗിക്കാം:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **പാഠങ്ങൾ:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **പാഠങ്ങൾ:** "githubmodels" (GitHub Models ജൂലൈ 2026 അവസാനം മുതൽ ഒഴിവാക്കപ്പെടുന്നു - അതിന്റെ പകരം Microsoft Foundry Models ഉപയോഗിക്കുക)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **പാഠങ്ങൾ:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - നിങ്ങളുടെ സ്വന്തം ഉപകരണത്തിൽ, ക്ലൗഡ് സബ്സ്ക്രിപ്ഷൻ കൂടാതെ, മുഴുവൻ ഓൺലൈൻ മോഡലുകൾ പ്രവർത്തിപ്പിക്കുക
   
- Python അല്ലെങ്കിൽ TypeScript ഫോണ്ടുകൾ കുറച്ച് അറിവ് ഉള്ളതായിരിക്കുക ഉപകാരകാരിയാണ് - \*പൂർണ്ണമായ തുടക്കക്കാർക്ക് ഈ [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) & [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) കോഴ്‌സുകൾ കാണുക
- ഈ മുഴുവൻ റിപ്പോസിറ്ററി നിങ്ങളുടെ സ്വന്തം GitHub അക്കൗണ്ടിലേക്ക് [fork ചെയ്യാൻ GitHub അക്കൗണ്ട്](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) വേണം

ഞങ്ങൾ **[കോഴ്‌സ് സജ്ജീകരണം](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** പാഠം തയ്യാറാക്കി, നിങ്ങള്ക്ക് നിങ്ങളുടെ ഡെവൽപ്മെന്റ് പരിസ്ഥിതി ക്രമീകരിക്കുന്നതിന് സഹായം നൽകുന്നു.

പിന്നീട് ആവർത്തനമായി കണ്ടെത്താൻ [ഈ റിപ്പോസിറ്ററി സ്റ്റാർ (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ചെയ്യുക.
