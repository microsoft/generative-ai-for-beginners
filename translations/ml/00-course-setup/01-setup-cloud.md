# ക്ലൗഡ് സജ്ജീകരണം ☁️ – GitHub കോഡ്‌സ്പേസുകൾ  

**നിങ്ങൾക്ക് ഏതെങ്കിലും പ്രാദേശിക ഇൻസ്റ്റലേഷൻ ആവശ്യമില്ലെങ്കിൽ ഈ ഗൈഡ് ഉപയോഗിക്കുക.**  
കോഡ്‌സ്പേസുകൾ നിങ്ങൾക്കു സൗജന്യമായ, ബ്രൗസർ അധിഷ്ഠിതമായ VS കോഡ് ഇൻസ്റ്റൻസ് നൽകുന്നു എല്ലാ ആശ്രിതങ്ങളും മുൻകൂർ ഇൻസ്റ്റാൾ ചെയ്താണ്.

## 2.  ഒരു കോഡ്‌സ്പേസ് സൃഷ്ടിക്കുക (ഒരു ക്ലിക്ക്)  

1. ഈ റെപ്പോ **ഫോർക്കുചെയ്‌തു** ഇല്ലാതുമുണ്ടാകുന്നു (**ഫോർക്** ബട്ടൺ മുകളിൽ വലത്തിരുവശത്ത്).  
2. നിങ്ങളുടെ ഫോർക്കിലുള്ളിൽ **കോഡ് ▸ കോഡ്‌സ്പേസുകൾ ▸ മെയിനിൽ കോഡ്‌സ്‌പേസ് സൃഷ്ടിക്കൂ** ക്ലിക്ക് ചെയ്യുക.  
   ![കോഡ്‌സ്‌പേസ് സൃഷ്ടിക്കാൻ ബട്ടണുകൾ കാണിക്കുന്ന ഡയലോഗ്](../../../translated_images/ml/who-will-pay.4c0609b1c7780f44.webp)  

✅ ഒരു ബ്രൗസർ VS കോഡ് വിൻഡോ തുറക്കുകയും ഡെവ് കണ്ടെയിനർ നിർമ്മാണം ആരംഭിക്കുകയും ചെയ്യും.  
ഇത് ആദ്യമായി **ഏകദേശം 2 മിനിറ്റ്** സമയമേടുക്കൂ.  

## 3. നിങ്ങളുടെ API കീ ചേർക്കുക (സുരക്ഷിത മാർഗം)  

### ഓപ്‌ഷൻ A കോഡ്‌സ്പേസുകൾ സീക്രെറ്റുകൾ — ശുപാർശ ചെയ്യുന്നു  

1. ⚙️ ഗിയർ ഐക്കൺ -> കമാൻഡ് പാലറ്റ് -> കോഡ്‌സ്പേസുകൾ : ഉപയോക്തൃ രഹസ്യം മാനേജ് ചെയ്യുക -> പുതിയ രഹസ്യം ചേർക്കുക.  
2. പേര്: OPENAI_API_KEY  
3. മൂല്യം: നിങ്ങളുടെ കീ പേസ്റ്റ് ചെയ്യുക → രഹസ്യം ചേർക്കുക  

അതുകെ സരിയാകും—നമ്മുടെ കോഡ് അത് ഓട്ടോമാറ്റിക്കായി സ്വീകരിക്കും.  

### ഓപ്‌ഷൻ B .env ഫയൽ (നിങ്ങൾക്ക് അത്യാവശ്യമെങ്കിൽ)  

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=നിങ്ങളുടെ_കീ_ഇവിടെയാണ്_എന്ന്_പൂരിപ്പിക്കുക
```
