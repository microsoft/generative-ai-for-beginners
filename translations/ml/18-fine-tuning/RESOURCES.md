# സ്വയം മാർഗനിർദ്ദേശ പഠനത്തിന് ആവശ്യമായ വിഭവങ്ങൾ

പാഠം നിയന്ത്രണ വാക്കുകളും ട്യൂട്ടോറിയലുകളും സംബന്ധിച്ച റഫറൻസുകൾ ആയി OpenAI യുടെയും Microsoft Foundry യുടെയും മുഖ്യ വിഭവങ്ങൾ ഉപയോഗിച്ചാണ് നിർമ്മിച്ചത്. നിങ്ങളുടെ സ്വന്തം സ്വയം മാർഗനിർദ്ദേശ പഠന യാത്രകൾക്കായി ഇതാ ഒരു സമ്പൂർണമല്ലാത്ത പട്ടിക. താഴെയുള്ള ഓരോ ലിങ്കും നിലവിലെ, പിന്തുടരുന്ന വസ്തുക്കൾക്ക് ആണ് കാണിക്കുന്നത്.

## 2. രണ്ടാം ജോതി വിഭവങ്ങൾ
