# Generatieve AI voor Beginners (Versie 3) - Een Cursus

Leer de basisprincipes van het bouwen van Generatieve AI-toepassingen met onze 21-lessen durende uitgebreide cursus door Microsoft Cloud Advocates.

## 🌱 Aan de slag

Deze cursus bestaat uit 21 lessen. Elke les behandelt een eigen onderwerp, dus begin waar je maar wilt!

Lessen zijn gelabeld als "Leer"-lessen die een Generatieve AI-concept uitleggen of "Bouw"-lessen die een concept uitleggen en codevoorbeelden in zowel **Python** als **TypeScript** bieden indien mogelijk.

Voor .NET-ontwikkelaars, bekijk [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Elke les bevat ook een sectie "Blijf Leren" met aanvullende leermiddelen.

## Wat je nodig hebt
### Om de code van deze cursus uit te voeren, kun je een van de volgende gebruiken:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Lessen:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Lessen:** "githubmodels" (GitHub Models wordt aan het eind van juli 2026 stopgezet - gebruik in plaats daarvan Microsoft Foundry Models)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Lessen:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Voer modellen volledig offline uit op je eigen apparaat, geen cloud-abonnement vereist
   
- Basiskennis van Python of TypeScript is nuttig - \*Voor absolute beginners bekijk deze [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) en [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) cursussen
- Een GitHub-account om [deze hele repo te forken](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) naar je eigen GitHub-account

We hebben een **[Cursus Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** les gemaakt om je te helpen bij het opzetten van je ontwikkelomgeving.

Vergeet niet om [deze repo te voorzien van een ster (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) zodat je hem later gemakkelijker terugvindt.
