# क्लाउड सेटअप ☁️ – GitHub Codespaces

**यदि आप कुछ भी लोकल पर इंस्टॉल नहीं करना चाहते हैं तो इस गाइड का उपयोग करें।**  
Codespaces आपको एक मुफ्त, ब्राउज़र-आधारित VS कोड इंस्टेंस देता है जिसमें सभी निर्भरता पहले से इंस्टॉल होती हैं।

---

## 1.  क्यों Codespaces?

| लाभ | इसका आपके लिए मतलब क्या है |
|---------|----------------------|
| ✅ कोई इंस्टालेशन नहीं | Chromebook, iPad, स्कूल लैब पीसी… पर काम करता है |
| ✅ पहले से बना हुआ डेवलपमेंट कंटेनर | Python 3, Node.js, .NET, Java पहले से अंदर हैं |
| ✅ मुफ्त कोटा | पर्सनल अकाउंट को मिलता है **120 कोर-घंटे / 60 GB-घंटे प्रति माह** |

> 💡 **टिप**  
> अपना कोटा स्वस्थ रखें **रोक कर** या **हटाकर** निष्क्रिय codespaces  
> (देखें ▸ कमांड पैलेट ▸ *Codespaces: Stop Codespace*).

---

## 2.  एक Codespace बनाएं (एक क्लिक में)

1. इस रिपोज़िटरी को **Fork** करें (ऊपरी-दाएँ **Fork** बटन)।  
2. अपने Fork में, क्लिक करें **Code ▸ Codespaces ▸ मुख्य पर codespace बनाएं**।  
   ![डायलॉग जिसमें codespace बनाने के बटन दिखाए गए हैं](../../../translated_images/hi/who-will-pay.4c0609b1c7780f44.webp)

✅ एक ब्राउज़र VS कोड विंडो खुलेगी और डेवलपमेंट कंटेनर बनना शुरू हो जाएगा।
यह पहली बार में **~2 मिनट** लेता है।

## 3. अपनी API कुंजी जोड़ें (सुरक्षित तरीका)

### विकल्प A Codespaces Secrets — अनुशंसित

1. ⚙️ गियर आइकन -> कमांड पैलेट -> Codespaces : Manage user secret -> नई सीक्रेट जोड़ें।
2. नाम: OPENAI_API_KEY
3. मान: अपनी कुंजी चिपकाएं → सीक्रेट जोड़ें

बस इतना ही — हमारा कोड इसे स्वचालित रूप से पकड़ लेगा।

### विकल्प B .env फ़ाइल (यदि आपको वास्तव में इसकी आवश्यकता है)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=your_key_here भरें
```
