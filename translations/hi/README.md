# शुरुआती लोगों के लिए जनरेटिव एआई (संस्करण 3) - एक कोर्स

माइक्रोसॉफ्ट क्लाउड एडवोकेट्स द्वारा प्रस्तुत 21 पाठों वाले व्यापक कोर्स के साथ जनरेटिव एआई एप्लिकेशन बनाने की मूल बातें सीखें।

## 🌱 शुरुआत करना

इस कोर्स में 21 पाठ हैं। प्रत्येक पाठ अपने विषय को कवर करता है इसलिए आप कहीं से भी शुरू कर सकते हैं!

पाठ "सीखें" के रूप में लेबल किए गए हैं जो जनरेटिव एआई सिद्धांत समझाते हैं या "बनाएं" पाठ जो एक सिद्धांत और कोड उदाहरण दोनों समझाते हैं, जहाँ संभव हो, **Python** और **TypeScript** में।

.NET डेवलपर्स के लिए [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) देखें!

प्रत्येक पाठ में अतिरिक्त सीखने के उपकरणों के साथ एक "शिक्षण जारी रखें" सेक्शन भी शामिल है।
