# هوش مصنوعی مولد برای مبتدیان (نسخه ۳) - یک دوره آموزشی

اصول ساخت برنامه‌های هوش مصنوعی مولد را با دوره جامع ۲۱ درسی ما توسط مشاوران ابر مایکروسافت بیاموزید.

## 🌱 شروع به کار

این دوره شامل ۲۱ درس است. هر درس موضوع خاص خود را پوشش می‌دهد، پس هر جایی که دوست داشتید شروع کنید!

دروس به دو دسته "یادگیری" که مفهوم هوش مصنوعی مولد را توضیح می‌دهند یا "ساخت" که مفهوم و مثال‌های کد را هم به **Python** و هم **TypeScript** در صورت امکان توضیح می‌دهند، تقسیم شده‌اند.

برای توسعه‌دهندگان .NET دوره [هوش مصنوعی مولد برای مبتدیان (نسخه .NET)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) را ببینید!

هر درس همچنین بخشی به نام "ادامه یادگیری" دارد که ابزارهای یادگیری بیشتری شامل می‌شود.

## آنچه نیاز دارید
### برای اجرای کد این دوره، می‌توانید از یکی از موارد زیر استفاده کنید:
 - [خدمات Azure OpenAI](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **دروس:** "aoai-assignment"
 - [مدل‌های Microsoft Foundry](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **دروس:** "githubmodels" (مدل‌های گیت‌هاب تا پایان ژوئیه ۲۰۲۶ بازنشسته می‌شوند - به جای آن از مدل‌های Microsoft Foundry استفاده کنید)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **دروس:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - اجرای مدل‌ها به طور کامل به صورت آفلاین در دستگاه خودتان، بدون نیاز به اشتراک ابری
   
- دانش پایه‌ای از Python یا TypeScript مفید است - \*برای مبتدیان مطلق این [دوره Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) و [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) را بررسی کنید
- یک حساب گیت‌هاب برای [فورک کل این مخزن](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) به حساب گیت‌هاب خودتان

ما درسی به نام **[راه‌اندازی دوره](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** ایجاد کرده‌ایم تا به شما در راه‌اندازی محیط توسعه کمک کند.

فراموش نکنید که این مخزن را [ستاره (🌟) بدهید](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) تا پیدا کردن آن بعداً آسان‌تر باشد.
