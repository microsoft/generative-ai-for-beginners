# Generativní AI pro začátečníky (Verze 3) - Kurz

Naučte se základy tvorby aplikací generativní AI s naším komplexním kurzem o 21 lekcích od Microsoft Cloud Advocates.

## 🌱 Začínáme

Tento kurz obsahuje 21 lekcí. Každá lekce pokrývá vlastní téma, takže začněte, kde chcete!

Lekce jsou označeny buď jako lekce "Learn" vysvětlující koncept generativní AI, nebo lekce "Build", které vysvětlují koncept a obsahují příklady kódu v obou jazycích **Python** a **TypeScript**, pokud je to možné.

Pro vývojáře .NET navštivte [Generativní AI pro začátečníky (.NET edice)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Každá lekce také obsahuje sekci "Keep Learning" s dalšími nástroji pro učení.
