# Generativní AI pro začátečníky (Verze 3) - Kurz

Naučte se základy tvorby aplikací s generativní AI s naším komplexním kurzem o 21 lekcích od Microsoft Cloud Advocates.

## 🌱 Začínáme

Tento kurz má 21 lekcí. Každá lekce pokrývá své vlastní téma, takže můžete začít, kde chcete!

Lekce jsou označeny buď jako "Learn" lekce, které vysvětlují koncept generativní AI, nebo jako "Build" lekce, které vysvětlují koncept a příklady kódu v **Pythonu** a **TypeScriptu**, pokud je to možné.

Pro vývojáře .NET se podívejte na [Generativní AI pro začátečníky (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Každá lekce také obsahuje sekci "Keep Learning" s dalšími nástroji pro učení.
