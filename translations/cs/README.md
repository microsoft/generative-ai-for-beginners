# Generativní AI pro začátečníky (Verze 3) - Kurz

Naučte se základy tvorby aplikací generativní AI s naším komplexním kursem o 21 lekcích od Microsoft Cloud Advocates.

## 🌱 Začínáme

Tento kurz má 21 lekcí. Každá lekce pokrývá vlastní téma, takže začněte, kde chcete!

Lekce jsou označeny jako buď "Learn" lekce vysvětlující koncept generativní AI, nebo "Build" lekce, které vysvětlují koncept a ukázky kódu v **Pythonu** a **TypeScriptu**, pokud je to možné.

Pro vývojáře .NET si prohlédněte [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Každá lekce také obsahuje sekci "Keep Learning" s dalšími nástroji pro učení.

## Co potřebujete
### Pro spuštění kódu z tohoto kurzu můžete použít některé z následujících: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Lekce:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Lekce:** "githubmodels" (GitHub Models bude ukončen na konci července 2026 - místo toho používejte Microsoft Foundry Models)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Lekce:** "oai-assignment" 
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Spouštějte modely plně offline na vlastním zařízení, není potřeba cloudové předplatné
   
- Základní znalost Pythonu nebo TypeScriptu je užitečná - \*Pro úplné začátečníky prohlédněte tyto kurzy [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) a [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- GitHub účet pro [forkování celého tohoto repozitáře](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) na váš vlastní GitHub účet

Vytvořili jsme lekci **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**, která vám pomůže s nastavením vývojového prostředí.

Nezapomeňte [označit hvězdičkou (🌟) tento repozitář](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), abyste ho později snadno našli.
