# Generatiivne tehisintellekt algajatele (versioon 3) - kursus

Õppige generatiivsete tehisintellekti rakenduste ehitamise põhialuseid meie Microsoft Cloud Advocatesi 21-õppetunnise kursuse abil.

## 🌱 Alustamine

Sellel kursusel on 21 õppetundi. Iga õppetund hõlmab oma teemat, nii et alustage sealt, kus soovite!

Õppetunnid on märgistatud kas "Õpi" õppetundidena, mis seletavad generatiivse tehisintellekti kontseptsiooni, või "Ehitada" õppetundidena, mis seletavad kontseptsiooni ja koodinäiteid nii **Pythonis** kui ka **TypeScriptis**, kui võimalik.

.NET arendajatele vaadake [Generatiivne tehisintellekt algajatele (.NET väljaanne)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Iga õppetund sisaldab ka "Jätka õppimist" sektsiooni lisavahenditega õppimiseks.

## Mida vajate
### Selle kursuse koodi käivitamiseks võite kasutada kas:
 - [Azure OpenAI teenus](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Õppetunnid:** "aoai-assignment"
 - [Microsoft Foundry mudelid](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Õppetunnid:** "githubmodels" (GitHub Models lõpetab töö 2026. aasta juuli lõpus – kasutage selle asemel Microsoft Foundry mudelid)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Õppetunnid:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Käivitage mudelid täiesti võrguühenduseta oma seadmes, pilveteenuse tellimuseta
   
- Põhilised Python’i või TypeScripti teadmised on abiks – \*Algajatele soovitame neid [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ja [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) kursusi
- GitHubi konto, et [forkida see kogu hoidla](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) oma GitHubi kontole

Oleme loonud õppetunni **[Kursuse seadistamine](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)**, mis aitab teil arenduskeskkonnas seadistust teha.

Ärge unustage [anda sellele hoidla(tärni🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst), et seda hiljem hõlpsamini leida.
