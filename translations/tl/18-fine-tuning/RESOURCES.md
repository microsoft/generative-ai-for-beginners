# Mga Mapagkukunan Para sa Sariling Pagsasanay

Ang aralin ay binuo gamit ang mga pangunahing mapagkukunan mula sa OpenAI at Microsoft Foundry bilang mga sanggunian para sa terminolohiya at mga tutorial. Narito ang isang hindi kumpletong listahan para sa iyong sariling mga paglalakbay sa sariling pagsasanay. Bawat link sa ibaba ay tumuturo sa kasalukuyang, suportadong materyal.

## 1. Pangunahing Mga Mapagkukunan

| Pamagat/Link | Paglalarawan |
| :--- | :--- |
| [Fine-tuning with OpenAI Models](https://platform.openai.com/docs/guides/fine-tuning?WT.mc_id=academic-105485-koreyst) | Pinapabuti ng fine-tuning ang few-shot learning sa pamamagitan ng pagsasanay gamit ang mas maraming mga halimbawa kaysa sa maaaring isama sa prompt - nagtitipid ng gastos, pinapabuti ang kalidad ng tugon, at nagpapahintulot ng mas mababang latency sa mga kahilingan. **Kumuha ng pangkalahatang ideya ng fine-tuning mula sa OpenAI.** |
| [When to use Microsoft Foundry fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/concepts/fine-tuning-considerations?WT.mc_id=academic-105485-koreyst) | Unawain **kung ano ang fine-tuning (konsepto)**, bakit ito dapat isaalang-alang, anong datos ang gagamitin, at paano sukatin ang kalidad - pati na rin kung kailan angkop ang SFT, DPO, o RFT. |
| [Customize a model with fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?WT.mc_id=academic-105485-koreyst) | Ang end-to-end **paano-gawin (proseso)** para sa fine-tuning sa Microsoft Foundry gamit ang portal, ang OpenAI / Foundry Python SDK, o ang REST API - sumasaklaw sa paghahanda ng datos, pagsasanay, checkpoint, at deployment. |
| [Continuous fine-tuning](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning?WT.mc_id=academic-105485-koreyst#perform-continuous-fine-tuning) | Ang paulit-ulit na proseso ng pagpili ng isang modelong nauna nang na-fine-tuned bilang base model at **karagdagang pag-fine-tune** nito sa mga bagong set ng mga halimbawa sa pagsasanay. |
| [Fine-tuning with tool (function) calling](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/fine-tuning-functions?WT.mc_id=academic-105485-koreyst) | Pinapabuti ng fine-tuning ng iyong modelo **gamit ang mga halimbawa ng tool-calling** ang output - mas tumpak, pare-pareho, at may pare-parehong format na mga tugon gamit ang mas kaunting mga token sa prompt. |
| [Fine-tuning models: Microsoft Foundry guidance](https://learn.microsoft.com/azure/ai-foundry/foundry-models/concepts/models-sold-directly-by-azure?WT.mc_id=academic-105485-koreyst#fine-tuning-models) | Tingnan kung **aling mga modelo ang maaaring i-fine-tune**, ang mga metodong sinusuportahan nila (SFT / DPO / RFT), at ang mga rehiyon kung saan sila magagamit. |
| [Fine-tuning overview: techniques and modalities](https://learn.microsoft.com/azure/ai-foundry/concepts/fine-tuning-overview?WT.mc_id=academic-105485-koreyst) | Ihambing ang tatlong mga teknik sa pagsasanay (SFT, DPO, RFT) at ang dalawang modality (serverless vs. managed compute), na may patnubay sa pagpili ng base model at paano magsimula. |
| **Tutorial**: [Fine-tune a model in Microsoft Foundry](https://learn.microsoft.com/azure/ai-foundry/openai/tutorials/fine-tune?WT.mc_id=academic-105485-koreyst) | Gumawa ng sample na dataset, maghanda para sa fine-tuning, magpatakbo ng fine-tuning job sa kasalukuyang suportadong modelo tulad ng `gpt-4.1-mini`, at i-deploy ang fine-tuned na modelo sa Azure. |
| **Tutorial**: [Fine-tune models with serverless API deployments](https://learn.microsoft.com/azure/ai-foundry/how-to/fine-tune-serverless?WT.mc_id=academic-105485-koreyst) | Iakma ang mga open at partner models (Phi, Llama, Mistral, at iba pa) sa iyong mga dataset _gamit ang isang low-code, UI-based workflow_ sa Microsoft Foundry. |
| **Tutorial**: [Fine-tune Hugging Face models on Azure Databricks](https://learn.microsoft.com/azure/databricks/machine-learning/train-model/huggingface/fine-tune-model?WT.mc_id=academic-105485-koreyst) | Mag-fine-tune ng Hugging Face model gamit ang `transformers` library sa isang GPU gamit ang Azure Databricks at Hugging Face Trainer. |
| **Training**: [Fine-tune a foundation model with Azure Machine Learning](https://learn.microsoft.com/training/modules/finetune-foundation-model-with-azure-machine-learning/?WT.mc_id=academic-105485-koreyst) | Ang catalog ng Azure Machine Learning models ay nag-aalok ng maraming open-source models na maaari mong i-fine-tune. Bahagi ng [Azure ML Generative AI Learning Path](https://learn.microsoft.com/training/paths/work-with-generative-models-azure-machine-learning/?WT.mc_id=academic-105485-koreyst). |
| **Tutorial**: [Azure OpenAI fine-tuning with Weights & Biases](https://docs.wandb.ai/guides/integrations/azure-openai-fine-tuning?WT.mc_id=academic-105485-koreyst) | Subaybayan at suriin ang mga fine-tuning runs sa Azure gamit ang W&B. Pinapalawak ang OpenAI fine-tuning guide na may mga hakbang para sa Azure at pagsubaybay sa eksperimento. |

## 2. Pangalawang mga Mapagkukunan

Kinukuha ng seksyong ito ang karagdagang mga mapagkukunan na sulit tuklasin na hindi namin nagkaroon ng oras na talakayin sa aralin. Gamitin ang mga ito upang bumuo ng iyong sariling kadalubhasaan sa paksang ito.

| Pamagat/Link | Paglalarawan |
| :--- | :--- |
| **OpenAI Cookbook**: [Paghahanda ng datos at pagsusuri para sa fine-tuning ng chat model](https://cookbook.openai.com/examples/chat_finetuning_data_prep?WT.mc_id=academic-105485-koreyst) | I-preprocess at suriin ang isang chat dataset bago ang fine-tuning: tingnan kung may mga error sa format, kumuha ng mga pangunahing estadistika, at tantiyahin ang bilang ng mga token (at gastos). Kasama ng [OpenAI fine-tuning guide](https://platform.openai.com/docs/guides/fine-tuning?WT.mc_id=academic-105485-koreyst). |
| **OpenAI Cookbook**: [Fine-tuning para sa Retrieval Augmented Generation (RAG) gamit ang Qdrant](https://cookbook.openai.com/examples/fine-tuned_qa/ft_retrieval_augmented_generation_qdrant?WT.mc_id=academic-105485-koreyst) | Isang komprehensibong halimbawa ng fine-tuning ng OpenAI models para sa RAG, na nagsasama ng Qdrant at few-shot learning upang mapabuti ang performance at mabawasan ang mga peke. |
| **OpenAI Cookbook**: [Fine-tuning GPT gamit ang Weights & Biases](https://cookbook.openai.com/examples/third_party/gpt_finetuning_with_wandb?WT.mc_id=academic-105485-koreyst) | Gamitin ang W&B para subaybayan ang pagsasanay at fine-tuning ng modelo. Basahin muna ang kanilang [OpenAI Fine-Tuning](https://docs.wandb.ai/guides/integrations/openai-fine-tuning/?WT.mc_id=academic-105485-koreyst) guide, pagkatapos subukan ang Cookbook exercise. |
| **Hugging Face Tutorial**: [Paano i-Fine-Tune ang mga LLM gamit ang Hugging Face TRL](https://www.philschmid.de/fine-tune-llms-in-2024-with-trl?WT.mc_id=academic-105485-koreyst) | Mag-fine-tune ng mga open na LLM gamit ang Hugging Face TRL, Transformers, at datasets: tukuyin ang kaso ng paggamit, mag-set up ng dev environment, maghanda ng dataset, mag-fine-tune, mag-evaluate, at mag-deploy. |
| **Hugging Face**: [AutoTrain Advanced](https://github.com/huggingface/autotrain-advanced?WT.mc_id=academic-105485-koreyst) | Isang no-code / low-code library mula sa Hugging Face para sa fine-tuning ng maraming uri ng modelo. Patakbuhin ito sa iyong sariling cloud, sa Hugging Face Spaces, o lokal gamit ang GUI, CLI, o YAML config. |
| **Unsloth**: [Patnubay sa Fine-tuning ng LLMs](https://docs.unsloth.ai/get-started/fine-tuning-llms-guide) | Isang open-source framework na nagpapadali sa lokal na fine-tuning ng LLM at reinforcement learning, na may mga handang gamitin na [notebooks](https://github.com/unslothai/notebooks?WT.mc_id=academic-105485-koreyst). |
