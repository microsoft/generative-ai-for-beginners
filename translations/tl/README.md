# Generative AI para sa Mga Nagsisimula (Bersyon 3) - Isang Kurso

Alamin ang mga pangunahing kaalaman sa pagbuo ng mga generative AI na aplikasyon sa aming 21-araling komprehensibong kurso mula sa Microsoft Cloud Advocates.

## üå± Pagsisimula

Ang kursong ito ay may 21 aralin. Bawat aralin ay sumasaklaw sa sarili nitong paksa kaya magsimula ka saan mo gustong!

Ang mga aralin ay may label na alinman sa "Alamin" na mga aralin na nagpapaliwanag ng isang konsepto ng Generative AI o "Bumuo" na mga aralin na nagpapaliwanag ng isang konsepto at nagbibigay ng mga halimbawa ng code sa parehong **Python** at **TypeScript** kapag maaari.

Para sa mga .NET Developer tingnan ang [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Ang bawat aralin ay may kasamang seksyong "Patuloy na Pag-aaral" na naglalaman ng mga karagdagang kasangkapan para sa pagkatuto.

## Ano ang Kailangan Mo
### Upang mapatakbo ang code ng kursong ito, maaari mong gamitin alinman sa: 
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Mga Aralin:** "aoai-assignment"
 - [GitHub Marketplace Model Catalog](https://aka.ms/genai-beginners/gh-models?WT.mc_id=academic-105485-koreyst) - **Mga Aralin:** "githubmodels"
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Mga Aralin:** "oai-assignment" 
   
- Makakatulong ang pangunahing kaalaman sa Python o TypeScript - \*Para sa mga ganap na nagsisimula, tingnan ang mga kursong ito na [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) at [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Isang account sa GitHub para [i-fork ang buong repo na ito](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) sa iyong sariling GitHub account

Nilikha namin ang isang **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** na aralin upang tulungan ka sa pag-set up ng iyong development environment.

Huwag kalimutang [i-star (üåü) ang repo na ito](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) para mas madali mo itong mahanap mamaya.
