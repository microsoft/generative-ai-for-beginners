# Generative AI para sa Mga Nagsisimula (Bersyon 3) - Isang Kurso

Alamin ang mga pundasyon ng pagbuo ng mga aplikasyon ng Generative AI gamit ang aming 21-lesson na komprehensibong kurso mula sa Microsoft Cloud Advocates.

## 🌱 Pagsisimula

Ang kurso na ito ay may 21 na aralin. Bawat aralin ay may sariling paksa kaya't simulan mo kung saan mo gusto!

Ang mga aralin ay pinangalanang "Learn" para sa mga naglalaman ng paliwanag tungkol sa konsepto ng Generative AI o "Build" para sa mga nagpapaliwanag ng konsepto at mga halimbawa ng code sa parehong **Python** at **TypeScript** kung posible.

Para sa mga .NET Developers, tingnan ang [Generative AI para sa Mga Nagsisimula (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst)!

Ang bawat aralin ay may seksyong "Keep Learning" na naglalaman ng karagdagang mga kasangkapan sa pag-aaral.

## Ano ang Kailangan Mo
### Para patakbuhin ang code ng kurso na ito, maaari mong gamitin ang alinman sa mga ito:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **Mga Aralin:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **Mga Aralin:** "githubmodels" (Ang GitHub Models ay magtatapos sa katapusan ng Hulyo 2026 - gamitin ang Microsoft Foundry Models bilang kapalit)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **Mga Aralin:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - Patakbuhin ang mga modelo nang ganap na offline sa sarili mong device, hindi kailangan ng cloud subscription
   
- Ang pangunahing kaalaman sa Python o TypeScript ay nakatutulong - \*Para sa mga ganap na nagsisimula tingnan ang mga kursong ito sa [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) at [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst)
- Isang GitHub account para [i-fork ang buong repo na ito](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) papunta sa sarili mong GitHub account

Nalikha namin ang isang **[Course Setup](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** na aralin para tulungan ka sa pag-set up ng iyong development environment.

Huwag kalimutang [bigyan ng bituin (🌟) ang repo na ito](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) upang mas madali mo itong mahanap mamaya.
