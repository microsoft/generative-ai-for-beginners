# ਸਰਚ ਐਪਲੀਕੇਸ਼ਨਾਂ ਦਾ ਨਿਰਮਾਣ

[![ਜਨਰੇਟਿਵ AI ਅਤੇ ਵੱਡੇ ਭਾਸ਼ਾ ਮਾਡਲਾਂ ਦੀ ਜਾਣੂ ਕਰਵਾਈ](../../../translated_images/pa/08-lesson-banner.8fff48c566dad08a.webp)](https://youtu.be/W0-nzXjOjr0?si=GcsqiTTvd7RKbo7V)

> > _ਇਸ ਪਾਠ ਦੀ ਵੀਡੀਓ ਦੇਖਣ ਲਈ ਉੱਪਰ ਦਿੱਤੀ ਤਸਵੀਰ 'ਤੇ ਕਲਿੱਕ ਕਰੋ_

LLMs ਸਿਰਫ਼ ਚੈਟਬੋਟਸ ਅਤੇ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਲਈ ਹੀ ਨਹੀਂ ਹਨ। ਇੰਬੈਡਿੰਗਜ਼ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ ਵੀ ਸੰਭਵ ਹੈ। ਇੰਬੈਡਿੰਗਜ਼ ਡੇਟਾ ਦੀ ਗਿਣਤੀ ਦੀ ਪ੍ਰਤੀਨਿਧਤਾ ਹੁੰਦੀ ਹੈ, ਜਿਨ੍ਹਾਂ ਨੂੰ ਵੈਕਟਰ ਵੀ ਕਹਿੰਦੇ ਹਨ, ਅਤੇ ਇਹ ਡੇਟਾ ਲਈ ਸੈਮੈਂਟਿਕ ਸਰਚ ਵਿੱਚ ਵਰਤੇ ਜਾ ਸਕਦੇ ਹਨ।

ਇਸ ਪਾਠ ਵਿੱਚ, ਤੁਸੀਂ ਸਾਡੇ ਸਿੱਖਿਆ ਸਟਾਰਟਅਪ ਲਈ ਇੱਕ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਜਾ ਰਹੇ ਹੋ। ਸਾਡਾ ਸਟਾਰਟਅਪ ਇੱਕ ਗੈਰ-ਨਫੇ ਵਾਲਾ ਸੰਗਠਨ ਹੈ ਜੋ ਵਿਕਾਸਸ਼ੀਲ ਦੇਸ਼ਾਂ ਵਿੱਚ ਛਾਤਰਾਂ ਨੂੰ ਮੁਫ਼ਤ ਸਿੱਖਿਆ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ। ਸਾਡੇ ਸਟਾਰਟਅਪ ਕੋਲ ਯੂਟਿਊਬ ਵੀਡੀਓਜ਼ ਦੀ ਇੱਕ ਵੱਡੀ ਗਿਣਤੀ ਹੈ, ਜੋ ਛਾਤਰ AI ਬਾਰੇ ਸਿੱਖਣ ਲਈ ਵਰਤ ਸਕਦੇ ਹਨ। ਸਾਡਾ ਸਟਾਰਟਅਪ ਇੱਕ ਐਸੀ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ ਚਾਹੁੰਦਾ ਹੈ ਜੋ ਛਾਤਰਾਂ ਨੂੰ ਕੋਈ ਪ੍ਰਸ਼ਨ ਟਾਈਪ ਕਰਕੇ ਯੂਟਿਊਬ ਵੀਡੀਓ ਖੋਜਣ ਦੀ ਆਗਿਆ ਦੇਵੇ।

ਉਦਾਹਰਣ ਵੱਜੋਂ, ਇੱਕ ਵਿਦਿਆਰਥੀ 'Jupyter Notebooks ਕੀ ਹਨ?' ਜਾਂ 'Azure ML ਕੀ ਹੈ' ਟਾਈਪ ਕਰ ਸਕਦਾ ਹੈ ਅਤੇ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਉਨ੍ਹਾਂ ਪ੍ਰਸ਼ਨਾਂ ਨਾਲ ਸਬੰਧਿਤ ਯੂਟਿਊਬ ਵੀਡੀਓਜ਼ ਦੀ ਸੂਚੀ ਵਾਪਸ ਕਰੇਗਾ, ਅਤੇ ਇਸ ਤੋਂ ਵੀ ਵਧੀਆ, ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਉਸ ਵੀਡੀਓ ਵਿਚ ਉਸ ਜਗ੍ਹਾ ਦਾ ਲਿੰਕ ਵੀ ਦੇਵੇਗਾ ਜਿੱਥੇ ਪ੍ਰਸ਼ਨ ਦਾ ਜਵਾਬ ਹੈ।

## ਪਰੀਚਯ

ਇਸ ਪਾਠ ਵਿੱਚ, ਅਸੀਂ ਕਵਰ ਕਰਾਂਗੇ:

- ਸੈਮੈਂਟਿਕ ਅਤੇ ਕੀਵਰਡ ਸਰਚ ਵਿੱਚ ਫਰਕ।
- ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼ ਕੀ ਹਨ।
- ਟੈਕਸਟ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਕਿਵੇਂ ਬਣਾਇਆ ਜਾਂਦਾ ਹੈ।
- ਟੈਕਸਟ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਵਿੱਚ ਕਿਵੇਂ ਖੋਜ ਕਰੀਏ।

## ਸਿੱਖਣ ਦੇ ਲਕੜੀ

ਇਸ ਪਾਠ ਪੂਰਾ ਕਰਨ ਦੇ ਬਾਅਦ, ਤੁਸੀਂ ਸਮਰੱਥ ਹੋਵੋਗੇ:

- ਸੈਮੈਂਟਿਕ ਅਤੇ ਕੀਵਰਡ ਸਰਚ ਵਿੱਚ ਅੰਤਰ ਦੱਸਣਾ।
- ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼ ਦੀ ਵਿਆਖਿਆ ਕਰਨਾ।
- ਇੰਬੈਡਿੰਗਜ਼ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਡੇਟਾ ਲਈ ਇੱਕ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ।

## ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਕਿਉਂ ਬਣਾਈਏ?

ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਨਾਲ ਤੁਸੀਂ ਸਮਝੋਗੇ ਕਿ ਇੰਬੈਡਿੰਗਜ਼ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਡੇਟਾ ਦੀ ਖੋਜ ਕਿਵੇਂ ਕੀਤੀ ਜਾਂਦੀ ਹੈ। ਤੁਸੀਂ ਇਹ ਵੀ ਸਿੱਖੋਗੇ ਕਿ ਇਕ ਐਸਾ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਕਿਵੇਂ ਬਣਾਇਆ ਜਾਵੇ ਜੋ ਛਾਤਰਾਂ ਨੂੰ ਜਾਣਕਾਰੀ ਛੇਤੀ ਮਿਲਣ ਵਿੱਚ ਮਦਦ ਕਰੇ।

ਇਸ ਪਾਠ ਵਿੱਚ ਮਾਇਕ੍ਰੋਸੌਫਟ [AI ਸ਼ੋ](https://www.youtube.com/playlist?list=PLlrxD0HtieHi0mwteKBOfEeOYf0LJU4O1) ਯੂਟਿਊਬ ਚੈਨਲ ਦੇ ਟ੍ਰਾਂਸਕ੍ਰਿਪਟ ਲਈ ਇਕ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਸ਼ਾਮਲ ਹੈ। AI ਸ਼ੋ ਉਹ ਯੂਟਿਊਬ ਚੈਨਲ ਹੈ ਜੋ ਤੁਹਾਨੂੰ AI ਅਤੇ ਮਸ਼ੀਨ ਲਰਨਿੰਗ ਬਾਰੇ ਸ਼ਿਖਾਉਂਦਾ ਹੈ। ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਵਿੱਚ ਹਰ ਯੂਟਿਊਬ ਟ੍ਰਾਂਸਕ੍ਰਿਪਟ ਦੀਆਂ ਇੰਬੈਡਿੰਗਜ਼ ਸ਼ਾਮਲ ਹਨ, ਤੱਕ ਕਿ ਅਕਤੂਬਰ 2023 ਤੱਕ। ਤੁਸੀਂ ਸਾਡੇ ਸਟਾਰਟਅਪ ਲਈ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਲਈ ਇਸ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਦੀ ਵਰਤੋਂ ਕਰੋਗੇ। ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਪ੍ਰਸ਼ਨ ਦੇ ਜਵਾਬ ਵਾਲੀ ਜਗ੍ਹਾ ਵਿੱਚ ਵੀਡੀਓ ਦਾ ਲਿੰਕ ਵੀ ਵਾਪਸ ਕਰਦਾ ਹੈ। ਇਹ ਛਾਤਰਾਂ ਲਈ ਜਾਣਕਾਰੀ ਛੇਤੀ ਲੱਭਣ ਦਾ ਵਧੀਆ ਤਰੀਕਾ ਹੈ।

ਹੇਠਾਂ ਇੱਕ ਸੈਮੈਂਟਿਕ ਕੁਏਰੀ ਦਾ ਉਦਾਹਰਣ ਹੈ ਜਿਸਦਾ ਪ੍ਰਸ਼ਨ ਹੈ 'ਕੀ ਤੁਸੀਂ rstudio ਨੂੰ Azure ML ਨਾਲ ਵਰਤ ਸਕਦੇ ਹੋ?'. ਯੂਟਿਊਬ URL ਨੂੰ ਦੇਖੋ, ਤੁਹਾਨੂੰ ਵੇਖਣ ਨੂੰ ਮਿਲੇਗਾ ਕਿ URL ਵਿੱਚ ਇੱਕ ਟਾਈਮਸਟੈਂਪ ਹੈ ਜੋ ਤੁਹਾਨੂੰ ਵੀਡੀਓ ਵਿੱਚ ਉਸ ਜਗ੍ਹਾ ਤੇ ਲੈ ਜਾਂਦਾ ਹੈ ਜਿੱਥੇ ਪ੍ਰਸ਼ਨ ਦਾ ਜਵਾਬ ਹੈ।

## ਸੈਮੈਂਟਿਕ ਸਰਚ ਕੀ ਹੈ?

ਹੁਣ ਤੁਸੀਂ ਸੋਚ ਰਹੇ ਹੋਵੇਗਾ, ਸੈਮੈਂਟਿਕ ਸਰਚ ਕੀ ਹੈ? ਸੈਮੈਂਟਿਕ ਸਰਚ ਇੱਕ ਐਸੀ ਖੋਜ ਤਕਨੀਕ ਹੈ ਜੋ ਕੁਏਰੀ ਦੇ ਸ਼ਬਦਾਂ ਦੇ ਅਰਥ ਜਾਂ ਮਤਲਬ ਨੂੰ ਵਰਤਦੀ ਹੈ ਤਾਂ ਜੋ ਸੰਬੰਧਿਤ ਨਤੀਜੇ ਵਾਪਸ ਕਰੇ।

ਇਹ ਇੱਕ ਸੈਮੈਂਟਿਕ ਖੋਜ ਦਾ ਉਦਾਹਰਣ ਹੈ। ਮਾਨ ਲਓ ਤੁਸੀਂ ਇੱਕ ਕਾਰ ਖਰੀਦਣ ਤਾਂ ਕੰਸ਼ਿਸ਼ ਕਰ ਰਹੇ ਹੋ, ਤਾਂ ਤੁਸੀਂ 'ਮੇਰੀ ਸੁਪਨਾ ਵਾਲੀ ਕਾਰ' ਖੋਜ ਸਕਦੇ ਹੋ, ਸੈਮੈਂਟਿਕ ਖੋਜ ਸਮਝਦਾ ਹੈ ਕਿ ਤੁਸੀਂ ਕਾਰ ਦੇ ਬਾਰੇ ਸੁਪਨੇ ਨਹੀਂ ਦੇਖ ਰਹੇ, ਬਲਕਿ ਤੁਸੀਂ ਆਪਣੀ 'ਆਦਰਸ਼' ਕਾਰ ਖਰੀਦਣਾ ਚਾਹੁੰਦੇ ਹੋ। ਸੈਮੈਂਟਿਕ ਖੋਜ ਤੁਹਾਡੇ ਇਰਾਦੇ ਨੂੰ ਸਮਝਦਾ ਹੈ ਅਤੇ ਸੰਬੰਧਿਤ ਨਤੀਜੇ ਵਾਪਸ ਕਰਦਾ ਹੈ। ਇਸਦਾ ਬਦਲ ਕੇਵਲ 'ਕੀਵਰਡ ਸਰਚ' ਹੁੰਦੀ ਹੈ ਜੋ ਲਿਟਰਲੀ ਕਾਰ ਦੇ ਬਾਰੇ ਸੁਪਨੇ ਖੋਜਦੀ ਹੈ ਅਤੇ ਅਕਸਰ ਗਲਤ ਨਤੀਜੇ ਲਿਆਉਂਦੀ ਹੈ।

## ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼ ਕੀ ਹਨ?

[ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼](https://en.wikipedia.org/wiki/Word_embedding?WT.mc_id=academic-105485-koreyst) ਇੱਕ ਟੈਕਸਟ ਪ੍ਰਤੀਨਿਧਿਤਾ ਤਕਨੀਕ ਹੈ ਜੋ [ਕੁਦਰਤੀ ਭਾਸ਼ਾ ਸੰਸਦ](https://en.wikipedia.org/wiki/Natural_language_processing?WT.mc_id=academic-105485-koreyst) ਵਿੱਚ ਵਰਤੀ ਜਾਂਦੀ ਹੈ। ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼ ਟੈਕਸਟ ਦੇ ਸੈਮੈਂਟਿਕ ਨੰਬਰਾਤਮਕ ਪ੍ਰਤੀਨਿਧਿਤਾ ਹਨ। ਇੰਬੈਡਿੰਗਜ਼ ਡੇਟਾ ਨੂੰ ਇਸ ਤਰ੍ਹਾਂ ਪ੍ਰਤੀਨਿਧਿਤਾ ਕਰਨ ਲਈ ਵਰਤੇ ਜਾਂਦੇ ਹਨ ਜੋ ਮਸ਼ੀਨ ਲਈ ਸਮਝਣਾ ਆਸਾਨ ਹੋਵੇ। ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼ ਬਣਾਉਣ ਲਈ ਕਈ ਮਾਡਲ ਹਨ, ਇਸ ਪਾਠ ਵਿੱਚ ਅਸੀਂ OpenAI ਇੰਬੈਡਿੰਗ ਮਾਡਲ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੰਬੈਡਿੰਗ ਜਨਰੇਟ ਕਰਨ ’ਤੇ ਧਿਆਨ ਦੇਵਾਂਗੇ।

ਇੱਥੇ ਇੱਕ ਉਦਾਹਰਣ ਹੈ, ਸੋਚੋ ਕਿ ਹੇਠਾਂ ਦਿੱਤਾ ਟੈਕਸਟ AI ਸ਼ੋ ਯੂਟਿਊਬ ਚੈਨਲ ਦੇ ਇੱਕ ਏਪੀਸੋਡ ਦੀ ਟ੍ਰਾਂਸਕ੍ਰਿਪਟ ਵਿੱਚ ਹੈ:

```text
Today we are going to learn about Azure Machine Learning.
```

ਅਸੀਂ ਟੈਕਸਟ ਨੂੰ OpenAI ਇੰਬੈਡਿੰਗ API ਨੂੰ ਭੇਜਾਂਗੇ ਅਤੇ ਇਹ 1536 ਨੰਬਰਾਂ ਦਾ ਇੱਕ ਵੈਕਟਰ ਵਾਪਸ ਕਰੇਗਾ। ਵੈਕਟਰ ਵਿੱਚ ਹਰ ਨੰਬਰ ਟੈਕਸਟ ਦੇ ਇੱਕ ਵੱਖਰੇ ਪੱਖ ਨੂੰ ਦਰਸਾਉਂਦਾ ਹੈ। ਛੋਟਾ ਕਰਨ ਲਈ, ਇੱਥੇ ਵੈਕਟਰ ਵਿੱਚ ਪਹਿਲੇ 10 ਨੰਬਰ ਦਿੱਤੇ ਗਏ ਹਨ।

## ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਕਿਵੇਂ ਬਣਾਇਆ ਜਾਂਦਾ ਹੈ?

ਇਸ ਪਾਠ ਲਈ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਪਾਈਥਨ ਸਕ੍ਰਿਪਟਾਂ ਦੀ ਇੱਕ ਲੜੀ ਨਾਲ ਬਣਾਇਆ ਗਿਆ। ਤੁਸੀਂ ਇਹ ਸਕ੍ਰਿਪਟਾਂ ਅਤੇ ਹਦਾਇਤਾਂ `'scripts'` ਫੋਲਡਰ ਵਿੱਚ [README](./scripts/README.md?WT.mc_id=academic-105485-koreyst) ਵਿੱਚ ਪ੍ਰਾਪਤ ਕਰ ਸਕਦੇ ਹੋ। ਇਹ ਸਕ੍ਰਿਪਟਾਂ ਚਲਾਉਣ ਦੀ ਲੋੜ ਨਹੀਂ ਜੇਕਰ ਤੁਸੀਂ ਇਹ ਪਾਠ ਪੂਰਾ ਕਰ ਰਹੇ ਹੋ ਕਿਉਂਕਿ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਤੁਹਾਨੂੰ ਦਿੱਤਾ ਗਿਆ ਹੈ।

2. [OpenAI Functions](https://learn.microsoft.com/azure/ai-foundry/openai/how-to/function-calling?WT.mc_id=academic-105485-koreyst) ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹੋਏ, ਪਹਿਲੇ 3 ਮਿੰਟਾਂ ਦੀ ਟ੍ਰਾਂਸਕ੍ਰਿਪਟ ਤੋਂ ਬੋਲਣ ਵਾਲੇ ਦਾ ਨਾਮ ਲੱਭਣ ਦਾ ਯਤਨ ਕੀਤਾ ਜਾਂਦਾ ਹੈ। ਹਰ ਵੀਡੀਓ ਲਈ ਬੋਲਣ ਵਾਲੇ ਦਾ ਨਾਮ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ `embedding_index_3m.json` ਵਿੱਚ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ।
3. ਟ੍ਰਾਂਸਕ੍ਰਿਪਟ ਟੈਕਸਟ ਨੂੰ **3 ਮਿੰਟ ਦੇ ਹਿੱਸਿਆਂ** ਵਿੱਚ ਵੰਡਿਆ ਜਾਂਦਾ ਹੈ। ਇਸ ਹਿੱਸੇ ਵਿੱਚ ਅਗਲੇ ਹਿੱਸੇ ਤੋਂ ਕੁਝ 20 ਸ਼ਬਦ ਓਵਰਲੈਪ ਹੁੰਦੇ ਹਨ ਤਾਂ ਜੋ ਇੰਬੈਡਿੰਗ ਕੱਟੀ ਨਾ ਜਾਵੇ ਅਤੇ ਖੋਜ ਨੂੰ ਬਹੁਤਰੀਨ ਸੰਦਰਭ ਮਿਲੇ।
4. ਹਰ ਟੈਕਸਟ ਹਿੱਸਾ OpenAI Chat API ਨੂੰ 60 ਸ਼ਬਦਾਂ ਵਿੱਚ ਸਰੰਨ੍ਹ ਕਰਨ ਲਈ ਭੇਜਿਆ ਜਾਂਦਾ ਹੈ। ਇਹ ਸੰਖੇਪ ਵੀ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ `embedding_index_3m.json` ਵਿੱਚ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ।
5. ਅੰਤ ਵਿੱਚ, ਹਿੱਸਾ ਟੈਕਸਟ OpenAI ਇੰਬੈਡਿੰਗ API ਨੂੰ ਭੇਜਿਆ ਜਾਂਦਾ ਹੈ। ਇੰਬੈਡਿੰਗ API 1536 ਨੰਬਰਾਂ ਦਾ ਇੱਕ ਵੈਕਟਰ ਵਾਪਸ ਕਰਦਾ ਹੈ ਜੋ ਹਿੱਸੇ ਦੇ ਸੈਮੈਂਟਿਕ ਅਰਥ ਨੂੰ ਦਰਸਾਉਂਦਾ ਹੈ। ਹਿੱਸਾ ਅਤੇ OpenAI ਇੰਬੈਡਿੰਗ ਵੈਕਟਰ ਇਕਠੇ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ `embedding_index_3m.json` ਵਿੱਚ ਜਮ੍ਹਾਂ ਕੀਤੇ ਜਾਂਦੇ ਹਨ।

### ਵੈਕਟਰ ਡੇਟਾਬੇਸ

ਸਿੱਖਣ ਦੀ ਸਹੂਲਤ ਲਈ, ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਏਕ JSON ਫਾਈਲ `embedding_index_3m.json` ਵਿੱਚ ਸਾਝਾ ਕੀਤਾ ਗਿਆ ਹੈ ਅਤੇ ਪੈਂਡਾਸ ਡੇਟਾ ਫ੍ਰੇਮ ਵਿੱਚ ਲੋਡ ਕੀਤਾ ਗਿਆ ਹੈ। ਪਰ ਉਤਪਾਦਨ ਵਿੱਚ, ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਨੂੰ ਵੈਕਟਰ ਡੇਟਾਬੇਸ ਵਿੱਚ ਸਟੋਰ ਕੀਤਾ ਜਾਵੇਗਾ ਜਿਵੇਂ ਕਿ [Azure Cognitive Search](https://learn.microsoft.com/training/modules/improve-search-results-vector-search?WT.mc_id=academic-105485-koreyst), [Redis](https://cookbook.openai.com/examples/vector_databases/redis/readme?WT.mc_id=academic-105485-koreyst), [Pinecone](https://cookbook.openai.com/examples/vector_databases/pinecone/readme?WT.mc_id=academic-105485-koreyst), [Weaviate](https://cookbook.openai.com/examples/vector_databases/weaviate/readme?WT.mc_id=academic-105485-koreyst) ਹੋਰ।

## ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਨੂੰ ਸਮਝਣਾ

ਅਸੀਂ ਟੈਕਸਟ ਇੰਬੈਡਿੰਗਜ਼ ਬਾਰੇ ਸਿੱਖਿਆ ਪ੍ਰਾਪਤ ਕੀਤੀ ਹੈ, ਅਗਲਾ ਕਦਮ ਇਹ ਸਿੱਖਣਾ ਹੈ ਕਿ ਇੰਬੈਡਿੰਗਜ਼ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਡੇਟਾ ਦੀ ਖੋਜ ਕਿਵੇਂ ਕੀਤੀ ਜਾਵੇ ਅਤੇ ਖਾਸ ਕਰਕੇ ਇੱਕ ਦਿੱਤੀ ਕੁਏਰੀ ਲਈ ਸਭ ਤੋਂ ਸਮਾਨ ਇੰਬੈਡਿੰਗਜ਼ ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਕਿਵੇਂ ਲੱਭੀ ਜਾਵੇ।

### ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਕੀ ਹੈ?

ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਦੋ ਵੈਕਟਰਾਂ ਦੇ ਵਿਚਕਾਰ ਸਮਾਨਤਾ ਦਾ ਮਾਪ ਹੈ, ਜਿਸ ਨੂੰ ਤੁਸੀਂ ‘ਨੀਅਰੇਸਟ ਨੇਬਰ ਸਰਚ’ ਵੀ ਕਹਿ ਸਕਦੇ ਹੋ। ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਖੋਜ ਕਰਨ ਲਈ ਤੁਹਾਨੂੰ OpenAI ਇੰਬੈਡਿੰਗ API ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਕੁਏਰੀ ਟੈਕਸਟ ਲਈ ਵੈਕਟਰ ਬਣਾਉਣਾ ਪੈਂਦਾ ਹੈ। ਫਿਰ ਕੁਏਰੀ ਵੈਕਟਰ ਅਤੇ ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਵਿੱਚ ਹਰ ਵੈਕਟਰ ਵਿਚਕਾਰ ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਨਿਕਾਲੋ। ਯਾਦ ਰੱਖੋ, ਇੰਬੈਡਿੰਗ ਇੰਡੈਕਸ ਵਿੱਚ ਹਰ ਯੂਟਿਊਬ ਟ੍ਰਾਂਸਕ੍ਰਿਪਟ ਟੈਕਸਟ ਹਿੱਸੇ ਲਈ ਇੱਕ ਵੈਕਟਰ ਹੁੰਦਾ ਹੈ। ਆਖ਼ਰ ਵਿੱਚ ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਦੇ ਰੂਪ ਵਿੱਚ ਨਤੀਜਿਆਂ ਨੂੰ ਛਾਂਟੋ ਅਤੇ ਸਭ ਤੋਂ ਵੱਧ ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਵਾਲੇ ਟੈਕਸਟ ਹਿੱਸੇ ਕੁਏਰੀ ਲਈ ਸਭ ਤੋਂ ਸਮਾਨ ਹੋਣਗੇ।

ਗਣਿਤੀ ਦ੍ਰਿਸ਼ਟੀਕੋਣ ਤੋਂ, ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਦੋ ਵੈਕਟਰਾਂ ਦੇ ਵਿਚਕਾਰ ਬਹੁਮਾਤ੍ਰੀਕ ਅੰਗਲ ਦੇ ਕੋਸਾਈਨ ਨੂੰ ਮਾਪਦੀ ਹੈ। ਇਹ ਮਾਪ ਕਿਉਂ ਲਾਭਦਾਇਕ ਹੈ ਕਿ ਜੇਕਰ ਦੋ ਦਸਤਾਵੇਜ਼ Euclidean ਦੂਰੀ ਵੱਲੋਂ ਲੰਬੇ ਤੁਰੇ ਹੋਣ, ਤਾਂ ਵੀ ਉਹਨਾਂ ਦੇ ਵਿਚਕਾਰ ਅੰਗਲ ਛੋਟਾ ਹੋ ਸਕਦਾ ਹੈ ਅਤੇ ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਵੱਧ ਹੋ ਸਕਦੀ ਹੈ। ਕੋਸਾਈਨ ਸਮਾਨਤਾ ਸਮੀਕਰਨ ਬਾਰੇ ਹੋਰ ਜਾਣਕਾਰੀ ਲਈ ਵੇਖੋ [ਕੋਸਾਈਨ ਸਮਾਨਤਾ](https://en.wikipedia.org/wiki/Cosine_similarity?WT.mc_id=academic-105485-koreyst)।

## ਆਪਣੀ ਪਹਿਲੀ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ

ਅਗਲਾ, ਅਸੀਂ ਇੰਬੈਡਿੰਗਜ਼ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਕਿਵੇਂ ਬਣਾਈਏ ਸਿੱਖਾਂਗੇ। ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਵਿੱਦਿਆਰਥੀਆਂ ਨੂੰ ਪ੍ਰਸ਼ਨ ਟਾਈਪ ਕਰਕੇ ਵੀਡੀਓ ਖੋਜਣ ਦੀ ਆਗਿਆ ਦੇਵੇਗੀ। ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਪ੍ਰਸ਼ਨ ਨਾਲ ਤਾਲਮੇਲ ਰੱਖਣ ਵਾਲੀਆਂ ਵੀਡੀਓਜ਼ ਦੀ ਸੂਚੀ ਵਾਪਸ ਕਰੇਗੀ। ਇਸਦੇ ਨਾਲ ਨਾਲ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਸਵਾਲ ਦੇ ਜਵਾਬ ਹਨ ਵਾਲੀ ਵੀਡੀਓ ਦੀ ਜਗ੍ਹਾ ਦਾ ਲਿੰਕ ਵੀ ਵਾਪਸ ਕਰੇਗੀ।

ਇਹ ਹੱਲ Windows 11, macOS, ਅਤੇ Ubuntu 22.04 'ਤੇ Python 3.10 ਜਾਂ ਉਸ ਤੋਂ ਬਾਅਦ ਵਰਜਨ ਨਾਲ ਬਣਾਇਆ ਅਤੇ ਟੈਸਟ ਕੀਤਾ ਗਿਆ। ਤੁਸੀਂ Python [python.org](https://www.python.org/downloads/?WT.mc_id=academic-105485-koreyst) ਤੋਂ ਡਾਊਨਲੋਡ ਕਰ ਸਕਦੇ ਹੋ।

## ਅਸਾਈਨਮੈਂਟ - ਵਿਦਿਆਰਥੀਆਂ ਲਈ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ

ਅਸੀਂ ਇਸ ਪਾਠ ਦੀ ਸ਼ੁਰੂਆਤ ਵਿੱਚ ਆਪਣੇ ਸਟਾਰਟਅਪ ਨੂੰ ਜਾਣੂ ਕਰਵਾਇਆ। ਹੁਣ ਸਮਾਂ ਹੈ ਕਿ ਛਾਤਰਾਂ ਨੂੰ ਉਹਨਾਂ ਦੀਆਂ ਮੁਲਾਂਕਣ ਲਈ ਇੱਕ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਦੇ ਯੋਗ ਬਣਾਈਏ।

ਇਸ ਅਸਾਈਨਮੈਂਟ ਵਿੱਚ, ਤੁਸੀਂ ਉਹ Azure OpenAI ਸੇਵਾਵਾਂ ਬਣਾਉਗੇ ਜੋ ਸਰਚ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣ ਲਈ ਵਰਤੀ ਜਾਣਗੀਆਂ। ਤੁਸੀਂ ਹੇਠ ਲਿਖੀਆਂ Azure OpenAI ਸੇਵਾਵਾਂ ਬਣਾਉਗੇ। ਇਸ ਅਸਾਈਨਮੈਂਟ ਨੂੰ ਪੂਰਾ ਕਰਨ ਲਈ ਤੁਹਾਨੂੰ Azure ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਦੀ ਲੋੜ ਹੋਵੇਗੀ।

### Azure ਕਲਾਉਡ ਸ਼ੈੱਲ ਸ਼ੁਰੂ ਕਰੋ

1. [Azure ਪੋਰਟਲ](https://portal.azure.com/?WT.mc_id=academic-105485-koreyst) ਵਿੱਚ ਲਾਗਇਨ ਕਰੋ।
2. Azure ਪੋਰਟਲ ਦੇ ਉੱਪਰ-ਸੱਜੇ ਕੋਨੇ ਵਿੱਚ ਕਲਾਉਡ ਸ਼ੈੱਲ ਆਈਕਾਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
3. ਵਾਤਾਵਰਨ ਦੀ ਕਿਸਮ ਲਈ **Bash** ਚੁਣੋ।

#### ਇੱਕ ਰਿਸੋਰਸ ਗਰੁੱਪ ਬਣਾਉ

> ਇਹ ਹਦਾਇਤਾਂ ਲਈ ਅਸੀਂ "semantic-video-search" ਨਾਂ ਦੇ ਰਿਸੋਰਸ ਗਰੁੱਪ ਨੂੰ East US ਵਿੱਚ ਵਰਤ ਰਹੇ ਹਾਂ।
> ਤੁਸੀਂ ਰਿਸੋਰਸ ਗਰੁੱਪ ਦਾ ਨਾਮ ਬਦਲ ਸਕਦੇ ਹੋ, ਪਰ ਜਦੋਂ ਰਿਸੋਰਸ ਲਈ ਸਥਾਨ ਬਦਲ ਰਹੇ ਹੋ,
> [ਮਾਡਲ ਉਪਲਬਧਤਾ ਟੇਬਲ](https://aka.ms/oai/models?WT.mc_id=academic-105485-koreyst) ਨੂੰ ਚੈੱਕ ਕਰੋ।

```shell
az group create --name semantic-video-search --location eastus
```

#### ਇੱਕ Azure OpenAI ਸੇਵਾ ਰਿਸੋਰਸ ਬਣਾਉ

Azure ਕਲਾਉਡ ਸ਼ੈੱਲ ਤੋਂ ਹੇਠ ਲਿਖਿਆ ਕਮਾਂਡ ਚਲਾਓ ਤਾਂ ਜੋ ਇੱਕ Azure OpenAI ਸੇਵਾ ਰਿਸੋਰਸ ਬਣਾਯਾ ਜਾ ਸਕੇ।

```shell
az cognitiveservices account create --name semantic-video-openai --resource-group semantic-video-search \
    --location eastus --kind OpenAI --sku s0
```

#### ਇਸ ਐਪਲੀਕੇਸ਼ਨ ਵਿੱਚ ਵਰਤੋਂ ਲਈ ਐਂਡਪੋਇੰਟ ਅਤੇ ਕੁੰਜੀਆਂ ਪ੍ਰਾਪਤ ਕਰੋ

Azure ਕਲਾਉਡ ਸ਼ੈੱਲ ਤੋਂ ਹੇਠ ਲਿਖੇ ਕਮਾਂਡ ਚਲਾਓ ਤਾਂ ਜੋ Azure OpenAI ਸੇਵਾ ਰਿਸੋਰਸ ਲਈ ਐਂਡਪੋਇੰਟ ਅਤੇ ਕੁੰਜੀਆਂ ਪ੍ਰਾਪਤ ਕੀਤੀਆਂ ਜਾ ਸਕਣ।

```shell
az cognitiveservices account show --name semantic-video-openai \
   --resource-group  semantic-video-search | jq -r .properties.endpoint
az cognitiveservices account keys list --name semantic-video-openai \
   --resource-group semantic-video-search | jq -r .key1
```

#### OpenAI ਇੰਬੈਡਿੰਗ ਮਾਡਲ ਡਿਪਲੌਯ ਕਰੋ

Azure ਕਲਾਉਡ ਸ਼ੈੱਲ ਤੋਂ ਹੇਠ ਲਿਖਿਆ ਕਮਾਂਡ ਚਲਾਓ ਤਾਂ ਜੋ OpenAI ਇੰਬੈਡਿੰਗ ਮਾਡਲ ਡਿਪਲੌਯ ਕੀਤਾ ਜਾ ਸਕੇ।

```shell
az cognitiveservices account deployment create \
    --name semantic-video-openai \
    --resource-group  semantic-video-search \
    --deployment-name text-embedding-ada-002 \
    --model-name text-embedding-ada-002 \
    --model-version "2"  \
    --model-format OpenAI \
    --sku-capacity 100 --sku-name "Standard"
```
