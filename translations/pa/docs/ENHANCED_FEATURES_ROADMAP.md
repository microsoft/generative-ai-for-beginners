# ਸੁਧਾਰਵਾਧਿਕ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਅਤੇ ਸੁਧਾਰਾਂ ਦਾ ਰੋਡਮੈਪ

ਇਹ ਦਸਤਾਵੇਜ਼ ਜੇਨੇਰੇਟਿਵ ਏਆਈ ਫੋਰ ਬਿਗਿਨਰਜ਼ ਕਰਿਕੁਲਮ ਲਈ ਸਿਫ਼ਾਰਸ਼ੀ ਸੁਧਾਰਾਂ ਅਤੇ ਇੰਹੈਂਸਮੈਂਟਾਂ ਬਾਰੇ ਵਿਸਥਾਰ ਨਾਲ ਸੰਪੂਰਨ ਕੋਡ ਸਮੀਖਿਆ ਅਤੇ ਉਦਯੋਗ ਦੀਆਂ ਸ਼੍ਰੇਸ਼ਠ ਪ੍ਰਥਾਵਾਂ ਦੇ ਵਿਸ਼ਲੇਸ਼ਣ ਅਧਾਰਿਤ ਹੈ।

## ਏਗਜ਼ਿਕਯੂਟਿਵ ਸਾਰ

ਕੋਡਬੇਸ ਦੀ ਸੁਰੱਖਿਆ, ਕੋਡ ਕਵਾਲਿਟੀ ਅਤੇ ਸਿੱਖਿਆ ਦੀ ਪ੍ਰਭਾਵਸ਼ੀਲਤਾ ਲਈ ਵਿਸ਼ਲੇਸ਼ਣ ਕੀਤਾ ਗਿਆ ਹੈ। ਇਹ ਦਸਤਾਵੇਜ਼ ਤੁਰੰਤ ਫਿੱਟਸ, ਨਜ਼ਦੀਕੀ ਸੁਧਾਰਾਂ, ਅਤੇ ਭਵਿੱਖੀ ਇੰਹੈਂਸਮੈਂਟਾਂ ਲਈ ਸਿਫ਼ਾਰਸ਼ਾਂ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

## 4. ਏਪੀਆਈ ਆਧੁਨਿਕੀਕਰਨ

### 4.1 ਪੁਰਾਣੇ ਏਪੀਆਈ ਪੈਟਰਨ (ਮਾਈਗਰੇਸ਼ਨ ਪੂਰਾ)

ਸਾਰੇ ਪਾਇਥਨ ਅਤੇ ਟਾਈਪਸਕ੍ਰਿਪਟ **ਚੈਟ** ਨਮੂਨੇ ਚੈਟ ਕੰਪਲੀਸ਼ਨ ਏਪੀਆਈ ਤੋਂ **ਰਿਸਪਾਂਸ ਏਪੀਆਈ** (`client.responses.create(...)` → `response.output_text`) ਵੱਲ ਮਾਈਗਰੇਟ ਕੀਤੇ ਗਏ ਹਨ।

| ਪਹਿਲਾ ਪੈਟਰਨ | ਨਵਾਂ ਪੈਟਰਨ | ਸਥਿਤੀ |
|-------------|-------------|--------|
| `openai.api_type = "azure"` / `AzureOpenAI()` (ਚੈਟ) | `OpenAI(base_url="<endpoint>/openai/v1/")` (ਰਿਸਪਾਂਸ ਏਪੀਆਈ) | ਪੂਰਾ |
| `openai.ChatCompletion.create()` / `client.chat.completions.create()` | `client.responses.create(input=...)` → `response.output_text` | ਪੂਰਾ |
| `@azure/openai` `OpenAIClient.getChatCompletions()` (ਟਾਈਪਸਕ੍ਰਿਪਟ) | `openai` ਪੈਕੇਜ `client.responses.create()` → `response.output_text` | ਪੂਰਾ |
| `df.append()` (ਪੈਂਡਸ) | `pd.concat()` | ਪੂਰਾ |

> **ਨੋਟ:** ਮਾਇਕਰੋਸਾਫਟ ਫਾਉਂਡਰੀ ਮਾਡਲਾਂ ਜੋ `azure-ai-inference` / `@azure-rest/ai-inference` SDK (`client.complete()`) ਵਰਤਦੇ ਹਨ, ਮਾਡਲ ਇਨਫ਼ਰੈਂਸ ਏਪੀਆਈ 'ਤੇ ਰਹਿੰਦੇ ਹਨ ਜੋ ਰਿਸਪਾਂਸ ਏਪੀਆਈ ਦਾ ਸਮਰਥਨ ਨਹੀਂ ਕਰਦਾ। `AzureOpenAI()` ਜਿੱਥੇ ਲਾਗੂ ਹੈ (ਐਮਬੈਡਿੰਗ ਅਤੇ ਚਿੱਤਰ ਉਤਪਾਦਨ ਲਈ) ਉਥੇ ਜਾਰੀ ਰੱਖਿਆ ਗਿਆ ਹੈ।

### 4.2 ਨਵੇਂ ਏਪੀਆਈ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਦਿਖਾਉਣ ਲਈ

## 5. ਇੰਫਰਾਸਟਰੱਕਚਰ ਸੁਧਾਰ

### 5.1 CI/CD ਸੁਧਾਰ

[`.github/workflows/code-quality.yml`](../../../.github/workflows/code-quality.yml) ਵਿੱਚ ਅਮਲ ਕੀਤਾ ਗਿਆ: Python ਲਿੰਟਿੰਗ/ਫਾਰਮੈਟਿੰਗ (Ruff + Black) ਮੈਨਟੇਨ ਕੀਤੇ ਗਏ `shared/` ਯੂਟਿਲਿਟੀ ਮਾਡਿਊਲ 'ਤੇ **ਲਾਗੂ** ਹੁੰਦਾ ਹੈ ਅਤੇ ਬਾਕੀ ਕਰਿਕੁਲਮ 'ਤੇ **ਸਲਾਹਕਾਰੀ** ਤੌਰ 'ਤੇ ਚਲਦਾ ਹੈ, ਨਾਲ ਹੀ ਜਾਵਾਸਕ੍ਰਿਪਟ/ਟਾਈਪਸਕ੍ਰਿਪਟ ਲਈ ਸਲਾਹਕਾਰੀ ESLint ਪਾਸ। ਉਦਾਹਰਣਤਮਕ ਮੂਲ ਰੇਖਾ ਇਹ ਸੀ:

```yaml
# .github/workflows/code-quality.yml
name: Code Quality

on: [push, pull_request]

jobs:
  python-lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.10'
      - run: pip install ruff black mypy
      - run: ruff check .
      - run: black --check .

  js-lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm ci
      - run: npx eslint .
```

### 5.2 ਸੁਰੱਖਿਆ ਸਕੈਨਿੰਗ

[`.github/workflows/security.yml`](../../../.github/workflows/security.yml) ਵਿੱਚ ਅਮਲ ਕੀਤਾ ਗਿਆ: Python ਅਤੇ ਜਾਵਾਸਕ੍ਰਿਪਟ/ਟਾਈਪਸਕ੍ਰਿਪਟ ਲਈ CodeQL ਵਿਸ਼ਲੇਸ਼ਣ (ਪੁਸ਼, ਪੁੱਲ ਰਿਕਵੇਸਟ, ਅਤੇ ਹਫਤਾਵਾਰੀ ਸਮਾਂ-ਸੂਚੀ 'ਤੇ) ਨਾਲ ਪੁੱਲ ਰਿਕਵੇਸਟ ਤੇ ਨਿਰਭਰਤਾ ਸਮੀਖਿਆ। ਉਦਾਹਰਣਤਮਕ ਮੂਲ ਰੇਖਾ ਇਹ ਸੀ:

```yaml
# .github/workflows/security.yml
name: Security Scan

on: [push, pull_request]

jobs:
  codeql:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: github/codeql-action/init@v3
        with:
          languages: javascript, python
      - uses: github/codeql-action/analyze@v3

  dependency-review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/dependency-review-action@v4
```

### 6.1 ਡੈਵਕੰਟੇਨਰ ਸੁਧਾਰ

[`.devcontainer/devcontainer.json`](../../../.devcontainer/devcontainer.json) ਅਤੇ [`.devcontainer/post-create.sh`](../../../.devcontainer/post-create.sh) ਵਿੱਚ ਅਮਲ ਕੀਤਾ ਗਿਆ: ਕੰਟੇਨਰ ਹੁਣ Pylance, Black ਫਾਰਮੈਟਰ, Ruff, ESLint, Prettier, ਅਤੇ Copilot ਐਕਸਟੈਂਸ਼ਨਾਂ ਨੂੰ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ, ਰਿਪੋ ਦੇ Black/Prettier ਕਨਫਿਗ ਨਾਲ ਫਾਰਮੈਟ-ਆਨ-ਸੇਵ ਯੋਗ ਕਰਦਾ ਹੈ, ਅਤੇ ਡਿਵੈਲਪਰ ਟੂਲਿੰਗ ਦੀ ਸੈਟਅਪ ਕਰਦਾ ਹੈ (`ruff`, `black`, `mypy`, `pytest`) ਤਾਂ ਜੋ [ਕੋਡ-ਗੁਣਵੱਤਾ ਵਰਕਫਲੋ](../../../.github/workflows/code-quality.yml) ਨੂੰ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਦੁਹਰਾਇਆ ਜਾ ਸਕੇ। `mcr.microsoft.com/devcontainers/universal` ਬੇਸ ਇਮੀਜ਼ ਵਿੱਚ ਪਹਿਲਾਂ ਹੀ Python ਅਤੇ Node ਸ਼ਾਮਲ ਹਨ, ਇਸ ਲਈ ਹੋਰ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਦੀ ਲੋੜ ਨਹੀਂ। ਉਦਾਹਰਣਤਮਕ ਮੂਲ ਰੇਖਾ ਇਹ ਸੀ:

```json
{
  "name": "Generative AI for Beginners",
  "image": "mcr.microsoft.com/devcontainers/universal:2",
  "features": {
    "ghcr.io/devcontainers/features/python:1": {
      "version": "3.11"
    },
    "ghcr.io/devcontainers/features/node:1": {
      "version": "20"
    }
  },
  "customizations": {
    "vscode": {
      "extensions": [
        "ms-python.python",
        "ms-python.vscode-pylance",
        "ms-toolsai.jupyter",
        "dbaeumer.vscode-eslint",
        "esbenp.prettier-vscode",
        "github.copilot"
      ],
      "settings": {
        "python.formatting.provider": "black",
        "editor.formatOnSave": true
      }
    }
  },
  "postCreateCommand": "pip install -e .[dev] && npm install"
}
```
