# ਕਲਾਉਡ ਸੈਟਅੱਪ ☁️ – GitHub ਕੋਡਸਪੇਸਜ਼

**ਜੇ ਤੁਸੀਂ ਕੁਝ ਵੀ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਇੰਸਟਾਲ ਨਹੀਂ ਕਰਨਾ ਚਾਹੁੰਦੇ ਤਾਂ ਇਸ ਗਾਈਡ ਦੀ ਵਰਤੋਂ ਕਰੋ।**  
ਕੋਡਸਪੇਸਜ਼ ਤੁਹਾਨੂੰ ਇੱਕ ਮੁਫ਼ਤ, ਬ੍ਰਾਊਜ਼ਰ-ਅਧਾਰਤ VS ਕੋਡ ਇੰਸਟੈਂਸ ਦਿੰਦਾ ਹੈ ਜਿਸ ਵਿੱਚ ਸਾਰੇ ਡਿਪੈਂਡੈਂਸੀਜ਼ ਪਹਿਲਾਂ ਹੀ ਇੰਸਟਾਲ ਹੁੰਦੇ ਹਨ।

> (ਵੇਖੋ ▸ ਕਮਾਂਡ ਪੈਲੇਟ ▸ *Codespaces: Stop Codespace*)।

## 2.  ਇੱਕ ਕੋਡਸਪੇਸ ਬਣਾਓ (ਇੱਕ ਕਲਿੱਕ ਵਿੱਚ)

1. ਇਸ ਰੇਪੋ ਨੂੰ **ਫੋਰਕ** ਕਰੋ (ਸਿਖਰ-ਸੱਜੇ ਕੋਣ ਦੇ **Fork** ਬਟਨ ਤੱਤੇ)।  
2. ਆਪਣੇ ਫੋਰਕ ਵਿੱਚ, ਕਲਿੱਕ ਕਰੋ **Code ▸ Codespaces ▸ Create codespace on main**।  
   ![Dialog showing buttons to create a codespace](../../../translated_images/pa/who-will-pay.4c0609b1c7780f44.webp)

✅ ਇੱਕ ਬ੍ਰਾਊਜ਼ਰ VS ਕੋਡ ਵਿੰਡੋ ਖੁਲਦੀ ਹੈ ਅਤੇ ਡੈਵ ਕੰਟੇਨਰ ਦਾ ਨਿਰਮਾਣ ਸ਼ੁਰੂ ਹੋ ਜਾਂਦਾ ਹੈ।
ਇਹ ਪਹਿਲੀ ਵਾਰੀ ਵਿੱਚ **~2 ਮਿੰਟ** ਲੈਂਦਾ ਹੈ।

## 3. ਆਪਣੀ API ਕੁੰਜੀ ਜੋੜੋ (ਸੁਰੱਖਿਅਤ ਤਰੀਕਾ)

### ਵਿਕਲਪ A ਕੋਡਸਪੇਸ ਸਿਕਰੇਟਸ – ਸਿਫਾਰਸ਼ੀ

1. ⚙️ ਗੀਅਰ ਆਈਕਨ -> ਕਮਾਂਡ ਪੈਲੇਟ-> Codespaces : Manage user secret -> ਨਵਾਂ ਸਿਕਰੇਟ ਜੋੜੋ।
2. ਨਾਮ: OPENAI_API_KEY
3. ਮੁੱਲ: ਆਪਣੀ ਕੁੰਜੀ ਪੇਸਟ ਕਰੋ → Add secret

ਬਸ—ਸਾਡਾ ਕੋਡ ਇਸਨੂੰ ਖੁਦ ਬਖੁਦ ਲੈ ਲਵੇਗਾ।

### ਵਿਕਲਪ B .env ਫਾਈਲ (ਜੇ ਤੁਹਾਨੂੰ ਵਾਕਈ ਲੋੜ ਹੋਵੇ ਤਾਂ)

```bash
cp .env.copy .env
code .env         # OPENAI_API_KEY=ਤੁਹਾਡੀ_ਚਾਬੀ_ਇੱਥੇ ਭਰੋ
```
