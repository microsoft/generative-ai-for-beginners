# ਬਿਗਿਨਰਜ਼ ਲਈ ਜਨਰੇਟਿਵ ਏਆਈ (ਵਰਜ਼ਨ 3) - ਇੱਕ ਕੋਰਸ

## 🌱 ਸ਼ੁਰੂਆਤ ਕਰਨਾ

.NET ਵਿਕਾਸਕਾਰਾਂ ਲਈ [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ਵੇਖੋ!
