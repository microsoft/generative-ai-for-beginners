# ਨਵਾਂਤਮ ਐਆਈ ਲਈ ਸ਼ੁਰੂਆਤੀ (ਵਰਜਨ 3) - ਇੱਕ ਕੋਰਸ

ਮਾਇਕ੍ਰੋਸੌਫਟ ਕਲਾਊਡ ਵਕੀਲਾਂ ਵੱਲੋਂ ਸਾਡਾ 21-ਸਬਕ ਵਿਆਪਕ ਕੋਰਸ ਦੇ ਨਾਲ ਜੇਨੇਰੇਟਿਵ ਐਆਈ ਐਪਲੀਕੇਸ਼ਨਾਂ ਬਣਾਉਣ ਦੇ ਬੁਨਿਆਦੀ ਸਿਧਾਂਤ ਸਿੱਖੋ।

## 🌱 ਸ਼ੁਰੂਆਤ ਕਰਨਾ

ਇਸ ਕੋਰਸ ਵਿਚ 21 ਸਬਕ ਹਨ। ਹਰ ਸਬਕ ਆਪਣਾ ਵਿਸ਼ਾ ਕਵਰ ਕਰਦਾ ਹੈ, ਇਸ ਲਈ ਕਿਸੇ ਵੀ ਸਥਾਨ ਤੋਂ ਸ਼ੁਰੂ ਕਰੋ!

ਸਬਕਾਂ ਨੂੰ "ਸਿੱਖੋ" ਅਤੇ "ਬਣਾਓ" ਵਜੋਂ ਲੇਬਲ ਕੀਤਾ ਗਿਆ ਹੈ ਜਿੱਥੇ "ਸਿੱਖੋ" ਸਬਕ ਜੇਨੇਰੇਟਿਵ ਐਆਈ ਸੰਕਲਪ ਨੂੰ ਸਮਝਾਉਂਦੇ ਹਨ ਜਾਂ "ਬਣਾਓ" ਸਬਕ ਸੰਕਲਪ ਅਤੇ ਕੋਡ ਉਦਾਹਰਣ ਦਿੰਦੇ ਹਨ ਜੋ ਸੰਭਵ ਹੋਣ 'ਤੇ ਦੋਹਾਂ **Python** ਅਤੇ **TypeScript** ਵਿੱਚ ਹੁੰਦੇ ਹਨ।

.NET ਵਿਕਾਸਕਾਰਾਂ ਲਈ [Generative AI for Beginners (.NET Edition)](https://github.com/microsoft/Generative-AI-for-beginners-dotnet?WT.mc_id=academic-105485-koreyst) ਨੂੰ ਦੇਖੋ!

ਹਰ ਸਬਕ ਵਿੱਚ ਇੱਕ "ਸਿੱਖਦੇ ਰਹੋ" ਸੈਕਸ਼ਨ ਵੀ ਹੁੰਦਾ ਹੈ ਜਿਸ ਵਿੱਚ ਵਾਧੂ ਸਿੱਖਣ ਵਾਲੇ ਸਾਧਨ ਹਨ।

## ਤੁਹਾਨੂੰ ਕੀ ਲੋੜ ਹੈ
### ਇਸ ਕੋਰਸ ਦਾ ਕੋਡ ਚਲਾਉਣ ਲਈ, ਤੁਸੀਂ ਇਹਨਾਂ ਵਿੱਚੋਂ ਕੋਈ ਵੀ ਵਰਤੀ ਹੋ ਸਕਦੇ ਹੋ:
 - [Azure OpenAI Service](https://aka.ms/genai-beginners/azure-open-ai?WT.mc_id=academic-105485-koreyst) - **ਸਬਕ:** "aoai-assignment"
 - [Microsoft Foundry Models](https://ai.azure.com/catalog/models?WT.mc_id=academic-105485-koreyst) - **ਸਬਕ:** "githubmodels" (GitHub Models ਜੁਲਾਈ 2026 ਦੇ ਅਖੀਰ 'ਤੇ ਰਿਟਾਇਰ ਹੋ ਰਹੇ ਹਨ - ਸਥਾਨ ਤੇ Microsoft Foundry Models ਵਰਤੋ)
 - [OpenAI API](https://aka.ms/genai-beginners/open-ai?WT.mc_id=academic-105485-koreyst) - **ਸਬਕ:** "oai-assignment"
 - [Foundry Local](https://foundrylocal.ai?WT.mc_id=academic-105485-koreyst) - ਆਪਣੇ ਡਿਵਾਈਸ 'ਤੇ ਪੂਰੀ ਤਰ੍ਹਾਂ ਆਫਲਾਈਨ ਮਾਡਲ ਚਲਾਓ, ਕੋਈ ਕਲਾਊਡ ਸਬਸਕ੍ਰਿਪਸ਼ਨ ਜ਼ਰੂਰੀ ਨਹੀਂ
   
- Python ਜਾਂ TypeScript ਦੀ ਬੁਨਿਆਦੀ ਜਾਣਕਾਰੀ ਮਦਦਗਾਰ ਹੈ - \*ਪੂਰੀ ਤਰ੍ਹਾਂ ਸ਼ੁਰੂਆਤੀ ਲਈ ਇਹਨਾਂ [Python](https://aka.ms/genai-beginners/python?WT.mc_id=academic-105485-koreyst) ਅਤੇ [TypeScript](https://aka.ms/genai-beginners/typescript?WT.mc_id=academic-105485-koreyst) ਕੋਰਸ ਦੇਖੋ
- ਇੱਕ GitHub ਖਾਤਾ ਇਸ ਪੂਰੇ ਰਿਪੋ ਨੂੰ [ਫੋਰਕ ਕਰਨ ਲਈ](https://aka.ms/genai-beginners/github?WT.mc_id=academic-105485-koreyst) ਆਪਣਾ ਖਾਤਾ ਬਣਾਓ

ਅਸੀਂ ਤੁਹਾਡੀ ਵਿਕਾਸ ਪ੍ਰਦਾਤਾ ਵਾਤਾਵਰਣ ਸੈੱਟਅਪ ਵਿੱਚ ਮਦਦ ਲਈ ਇੱਕ **[ ਕੋਰਸ ਸੈੱਟਅਪ](./00-course-setup/README.md?WT.mc_id=academic-105485-koreyst)** ਸਬਕ ਬਣਾਇਆ ਹੈ।

ਭਵਿੱਖ ਵਿਚ ਇਸ ਰਿਪੋ ਨੂੰ ਅਸਾਨੀ ਨਾਲ ਲੱਭਣ ਲਈ [ਇਸਨੂੰ ਤਾਰਾ (🌟) ਦਿਓ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ਨਾ ਭੁੱਲੋ।
