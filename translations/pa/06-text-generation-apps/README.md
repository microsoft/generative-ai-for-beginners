# ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ

ਤੁਸੀਂ ਇਸ ਕੋਰਸ ਵਿੱਚ ਅਜੇ ਤੱਕ ਦੇਖਿਆ ਹੈ ਕਿ ਪ੍ਰੋੰਪਟ ਵਰਗੇ ਮੁੱਖ ਸੰਕਲਪ ਹਨ ਅਤੇ ਇੱਥੇ ਤੱਕ ਕਿ "ਪ੍ਰੋੰਪਟ ਇੰਜੀਨੀਅਰਿੰਗ" ਨਾਂਕ ਦੀ ਪੂਰੀ ਵਿਧਾ ਹੈ। ਕਈ ਟੂਲ ਜਿਵੇਂ ਕਿ ChatGPT, Office 365, Microsoft Power Platform ਆਦਿ ਤੁਹਾਨੂੰ ਕੁਝ ਕਰਨ ਲਈ ਪ੍ਰੋੰਪਟ ਵਰਤਣ ਦਾ ਸਮਰਥਨ ਦਿੰਦੇ ਹਨ।

ਇੱਕ ਐਪ ਵਿੱਚ ਇਸ ਤਰ੍ਹਾਂ ਦਾ ਅਨੁਭਵ ਜੋੜਨ ਲਈ, ਤੁਹਾਨੂੰ ਪ੍ਰੋੰਪਟ, ਪੂਰਨਤਾ ਅਤੇ ਇੱਕ ਲਾਇਬ੍ਰੇਰੀ ਚੁਣਨ ਵਰਗੇ ਸੰਕਲਪਾਂ ਨੂੰ ਸਮਝਣ ਦੀ ਜ਼ਰੂਰਤ ਹੈ। ਇਹੀ ਹੈ ਜੋ ਤੁਸੀਂ ਇਸ ਅਧਿਆਇ ਵਿੱਚ ਸਿੱਖੋਗੇ।

## ਪਰਿਚਯ

ਇਸ ਅਧਿਆਇ ਵਿੱਚ, ਤੁਸੀਂ:

- openai ਲਾਇਬ੍ਰੇਰੀ ਅਤੇ ਇਸਦੇ ਮੁੱਖ ਸੰਕਲਪਾਂ ਬਾਰੇ ਸਿੱਖੋਗੇ।
- openai ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਬਣਾਉਣਾ।
- ਪ੍ਰੋੰਪਟ, ਤਾਪਮਾਨ, ਅਤੇ ਟੋਕਨ ਵਰਗੇ ਸੰਕਲਪਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਬਣਾਉਣ ਦੀ ਸਮਝ ਪਾਉਣਾ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਪਾਠ ਦੇ ਅੰਤ ਵਿੱਚ, ਤੁਸੀਂ ਸਮਝਾ ਸਕੋਗੇ:

- ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਕੀ ਹੈ।
- openai ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਇੱਕ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਬਣਾਉਣਾ।
- ਆਪਣੇ ਐਪ ਨੂੰ ਵੱਧ ਜਾਂ ਘੱਟ ਟੋਕਨ ਵਰਤਣ ਲਈ ਕਨਫਿਗਰ ਕਰਨਾ ਅਤੇ ਤਾਪਮਾਨ ਬਦਲਣਾ, ਇੱਕ ਵੱਖਰੇ ਨਤੀਜੇ ਲਈ।

## ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਕੀ ਹੈ?

ਆਮ ਤੌਰ 'ਤੇ ਜਦੋਂ ਤੁਸੀਂ ਇੱਕ ਐਪ ਬਣਾਉਂਦੇ ਹੋ ਤਾਂ ਇਸ ਵਿੱਚ ਕਿਸੇ ਕਿਸਮ ਦਾ ਇੰਟਰਫੇਸ ਹੁੰਦਾ ਹੈ ਜਿਵੇਂ ਕਿ:

- ਕਮਾਂਡ-ਅਧਾਰਿਤ। ਕਨਸੋਲ ਐਪ ਆਮ ਤੌਰ 'ਤੇ ਐਪ ਹੁੰਦੇ ਹਨ ਜਿੱਥੇ ਤੁਸੀਂ ਇੱਕ ਕਮਾਂਡ ਟਾਈਪ ਕਰਦੇ ਹੋ ਅਤੇ ਇਹ ਇੱਕ ਕੰਮ ਕਰਦਾ ਹੈ। ਉਦਾਹਰਨ ਲਈ, `git` ਇੱਕ ਕਮਾਂਡ-ਅਧਾਰਿਤ ਐਪ ਹੈ।
- ਯੂਜ਼ਰ ਇੰਟਰਫੇਸ (UI)। ਕੁਝ ਐਪ ਵਿੱਚ ਗ੍ਰਾਫਿਕਲ ਯੂਜ਼ਰ ਇੰਟਰਫੇਸ (GUIs) ਹੁੰਦੇ ਹਨ ਜਿੱਥੇ ਤੁਸੀਂ ਬਟਨ ਕਲਿਕ ਕਰਦੇ ਹੋ, ਟੈਕਸਟ ਇਨਪੁੱਟ ਕਰਦੇ ਹੋ, ਵਿਕਲਪ ਚੁਣਦੇ ਹੋ ਅਤੇ ਹੋਰ ਕੁਝ।

### ਕਨਸੋਲ ਅਤੇ UI ਐਪ ਸੀਮਿਤ ਹਨ

ਇਸ ਨੂੰ ਇੱਕ ਕਮਾਂਡ-ਅਧਾਰਿਤ ਐਪ ਨਾਲ ਤੁਲਨਾ ਕਰੋ ਜਿੱਥੇ ਤੁਸੀਂ ਇੱਕ ਕਮਾਂਡ ਟਾਈਪ ਕਰਦੇ ਹੋ:

- **ਇਹ ਸੀਮਿਤ ਹੈ**। ਤੁਸੀਂ ਕਿਸੇ ਵੀ ਕਮਾਂਡ ਨੂੰ ਟਾਈਪ ਨਹੀਂ ਕਰ ਸਕਦੇ, ਸਿਰਫ ਉਹ ਜੋ ਐਪ ਸਹਾਇਕ ਕਰਦਾ ਹੈ।
- **ਭਾਸ਼ਾ ਵਿਸ਼ੇਸ਼**। ਕੁਝ ਐਪ ਕਈ ਭਾਸ਼ਾਵਾਂ ਦਾ ਸਮਰਥਨ ਕਰਦੇ ਹਨ, ਪਰ ਮੂਲ ਰੂਪ ਵਿੱਚ ਐਪ ਇੱਕ ਵਿਸ਼ੇਸ਼ ਭਾਸ਼ਾ ਲਈ ਬਣਾਇਆ ਗਿਆ ਹੈ, ਭਾਵੇਂ ਤੁਸੀਂ ਹੋਰ ਭਾਸ਼ਾ ਸਮਰਥਨ ਜੋੜ ਸਕਦੇ ਹੋ।

### ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਦੇ ਫਾਇਦੇ

ਤਾਂ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਕਿਵੇਂ ਵੱਖਰਾ ਹੈ?

ਇੱਕ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਵਿੱਚ ਤੁਹਾਡੇ ਕੋਲ ਵੱਧ ਲਚਕਦਾਰਤਾ ਹੈ, ਤੁਸੀਂ ਕਮਾਂਡਾਂ ਦੇ ਸੈੱਟ ਜਾਂ ਕਿਸੇ ਵਿਸ਼ੇਸ਼ ਇਨਪੁੱਟ ਭਾਸ਼ਾ ਤੱਕ ਸੀਮਿਤ ਨਹੀਂ ਹੋ। ਇਸਦੀ ਬਜਾਏ, ਤੁਸੀਂ ਐਪ ਨਾਲ ਇੰਟਰਐਕਟ ਕਰਨ ਲਈ ਕੁਦਰਤੀ ਭਾਸ਼ਾ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ। ਇੱਕ ਹੋਰ ਫਾਇਦਾ ਇਹ ਹੈ ਕਿ ਕਿਉਂਕਿ ਤੁਸੀਂ ਪਹਿਲਾਂ ਹੀ ਇੱਕ ਡਾਟਾ ਸ੍ਰੋਤ ਨਾਲ ਇੰਟਰਐਕਟ ਕਰ ਰਹੇ ਹੋ ਜਿਸਨੂੰ ਜਾਣਕਾਰੀ ਦੇ ਵਿਸ਼ਾਲ ਕੋਰਪਸ 'ਤੇ ਪ੍ਰਸ਼ਿਕਸ਼ਿਤ ਕੀਤਾ ਗਿਆ ਹੈ, ਜਦਕਿ ਇੱਕ ਪਾਰੰਪਰਿਕ ਐਪ ਇੱਕ ਡਾਟਾਬੇਸ ਵਿੱਚ ਜੋ ਕੁਝ ਹੈ ਉਸ 'ਤੇ ਸੀਮਿਤ ਹੋ ਸਕਦਾ ਹੈ।

### ਮੈਂ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਨਾਲ ਕੀ ਬਣਾ ਸਕਦਾ ਹਾਂ?

ਕਈ ਚੀਜ਼ਾਂ ਹਨ ਜੋ ਤੁਸੀਂ ਬਣਾ ਸਕਦੇ ਹੋ। ਉਦਾਹਰਨ ਲਈ:

- **ਇੱਕ ਚੈਟਬੋਟ**। ਇੱਕ ਚੈਟਬੋਟ ਜੋ ਵਿਸ਼ਿਆਂ ਬਾਰੇ ਸਵਾਲਾਂ ਦੇ ਜਵਾਬ ਦਿੰਦਾ ਹੈ, ਜਿਵੇਂ ਕਿ ਤੁਹਾਡੀ ਕੰਪਨੀ ਅਤੇ ਇਸਦੇ ਉਤਪਾਦ ਇੱਕ ਚੰਗਾ ਮੈਚ ਹੋ ਸਕਦੇ ਹਨ।
- **ਮਦਦਗਾਰ**। LLMs ਟੈਕਸਟ ਦਾ ਸੰਖੇਪ ਕਰਨ, ਟੈਕਸਟ ਤੋਂ ਦ੍ਰਿਸ਼ਟੀਕੋਣ ਪ੍ਰਾਪਤ ਕਰਨ, ਰਿਜ਼ਿਊਮ ਵਰਗਾ ਟੈਕਸਟ ਤਿਆਰ ਕਰਨ ਵਿੱਚ ਮਹਾਨ ਹਨ।
- **ਕੋਡ ਸਹਾਇਕ**। ਤੁਸੀਂ ਜਿਸ ਭਾਸ਼ਾ ਮਾਡਲ ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹੋ ਉਸ ਦੇ ਆਧਾਰ 'ਤੇ, ਤੁਸੀਂ ਇੱਕ ਕੋਡ ਸਹਾਇਕ ਬਣਾ ਸਕਦੇ ਹੋ ਜੋ ਤੁਹਾਨੂੰ ਕੋਡ ਲਿਖਣ ਵਿੱਚ ਮਦਦ ਕਰਦਾ ਹੈ। ਉਦਾਹਰਨ ਲਈ, ਤੁਸੀਂ GitHub Copilot ਦੇ ਨਾਲ ਨਾਲ ChatGPT ਵਰਗੇ ਉਤਪਾਦ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਤੁਹਾਨੂੰ ਕੋਡ ਲਿਖਣ ਵਿੱਚ ਮਦਦ ਕਰਨ ਲਈ ਵਰਤ ਸਕਦੇ ਹੋ।

## ਮੈਂ ਕਿਵੇਂ ਸ਼ੁਰੂ ਕਰ ਸਕਦਾ ਹਾਂ?

ਅੱਛਾ, ਤੁਹਾਨੂੰ ਇੱਕ LLM ਨਾਲ ਇੰਟੀਗ੍ਰੇਟ ਕਰਨ ਦਾ ਇੱਕ ਤਰੀਕਾ ਲੱਭਣ ਦੀ ਜ਼ਰੂਰਤ ਹੈ ਜੋ ਆਮ ਤੌਰ 'ਤੇ ਹੇਠਾਂ ਦਿੱਤੇ ਦੋ ਦ੍ਰਿਸ਼ਟੀਕੋਣਾਂ ਨੂੰ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ:

- ਇੱਕ API ਦੀ ਵਰਤੋਂ ਕਰੋ। ਇੱਥੇ ਤੁਸੀਂ ਆਪਣੀ ਪ੍ਰੋੰਪਟ ਦੇ ਨਾਲ ਵੈਬ ਬੇਨਤੀ ਬਣਾਉਂਦੇ ਹੋ ਅਤੇ ਜਨਰੇਟ ਕੀਤਾ ਟੈਕਸਟ ਵਾਪਸ ਪ੍ਰਾਪਤ ਕਰਦੇ ਹੋ।
- ਇੱਕ ਲਾਇਬ੍ਰੇਰੀ ਦੀ ਵਰਤੋਂ ਕਰੋ। ਲਾਇਬ੍ਰੇਰੀਆਂ API ਕਾਲਾਂ ਨੂੰ ਕੈਪਸੂਲ ਕਰਕੇ ਉਨ੍ਹਾਂ ਨੂੰ ਵਰਤਣ ਲਈ ਆਸਾਨ ਬਣਾਉਣ ਵਿੱਚ ਮਦਦ ਕਰਦੀਆਂ ਹਨ।

## ਲਾਇਬ੍ਰੇਰੀਆਂ/SDKs

LLMs ਨਾਲ ਕੰਮ ਕਰਨ ਲਈ ਕੁਝ ਚੰਗੀਆਂ ਜਾਣੀਆਂ ਲਾਇਬ੍ਰੇਰੀਆਂ ਹਨ ਜਿਵੇਂ ਕਿ:

- **openai**, ਇਹ ਲਾਇਬ੍ਰੇਰੀ ਤੁਹਾਡੇ ਮਾਡਲ ਨਾਲ ਜੁੜਨ ਅਤੇ ਪ੍ਰੋੰਪਟ ਭੇਜਣ ਲਈ ਆਸਾਨ ਬਣਾਉਂਦੀ ਹੈ।

ਫਿਰ ਉੱਚੇ ਪੱਧਰ 'ਤੇ ਕੰਮ ਕਰਨ ਵਾਲੀਆਂ ਲਾਇਬ੍ਰੇਰੀਆਂ ਹਨ ਜਿਵੇਂ ਕਿ:

- **Langchain**। Langchain ਚੰਗੀ ਤਰ੍ਹਾਂ ਜਾਣੀ ਜਾਂਦੀ ਹੈ ਅਤੇ Python ਦਾ ਸਮਰਥਨ ਕਰਦੀ ਹੈ।
- **Semantic Kernel**। Semantic Kernel Microsoft ਦੁਆਰਾ ਇੱਕ ਲਾਇਬ੍ਰੇਰੀ ਹੈ ਜੋ C#, Python, ਅਤੇ Java ਭਾਸ਼ਾਵਾਂ ਦਾ ਸਮਰਥਨ ਕਰਦੀ ਹੈ।

## openai ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਪਹਿਲਾ ਐਪ

ਆਓ ਦੇਖਦੇ ਹਾਂ ਕਿ ਅਸੀਂ ਆਪਣਾ ਪਹਿਲਾ ਐਪ ਕਿਵੇਂ ਬਣਾਉਂਦੇ ਹਾਂ, ਕੀ ਲਾਇਬ੍ਰੇਰੀਆਂ ਦੀ ਜ਼ਰੂਰਤ ਹੈ, ਕਿੰਨੀ ਲੋੜ ਹੈ ਅਤੇ ਆਦਿ।

### openai ਇੰਸਟਾਲ ਕਰੋ

OpenAI ਜਾਂ Azure OpenAI ਨਾਲ ਇੰਟਰਐਕਟ ਕਰਨ ਲਈ ਬਾਹਰ ਕਈ ਲਾਇਬ੍ਰੇਰੀਆਂ ਹਨ। ਕਈ ਪ੍ਰੋਗਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਵਰਤਣਾ ਸੰਭਵ ਹੈ ਜਿਵੇਂ ਕਿ C#, Python, JavaScript, Java ਅਤੇ ਹੋਰ। ਅਸੀਂ `openai` Python ਲਾਇਬ੍ਰੇਰੀ ਦੀ ਵਰਤੋਂ ਕਰਨ ਲਈ ਚੁਣਿਆ ਹੈ, ਇਸ ਲਈ ਅਸੀਂ ਇਸਨੂੰ ਇੰਸਟਾਲ ਕਰਨ ਲਈ `pip` ਦੀ ਵਰਤੋਂ ਕਰਾਂਗੇ।

### ਇੱਕ ਸਰੋਤ ਬਣਾਓ

ਤੁਹਾਨੂੰ ਹੇਠਾਂ ਦਿੱਤੇ ਕਦਮ ਕਰਨ ਦੀ ਜ਼ਰੂਰਤ ਹੈ:

- Azure 'ਤੇ ਇੱਕ ਖਾਤਾ ਬਣਾਓ [https://azure.microsoft.com/free/](https://azure.microsoft.com/free/?WT.mc_id=academic-105485-koreyst)।
- Azure OpenAI ਤੱਕ ਪਹੁੰਚ ਪ੍ਰਾਪਤ ਕਰੋ। [https://learn.microsoft.com/azure/ai-services/openai/overview#how-do-i-get-access-to-azure-openai](https://learn.microsoft.com/azure/ai-services/openai/overview#how-do-i-get-access-to-azure-openai?WT.mc_id=academic-105485-koreyst) 'ਤੇ ਜਾਓ ਅਤੇ ਪਹੁੰਚ ਲਈ ਅਰਜ਼ੀ ਦਿਓ।

  > [!NOTE]
  > ਲਿਖਣ ਦੇ ਸਮੇਂ, ਤੁਹਾਨੂੰ Azure OpenAI ਤੱਕ ਪਹੁੰਚ ਲਈ ਅਰਜ਼ੀ ਦੇਣ ਦੀ ਜ਼ਰੂਰਤ ਹੈ।

- Python ਇੰਸਟਾਲ ਕਰੋ <https://www.python.org/>
- ਇੱਕ Azure OpenAI ਸਰਵਿਸ ਸਰੋਤ ਬਣਾਇਆ ਹੈ। ਇੱਕ ਸਰੋਤ ਬਣਾਉਣ ਲਈ ਇਸ ਗਾਈਡ ਨੂੰ ਵੇਖੋ [create a resource](https://learn.microsoft.com/azure/ai-services/openai/how-to/create-resource?pivots=web-portal?WT.mc_id=academic-105485-koreyst)।

### API ਕੁੰਜੀ ਅਤੇ ਅੰਤ ਬਿੰਦੂ ਲੱਭੋ

ਇਸ ਮੋੜ 'ਤੇ, ਤੁਹਾਨੂੰ ਆਪਣੀ `openai` ਲਾਇਬ੍ਰੇਰੀ ਨੂੰ ਦੱਸਣ ਦੀ ਜ਼ਰੂਰਤ ਹੈ ਕਿ ਕਿਸ API ਕੁੰਜੀ ਦੀ ਵਰਤੋਂ ਕਰਨੀ ਹੈ। ਆਪਣੀ API ਕੁੰਜੀ ਲੱਭਣ ਲਈ, ਆਪਣੇ Azure OpenAI ਸਰੋਤ ਦੇ "Keys and Endpoint" ਭਾਗ ਵਿੱਚ ਜਾਓ ਅਤੇ "Key 1" ਮੁੱਲ ਨੂੰ ਕਾਪੀ ਕਰੋ।

ਹੁਣ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਇਹ ਜਾਣਕਾਰੀ ਕਾਪੀ ਕੀਤੀ ਗਈ ਹੈ, ਆਓ ਲਾਇਬ੍ਰੇਰੀਆਂ ਨੂੰ ਇਸ ਦੀ ਵਰਤੋਂ ਕਰਨ ਦੀ ਹਦਾਇਤ ਦਿੰਦੇ ਹਾਂ।

> [!NOTE]
> ਆਪਣੀ API ਕੁੰਜੀ ਨੂੰ ਆਪਣੇ ਕੋਡ ਤੋਂ ਵੱਖ ਕਰਨ ਦੀ ਸਲਾਹ ਦਿੱਤੀ ਜਾਂਦੀ ਹੈ। ਤੁਸੀਂ ਵਾਤਾਵਰਨ ਚਰਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਕਰ ਸਕਦੇ ਹੋ।
>
> - ਵਾਤਾਵਰਨ ਚਰ `OPENAI_API_KEY` to your API key.
>   `export OPENAI_API_KEY='sk-...'` ਸੈਟ ਕਰੋ

### Azure ਕਨਫਿਗਰੇਸ਼ਨ ਸੈਟਅਪ

ਜੇਕਰ ਤੁਸੀਂ Azure OpenAI ਦੀ ਵਰਤੋਂ ਕਰ ਰਹੇ ਹੋ, ਤਾਂ ਇੱਥੇ ਤੁਹਾਨੂੰ ਕਨਫਿਗਰੇਸ਼ਨ ਸੈਟਅਪ ਕਰਨ ਦਾ ਤਰੀਕਾ ਦਿੱਤਾ ਗਿਆ ਹੈ:

ਉੱਪਰ ਅਸੀਂ ਹੇਠਾਂ ਦਿੱਤੇ ਸੈਟ ਕਰ ਰਹੇ ਹਾਂ:

- `api_type` to `azure`. This tells the library to use Azure OpenAI and not OpenAI.
- `api_key`, this is your API key found in the Azure Portal.
- `api_version`, this is the version of the API you want to use. At the time of writing, the latest version is `2023-05-15`.
- `api_base`, this is the endpoint of the API. You can find it in the Azure Portal next to your API key.

> [!NOTE] > `os.getenv` is a function that reads environment variables. You can use it to read environment variables like `OPENAI_API_KEY` and `API_BASE`. Set these environment variables in your terminal or by using a library like `dotenv`.

## Generate text

The way to generate text is to use the `Completion` ਕਲਾਸ। ਇੱਥੇ ਇੱਕ ਉਦਾਹਰਨ ਹੈ:

ਉੱਪਰ ਦਿੱਤੇ ਕੋਡ ਵਿੱਚ, ਅਸੀਂ ਇੱਕ completion object ਬਣਾਉਂਦੇ ਹਾਂ ਅਤੇ ਉਸ ਮਾਡਲ ਨੂੰ ਪਾਸ ਕਰਦੇ ਹਾਂ ਜਿਸਦੀ ਅਸੀਂ ਵਰਤੋਂ ਕਰਨਾ ਚਾਹੁੰਦੇ ਹਾਂ ਅਤੇ ਪ੍ਰੋੰਪਟ ਨੂੰ। ਫਿਰ ਅਸੀਂ ਜਨਰੇਟ ਕੀਤਾ ਟੈਕਸਟ ਪ੍ਰਿੰਟ ਕਰਦੇ ਹਾਂ।

### ਚੈਟ completions

ਅਜੇ ਤੱਕ, ਤੁਸੀਂ ਦੇਖਿਆ ਹੈ ਕਿ ਅਸੀਂ `Completion` to generate text. But there's another class called `ChatCompletion` ਦੀ ਵਰਤੋਂ ਕਰ ਰਹੇ ਹਾਂ ਜੋ ਚੈਟਬੋਟਸ ਲਈ ਵਧੇਰੇ ਸੁਹਜ ਹੈ। ਇੱਥੇ ਇਸਦੀ ਵਰਤੋਂ ਦਾ ਇੱਕ ਉਦਾਹਰਨ ਹੈ:

ਇਸ ਕਾਰਜਸ਼ੀਲਤਾ ਬਾਰੇ ਹੋਰ ਜਾਣਕਾਰੀ ਇੱਕ ਆਉਣ ਵਾਲੇ ਅਧਿਆਇ ਵਿੱਚ।

## ਅਭਿਆਸ - ਤੁਹਾਡਾ ਪਹਿਲਾ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ

ਹੁਣ ਜਦੋਂ ਅਸੀਂ ਸਿੱਖ ਲਿਆ ਕਿ openai ਨੂੰ ਕਿਵੇਂ ਸੈਟਅਪ ਅਤੇ ਕਨਫਿਗਰ ਕਰਨਾ ਹੈ, ਇਹ ਤੁਹਾਡਾ ਪਹਿਲਾ ਟੈਕਸਟ ਜਨਰੇਸ਼ਨ ਐਪ ਬਣਾਉਣ ਦਾ ਸਮਾਂ ਹੈ। ਆਪਣਾ ਐਪ ਬਣਾਉਣ ਲਈ, ਇਹ ਕਦਮ ਅਪਣਾਓ:

1. ਇੱਕ ਵਰਚੁਅਲ ਵਾਤਾਵਰਨ ਬਣਾਓ ਅਤੇ openai ਇੰਸਟਾਲ ਕਰੋ:

   > [!NOTE]
   > ਜੇ ਤੁਸੀਂ Windows ਦੀ ਵਰਤੋਂ ਕਰ ਰਹੇ ਹੋ ਤਾਂ `venv\Scripts\activate` instead of `source venv/bin/activate`.

   > [!NOTE]
   > Locate your Azure OpenAI key by going to [https://portal.azure.com/](https://portal.azure.com/?WT.mc_id=academic-105485-koreyst) and search for `Open AI` and select the `Open AI resource` and then select `Keys and Endpoint` and copy the `Key 1` ਮੁੱਲ ਟਾਈਪ ਕਰੋ।

1. _app.py_ ਫਾਈਲ ਬਣਾਓ ਅਤੇ ਇਸ ਵਿੱਚ ਹੇਠਾਂ ਦਿੱਤਾ ਕੋਡ ਦਿਓ:

   > [!NOTE]
   > ਜੇ ਤੁਸੀਂ Azure OpenAI ਦੀ ਵਰਤੋਂ ਕਰ ਰਹੇ ਹੋ, ਤਾਂ ਤੁਹਾਨੂੰ `api_type` to `azure` and set the `api_key` ਨੂੰ ਆਪਣੇ Azure OpenAI ਕੁੰਜੀ 'ਤੇ ਸੈਟ ਕਰਨ ਦੀ ਜ਼ਰੂਰਤ ਹੈ।

   ਤੁਹਾਨੂੰ ਹੇਠਾਂ ਦਿੱਤੇ ਜੇਹੇ ਨਤੀਜੇ ਦੇਖਣੇ ਚਾਹੀਦੇ ਹਨ:

## ਵੱਖਰੇ ਚੀਜ਼ਾਂ ਲਈ ਵੱਖਰੇ ਕਿਸਮ ਦੇ ਪ੍ਰੋੰਪਟ

ਹੁਣ ਤੁਸੀਂ ਦੇਖਿਆ ਹੈ ਕਿ ਇੱਕ ਪ੍ਰੋੰਪਟ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਟੈਕਸਟ ਕਿਵੇਂ ਜਨਰੇਟ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ। ਤੁਹਾਡੇ ਕੋਲ ਇੱਕ ਪ੍ਰੋਗਰਾਮ ਹੈ ਜੋ ਚਲ ਰਿਹਾ ਹੈ ਜੋ ਤੁਸੀਂ ਸੋਧ ਅਤੇ ਬਦਲ ਸਕਦੇ ਹੋ ਤਾਂ ਜੋ ਵੱਖਰੇ ਕਿਸਮ ਦੇ ਟੈਕਸਟ ਜਨਰੇਟ ਕੀਤੇ ਜਾ ਸਕਣ।

ਪ੍ਰੋੰਪਟਾਂ ਨੂੰ ਸਾਰੇ ਕਿਸਮ ਦੇ ਕੰਮਾਂ ਲਈ ਵਰਤਿਆ ਜਾ ਸਕਦਾ ਹੈ। ਉਦਾਹਰਨ ਲਈ:

- **ਟੈਕਸਟ ਦੀ ਇੱਕ ਕਿਸਮ ਜਨਰੇਟ ਕਰੋ**। ਉਦਾਹਰਨ ਲਈ, ਤੁਸੀਂ ਇੱਕ ਕਵਿਤਾ, ਇੱਕ ਕਵੀਤਾ ਲਈ ਸਵਾਲ ਆਦਿ ਜਨਰੇਟ ਕਰ ਸਕਦੇ ਹੋ।
- **ਜਾਣਕਾਰੀ ਲੁੱਕਅਪ**। ਤੁਸੀਂ ਜਾਣਕਾਰੀ ਲਈ ਪ੍ਰੋੰਪਟਾਂ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ ਜਿਵੇਂ ਕਿ ਹੇਠਾਂ ਦਿੱਤੇ ਉਦਾਹਰਨ 'What does CORS mean in web development?'।
- **ਕੋਡ ਜਨਰੇਟ ਕਰੋ**। ਤੁਸੀਂ ਕੋਡ ਜਨਰੇਟ ਕਰਨ ਲਈ ਪ੍ਰੋੰਪਟਾਂ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ, ਉਦਾਹਰਨ ਲਈ ਇੱਕ ਵੈੱਬ ਐਪ ਵਰਗਾ ਪੂਰਾ ਪ੍ਰੋਗਰਾਮ ਜਨਰੇਟ ਕਰਨਾ।

## ਇੱਕ ਵਧੇਰੇ ਵਿਅਵਹਾਰਿਕ ਉਪਯੋਗ ਮਾਮਲਾ: ਇੱਕ ਰਸੋਈ ਨਿਰਮਾਤਾ

ਕਲਪਨਾ ਕਰੋ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਘਰ ਵਿੱਚ ਸਮੱਗਰੀ ਹੈ ਅਤੇ ਤੁਸੀਂ ਕੁਝ ਪਕਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ। ਇਸ ਲਈ, ਤੁਹਾਨੂੰ ਇੱਕ ਰਸੋਈ ਦੀ ਜ਼ਰੂਰਤ ਹੈ। ਇੱਕ ਤਰੀਕਾ ਰਸੋਈਆਂ ਲੱਭਣ ਦਾ ਇੱਕ ਖੋਜ ਇੰਜਨ ਦੀ ਵਰਤੋਂ ਕਰਨਾ ਹੈ ਜਾਂ ਤੁਸੀਂ ਇਸ ਲਈ ਇੱਕ LLM ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ।

ਤੁਸੀਂ ਇੱਕ ਪ੍ਰੋੰਪਟ ਇਸ ਤਰ੍ਹਾਂ ਲਿਖ ਸਕਦੇ ਹੋ:

> "ਮੈਨੂੰ ਹੇਠਾਂ ਦਿੱਤੀ ਸਮੱਗਰੀ ਦੇ ਨਾਲ ਇੱਕ ਵਿਅੰਜਨ ਲਈ 5 ਰਸੋਈਆਂ ਦਿਖਾਓ: ਚਿਕਨ, ਆਲੂ, ਅਤੇ ਗਾਜਰ। ਪ੍ਰਤੀ ਰਸੋਈ, ਵਰਤੀਆਂ ਜਾਣ ਵਾਲੀਆਂ ਸਾਰੀਆਂ ਸਮੱਗਰੀਆਂ ਦੀ ਸੂਚੀ ਦਿਓ"

ਉਪਰੋਕਤ ਪ੍ਰੋੰਪਟ ਦੇ ਅਨੁਸਾਰ, ਤੁਹਾਨੂੰ ਇਸ ਜੇਹਾ ਜਵਾਬ ਮਿਲ ਸਕਦਾ ਹੈ:

ਇਹ ਨਤੀਜਾ ਵਧੀਆ ਹੈ, ਮੈਨੂੰ ਪਕਾਉਣ ਲਈ ਪਤਾ ਹੈ। ਇਸ ਮੋੜ 'ਤੇ, ਕੀ ਲਾਭਦਾਇਕ ਸੁਧਾਰ ਹੋ ਸਕਦੇ ਹਨ:

- ਉਹ ਸਮੱਗਰੀਆਂ ਫਿਲਟਰ ਕਰਨਾ ਜੋ ਮੈਨੂੰ ਪਸੰਦ ਨਹੀਂ ਹਨ ਜਾਂ ਜਿਨ੍ਹਾਂ ਨਾਲ ਮੈਨੂੰ ਅਲਰਜੀ ਹੈ।
- ਇੱਕ ਖਰੀਦਦਾਰੀ ਸੂਚੀ ਤਿਆਰ ਕਰੋ, ਜੇਕਰ ਮੇਰੇ ਕੋਲ ਘਰ ਵਿੱਚ ਸਾਰੀਆਂ ਸਮੱਗਰੀਆਂ ਨਹੀਂ ਹਨ।

ਉਪਰੋਕਤ ਮਾਮਲਿਆਂ ਲਈ, ਆਓ ਇੱਕ ਹੋਰ ਪ੍ਰੋੰਪਟ ਜੋੜੀਏ:

> "ਕਿਰਪਾ ਕਰਕੇ ਰਸੋਈਆਂ ਵਿੱਚੋਂ ਲਸਣ ਨੂੰ ਹਟਾਓ ਕਿਉਂਕਿ ਮੈਨੂੰ ਅਲਰਜੀ ਹੈ ਅਤੇ ਇਸਨੂੰ ਕੁਝ ਹੋਰ ਨਾਲ ਬਦਲ ਦਿਓ। ਇਨ੍ਹਾਂ ਵਿੱਚੋਂ ਕਿਸੇ ਵੀ ਦੀ ਸੂਚੀ ਤਿਆਰ ਕਰੋ, ਇਹ ਸਮੱਗਰੀਆਂ ਜਿਨ੍ਹਾਂ ਨੂੰ ਮੈਂ ਘਰ ਵਿੱਚ ਚਿਕਨ, ਆਲੂ ਅਤੇ ਗਾਜਰ ਹੋਣ ਦੇ ਨਜ਼ਰ ਵਿੱਚ ਰੱਖ ਕੇ ਵਰਤ ਚੁੱਕਾ ਹਾਂ।"

ਹੁਣ ਤੁਹਾਡੇ ਕੋਲ ਇੱਕ ਨਵਾਂ ਨਤੀਜਾ ਹੈ, ਜਿਸ ਵਿੱਚ:

ਇਹ ਤੁਹਾਡੀਆਂ ਪੰਜ ਰਸੋਈਆਂ ਹਨ, ਜਿਨ੍ਹਾਂ ਵਿੱਚ ਲਸਣ ਦਾ ਕੋਈ ਜ਼ਿਕਰ ਨਹੀਂ ਹੈ ਅਤੇ ਤੁਹਾਡੇ ਕੋਲ ਇੱਕ ਖਰੀਦਦਾਰੀ ਸੂਚੀ ਵੀ ਹੈ ਜੋ ਇਹ ਨਜ਼ਰ ਵਿੱਚ ਰੱਖ ਕੇ ਹੈ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਘਰ ਵਿੱਚ ਕੀ ਹੈ।
